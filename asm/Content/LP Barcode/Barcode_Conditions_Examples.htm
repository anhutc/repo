Regular expressions (ECMAScript syntax)

Examples of metacharacters:

[A-Z]

Single uppercase character.

[a-z]

Single lowercase character.

[0-9]

Single numeric digit.

[^a]

^ negates the string that follows: Anything except "a".

{n,m}

The preceding element must occur at least n times and may occur up to a maximum of m times.

{n,}

n times or more often

?

The preceding element must occur once or not at all.

+

The preceding element must occur once or several times.

*

The preceding element must occur once, several times or not at all.

Examples of regular expressions:

ABC[0-9]{6}

All barcodes that begin with the exact string "ABC" followed by exactly 6 digits.

((abc)|(ABC))[1-3]+

All barcodes that start with the string "ABC" or "abc" followed by at least 1 digit in the range 1 through 3.

 

Text expressions

Available wildcards:

?

Any single character.
* Any string.

Examples of text expressions:

A?X*

All barcodes that start with the character "A" and where the 3rd character is "X", followed by any number of additional characters.
*A The last character must be "A".
