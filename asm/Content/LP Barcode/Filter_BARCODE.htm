Barcode (PCB): Defining filters

A filter or a combination of various filters allows you to define precisely the information from the raw barcode data that you require as the output format for traceability. The length of the result is limited to 40 characters.

 

Proceed as follows to define a new filter:

  1. Under Properties, choose Barcode filter.

  2. Click the Add filter button.

  3. Define the criteria for the new filter.

     

To specify the output format of a barcode, proceed as follows:

  1. Under Properties, choose Barcode filter.

  2. Enter the filters to be used in the Format field.

 

%x corresponds to Filter x.

You can also combine the predefined filters or supplement them with freely selectable text.

 

Example:

Raw barcode data: A5E00123456AAT-U02112010

Combinations of the specified criteria can also be used:

E.g. Start text AAT, Start offset 2: 2 characters are omitted after the string "AAT". The barcode is then evaluated -> 02112010.

 

* can be used as a wildcard for any string.
