Display

 

You can change the display with the following buttons:

Larger: Enlarges the camera image around the center of the image.

Smaller: Reduces the size of the camera image.

Center:
The image is centered.
This function can only be selected if the camera image had previously been moved or enlarged.

Pseudocolors: Represents grayscale values using a color palette.

Crosshair: Displays a crosshair in the current image to facilitate orientation.

1:1 representation of the camera image.

More:

If, in order to display the camera image as large as possible, not all the buttons are shown, this button allows you to display and use the hidden buttons.

 

Moves the camera image in the direction of the arrow.
