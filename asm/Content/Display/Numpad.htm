Numpad

How to make numeric entries using the touch screen:

  1. Open the numeric keypad by clicking the Numpad icon

  2. Select the required sequence of numbers using the number keys.

  3. Correct your entries, if need be, using the cursor control keys.

  4. Confirm the value you have entered.

  5. Close the numeric keypad by clicking the Numpad icon again :

 

Cursor control keys:

The last digit is deleted.

The current number is deleted.

The current number is accepted.
