Buttons (components, geometry view)

Capture new image

A new image is captured and a new measurement is performed.

New measurement

The newly captured image is measured again and the result is displayed below the image.

Show height map

(Only shown if stereo measurement has been enabled).

The brighter a point appears in the representation, the nearer it is to the camera.

Switching to this view improves visibility of the connections.

Because the height map is used to identify the location of the component, the learning step is also carried out using the height map.

Component shape wizard

Starts a wizard that provides a simple way for you to describe the geometry data of components.

 

Reset

All changes are reset to their initial status when SIPLACE Vision was started.

Cancels the entries most recently made.

If the button is pressed after closing a wizard, all the changes made in the wizard are reset.

Restores an entry which had been canceled with Undo.

If changes made in a wizard are reset, all the changes are redone.
