Buttons (components, illumination view)

Capture new image

A new image is captured and a new measurement is performed.

New measurement

The newly captured image is measured again and the result is displayed below the image.

Optimize illumination

(Only available if a component is described as "Nonstandard" or "Connector" and the "OSC package" license is available on the programming system.)

Opens the wizard for setting improved illumination for the component and the camera used in order to ensure that the measurement is as robust as possible.

First describe all the connections and, if necessary, specify whether a connection appears bright or dark compared with the background.

 

Pick up component

The component is picked up by the feeder and placed in front of the camera.

Feed components (SIPLACE BulkFeeder X)

Components are shaken onto the pick-up plate and a new camera image is acquired.

 

Default illumination values

Resets the values recommended by the vendor for the current component type.

Reset

All changes are reset to their initial status when SIPLACE Vision was started.

Cancels the entries most recently made.

If the button is pressed after closing a wizard, all the changes made in the wizard are reset.

Restores an entry which had been canceled with Undo.

If changes made in a wizard are reset, all the changes are redone.
