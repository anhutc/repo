Initiating a station-wise job change

How to carry out a job change:

  1. Enter the new job at the SIPLACE Pro Line Control user interface and select the "Station-wise Download" option.

  2. Go to the first station of the line, wait until the last board of the old job (job A) has entered the station and then click the button .

    The conveyor interface is locked:

    As long as the following button is displayed, you can undo the action by clicking the button again: .
    The input section is then blocked for the new boards and the new job B is not yet active: .


  3. If conversion work or conveyor width adjustment is necessary for job B, the line is emptied and width adjustment is started for the conveyor:

    Perform the conversion work and confirm this operation by pressing the Start button.

    .

    If neither conversion work nor width adjustment are necessary, the boards for the new job B can be moved in and produced as soon as the last board from job A has been moved from the input section to the processing area. .


    The boards of job B remain in the machine until the job change has taken place on the downstream machine. This is done automatically.

     

 

Special case: If a station needs to be restarted, is displayed when a board moves into the downstream station. For each board that moves into the machine, you must specify whether it is the last board of job A. Do this by clicking the button and choosing the appropriate entry from the drop-down menu.

Only after the last board has been confirmed is the input conveyor locked, allowing you to proceed.
