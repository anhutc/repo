Board camera (SST 34): Illumination settings for the board camera (standard digital)

Inappropriate illumination settings can lead to poor contrast. This can result in unusually lengthy measuring times or poor recognition quality for ink spots or fiducials. With taught fiducials or ink spots, the determined model may be unsuitable if the training image was incorrectly illuminated. It is therefore important to set the illumination for the board camera to the best possible value.

A new camera image is acquired, the measurement carried out and the result displayed each time the illumination setting is changed. This means that you can immediately react to any improvement or worsening as a result of the illumination settings.

 

How to adjust the board camera illumination:

  1. Switch to the Illumination setting view .

  2. Select the illumination type which delivers the best contrast in the camera image:
    Red steep, Red diffuse, Red diffuse and steep and Blue.

  3. Set the required intensity (brightness value) for the selected illumination type using the rotary control:

  • For fiducials covered with solder stop resist, higher brightness values should be set.

  • Blue, oblique lighting is best for recognizing barcodes that have been laser-etched into the board.

 

Exception: Data Matrix barcode

In this application scenario, it may make sense to combine red and blue illumination to achieve the best possible contrast between the modules and the background.

An improvement can be expected if the modules appear light against the background or dark against the background under both red and blue illumination.

Caution: Blurring can occur at the edges of the image in particular.
