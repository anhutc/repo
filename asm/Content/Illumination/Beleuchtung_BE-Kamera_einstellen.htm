Setting the component camera illumination

 

How to adjust the component camera illumination:

  1. Switch to the Illumination adjustment view:
  2. Select an entry from the list of illumination angles:
    E.g. steep, medium, flat.
  3. Set the required intensity (brightness value) for the selected illumination type using the rotary control:

 

A new camera image is acquired, the measurement carried out and the result displayed each time the illumination setting is changed. This means that you can immediately react to any improvement or worsening as a result of the illumination settings.

 

There are special recommended settings for the individual camera types:
