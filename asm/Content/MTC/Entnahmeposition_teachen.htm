MTC: Teaching the removal position

 

  1. Select the feed axis of the relevant tower.
  2. Press Teach remove position and then Next.
  3. Press Next again to start the reference run.
  4. Press the button for the cassette whose removal position is to be taught and confirm your selection by pressing Next. The tower is moved to the refill position for the selected cassette.
  5. Insert a waffle pack tray carrier in the required level and confirm that you have done this by pressing Next. The tower is moved to the removal position for the selected cassette.
  6. If necessary, teach the removal position using the and buttons. (Every press of these buttons moves the feed axis one step of the selected step size in the corresponding direction.)
  7. Press Engage catch into WPTC to check the current position.
  8. Press Finish to store the current position as the removal position for the selected cassette in the MTC's internal machine data.
