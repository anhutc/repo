Maximal vertical acceleration

Waffle pack tray carriers with the specified component may only be accelerated by this maximum value when being lifted or lowered.

Maximal horizontal acceleration

Waffle pack tray carriers with the specified component may only be accelerated by this maximum value when being moved out or in.

Unit

You can enter the acceleration force in g (g = 9.81 m/s2), in m/s2 or in mm/s2.

The possible value ranges are preset in SIPLACE Pro.

OK

The changed acceleration values are applied and sent to SIPLACE Pro and the window is closed.

Cancel

The changed acceleration values are rejected and the window is closed.
