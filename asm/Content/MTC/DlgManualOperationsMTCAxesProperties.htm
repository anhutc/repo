Functions

Perform reference run

Moves the selected axis to a defined position.

Positioning...

Opens the Positioning dialog box where you can specify the target position for the selected axis. The machine function for moving the axis to the target position is started after you confirm your entry with Accept. The current position of the axis is then displayed.

Machine data

Shows the parameters that can be changed and the associated values stored in the machine data for the selected axis.

Edit...

Opens a dialog box for entering a value for the selected parameter within the specified value range.

Submit changes

Starts the machine function for storing the changed parameters in the machine data.

Undo changes

The changed parameters are reset to the values from the machine data.
