MTC: Teaching the refill position

 

  1. Select the lifting axis of the relevant tower.
  2. Press Teach refill position and then Next.
  3. Press Next again to start the reference run.
  4. Press the button for the cassette whose refill position is to be taught and confirm your selection by pressing Next. The tower is moved to the refill position for the selected cassette.
  5. If necessary, teach the refill position using the and buttons. (Every press of these buttons moves the lifting axis one step of the selected step size in the corresponding direction.)
  6. Press Finish to store the current position as the refill position for the selected cassette in the MTC's internal machine data.
