MTC: Teaching the feed axis

 

Teach zero position

When you select the function, the reference run is performed and the zero point correction value is determined after you press Next twice to confirm.

A message then appears indicating the procedure to be followed.  

See

Teach zero position of the feed axis (method).

Teach remove position

When you select the function, the reference run is performed after you press Next twice to confirm. The buttons for selecting the cassette whose removal position you want to teach are then displayed.

See

Teach removal position (method).

Teach WPTC alignment position 

Notes on the procedure are then displayed. You must confirm each of these with Next. After all the steps have been performed and the machine functions have been executed, press Finish to accept the current position as the WPTC alignment position.

See

Teach WPTC alignment position (method).

Limit switch -

Moves the feed axis to the limit switch on the transfer side in order to re-determine the minimum position which can be reached. A reference run is then performed with the axis.

Limit switch +

Moves the feed axis to the limit switch on the tower side in order to re-determine the maximum position which can be reached. A reference run is then performed with the axis.
