Component

Component shape

Name of the component shape.

Enable

Activates the waffle pack tray again.
A waffle pack tray is automatically deactivated if the description of the waffle pack tray does not match the specification from SIPLACE Pro.

The filling level cannot be changed nor can position correction be performed as long as a waffle pack tray is deactivated.

Set full

Confirms that the waffle pack tray has been completely filled with components.

Set empty

Confirms that the waffle pack tray has been emptied.

Change filling level...

Sets the number of components in the waffle pack tray:

In the representation of the waffle pack tray, first select the component up to which the tray is filled and then press Set full.

Position correction... (Can only be enabled as of the "Service" activity level.)

Opens a dialog box for manually correcting the component pickup position.
