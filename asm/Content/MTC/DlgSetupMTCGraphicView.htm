Setup > Locations > MTC x Waffle pack tray detail view

Level

Refill position

Move to refill position

Moves to the position at which the waffle pack trays of all levels can be exchanged.

While this process is taking place, it is not possible to pick up components from the MTC.

Quit refilling mode

Moves the entire MTC back into the processing position.
It is now possible to pick up components again.

Check WPTC occupancy

The waffle pack tray carriers present in the tower are compared with those specified in the setup.

The position of the waffle pack tray carrier (cassette, level) in the tower is then displayed. If waffle pack tray carriers specified by the setup are missing, this is displayed.

Provide level

The lifting axis moves the selected level to the position of the feed axis and the waffle pack tray carrier is then pushed into the staging position .

Confirm exchange of cassette

The cassette has been loaded and pushed into the MTC as per the required setup.
The maximum number of components is entered as the current component level and the waffle pack trays are taken back into operation.

Confirm exchange of tray carrier

The waffle pack tray carrier has been loaded and pushed into the relevant level as per the required setup.
The maximum number of components is entered as the current component level and the waffle pack trays are taken back into operation.

Error list (only displayed if an error has occurred)

Brief description of an error for the affected waffle pack tray.

Tray at position x

Component name

Name of the component shape

If you press this button, a dialog box opens showing all the information available on the component:

The complete component name and the component shape name together with a descriptive text for each (freely-selectable comment stored in SIPLACE Pro).

Enable

Activates the selected waffle pack tray again.
A waffle pack tray is automatically deactivated if the description of the waffle pack tray does not match the specification from SIPLACE Pro.

The filling level cannot be changed nor can position correction be performed as long as a waffle pack tray is deactivated.

Set full

Confirms that the waffle pack tray has been completely filled with components.

Set empty

Confirms that the waffle pack tray has been emptied.

Change filling level...

Sets the number of components in the waffle pack tray:

In the representation of the waffle pack tray, first select the component up to which the tray is filled and then press OK.

Position correction (Can only be enabled as of the "Service" activity level.)

Correct position...

Opens a dialog box for manually correcting the component pickup position.

The table shows the deviation from the corresponding specified value.

Reset correction

Resets the pickup position to the value specified by the setup.
