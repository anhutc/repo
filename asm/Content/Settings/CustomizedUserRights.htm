Setting customized activity levels for individual buttons

Under certain circumstances, it may be necessary to adjust the activity levels proposed by the manufacturer for certain buttons and text boxes.

When doing so, however, it is not possible to assign a higher activity level than the one at which you are logged in.

 

To change the activity level for a button:

  1. Switch to the view Settings -> User settings .

  2. Switch to the "Service" activity level.

  3. Switch back to the view in which you wish to change the setting for a button.

  4. Move the mouse pointer over the button and right-click.

  5. Choose the required activity level and click OK.

To change the activity level for a text box:

  1. Switch to the view Settings -> User settings .

  2. Switch to the "Service" activity level.

  3. Switch back to the view in which you wish to change the setting for a text box.

  4. Move the mouse pointer over the descriptive text and right-click.

  5. Choose the required activity level and click OK.
