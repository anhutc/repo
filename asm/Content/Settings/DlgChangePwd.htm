Change password

 

Old password

Enter the password currently set.

You do not need to make an entry here the first time you change the password or when the field is disabled, as no password is set by default.

New password

Enter a new password that meets the required guidelines.

Confirm password

Confirm the change by entering the new password here again.

OK

Applies the current setting, checks the entries and closes the window.

Abort

The entries are discarded and the password is not changed.

Changing the password:

  1. Enter the password currently set in the Old password field.

  2. Enter the new password in the New password field.

  3. Repeat the new password in the Confirm password field.

  4. Press OK.

    The new password is valid immediately.
