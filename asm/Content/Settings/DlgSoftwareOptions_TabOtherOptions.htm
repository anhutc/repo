Activate whisper configuration file

The information on the ink spots is read in from a file (e.g. from the manufacturer of the board or from an inspection system). This means that it is not necessary to read the ink spots with the board camera at every station. In order to use this functionality, you must select an appropriate configuration file.

The configuration file is named in accordance with the barcode of the board.

The first time this option is enabled, it only takes effect the next time a placement program is specified.

Time synchronization

Activate time synchronization with programming system

Terminates the Microsoft Windows time synchronization service and establishes a connection to the SNTP (Simple Network Time Protocol) time server of the programming system computer on which SIPLACE Pro Line Control is running. This time server then provides the system time for the station computer.
The system time is updated when this option is enabled and every time the programming system is restarted or when a job is specified.

Note: If you are using a different time synchronization service, this must be disabled before you enable "Time synchronization with programming system".
