If one of the specified scenarios occurs, you can choose whether the board is stopped, and if so at what position it should be stopped with a prompt to perform visual inspection.

If placement status is unknown:

Automatic repair placement, without inspection

If it is not possible to reliably determine whether a component has been placed or not, a component is placed at that position again and no prompt is issued. The board is not stopped for inspection.

No repair placement, without inspection

If it is not possible to reliably determine whether a component has been placed or not, no component is placed at that position again. The board is not stopped for inspection.

 

Reset to default values

The recommended settings are reactivated.
