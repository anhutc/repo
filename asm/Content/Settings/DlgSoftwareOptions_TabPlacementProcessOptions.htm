Settings > Software options: Placement process

Recognition

Skip board position recognition

Only for test run (board with foil):
Only the ink spots are measured, not the board fiducials.

Skip component recognition

Recognition with the Vision system is not carried out.

Skip alignment of component using top surface feature

The precise position of the component (X, Y and angle) is not determined on the basis of specified top surface features (e.g. active regions on LED components), even though this measurement step has been specified in SIPLACE Pro.

Dipping and dispensing

Skip dipping

Even the components selected for dipping are no longer dipped in the dipping medium.

Skip glue dispensing

Dispensing is not performed, even for those components for which one or more glue dots have been specified.

Place

Stop placement after pickup failure

The machine continues working with the other placement heads.

The machine stops as soon as no further component can be picked up by a placement head.

The empty track error is shown on a red background in the error line and the indicator lamp on the corresponding side of the machine shows a permanent white light. The track affect must be refilled as soon as possible to avoid the machine stopping.

Skip placement 

The board is only measured.

Placement sequence

Prioritize components already provided

Intended to avoid unnecessary changes of tray.
If a tray is already in the pickup position, all components that are required from this tray and are permitted to be placed are first picked up. Only after this has been done will components be taken from a different tray from the same tray changer.

Inspection areas

Skip inspection

Inspection of the inspection areas will not be performed.
