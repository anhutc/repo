Settings > Software options: Machine configuration

The settings you can make here are for test purposes only!

Options

Skip producibility check

For internal test purposes only.

Skip processing area 1/2

Components are not placed on the board in processing area 1 or 2.

Skip automatic feeder eSW download

When a feeder is set up in the setup area, no check is made as to whether the most recent firmware is loaded and no newer version can be loaded onto the feeder.

Skip auto-configuration

The version of the embedded software is not checked, nor is any corresponding update performed, nor is the machine configuration detected.

Default: The options are deactivated:
