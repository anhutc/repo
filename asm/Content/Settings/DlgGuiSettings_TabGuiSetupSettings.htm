Settings > GUI settings: Setup

Settings

Confirm refill location

Displays a prompt asking the operator to confirm refilling.

Show entire setup

The LEDs of all feeders that are set up are active.

Show current board only

Only the LEDs of the feeders required to assemble the current board are active.
The components that are not required for the current board are grayed out in the setup view.

Suppress 'magnetic support pins' warning on narrowing conveyor lane width

(Only displayed if smart pin support is not supported on the machine.)

Before the conveyor lane width is reduced, you are prompted to remove the magnetic support pins and to confirm that you have done so.

You are not prompted to remove the magnetic support pins.

Display short tape handling solution

If the specification has been made for a feeder that components are to be picked up from it until the last component has been removed, and if the error message "Cover foil torn" is reported, the detailed message offers the additional solution of continuing to pick up from this feeder despite the fact that the foil is torn.

(Default)
