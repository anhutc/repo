Settings > Machine options : Fault detection

If oscillating board is detected

"Oscillating" boards are detected by discrepancies in the placement heights (the Z axis travels further down or less far down than expected).

To ensure the greatest possible placement accuracy, you should always use board support if this problem arises. If Smart pin support is enabled, check the positions of the support pins.

Here, you can choose how the machine is to react if oscillating boards are detected:

Ignore fault

No message is issued and placement continues without interruption.

Reduce speed

Placement speed is reduced automatically.

Stop production

The machine is stopped and an error message is issued.
Production continues after the Start button has been pressed.

Threshold [mm]

If the placement plane deviates by more than the value specified here, the system detects an oscillating board.

Automatic feeder pitch detection for all tracks

During the initial attempts to pick up a component from a feeder, the machine should automatically detect what feeder pitch is to be set.

The setting made here applies to all feeders of the corresponding type

- after a new job has been specified
- after a feeder has been mounted
- after a splice has been detected
- after an empty track error has been issued

Feeder type (e.g. 8 mm feeder)

Automatic detection is performed for all feeders of the specified type as soon as one of the criteria listed above is met.

No automatic detection is performed (default).

If erorrs are detected during conveyor reference run

If an error occurs on one lane during the reference run, and if this lane cannot be used for production, it is possible to proceed with production using the other lane only.

Block input section and deactivate related conveyor lane

Emergency operation: The affected lane is locked and the other lane is used for production.

The machine stops (default).

Support pins

Check positions of support pins periodically

The check is carried out.

No inspection is performed (default).

Time interval [min.]:

Here you can specify the time interval after which the positions are to be inspected. Default: 60 minutes.

After the specified time interval has elapsed, the input conveyor is locked and all the boards remaining in the machine are completely assembled and transported out of the machine. The support pins are then inspected in order to identify whether they are still located at their respective specified positions. If this is not the case, they are moved to their original positions.
Placement is then restarted

Steadiness of board position

Do not verify (default)

Default.

Verify using only board fiducials

Checks board clamping in the machine on the basis of the fiducials for board position recognition.
If the board was not clamped correctly, with the result that it moved during placement, you must check the clamping mechanism.

Verify using all fiducials

The fiducials for the various boards are checked in addition to the fiducials for board position recognition to ensure that all the panels are in the correct position.

Note: Because performance will deteriorate if additional checking is enabled, it is recommended that the check is only enabled in the event of placement offsets that have not been clarified.

Tolerance [mm]

Maximum permitted discrepancy between the fiducial position determined with the new measurement and the position from the original measurement.

Plausibility of fiducial measurement

Check plausibility of fiducial distances

The board fiducials are checked to see whether they are located at a distance from each other specified in SIPLACE Pro.

In the event of an error, check the description of the board fiducials and ensure that no disruptive structures (artifacts) have been measured.

Tolerance [mm]

Maximum permitted deviation from the distance between fiducials as specified in SIPLACE Pro.

The check is carried out. This is recommended in the event of a placement offset.

No inspection is performed (default).
