Settings > Fault thresholds

Pickup errors / Vision errors

Disable threshold for warnings

(Default) A warning in the form of a detailed error message is issued after the specified threshold is reached.

Preset warning threshold: 2 %, i.e. 10 of 500 pickup attempts have failed.

When the warning threshold has been reached, the pickup height is calibrated again, the pickup position that has been learned is reset and the component pocket is measured.

No warning is issued that the ratio between pick up errors and successful pickup operations has reached the specified threshold.

 

Disable threshold for errors

(Default) An error in the form of a detailed error message is issued after the specified threshold is reached.

Preset error threshold: 5 %, i.e. 25 of 500 pickup attempts have failed.

No error message is issued that the ratio between pick up errors and successful pickup operations has reached the specified threshold.


In order to obtain realistic information, calculation is only started after a specified minimum number of pickup attempts.

The error counter is reset after the error message has been issued.

You can change threshold values by clicking in the relevant text box, typing in the required value and clicking Accept.

The preset thresholds apply to all tracks unless specific values have been set for individual components in the component editor on the programming system.
