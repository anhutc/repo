Settings > Machine options

Nozzle scanning

These settings apply to all nozzles for which nozzle scanning is activated.

1. Select scanning level

Level 1: Check nozzle position

Only if it was possible to unambiguously detect the position of the nozzle is it possible to check the higher levels. This level cannot be disabled.

Level 2: Check for soiling

Soiling caused by solder paste or metal abrasion, for instance, and which increases the brightness of the front face so much that it can influence measurement of a component is detected.

Level 3: Check nozzle ducts

Soiling or blockage of the suction ducts can no longer be detected with a vacuum check in the case of ducts with small cross-sections.

Level 4: Check nozzle end surface

The brightness distribution on the front surface is an indicator of damage to the suction surface / front surface.

 

Activation of one level implicitly activates all lower levels (e.g. if level 3 is activated, levels 1 and 2 are also automatically activated).

This also applies to deactivation:

If level 2 is deactivated, levels 3 and 4 are also automatically deactivated.

 

2. Select self healing action

The function selected here is called as soon as a corresponding error message is issued.

None

Clean

The nozzle is moved to the air blast removal station.

Replace

The nozzle is rejected, and a new one is picked up and scanned.

Clean and replace

An attempt is made to clean the nozzle using an air blast. If, however, the subsequent nozzle scan still shows an error, the nozzle will be replaced.

3. Select action on error

For each selected level, you can specify what action the machine is to perform in the event of an error:

None

The error will be ignored.

Show warning

A warning is issued and placement continues.

Stop machine

Placement is stopped.

Restore default values

The recommended settings for how to proceed in the event of an error are restored.
