Use central authentication (can only be selected if a central authentication unit is connected)

It is only possible to log into the machine using a central authentication unit (e.g. card reader).

If the specified timeout for automatic logoff has expired or if the current user has explicitly logged off, only the functions for the 'Production' activity level are available.

Log out

The operator who is currently logged in with the card reader is logged off.

Log in

Customized activity levels

Reset

Resets the activity level settings recommended by the manufacturer.

Import...

Opens a dialog box in which you can select an XML file from which you can import the defined activity levels.

Distribute...

The currently applicable settings on this machine with respect to activity levels are sent to the programming system, which passes them to all machines in this line.

Export...

Opens a dialog box in which you can specify the file (XML format) in which the newly defined activity levels are to be stored.

Customized indicator lamps configuration

Configure...

Starts a tool that allows you to adjust the settings for the indicator devices to suit your application scenario. You can set both indicator lamps and buzzer signals individually.

Restart the machine after you have made the adjustments in order to activate the modified configuration.

If the settings are also to be used on machines other than the current machine, you must distribute the configuration using the programming system.

Distribute...

The currently applicable settings on this machine with respect to indicator devices are sent to the programming system, which passes them to all machines in this line. These settings then apply on all machines that have the same indicator lamp and buzzer configuration.

Language

All the languages currently installed on the machine are offered here.

If you want to switch to a different language, you must press the corresponding button and all texts are then displayed in the selected language.
