Settings > Machine options

General

Options

Skip feeder position recognition

The exact position of all feeders set up with components (maximum component width, 0.5 mm) and thus the exact pickup position are determined when a new setup is transferred. (Default)

The exact pickup position is only determined the first time a component is picked up.

Transport through conveyor 1/2

The boards are not assembled in the specified conveyor.

Skip ink spot check conveyor 1/2

Boards marked with an ink spot are not assembled. Requirement: A board type with ink spot recognition is available.

No ink spot check is carried out. Every board is assembled. One reason for switching the option off may be a board batch with bad ink spots.

Skip nozzle scanning (can only be enabled in the "SIPLACE Service" activity level)

Default: Nozzle scanning is carried out.

Nozzle scanning is deactivated.

Skip whispering actions

Default: Whispering data is requested from the upstream machine.

The current station does not request whispering data from the upstream machine.

Skip measurement of fiducials after safety circuit interruption

Default: After the safety circuit has been opened and closed again, two board fiducials are measured again. This is to ensure placement accuracy.

This setting is only recommended if board fiducials are covered by components and can therefore no longer be measured.
