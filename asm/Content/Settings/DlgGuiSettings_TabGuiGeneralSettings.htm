Settings > GUI settings: General

 

Threshold settings

Warning level for trays [min.]

Each time a component is picked up from a tray, the system calculates how long complete boards can still be assembled with the remaining components of this type.

If this time falls below the time specified here, and the relevant trays are not being monitored by the Setup Center, a warning is issued that the components need to be refilled shortly.

Automatic logoff

Enable automatic logoff

If the Production view has been started, and at least the activity level "Machine Service" has been set and if no operator action has been registered during the period set under Timeout, the system logs off the current user and switches to the "Production" activity level.

As soon as the counter is activated, the time remaining before automatic logoff is shown next to the activity level (Minutes:Seconds).

This area is highlighted in orange during the last 15 seconds before logoff.

Timeout [min.]

Time in minutes after which the system switches to the "Production" activity level.

Virtual keyboard

If the system expects text to be entered using a keyboard, a keyboard is displayed on top of the current view. You can move this keyboard wherever you wish. A slider allows you to control the transparency of the keyboard.

You can also specify that the keyboard is not to be shown. Do this by checking the relevant box:

Disable full keyboard

Disable numeric keyboard

User Interface

Request reason on shutting down machine

Every time the machine is shut down, the system asks for a reason for statistical purposes: Error or regular shutdown.

Switch displays

If the screens have been configured incorrectly, with the result that the board transport direction does not correspond to the transport direction shown on the screens, you must change the transport direction on the two screens.
