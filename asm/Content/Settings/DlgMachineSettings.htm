Settings > Machine settings

Operation mode

(Different operation modes appear here depending on the machine type.)

SIPLACE Pro

A connection is established to the SIPLACE Pro programming system.

E Pro

It is possible to create and specify placement programs and configure lines directly at the machine.

Standalone

The machine is not under the control of a higher-level computer.

New board ...

Opens a dialog box in which you can specify a new board type.

(Can only be selected on the Standalone control mode.)

Automatically switch off the compressed air

The switch off mechanism is activated

The compressed air for the machine is switched off after the machine has been idle for the specified number of seconds.

The compressed air remains activated even when the machine comes to a standstill.

Switch off after [s]

In the field, enter the number of seconds after which the compressed air is to be switched off when the machine is idle.

Operating system

Shut down machine...

Shuts down the station computer completely within a few seconds.

All active programs are canceled.

The computer can subsequently be turned off.

Caution! The data will not be saved!

Restart station software...

Closes all applications and restarts the station software.

Caution! The data will not be saved!

Switch to operating system...

Switches to the Windows XP operating system.
The station computer continues to run in the background.

Create a bug report...

The bug report is stored under:
C:\Sirio\BugReports\archive\Log_DateTime.zip

Factory settings

Reset data to customer delivery state and shut down machine (only available for SIPLACE Service)

Deletes data generated during commissioning and resets any settings made to their default values. The machine is then shut down.

Site related settings

Elevation [m]

You should enter the height in meters above sea level of the machine location to obtain more accurate vacuum measurements.

Stationwise download settings

Product change

On confirmation

If the last board of the current job is loaded into a station, the operator at the station must initiate the job change. Then the changeover can be performed and once the changeover process has been confirmed, the new job can be processed.

Automatic

The new job is started without any intervention from the operator.

First board of new recipe

Stop at output section

The first board produced after a product change remains in the output section. This allows you to ensure that this board is only transported down the line after the necessary preparations have been made.

Pass on

The first board produced after a product change is transported on without waiting for confirmation.
