Setting nozzle scanning

To change the nozzle scanning settings:

  1. Switch to the view "Settings > Machine options": .

  2. Choose the Nozzle scanning tab.

  3. Make the settings you require.

    • Select the nozzle types for which nozzle scanning is to be carried out.

    • Select the nozzle scanning level.

    • Select the behavior in the event of an error: No reaction / Show warning / Stop machine.
