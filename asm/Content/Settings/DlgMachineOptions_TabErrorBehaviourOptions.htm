Settings > Machine options

Actions on events

This view allows you to specify what action is to be taken if an error message is issued:

Create bug report

If one of the error numbers listed here occurs, detailed machine information at the time when the error was triggered is stored.

(Bug reports are stored in the following folder:

C:\Sirio\BugReports\archive

Name of the report: Log_DateTime_error_ErrorNumber.zip)

Stop machine

If one of the error numbers listed here occurs, the machine is stopped.

Opens a numeric entry pad with which you can enter a 5-digit error number (30000 .. 39999).

Choose the entry you wish to remove from the list. The required action is then no longer taken if this error occurs.

Show help

Shows additional information on the error message selected in the list. The associated error number is displayed on the button.
