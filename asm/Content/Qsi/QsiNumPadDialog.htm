Virtual keyboard

As soon as you click in a field in which a number should be entered, a virtual keyboard appears which will allow you to enter numeric information.

How to make numeric entries using the touch screen:

  1. Select the required sequence of numbers using the number keys.

  2. Correct your entries, if need be, using the cursor control keys.

    Deletes the digit to the left of the cursor position.

    Moves the cursor one position to the left.

    Moves the cursor one position to the right.
  3. Confirm your entry by pressing the confirmation button .

 

To move the virtual keyboard:

Press and hold and move the virtual keyboard to the required position on the screen.

 

To change the opacity of the virtual keyboard:

Press and set the required opacity using the slider that appears.
