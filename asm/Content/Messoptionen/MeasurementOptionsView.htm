Measurement options

Glue dot recognition

On

Glue dots are present.

Off

No glue dots are present for this component shape.

Detection of top side of component

On

Measure pickup position before pick: Prior to each pickup operation, the component pickup position is determined by detecting the top side of the component.

Off

Detection of the top side of the component is not performed.

Detection of Pin1

On

An orientation mark is used to identify whether the component has been picked up in the wrong orientation.

Off

Detection of an orientation mark is not performed.

Pattern matching
(can only be activated for components of the type bare die)

On

The internal structure of the component is measured.

Off

The component is measured on the basis of its outline.

Detection of flipped components

On

Different methods can be used to compare certain areas of a component in order to detect whether a component is oriented correctly or is flipped.

Off

Detection of flipped components is not performed.

Detection of tilted chip components

On

In addition to the image for position recognition, a further image is captured using steep 0Â° illumination with an intensity of 100 %.

If a chip is picked up tilted on the nozzle, the connection surfaces appear darker in this image.

Off

Default.

Stereo measurement

On

To determine the precise position of THT pins, pattern features or gullwing leads, the component is presented in front of the camera in two positions.

Off

The component is only presented in front of the camera in one position and measured.

Barcode recognition

On

Recognition of a barcode on the underside of the component is performed using the component camera. You can specify a separate illumination for this purpose.

Chipping inspection

On

An inspection is carried out to detect any chipping/splintering along the edge of the component.

Edit parameters / Enable measurement option

Starts the corresponding wizard.

Disable measurement option

The selected option is disabled for the current measurement.

A measurement option can, however, only be disabled if the option is not explicitly required by SIPLACE Pro.
Glue dot detection cannot be deactivated. It is only possible to deactivate individual glue dots in the wizard.

 

Reset

Any changed values are rejected and those which were valid previously are used again.
