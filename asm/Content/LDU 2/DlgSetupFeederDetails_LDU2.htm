Disassemble

The linear dipping unit must be cleaned regularly to prevent it from becoming clogged. To do this, the unit must be dismantled:

1. Unhook flux tank

Moves the flux tank onto the dipping plate and then moves the dipping plate down.

This causes the flux tank to be unhooked by the squeegee unit.

2. Remove dipping plate

Moves the squeegee unit back to the park plate without the flux tank and then moves the dipping plate up with the flux tank.

Rotate the flux tank by 90° on the dipping plate and lift the dipping plate out of the LDU together with the flux tank.

3. Remove park plate and drip tray

Moves the Z axis down and moves the squeegee unit above it so that the park plate is freely accessible.

Remove the park plate and then the drip tray from the LDU.

After the LDU has been dismantled, the components can be cleaned.

Assemble

After the components of the LDU have been cleaned, they have to be reassembled:

1. Fit drip tray and park plate

Moves the Z axis down and moves the squeegee unit above it.

Fit the drip tray and then the park plate in the LDU.

2. Fit dipping plate

Moves the squeegee unit over the park plate and moves the Z axis up.

Fit the dipping plate.

3. Fit flux tank

Moves the Z axis and the dipping plate down and moves the squeegee unit forwards a little.

Place the flux tank on the dipping plate and push it hard up against the slightly raised park plate.

4. Move flux tank to park position

Moves the dipping plate to the zero position at which the flux tank is hooked into the squeegee unit and then moves the squeegee unit together with the flux tank onto the park plate.

If necessary, fill the flux tank with the appropriate flux.

Planarity

In order to ensure that the components can be dipped into the medium throughout the entire dipping area, the dipping area must not be tilted.

During measurement, a nozzle is moved to various measuring points in the dipping area in order to determine the height of each point. These values are compared with the height of a reference point.

Caution:

The dipping area must contain no flux during measurement, otherwise the nozzle will be ruined.

ID

1: Reference point

2: Measure point

3: Measure point

Absolute

The measured height of the relevant measuring point.

Relative

The deviation of the absolute height of measurement point 2 or 3 from the reference point. For reference point 1, this value is always 0.

Check planarity

Starts measurement of the dipping area.

Recommendation: Carry out this check after the linear dipping unit has been mounted.

Flux level sensor (only shown if the flux level sensor has been enabled in the programming system)

If it is present, the flux level sensor is integrated in the park plate.

The width of the flux surge is measured for every squeegee cycle.

Latest measured value

The value for the width of the "flux surge" determined by the flux level sensor.

Warning threshold

If this value is not reached, a warning is issued to the effect that the flux will soon need refilling.

Default: 4 mm

Threshold for 'Empty'

If this value is not reached, an error message is issued to the effect that the flux is exhausted and that the LDU 2 can no longer be used.

Default: 3 mm
