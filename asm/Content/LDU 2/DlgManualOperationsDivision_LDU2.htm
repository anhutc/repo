Location > X-Table > Dip module (LDU 2)

Functions

Activate dipping area

Activates the dipping area without going through the warm-up cycle.

The squeegee is moved back and forward once to smooth the surface of the flux or to apply new flux.

Refresh dipping medium

The squeegee is moved backwards and forwards the number of times specified under Cycles in the view Setup > X-Table > LDU in order to smooth the surface of the flux or to apply new flux.

Self-test

Performs various movements to check whether the reference values saved during initial startup / after complete assembly or replacement of parts (e.g. the end stoppers of the axes) are achieved within specified tolerances.

If this is not the case, you should check for wear or a part that needs to be replaced.

Prerequisite: The dipping plate and park plate must be fitted, but not the flux tank.

Reference run

Initializes all axes and moves them to a defined position.

If the reference run is started with an LDU 2 fitted, the squeegee unit must be located on the park plate.

Z height correction

Z [mm]:

Shows the correction value for the Z height of the dipping area that has been automatically determined.

Reset

Resets the automatically determined correction value for the Z height to the default value.

Flux level sensor

Set reference values for "flux empty"

Identifies various comparison values that allow the quantity of flux present to be determined subsequently.

This function must be executed after the park plate or flux level sensor has been replaced.

Prerequisite: The LDU 2 must be fully assembled and the flux tank must be empty.

Service functions

Set reference values

Records comparison values for all internal sensors following complete assembly of the LDU 2 or the replacement of parts. All previously stored comparison values are overwritten and the new values are stored in the EEPROM of the LDU 2.

Prerequisite: The dipping plate and park plate must be fitted, but the flux tank must not be present.

Set reference values for sensor dipping plate

A comparison value without the dipping plate is first determined, followed by a comparison value with the dipping plate installed.

These values can subsequently be compared to detect whether a dipping plate is present.

Prerequisite: The reference run has already been performed.

Align park plate and dipping plate

The height of the dipping plate must be adjusted to allow the flux tank to be moved from the park plate to the dipping plate and back without any difference in height.

Prerequisite: The dipping plate and park plate must be fitted, but the flux tank must not be present.

First choose an increment and then press or to move the dipping plate up or down by this amount.

Press Accept to save the set height when the two plates are at exactly the same height.

Automatic cavity depth

Adjust cavity of dipping plate...

If a dipping plate with automatic setting of the cavity is present, you can check whether the specified cavity depth has been set correctly.

To do this, you can select a defined depth and measure to check that this depth has been reached.
