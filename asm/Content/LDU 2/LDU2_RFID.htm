LDU 2: RFID (radio-frequency identification)

Identification of the dip plate using electromagnetic waves.

An RFID tag is attached to the underside of the dip plate. This is read by the RFID reader on the lifting table plate. This allows the dip plate and hence the cavity to be uniquely identified.

If it is not possible to identify the dip plate, you should check whether the RFID tag is faulty.

The RFID of the current dip plate is checked at regular intervals, so that it is possible to immediately detect whether the dip plate has been replaced.
