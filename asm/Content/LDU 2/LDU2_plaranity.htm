LDU 2: Checking planarity

In order to ensure that the components can be dipped into the medium throughout the entire dipping area, the dipping area must not be tilted.

During measurement, a nozzle is moved to various measuring points in the dipping area in order to determine the height of each point.

Caution:

The dipping area must contain no flux during measurement, otherwise the nozzle will be ruined.

To check the planarity of the linear dipping unit (LDU 2), proceed as follows:

  1. Switch to the view Setup > X-Table > , select the dip module and press Settings...

  2. Select the Planarity tab.

  3. Make sure that there is no flux on the dip plate.

  4. Press Check planarity to start measurement.

  5. Check the measured values.

  6. If the values are not correct, you can correct the inclination of the dip plate using the set screws.

  7. Repeat steps 5 - 7 until the measuring points are at the same height as the reference point.

  8. Press Close to close the dialog box.

Evaluating the measured heights:

Type

1: Reference point

2, 3: Measuring point

Absolute

The measured height of the relevant measuring point.

Relative

The deviation in Âµm of the absolute height of measurement point 2 or 3 from the reference point.
(This value is always 0 for reference point 1.)
