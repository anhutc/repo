LDU 2: Function keys and LEDs

Function keys:

• Press briefly to switch to the next mode (P0 -> P1, P1 -> P2, P2 -> P0).

• Press and hold when in P2 to switch the operating level (P2 -> P3).

• Switches back to the previous mode (P2 -> P1, P1 -> P0, ...).

• Deletes the current error (Er displayed).

Selects the current mode or start operation (e.g. start squeegee operation in P2).

Set a parameter (e.g. the speed for P3) and stop the warm-up operation (in P4, cycle=0).

LEDs:

Green

Status OK

Green (flashing rapidly)

Software is being downloaded.

Yellow

A warning has been issued. The status will soon turn to "not ready" (red), e.g. because too little flux is available or maintenance needs to be carried out, etc.

LDU not ready or only partially ready, LDU in programming mode.

Red

LDU is not initialized, is not warmed up or is in error mode.

Red (flashing rapidly)

The application software is invalid.

Off

The linear dipping unit is currently not in use. The 7-segment display is also switched off.
