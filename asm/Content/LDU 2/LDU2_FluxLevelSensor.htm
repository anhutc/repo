Flux level sensor

When the flux tank is moved, the flux surges at the rear edge of the tank. The width of this surge reflects the quantity of flux in the tank. This width is determined using the flux level sensor.

In the programming system, it is possible to specify threshold values. A warning or error is issued for the linear dipping unit 2 if the value falls below these thresholds.

Width of the flux surge:

The width at which the flux surge is in contact with the flux level sensor.

In order to use the flux level sensor, you have to record the reference values with an empty flux tank after replacement of the park plate or the flux level sensor:

  1. Mount the linear dipping unit (LDU 2) as specified.

  2. Switch to the view Manual operations > X-Table , select the linear dipping unit (LDU 2) and press Details...

  3. Make sure that the flux tank is empty.

  4. Then press Set reference values for "flux empty".
