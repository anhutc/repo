LDU 2: Status

Examples:

Not referenced

Press Reference run under Dipping area.

Not warmed up

Press Start warm-up cycle under Dipping area.

Warming up

The warm-up cycle has been started.

Dipping plate not fitted

Fit the specified dipping plate.

Wrong dipping plate fitted

Make sure that a dipping plate that corresponds to the specification from the programming system has been fitted.

Invalid RFID data of dipping plate

The dipping plate fitted could not be recognized.

Make sure that neither the RFID code nor the cover plate of the RFID reader is dirty or faulty.

Check that the correct dipping plate has been fitted.

Dipping medium low

Warning: The flux level that has been determined is lower than the warning threshold defined in the programming system.

 

Dipping medium empty

Error: The flux level that has been determined is lower than the error threshold defined in the programming system.

The dip module is deactivated and it is not possible to dip any more components.

Flux level sensor not referenced

Clean the linear dipping unit, reassemble it and switch to the view Location > X-Table > Dip module (LDU 2). Press Set reference values for "flux empty" under Flux level sensor.

Flux tank in upper position

The dipping plate was moved up to the removal position together with the flux tank.

Caution: risk of crash!

Error during dipping

Check the dipping area. If necessary, remove any components from the dipping medium.

Perform a reference run.

Error during flux inspection

When measuring the component after dipping, the system detected that it has slipped on the nozzle.

LDU 2: Hardware error

Replace the linear dipping unit and send it off for repair.

LDU 2: Cleaning

The linear dipping unit is disassembled and cleaned.

Component lost during dipping

If the error recurs repeatedly:

In the programming system, reduce the initial speed for upward movement after the component has been dipped.

Emergency stop

Release the emergency stop button on the LDU 2 and perform a reference run.

 

After a dipping plate that allows automatic setting of the cavity depth has been replaced, it is necessary to calibrate the cavity depth.
