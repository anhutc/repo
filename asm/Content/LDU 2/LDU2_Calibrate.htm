LDU 2: Aligning the dipping plate

Proceed as follows to align the dip plate with the park plate using the function keys:

  1. Mount the linear dipping unit as specified.

  2. Press the or button (for longer than 0.75 s) until P6 appears in the top 7-segment display.

  3. Press the button to start calibration.

  4. Press the button again to start adjustment of the two plates.

  5. Use the and buttons to set the dipping plate and park plate to exactly the same height.

  6. Press the button to save the current setting as the zero position.
