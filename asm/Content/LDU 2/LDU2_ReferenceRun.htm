Performing a reference run on the linear dipping unit (LDU 2)

Requirement for performing a reference run when the LDU is assembled:

The flux tank must be on the park plate. If necessary, push the squeegee unit onto the park plate and against the stopper.

To perform a reference run for the linear dipping unit (LDU 2) in the Setup view, proceed as follows:

  1. Mount the linear dipping unit 2 as specified.

  2. Switch to the view Setup > X-Table , select the dip module (LDU 2) and press Settings...

  3. Switch to the Dipping area tab.

  4. Press Reference run.

To perform a reference run for the linear dipping unit (LDU 2) using the Manual operations, proceed as follows:

  1. Mount the linear dipping unit as specified.

  2. Switch to the view Manual operations > X-Table , select the dip module (LDU 2) and press Details...

  3. Press Reference run.

Proceed as follows to perform a reference run on the linear dipping unit (LDU 2) using the function keys:

  1. Mount the linear dipping unit as specified.

  2. Press the or button (for longer than 0.75 s) until P1 appears in the top 7-segment display.

  3. Press the button to start a reference run.
