Taking the LDU 2 into operation

Proceed as follows to take the linear dipping unit (LDU 2) into operation:

  1. Mount the dip module as specified.

  2. If the LDU is dismantled, fully assemble it first.

  3. Switch to the view Setup > X-Table > , select the dip module and press Settings...

  4. Select the Dipping area tab.

  5. Start the reference run for the unit: Press Reference run.

  6. Switch to the Planarity tab.

  7. Check the planarity of the dip plate.

  8. Switch to the Dipping area tab.

  9. Fill the tank with the required flux.

  10. Press Start warm-up cycle.

The linear dipping unit is now ready for use.
