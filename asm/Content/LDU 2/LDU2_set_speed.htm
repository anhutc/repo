LDU 2: Setting the squeegee speed using the function keys

  1. Press the or button (for longer than 0.75 s) until P2 appears in the top 7-segment display.

  2. Then press and hold the button until P3 appears in the top display.

  3. Press the button again to start setting the speed.

  4. Press the button to raise the speed by 1 increment and the button to lower the speed by 1 increment.

  5. When the required speed is shown, press the button to set this speed.
