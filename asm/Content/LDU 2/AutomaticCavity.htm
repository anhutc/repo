Automatic setting of the cavity depth

In the programming system, it is possible to make a setting for component shapes specifying how deep the flux in which a component is dipped should be (cavity depth).

If a dipping plate that allows automatic setting of the cavity depth is used, the cavity on the dipping plate is set as specified in the programming system. Settings between 0 and 0.3 mm are possible.

Cavity depth

Automatic setting of the cavity depth can be selected for components with dimensions of 0.2 x 0.2 mm² through 15 x 25 mm².
