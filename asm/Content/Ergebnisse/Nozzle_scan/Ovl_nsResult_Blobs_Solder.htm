Nozzle scanning: Determined soiling

The aim is to reliably identify any solder present on the nozzle.

This is done by searching for bright patches in the image of the nozzle.

 

Areas with a red border Soiling detected.

 

Example:

Bright, shiny areas indicate soiling (e.g. with solder paste).

The nozzle is dirty.

 

Possible error message:

841

Nozzle dirty

Clean or replace the nozzle.

The machine is stopped if "Stop machine" is set for the scanning level "Check for soiling".
