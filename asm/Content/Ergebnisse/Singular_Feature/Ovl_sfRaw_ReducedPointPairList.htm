Recognition step: Subsampling

Coarse search in image with reduced resolution.

The resolution should only be reduced to a level where the object (e.g. fiducial, component pocket or nozzle) can still be clearly recognized.

 

Blue points

Test points on the object.

Yellow points

Test points on the background or on the tape.

Yellow connecting line

The contrast between the points of the pair of points which are connected is greater than the specified minimum value: minimum contrast.

Red connection line

The contrast between the points of the pair of points which are connected is less than the specified minimum value: minimum contrast.
