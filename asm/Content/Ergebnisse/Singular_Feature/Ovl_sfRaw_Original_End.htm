Recognition step: Result of coarse search

Determination of the rough position:

Coarse position recognition is carried out with pairs of test points. At each location in the image, the proportion of point pairs is determined where one point is inside the fiducial and one point is outside the fiducial.

The contrast between the two points is then evaluated. Only if this contrast is above a minimum threshold does the point pair return a proportion for determining the position of the fiducial.

 

Yellow frame

Search area (corresponds to fiducial size + twice the positional tolerance).

Blue points

Test points on the fiducial or component pocket.

Yellow points

Test points on the background.

Yellow connecting line

The contrast between the points of the pair of points which are connected is greater than the specified minimum value: minimum contrast.

Red connection line

The contrast between the points of the pair of points which are connected is less than the specified minimum value: minimum contrast.

 

The following parameters affect the result of this recognition step: Minimum contrast, subsampling, number of point pairs and distance between points.
