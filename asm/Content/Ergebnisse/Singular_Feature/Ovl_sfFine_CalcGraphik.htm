Recognition step: Fine search

Fine position detection by determining the edges of objects:

Blue crosses

Center of opposite edges. If the edges are good, all these points should be in line.

Yellow line

Scan line along which the system searches for the edges of the object.

 

Green crosses (can only be seen in zoom mode)

Detected object edge points.

Only those detected object edge points that are also marked with a blue dot are used to calculate the fiducial position.

 

Turquoise crosses

These crosses are only drawn if there are several candidates for the object edge on a line.
