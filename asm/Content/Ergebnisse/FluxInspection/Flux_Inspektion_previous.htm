Inspection to check that balls have been coated with flux

 

In order to ensure that all balls have been coated with flux after BGA components have been dipped, the Vision system inspects the balls after dipping.

The method used can detect whether any flux is present on the ball, but cannot determine whether the quantity of flux is sufficient.

To work with a new, improved flux inspection method, you need to reset the current recognition parameters to their default values.

 

BGA/balls without flux

BGA/balls with flux

 

A ball is classified either as coated or not coated.

For this classification, you can decide whether only the outer annulus of the ball or the entire ball is to be used for evaluation purposes and which method is to be used. In addition, you can define a threshold value as of which a ball is flagged as being coated.

 

Before you start placement using the flux inspection facility, you must specify the detection threshold for flux inspection.

As a minimum, the following cases should be tested in the component shape manipulator:

A component is not dipped, but flux inspection is activated:

Result: Flux inspection fails, in other words all connections should be flagged as insufficiently coated.

A component is dipped when flux inspection is activated.

Result: Flux inspection succeeds, in other words all connections should be flagged as sufficiently coated.

If these results are not achieved, the parameters must be adjusted.

 

Flux inspection can only be successful if a suitable flux or solder paste is used which causes a sufficient change in appearance between dipped and undipped components. To ensure that flux is detected at all, it is made "visible" by the addition of dark or light pigments.

Classification (with flat-angle illumination)

Balls of an undipped component

A clearly visible bright ring can be seen.

Balls of a component that has been dipped in flux (with a light pigment)

The balls appear closed and bright.

Balls of a component that has been dipped in flux (with a dark pigment)

The balls appear significantly darker (the bright rings become darker, virtually no change can be seen in the center of the ball).

Balls of a component dipped in solder paste

The balls appear significantly darker (the rings are darker, virtually no change in brightness can be seen in the center of the ball).
