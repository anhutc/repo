Recognition step: Flux inspection - comparison before/after dipping

The system switches between the image captured before dipping (reference) and the image captured after dipping (with flux).

Green border

Connections which have been sufficiently coated.

Red border

Connections that are insufficiently coated if the overall result is negative and the component cannot be placed.

Light blue border

Connections that are insufficiently coated if the component is still to be placed (fewer connections are marked than specified under Maximum number of uncoated connections and the specified value for Minimum gap between uncoated connections is observed).

Tip:

Magnifying the image region makes it easier to see the differences in brightness between the individual connections.
