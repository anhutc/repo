Recognition step: Flux inspection results

 

Connection borders

Result of the inspection for this connection.

 

 

 

Green

Connection is sufficiently coated.

Red

The connection has been identified as not coated.

Light blue

The connection was deemed to be insufficiently coated, but the settings (Maximum number of uncoated connections has not been exceeded and Minimum gap between uncoated connections is observed) mean that the connection would be accepted and will not exclude the component.

Inner surface

The result of the measurement of the difference in brightness before and after dipping is shown as a color gradient (green - yellow - red) and corresponds to the quality value for the inspection.

The greater the number of connections that are clearly shown in green, the better the component has been coated.

 

 

 

Green

The connection is well coated: The difference in brightness is above the defined threshold.

Yellow

The difference in brightness matches the specified threshold value Minimum difference from the reference image.

Red

The connection is not coated: The necessary difference in brightness is not present.

Dashed circle

The radius of this circle corresponds to the threshold Minimum gap between uncoated connections.

 

 

Red

The connection is not sufficiently coated and the gap to the next uncoated connection is less than the defined threshold (Minimum gap between uncoated connections).

Light blue

The connection is not sufficiently coated, but the gap to the next uncoated connection is greater than the defined threshold (Minimum gap between uncoated connections).

 

Examples:

Maximum number of uncoated connections = 0 (default)

More uncoated connections are detected than are permitted.

The component cannot be placed.

Maximum number of uncoated connections = 8 (default)

 

Minimum gap between uncoated connections = 2 mm

Uncoated connections are detected, but the number is less than the maximum permitted number.

However, because the minimum gap between 2 uncoated connections is not observed, the component cannot be placed.

 

If a component has already been checked with an earlier version of flux inspection:

  

Green

Balls which have been sufficiently coated.

Red

Balls which have not been sufficiently coated.

Values shown: Characteristic [ball number]
