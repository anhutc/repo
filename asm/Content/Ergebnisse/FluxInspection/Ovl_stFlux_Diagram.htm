Recognition step: Flux inspection - coating diagram

One "column" is shown for each connection (only visible when magnified).

The height of the column indicates, for one connection, the average difference in brightness between the images before and after dipping.

 

Yellow line

Threshold set: Minimum difference from the reference image

All connections for which the average difference in brightness between the images before and after dipping is larger than this value will be deemed to be sufficiently coated.

Green line

Corresponds to the value of the connection with the lowest difference in brightness.

The greater the difference between the yellow and green lines, the better the component has been coated.

Red line

Corresponds to the value of the connection with the smallest difference in brightness if this lies below the set threshold Minimum difference from the reference image.

If the red line is below the yellow line, the component will not be placed.

 
 Examples:  

The component can be placed.

One connection is not sufficiently coated.

Because the Maximum number of uncoated connections was set to 0, the component will not be placed.
