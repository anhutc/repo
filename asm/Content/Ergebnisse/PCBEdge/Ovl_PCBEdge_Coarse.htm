Board edge detection: Coarse search

A point pair filter is used to attempt to identify an outer edge.

All edges detected in the region of interest during this recognition step are drawn.

Yellow rectangle

Region of interest: The center of the board edge must be located in this region.

Green highlighting

Edge position detected within the region of interest.

Yellow highlighting

Detected edge position - outside the region of interest, but still within the tolerance range for the coarse search.

Red highlighting

Edge position detected - outside the region of interest.

No fine search is started for this edge position.

 

Result:

Number of coarse positions

Number of potential candidates for the board edge.

A fine search is started for each of these candidates.

Each of the candidates detected is numbered. This is the number specified in the fine search.
