Recognition parameters: Board edge

The following parameters can be set for detection of a board edge:

Parameter

Range of values

Default

Description

General settings

Minimum recognition quality

 

75 %

Threshold for the ratio between edge points found and edge points searched for.

Default: 75, at least 75 % must be found.

Brightness limit

.. 255

255

The brightness of the camera image is restricted to this value.

All points that are brighter are shown using the specified grayscale value.

Minimum contrast

5 .. 100

20

Specifies the minimum contrast between the board edge (edge point) and the background for the external contour of the substrate to be detected.

Contrast type

-1, 1

-1

1

Bright edge on dark background.
(corresponds to the setting Board is bright - yes)

-1

Dark edge on bright background.
(corresponds to the setting Board is bright - no)

Coarse search

Distance between points

3 .. 30

10

Specifies the distance between the points of a point pair. One point of each pair should lie on the background and one on the edge.

Fine search

Edge quality

1 .. 5

2

Defines the requirements on the edge points.
