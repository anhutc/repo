Board edge detection: Fine search

The coarse search may yield several potential candidates for a board edge.

For each candidate detected by the coarse search, scan lines are defined perpendicular to the expected board edge. The system searches for transitions between light and dark (sudden differences in grayscale values) along these scan lines. If the differences in grayscale values are large enough (larger than the specified Minimum contrast threshold), the position may be an edge position.

 

Green outline

Detected board edge: The result is above the quality threshold and the position is within the presentation tolerance.

Red outline

The search was not successful: The board edge was not detected.

Green crosses

Detected edge positions located on the calculated line.

Light blue crosses

Detected light/dark transitions that are not located on the calculated straight line and are thus not recognized as edge positions.

Yellow line

Scan line on which an edge position was detected.

Orange line

Scan line on which a clear light/dark transition was detected but where the position of the transition is too far from the calculated straight line.

Red line

Scan line on which no sufficiently large light/dark transition was detected.

 

Result:

Quality of current position

Ratio of detected edge points to expected edge points for the potential board edge shown.

This value is compared with the specified threshold "Minimum recognition quality".

If the result is above this threshold, the candidate is marked as a potential board edge.

If several valid board edge candidates are identified (green outline), the position hose center is closest to the specified position is selected as the board edge.
