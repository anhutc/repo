Recognition step: Result

Yellow rectangle

Region in which the center of the object to be measured (component, fiducial, ink spot, component pocket, nozzle, barcode or substrate) must be located.

Yellow wedge

Maximum possible angle of rotation (the X arrow must be located in this region).

Green outline

Component recognition: corresponds to the component body.

Component pocket recognition: corresponds to the dimensions of the component pocket.

Nozzle scanning: Corresponds to the dimensions of the nozzle and the nozzle channels.

Barcode: Corresponds to the specified size of the barcode.

Substrate: Indicates the edge of the substrate.

Green coordinate system

The object to be measured has been detected.

Red coordinate system

The object to be measured has not been detected.

Only for substrates:

Blue rectangles

Evaluation points on the edge of the substrate. Used to determine the exact X/Y position of the substrate.

Light blue rectangles

Evaluation position for the notch.

Only for board edges:

Green line with hatching

Detected board edge.

Glue dot.
