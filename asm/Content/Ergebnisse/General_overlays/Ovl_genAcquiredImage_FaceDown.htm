Recognition step: Camera image for detection of flipped components

Shows the raw camera image with special illumination for the detection of flipped components.

This illumination should be set in such a way that the ends of the leads are illuminated as well as possible, allowing the top and bottom of a component to be distinguished easily.

If this is not so, adjust the illumination.

 

The illumination can be set in the wizard for detection of flipped components under Setting the illumination for detection using gullwing leads.
