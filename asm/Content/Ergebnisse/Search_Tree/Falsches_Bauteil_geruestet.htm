Detection of components incorrectly set up

If an incorrect component is set up (e.g. SO16 instead of SO14: Same pitch, virtually identical connector descriptions, but more leads than the component described), it is possible for multiple potential component positions to be detected.

Solution: After the component position has been detected, a check is carried out as to whether there are other positions that match the description. If this is the case, an error message is issued and the component is not placed.

This possibility is excluded by additional algorithms with the following component shape types:
