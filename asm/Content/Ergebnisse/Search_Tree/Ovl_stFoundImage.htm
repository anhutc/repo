Recognition step: recognized model - result of position recognition

Comparison between filter result (detected features) and the features which were searched for:

1. Component recognized with quality level...

This mark is displayed if the component was found: Detected position of the center of the component.

Green crosses

Indicate the correlation between the lead description and the described model.

Green line, green points (for shield, non-standard or D-Pack)

Indicate the correlation between the edge description and the described model.

 

2. Component not recognized:

Indicates the specified position of the center of the component. The position could not be detected.

Red crosses

Indicate features which could not be detected.

Yellow crosses

Indicate the correlation between the lead description and the described model.

 

3. Some features were not found:

Indicates the specified position of the center of the component.

 

Red crosses

Indicate features which could not be detected.

Yellow crosses

Indicate the correlation between the lead description and the described model.

 

4. More than one position detected: Incorrect component set up

Marks the first detected position of the center of the component.

 

Green crosses

Marks features belonging to the first detected position.

Marks the second detected position of the center of the component.

Yellow crosses

Marks features belonging to the second detected position.
