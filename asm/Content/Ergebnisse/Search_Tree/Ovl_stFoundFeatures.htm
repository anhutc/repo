Recognition step: Features found (position recognition)

Marking of features detected by the filter.

Magenta crosses

 

Indicate the detected features.

(In the overall representation, the features found in the individual subareas are marked with different colors, e.g. cyan, orange, etc.).

Pink lines and points (for shield, non-standard or D-Pack)

Indicate the detected edges.

(In the overall representation, the edges found in the individual subareas are marked with different colors, e.g. cyan, orange, etc.).

Dark yellow frame

The component must be located in this area. The features are only searched for in this area (is not displayed for multiple measurements).

Red frame

If the component is so large that the complete component does not fit into the field of view of the camera, only a portion of the component is inspected. This is indicated by the red rectangle.

Blue frame

Areas which have already been evaluated.

Recognized with quality level

Quality level for recognition:

Value calculated from comparing the number of features found with the number of features searched for and from the quality level of the individual features found.

Threshold: Minimum quality on position measurement
