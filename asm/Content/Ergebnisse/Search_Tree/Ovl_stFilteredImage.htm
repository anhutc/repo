Recognition step: Filter result

Marks the positions at which the filter and the feature coincide:

The grayscale difference between the yellow and blue points is calculated. The points at which this value is greater than the minimum contrast value are shown as light points.

Green frame

This frame is placed around the points at which the grayscale difference value is greater than the specified minimum contrast.

The greater the difference, the lighter the point is displayed.

Yellow frames

Areas in which the features being searched for are expected. *

Filter point pairs
(yellow and dark blue points) and a light blue outline

Shows the filter used in the relevant search area.

Outer yellow frame

Search area for component.

Red frame

If the component is so large that the complete component does not fit into the field of view of the camera, only a portion of the component is inspected. This is indicated by the red rectangle.

Blue frame

Areas which have already been evaluated.

 

Evaluation of the filter results with the threshold minimum contrast of the feature.

 

*If regions of the image in which different filters are used overlap, a separate representation is used for each portion (filter result 1, filter result 2, etc.).

The portions not shown in the current image are marked in pale yellow.

 

Subsampling

The higher the subsampling level, the fewer pixels are used for marking in the filtered image.

Recommendation: In the case of larger features, a higher subsampling level can be used.
Advantage: Shorter evaluation time.
Disadvantage: Reduced accuracy.
