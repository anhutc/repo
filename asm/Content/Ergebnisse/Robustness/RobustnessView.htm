Robustness view

The purpose of this view is to synthetically manipulate the images of the current 2D measurement in order to provide information on the reliability of the current measurement in respect to any fluctuations that are to be expected and, on the basis of the measurement results, to identify how the measurements can be made more robust.

This analysis is not available for measurements from the 3D sensor.

 

A measurement is deemed to be robust if the object under inspection is recognized with a sufficient degree of quality despite slight fluctuations of the measurement parameters and if the same results are achieved when the object under inspection is in the same position in the image.

-

The results of position measurements must lie within values that have been specified internally.

-

In the case of ink spots, the type of ink spot recognized ("place" or "omit") must remain the same.

 

In order to determine the reliability of measurements despite fluctuations of the measurement parameters, the following parameters are adjusted slightly for each individual measurement:

Zoom

The image size is changed in order to model the height in Z and any manufacturing tolerances.

Check the height of the component and the geometry of the component.

Feature sizes

The specified feature sizes are increased or reduced in the image by the relevant factor.

Check the geometry of the features and whether the feature is fully illuminated. Also check the position of the features if necessary.

Rotation

Rotation of the image relative to the presentation angle set.

Check whether the angular tolerance has been entered correctly in the programming system (recognition step: presentation position).

Check the geometry of the features.

Note: Changes to the zoom factor, the feature sizes and the rotation are not possible in the case of multiple measurements.

The results of the individual measurements allow you to identify how to improve the robustness of the measurements.

The small, colored box indicates the status of the previous measurement if a robustness analysis has already been performed. This allows you to immediately see whether the changes that have currently been made to the settings result in an improvement or a deterioration.

To identify the recognition step in which a problem arises, you can select a particular measurement by clicking on the corresponding entry and view the results of the individual recognition steps.

 

Hint:

Set the parameters in such a way that as many measurements as possible are flagged in green in the middle of the range.
