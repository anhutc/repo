3D sensor: Model

Shows the orientation at which the component is measured.
In the case of symmetrical components, it is possible to identify the orientation on the basis if the overlaid coordinate system.

 

Scanning angle

Orientation at which the component is measured (in degrees).

 

Scan height correction

The discrepancy between the component orientation and the focal plane.

Normally, the component is presented at the focal plane of the 3D sensor at scan height 0. If this is not the case, the machine can learn the correct scan height from previous measurements and adjust it accordingly.

The correction value given here is the value by which the machine corrects the vertical position of the component.
