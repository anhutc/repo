Height difference

 

Average distance of the connections from the focal plane.

 

The sensor operates most accurately if the component is presented on the focal plane of the sensor.

 

Hint:
This value can be used in the programming system to correct the component height.
Example: A component has been described with a height of 2.5 mm and the height difference is -400 Âµm.
Correct the component height to 2.1 mm.
