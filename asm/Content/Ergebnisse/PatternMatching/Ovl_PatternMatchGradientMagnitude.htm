Pattern matching: Gradient value

The lighter a point is displayed in the gradient image, the greater the brightness change is at this location in the original image.

The lines shown correspond to the contours of the structures.

Pattern matching is based on points where the change in brightness is sufficiently marked.

 

If only a very small number of bright pixels can be seen in this image, the algorithm only has very few structures available to it for matching.

Tip: If this happens, check the illumination settings or the selected region of interest.
