Recognition step: Detection of flipped components - trained components

Two components can be shown here: The most recently taught component in the correct orientation and the most recently taught flipped component.

The brightness value determined in the region of interest is given for each component.

Example:

Evaluation using the brightness method

Example: Brightness threshold = 139, Flipped if brightness is above threshold: No

OK

The component is in the correct orientation. The brightness value determined in the region of interest is 99.

Flipped

The component is flipped. The brightness value determined in the region of interest is 179 and is above the brightness threshold.

Evaluation using the contrast method

OK

The standard deviation of the brightness distribution is less than the threshold.

Low contrast: There is a homogeneous range of brightness in the region of interest.

Flipped

The standard deviation of the brightness distribution is greater than the threshold.

High contrast: A wide variety of intensities occur in the region of interest.

Evaluation using gullwing connections

Example: Brightness threshold = 139, Flipped if brightness is above threshold: Yes

OK

The component is in the correct orientation. The brightness value determined in the regions of interest is 99.

Flipped

The component is flipped. The brightness value determined in the regions of interest is 179 and is above the brightness threshold.
