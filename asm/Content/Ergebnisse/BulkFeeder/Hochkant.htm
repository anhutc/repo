SIPLACE BulkFeeder X: Detection of upright components

When components are shaken out of the container, some of them may lie on edge (upright) and cannot therefore be placed. In order to ensure that these are not picked up, it is possible to enable upright detection for particularly small components in addition to dimension inspection (determination of the outer dimensions).

To do this, the vision system determines the average brightness level (possible values: 0 = dark through 255 = light) in a rectangle in the center of the connection areas on both sides of the component. The measured value is then compared with the brightness threshold (default: 128) that has been set in order to determine whether the component is lying correctly or is on edge.

If the measured value is greater than the brightness threshold, i.e. if the region of interest is brighter, the component is oriented correctly.

Components that are identified as being oriented incorrectly are shown under "Tilted components".

  • the closest brightness value above this threshold is 175 and is assigned to a component that is oriented correctly
  • the closest brightness value below this threshold is 145 and is assigned to a component that is on edge (upright)
