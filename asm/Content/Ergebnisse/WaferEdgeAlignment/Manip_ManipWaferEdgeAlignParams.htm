Recognition parameters: Substrate

The orientation of a substrate within the machine is determined by a notch in the substrate.

Two methods are available for detecting this notch:

1. Notch detection by contour tracing.

2. Notch detection by point pair filtering along the notch.

The second method is only used if the first does not return a result, as the second method takes longer than the first.

Notch detection by contour tracing:

Edge quality

1 .. 5

2

1: very good edge quality, the edge points are in line.
If edge points deviate from this, they are not taken into account when detecting the contour.

5: poor edge quality, the edge is "rough". Larger deviations in the external contour are tolerated.
