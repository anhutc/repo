Substrate: Notch detection by contour tracing

Determination of the position of the substrate means that the external contour is already known.

The system now checks where the external contour cannot be detected around this circle in order to detect the notch.

To do this, scan lines are defined perpendicular to the external contour. The system searches along these scan lines for any light/dark transitions present in the vicinity of the external contour.

Points are identified at which there is no sufficiently large change in the grayscale value (transition from the generally dark substrate to the lighter background). These locations are marked with yellow circles. If there are several adjacent locations marked in yellow, it is possible that there is a notch at this position.

The notch is deemed to have been detected if the majority of points in a sufficiently large area do not have a sudden change in grayscale value (shown in yellow).

Blue points

Change in grayscale value found: There could be an edge at this position.

Yellow points

No change in grayscale value found: There could be an notch at this position.

Green

Assumed edge position, marks the center of the notch.

Note on the representation: The blue and yellow dots are shown as circles at the position at which a notch is detected.

If there are several regions at which the notch may be located, the longest region is identified as the notch.

 

Result:

Quality of notch detection

If the determined quality is above the "Minimum recognition quality" threshold, the notch is deemed to have been found.
