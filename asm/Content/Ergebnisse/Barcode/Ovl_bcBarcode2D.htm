Recognition step: Modules (barcode)

Data matrix barcode:

Green circles

Bit in data matrix barcode: set.

Red circles

Bit in data matrix barcode: not set.

Blue circles

Finder pattern and alternating pattern:

Recognition pattern for position and orientation.

 

Depending on the definition, the red or green circles identify a bright or dark point.

Evaluation: The more points that correspond with the definition, the better the quality of the barcode.

 

If not many points correspond, you should attempt to improve the result by changing the illumination.
