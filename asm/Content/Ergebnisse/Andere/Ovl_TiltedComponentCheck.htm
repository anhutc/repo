Recognition step: Check for tilted chip components

This inspection can currently only be enabled for internal use.

 

If detection of tilted components is enabled, an additional image of the component is captured using steep 0° illumination.

Caution: Capture of the additional image reduces placement performance!

 

The top view shows the image with default illumination and the bottom view shows the image with 0° illumination.

Yellow

Outline of component with the measured component dimensions.

Light blue

Evaluated connection areas.

Orange

Brightness profile of the connection areas.

The maximum and minimum value of the corresponding brightness profile are output, together with the median value of the connection areas.

 

Evaluation: If the median value of the connection areas (0° illumination) is < 100, the component is marked as being tilted.
