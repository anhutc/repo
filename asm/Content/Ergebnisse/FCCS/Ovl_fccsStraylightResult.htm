Recognition step: Check external light

Detects whether scattered light / external light is affecting the measurement.

An image is captured without using any camera illumination.

The mean brightness in the region of interest is measured.

Yellow circles

Regions of interest when checking the camera for soiling.

Yellow rectangle

Region of interest for camera verification (illumination calibration),

Turquoise

Grayscale range of the entire region of interest

Yellow line

Threshold value for mean grayscale value measured in the corresponding region of interest.

Green line

Measured mean grayscale value is in the green range.

Red line

Measured mean grayscale value is in the red range. There is too much external light present. Measurements cannot be performed successfully.

Wherever possible, switch off external light sources or shade the machine from such light sources.

Green column

If the measured mean grayscale value in the region of interest is in this range, the result is OK. There is not too much external light.

Red column

The external light is too strong. The region of interest appears too bright.

If the image is too bright, this indicates that there is too much external light on the measured object.
