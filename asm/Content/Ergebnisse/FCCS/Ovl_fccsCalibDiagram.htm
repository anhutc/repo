Brightness calibration (FCCS): Results - Diagram

Inspection of the calibration tool for soiling.

Green bar (left)

Brightness gain set for the camera.

The upper and lower numbers specified indicate the range in which the gain can be set.

The middle number corresponds to the value actually set.

  

Gray bar

Indicates the medium brightness range with which the calibration should appear in the image.

  

Light blue line

Image brightness actually measured together with the associated value.

 

Blue bar

Indicates the range used to control the flash time.

Light background: Ideal range.

 

Red line

Actual value for controlling the flash time.
