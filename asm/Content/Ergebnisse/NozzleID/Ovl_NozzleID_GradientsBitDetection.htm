Recognition step: Gradient bit detection (nozzle ID)

Search for transitions between light and dark.

The gradient corresponds to the relevant grayscale value.

 

Blue

Brightness (profile):

Each value on this curve corresponds to the sum of the brightness values in one column of the image.

Green

Gradient:

The difference between the grayscale values of consecutive columns of the image. The greater the difference, the more marked the transition between light and dark.

The difference in brightness is particularly great at a peak. This is where a mark begins or ends.
