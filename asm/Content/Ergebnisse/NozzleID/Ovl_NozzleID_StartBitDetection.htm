Recognition step: Detection of start bit (nozzle ID)

A start bit can be identified by its size, because it is wider than all the other encoding bumps.

If a filter with the width of the start bit is overlaid over the calculated gradient, only the start bit produces a match, because the other bits are too narrow.

 

Blue

Filter result:

The higher the peak, the more likely it is that the start bit is at this location.

Green

Boundaries of the start bit.

Red

If a boundary of the start bit is shown in red, detection of the start bit has failed.
