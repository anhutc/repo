Recognition step: Detection of data bits (nozzle ID)

A filter with the width of the data bits is placed over the gradient curve derived from the transformed representation. If the peak is sufficiently high in the region, it is interpreted as 1, i.e. an encoding bump is detected.

 

Blue

Indicator of the probability that an encoding bump is located in this region.

Green line

If the peak lies above this line, an encoding bump is detected: 1.
 

If the peak lies between the green and the red lines, the region cannot be assigned on the basis of the peak. If, however, other unambiguous characteristics are present, assignment may still be possible under certain circumstances.

Red line

If the peak lies below this line, the region does not contain an encoding bump: 0.

Light blue

Indicates the start bit.

Example:

Code in the example: 0010 1000 0101 0100
