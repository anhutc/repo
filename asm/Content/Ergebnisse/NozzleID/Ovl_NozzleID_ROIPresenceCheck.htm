Recognition step: Presence of nozzle (nozzle ID)

The brightness distribution in the center of the image is used to determine whether a nozzle is in the garage.

If there is no nozzle in the garage, the base of the empty garage (metal plate) is illuminated, and this region of the image appears bright.

 

A number of different grayscale histograms were evaluated, averaged and stored for the purposes of similarity comparison for both cases (nozzle present, no nozzle present):

Green column

Measure of the similarity to an occupied garage.

Red column

Measure of the similarity to an empty garage.

 

The greater the difference between these two bars, the more reliable the result.

  • If an empty garage is detected (tall red bar), evaluation is terminated and the result is reported.
  • If a nozzle is detected (tall green bar), evaluation continues.
  • If the bars are of a similar height, evaluation is canceled with the result "not detected".
