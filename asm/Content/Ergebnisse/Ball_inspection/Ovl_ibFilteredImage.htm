Recognition step: Filter results (inspection)

Marks the positions at which the filter and the feature coincide:

The grayscale difference between the yellow and blue points is calculated. The points at which this value is greater than the minimum contrast value are shown as light points.

Green frame

This frame is placed around the points at which the grayscale difference value is greater than the specified minimum contrast.
The greater the difference, the lighter the point is displayed.

Yellow frame

Area in which the filter for a feature has been calculated.

Red frame

This frame is placed around the points at which the grayscale difference value is less than the specified minimum contrast.
The feature will not be recognized.

Blue frame

If the component is so large that the complete component does not fit into the field of view of the camera, only a portion of the component is inspected. The individual portions are indicated by blue frames.

 

Evaluation of the filter results with the threshold minimum contrast of the feature.
