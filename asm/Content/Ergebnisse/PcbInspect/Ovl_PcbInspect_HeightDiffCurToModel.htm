Recognition step: Height difference

Shows the differences between the current height map and the learned height map.

The height of each point of the learned height map (model) is compared with the height of the corresponding point in the current height map and the height difference is shown.

The higher a point is shown, the greater the height difference is at the corresponding point.

•

If all the solder deposits and components are recognized and there is no lost component on the board, a flat surface is shown here (no height difference):

•

If a large number of clear height differences, both positive and negative, can be seen, this may indicate that the position of the board has changed.

 

Inspection before placement:

Example:

The height difference is positive:

The height of the current board is greater than at the same point in the model.

A component was found that is not present in the model.

 

A component was lost.

Inspection before or after placement:

Example:

The height difference is negative:

The height of the current board is less than at the same point in the model.

A component is missing.
