Recognition step: Learned solder deposits / components

The results of the learning process are shown as a height map of the model inspection area. Learned solder deposits and components are shown as regions.

Solder paste inspection:

Turquoise

Teaching area.

Light green

Learned solder deposits.

Yellow frame

Complete inspection area.

Inspection before or after placement:

Turquoise

Teaching area.

Blue

Detected components.

Orange

Exclusion area.

Yellow frame

Complete inspection area.

Tips:

  • If it is not possible to learn existing solder deposits or components, you should check whether the current board is suitable for learning or is faulty and needs to be replaced.
  • If it is not possible to learn individual solder deposits or components within a group, you should place an additional teaching area precisely over this solder deposit or component.
