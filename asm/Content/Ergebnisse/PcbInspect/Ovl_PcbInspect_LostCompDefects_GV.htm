Recognition step: Lost components (grayscale image)

Displays the current inspection area.

Red border

A component that could not be found in the model.

At this point, the height difference between the model and the current board is so high that it is assumed that there is an additional component on the current board.

The component was probably lost during the placement operation and is now lying on the board.

Dark red border

A component that could not be found in the model.

At this point, the grayscale difference between the model and the current board is so high that it is assumed that there is an additional component on the current board.

 

Example:

Marked in red:

Significant height differences between the model and the current board have been determined at this point.

The current board has a component at a point at which there is no component in the model.

 

A component has been lost and is lying on the board.

Marked in dark red:

Significant brightness differences between the model and the current board have been determined at this point.
