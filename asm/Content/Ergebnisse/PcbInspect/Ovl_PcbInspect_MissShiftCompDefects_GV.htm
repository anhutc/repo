Recognition step: Result of component presence inspection

Inspection area before placement:

Checks whether all the components that must be located below the component that is to be placed are present at the specified position.

In addition, the positional offset compared with the corresponding component in the model is determined for each component. If this positional offset is greater than the calculated threshold (20 % of the size of the component), it is possible that the board may not function correctly.

Green

The component is present and positioned correctly.

Orange

The component is present, but out of position.

Red

A component is missing at this point.
