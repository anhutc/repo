Recognition step: Obscured areas

Blue frame

Regions in which there are areas which are clearly obscured as a result of the oblique illumination used. If there are components in these regions, it is possible that they will not be detected.

Tip:

  • Go to the view Learned solder deposits / components and check whether all the components that are present have been detected.
  • It is particularly difficult to detect components where obscured regions overlap.
  • If individual components are obscured by higher components in the vicinity, specify a teaching area for each of the obscured components. The size of this teaching area should correspond to the size of the component and the parameter Single component (identical size) should be set to Yes.
