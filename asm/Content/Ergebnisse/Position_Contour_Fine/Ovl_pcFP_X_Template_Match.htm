Recognition step: X template match (components without leads)

Determination of the center of the component in the X direction:

Black curve

The center of this curve above the red, horizontal line shows the X position of the component.

Clear result: The curve above the red, horizontal line is symmetrical.

Red vertical line

Result of the template method: Position of the component in the X direction.

Green vertical line

Result of the symmetry method: Position of the component in the X direction.
 

Red horizontal line

Auxiliary line for determining the center.

Light red horizontal line

Threshold in template for fine position determination.

Light blue bar

Minimum quality for template method.

Light green bar

Determined recognition quality.
If this bar is higher than the light blue bar,  the center in the X direction was detected.

Note:

If the green and red vertical lines are not in the same place, the detected positions are different. Possible causes include: Component edges are not clear (damaged, dirty, etc.), other negative influences.

Check the illumination.
