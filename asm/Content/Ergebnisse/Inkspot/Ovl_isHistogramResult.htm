Recognition step: Classification by histogram method

The model comprises a histogram for the "go" case ("place") and a histogram for the "no-go" case ("do not place").

During classification, the histogram of the region of interest of the current image and the deviation from the two histograms of the model are determined.

The image is assigned to the class where there is greater similarity to the histogram for that class.

If no assignment is possible, the machine is uncertain.

 

Example:

Left area:

Green histogram: Model for "go" case.

Red histogram: Model for "no-go" case.

Blue histogram: The current camera image.

 

Right area:

Blue bar on red background:

The height of the bar indicates the match with the red histogram for the "no-go" case (do not place).

Blue bar on green background:

The height of the bar indicates the match with green red histogram for the "go" case (place).

 

Result:

If the blue bar on the red background is higher, the system decides on "no-go" (do not place). If the blue bar on the green background is higher, the system decides on "go" (place).


In the example, the decision was "do not place".
