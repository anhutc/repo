Recognition step: Normalized template "place"

Normalized brightness representation:

In the displayed camera image for "place" ("go" case), the darkest point is shown with a grayscale value of 0 and the brightest point with a grayscale value of 255. This serves to compensate for brightness fluctuations in the line.

 

Example:

This shows the "go" template derived during teaching. This template is compared with the image to be evaluated by correlating it with areas of the image to be evaluated of the same size. Movement also creates a two-dimensional matching function in this case.
