Recognition step: Matching function "place"

This interim result provides a visual representation of the correlation with the "go" template.

This is a two-dimensional function whose results must be in the range -1 through +1. This function is represented in an image where the value range is from black (grayscale value 0, corresponds to a correlation value of -1) through white (grayscale value 255, corresponds to a correlation value of +1). A light maximum indicates a good correlation between the evaluated image and the template.
