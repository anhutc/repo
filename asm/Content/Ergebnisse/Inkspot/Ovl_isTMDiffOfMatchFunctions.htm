Recognition step: Difference between matching functions

Representation of the difference between matching function "place" and Matching function "do not place".

 

The location of the maximum is marked with a cross:

Green cross

Classifies the "go" case: "place"

Red cross

Classifies the "no-go" case: "do not place"

This view shows the difference between the two matching functions. At each point, the correlation value for the "do not place" template is subtracted from the correlation value for the "place" template. (Dark) minima thus indicate a high correlation between the image and the "do not place" template. Light maxima indicate a high correlation between the image and the "place" template.

In this case, a minimum is most obvious, and this is therefore taken to classify the component as "no-go" (it is marked with a red cross).
