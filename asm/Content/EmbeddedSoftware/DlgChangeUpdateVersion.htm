Change update version

<Name of subsystem> (BIOS) / (application x)

Installed version:

Firmware version installed on the selected subsystem.

Update version:

Firmware version available for download for the selected subsystem.

 

Accept for all subsystems with the same hardware ID

The selected update version is applied to all the subsystems shown in the table with the same hardware status.

The update version is only applied to the subsystem selected in the table.

 

Version format: [Variant ID].[Hardware ID].[Version].[Release]

The file name of the update version is made up as follows:

Variant ID: Shows the variant type:

0: Series status (no variant)

A: Special design solution

Hardware ID: Shows the subsystem hardware status.

Version: Shows the firmware version.

Release: Shows the release status of the firmware version.

 

Accept

Closes this dialog box and applies the settings for the update version.

Cancel

Closes the dialog box without saving the settings you have made.

 

Copy firmware file to embedded software update folder

Add update versions...

Opens a file selection window. This allows you to select an appropriate firmware file from any directory and copy it to the directory 'C:\EmbeddedSoftwareSetup' by pressing the Copy file button.
