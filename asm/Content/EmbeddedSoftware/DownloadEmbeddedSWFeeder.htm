Downloading the embedded software for a feeder

To update the embedded software for individual feeders:

  1. Opens the Service view.
  2. Press Embedded software.
  3. Press Update subsystem.
  4. Choose Feeder from the subsystems.
  5. Press the Next button.
  6. In the list, all the feeders are marked for which it is recommended that you update the embedded software.
  7. If you only want to update specific feeders:
    • Press Update no subsystem.
    • Choose the feeder(s) you wish to update.
    • Press Toggle update.
  8. Press Start embedded software update.
