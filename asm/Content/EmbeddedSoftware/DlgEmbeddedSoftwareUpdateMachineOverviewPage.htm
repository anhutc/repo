Update embedded software

BIOS

BIOS is the operating system for the subsystems.

Caution:If the BIOS download does not work, do not switch the machine off and do not attempt to carry out a hardware reset! Since the most recent BIOS is only saved in the RAM, a failed download in BIOS mode means that the subsystem would no longer be functional as a result of missing data.

Note:Attempt to load the BIOS again with the machine switched on.

Start embedded software update

The update of the embedded software is started with the selected options for those subsystems for which a more recent firmware version is available than that installed.

A dialog box opens indicating the progress and showing notes on the update. After the update has been completed successfully, you can close the dialog box by pressing Close.
