Singular feature

Evaluation of the description of geometrical forms:

The position in the image is determined at which the image correlates most closely to the stored geometrical model. Alongside position recognition, a quality value is determined which is a measure of how closely the image and the model correlate at the determined position.

This method is recommended for fiducials, as it avoids inaccuracies resulting from incorrectly placed reference points.

 

Advantage:

  • The fiducial or ink spot must not be at exactly the same position in the camera image.
    -> Compensation for transport tolerances
  • Insensitive to fluctuations in brightness/contrast
  • The reference point can always be precisely derived from the geometrical description and matches the board description

 

Disadvantage:

  • Cannot be used if artifacts with a similar structure are in the vicinity of the fiducial
  • Cannot be used when the "go" and "no-go" cases are not significantly different as a result of a change of the outer contour (e.g. with laser engraving)
