Template method

Evaluation of the teaching results for the "go" and "no-go" cases:

With this method, a 2D pattern is generated for the "go" and/or "no-go" case. This pattern comprises normalized brightness / normalized contrast images. For each image taught, a correlation value (measure of the correlation between the training image and the pattern) is calculated for the "go" and/or "no-go" case.

One-sided template method:

For each taught image, the correlation with the "go" or "no-go" sample is calculated.

Two-sided template method (when using laser engraving):

For each taught image, the difference between the similarities with the "go" and "no-go" patterns are calculated.

The difference is calculated as follows: Correlation value for "go" template - correlation value for "no-go" template.

This (correlation) value results in two value ranges for the "go" case and the "no-go" case. Ideally, these should be very distinct. The system now defines two threshold values which divide the value range into three categories: "place", "do not place" and "uncertain".

Whether the one-sided or two-sided template method is used can be set in the parameter list in the teaching interface.

 

Advantage:

  • The outer contour must be exactly described, as the entire area of the pattern is evaluated.
  • Transport tolerances can be compensated for if "Position recognition before ink spot recognition" is deactivated.

Disadvantage:

  • CPU intensive
  • Defined form required

Note:

The method is unsuitable for applications where the ink spot has been covered with a sticker or painted over. Under no circumstances should a correlation be drawn with an ink spot which has been painted over or which has had something stuck over it. This method is only recommended if a particular structure is modified by laser engraving for the "do not place" case. In this case, the two-sided template method should be used.

 

Example:

Training image 1: Square fiducial,

"Go" (place).

Training image 2: Square which has been struck through,

"No-go" (do not place).

 

Image which is to be classified
  1. Determination of the correlation value with the "no-go" template: E.g. 0.83
  2. Determination of the correlation value with the "go" template: E.g. 0.31
  3. Difference: E.g. 0.31 - 0.83 = -0.52
  4. Comparison with threshold values: E.g.: -1 through -0.27 "no-go"   -  0.2 through 1 "go"
    In the example, the decision is "do not place" (no-go).
