Definition of the geometrical form of fiducials and ink spots

If fiducials or ink spots are described in terms of their geometrical form, it is possible to use certain predefined patterns (e.g. circle, square, rectangle, cross, etc.). Depending on the fiducial shape selected, it is necessary to enter appropriate dimensional information in order to define the geometry of the fiducial as accurately as possible.

The dimensions of the fiducial defined in this way are overlaid over a camera image of the fiducial on the PCB. It is thus immediately clear whether the specified sizes are correct.

 

How to define the geometrical form of a fiducial or ink spot:

  1. Open the Select fiducial view.
  2. Select a conveyor lane.
    All the fiducials and ink spots specified in the panel for this conveyor lane are shown in the fiducial list.
  3. Select a gantry.
  4. Select the corresponding entry from the fiducial list:
    Click directly on the entry or select the entry using the arrow keys on the keyboard.
  5. Place a board in the selected conveyor.
  6. Click on the "Test and teach fiducial" icon .
    SIPLACE Vision is started and the board is moved to the relevant position.
  7. Click the "Geometry" icon .
  8. Choose the shape that best describes the fiducial or the ink spot: Different parameters are displayed, depending on your choice.
  9. Select all the parameters you want to change from the list one by one.
    The currently selected parameter is also selected in the representation on the left.
  10. A parameter can be changed once it has been selected.
    If the value is a numeric value, it can make sense to first set the increment using the arrow keys so that the parameter can be entered in as few steps as possible.
    Now click the or buttons repeatedly until the displayed value corresponds to the one you want or choose the numeric keypad and enter the required value directly.
  11. Test the settings by clicking the New measurement button.
  12. Start the Fiducial Wizard to learn the geometrical shape of the fiducial: Click Fiducial wizard.
  13. The Fiducial Wizard is opened with the default setting Learn fiducial as geometrical shape: Press Next.
  14. Select the region in which the fiducial is located in the camera image and press Next.
  15. After measurement was completed successfully, the result of the measurement is shown (in green). Press OK to accept the data and close the Fiducial Wizard.
