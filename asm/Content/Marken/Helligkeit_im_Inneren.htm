Inspecting the internal structure of circular fiducials

If there are any artifacts in the vicinity of the fiducial being detected and these artifacts closely resemble the outer contour of the fiducial, problems may arise when detecting the position of the board.

This happens especially when there are vias or spacer pins in the vicinity of circular fiducials.

 

Additional inspection is carried out in order to avoid incorrect placement:

 

1

Calculation of the average brightness of two regions within the fiducial:

close to the center and close to the edge.

 

The difference between these two brightness values is calculated.

Result:

> specified threshold

The 2nd check is carried out.

< specified threshold

The fiducial is recognized.

2

Calculation of the gradient value (value indicating the magnitude of the changes in the grayscale values) at all points within the fiducial:

 

The number of pixels with significant gradients is determined.

Result:

> specified number

The structure is not recognized as a fiducial.

< specified number

The fiducial is recognized.

 

 

Artifacts

    

Via

Outer contour: Circle, which approximately corresponds to the size of the fiducial shown at the bottom right.

 

The test for the average brightness fails. The via is not incorrectly recognized as a fiducial.

Spacer pin on the board

Outer contour: Circle.

 

The test for the average brightness fails.

 

The spacer pin is not incorrectly recognized as a fiducial.
     

Fiducials:

    

Fiducial with internal structure (significant differences in grayscale values).

 

The test for the average brightness fails, but the gradient test does not fail.

 

The fiducial is recognized.

Fiducial with constant brightness gradient

The test for the average brightness is taken to be passed.

 

The fiducial is recognized.
