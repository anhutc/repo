Teaching a fiducial

If the geometrical form of a fiducial prevents it from being described, the fiducial model can also be learned automatically by the recognition algorithm on the basis of a structure in the camera image. However, this structure must only appear once in the field of view of the camera and no other structure may appear with which it may be confused.

  • The fiducial model is created in a teaching process

  • In a training image, an area must be specified which is to be evaluated for creating the model.

  • During fiducial measurement, the location in the area to be evaluated is determined where there is the greatest correlation with the pattern which has been learned.

  • The reference point describes the orientation of the reported position relative to the pattern. It is either calculated automatically or specified manually.

 

When teaching a fiducial, the fiducial is recorded by the PCB camera. The vision evaluation unit then determines the fiducial structure parameters. The reference fiducials for the PCB to be assembled are thus defined as patterns. These patterns are sent to the programming system which in turn forwards them to all stations.

During placement, the PCB is loaded and PCB position recognition is performed. The stored pattern is compared with the actual fiducials and any deviations with respect to the position of the PCB are taken into account for the placement positions.

How to teach a fiducial:

  1. Open the Select fiducial view.

  2. Select a conveyor lane.
    All the fiducials specified in the panel for this conveyor lane are shown in the fiducial list.

  3. Select a gantry.

  4. Select the corresponding entry from the fiducial list:
    Click directly on the entry or select the entry using the arrow keys on the keyboard.

  5. Place a board in the selected conveyor.

  6. Click on the "Test and teach fiducial icon" .
    SIPLACE Vision is started and the board is moved to the relevant position.

  7. Click the "Geometry" icon .

  8. Click Fiducial wizard.

  9. Click Learn pattern.

  10. Set the illumination so that the fiducial can be clearly seen and press Next.

  11. Mark the area in which the system is to search for the fiducial:
    The region of interest must completely enclose the structure to be learned and should extend approximately 5-10 pixels beyond.

  12. Click Next.

  13. Define the reference point and press Next.
    The reference point is calculated and the model is taught and automatically measured.

  14. If you want to repeat the measurement for test purposes, click the New measurement button.
    This serves to determine the quality of the fiducial.

  15. Exit the editor.

Problems teaching the fiducial

  • Teaching failed:
    Possible causes:
    1. The fiducials are too small or the contrast is too poor.
    Try to describe the fiducial geometrically or select a different structure as a fiducial.
    2. The selected region of interest is too large.
    Reduce the size of the region of interest.
     

  • The reference point could not be set automatically (e.g. with asymmetrical fiducials). In this case, an error message is issued.
    Click the Set reference point manually button and specify the position by clicking on it in the search area in the camera image.

  • Measurement was unsuccessful:
    Start the results view .
    There you can establish the points at which problems occurred using the various results views.

Reference point:

The reference point must correspond to the position specified in the board description in the programming system.
