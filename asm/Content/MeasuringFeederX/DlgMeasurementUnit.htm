Generate report

Generates a report documenting the results of the measurements most recently performed for the components displayed.

You can enter your name and edit or add to the existing comment.

The report is saved under:

MeasuringFeeders_<MachineType>_<MID>_<Date>_<Time>.html

You can view the reports under LineDiagnostics in the programming system.

Tip: If a MeasuringFeeder X is to be verified, you can specify a job with reference components and measure this.

Last measurement of selected component

  Value Range
Expected Nominal value: Value given in the component specification. Permitted range: The tolerance for the component in respect of the nominal value or the maximum and minimum values as specified in the programming system.
Measured The measured value. Accuracy range: The range within which the determined value lies after accounting for measurement tolerance of the measuring device

The permitted range and the accuracy range must overlap for the measurement to be classed as OK.
