Electric measurement of component failed

An internal measuring error means that it was not possible to measure the component.

The measuring board is faulty.

Service engineer: Replace the measuring board and carry out zero calibration.

The contact module is faulty.

Remove the MeasuringFeeder X and send it off for repair.

Start the self-test.

If problem still exists:

Remove the MeasuringFeeder X and send it for repair, quoting the internal codes shown.

You can find the codes at Manual operations > Location x > MeasuringFeeder X  (Status > Internal codes).
