Endurance run for individual axes:

The axis moves backward and forward between two positions within the minimum and maximum travel range. It is possible to configure both these positions and the speed and acceleration.

Feed axis

WPC endurance run

Simulates a stress test under real conditions.

The system checks whether all levels are occupied by waffle pack tray carriers before each endurance run.

The WPC endurance run combines movement of several axes. This allows several different waffle pack tray carriers to be provided in sequence or in random order.
A list allows you to select the waffle pack tray carriers involved and a further list allows you to set the parameters for the endurance run.
All waffle pack tray carriers are selected by default.

Execute endurance run

The dialog box for selecting the start time for the endurance run is opened.

Note: During the endurance run, the Execute endurance run button and the buttons are shown on a green background. If the endurance run is halted during execution, they are shown on an orange background.
