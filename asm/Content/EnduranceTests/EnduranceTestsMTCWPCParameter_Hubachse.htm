Endurance runs for MTC / WPC: Parameters for the lifting axis and feed axis

Parametrize  

Parameter

Value

Wait time between Cycles [ms]

This is the wait time in the output area after each waffle pack tray carrier has been provided.

Number of cycles

Specified number of repetitions until the endurance run is automatically terminated.

Provide levels at random

Yes

The waffle pack tray carriers to be provided are selected randomly.

No

 

The waffle pack tray carriers are provided in sequence.

(E.g. on the MTC: Starting with tower 1, lowest level through tower 2, highest level).

Output waffle pack tray carrier to the non-stop-operation module.

Frequency

The frequency with which a waffle pack tray carrier is provided and moved to the non-stop-operation module.

Level

Specifies the waffle pack tray carrier.

Randomized level

A random waffle pack tray carrier is provided, rather than the waffle pack tray carrier of the selected level.

Speed

Multiple of [0.01m/s]

Start position [mm]

Specified start position of the lifting axis (must lie within the valid travel range).

End position [mm]

Specified target position of the lifting axis (must lie within the valid travel range).
