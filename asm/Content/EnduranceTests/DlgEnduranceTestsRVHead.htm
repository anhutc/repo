Head endurance runs

Head endurance run

The endurance run is performed with all head axes.

DP axis endurance run / Z axis endurance run / Star axis endurance run

The endurance run is carried out on the corresponding axis.

Star axis relative endurance run 

The endurance run is carried out on the star axis. In this case, the star axis is only moved to the positions of the segments selected under the subsystem Segments.

Star axis APP / Z axis APP (APP= Axis Performance Protocol: axis-specific time/travel chart)

The endurance run is carried out on the corresponding axis. In this case, the axis is moved to the positions defined in the file "PlaceHeadTypes.xml". The definitions can be modified directly in the file, which is stored under "C:\Sirio\Type".

Nozzle changer endurance run

Nozzles are picked up from the selected magazines and then returned using the activated segments.

Make sure that there is no nozzle on the head before the endurance run is started.

Execute endurance run

The dialog box for selecting the start time for the endurance run is opened.

Note: During the endurance run, the Execute endurance run button and the buttons are shown on a green background. If the endurance run is halted during execution, they are shown on an orange background.

Parameter Value
Wait time between Cycles [ms] Specified waiting time between the individual cycles.

Number of cycles

Specified number of cycles until the endurance run is automatically terminated.

Edit...

A dialog box opens in which you can change the preset values for the selected parameters.

  1. Enter the required value within the permitted range or select the required option from the drop-down list.
  2. Press the Accept button to confirm your entry or selection or press Cancel to cancel the operation.

Choose subsystems

Here you can choose the subsystems that are to be incorporated in the endurance run for the selected endurance run option. Depending on the endurance run option selected, a greater or lesser number of subsystems will be available for selection.

Segments

When this subsystem is selected, the segments which are relevant for the selected endurance run option are automatically activated. Some endurance run options allow the required segments to be activated and deactivated individually for the endurance run.

Z axis

When the Z axis is selected, the parameters for "Z axis down", "Component sensor" and "Z axis up" are shown in the parameter table and can be edited. The Z axis is moved up and down using the parameters set during the endurance run.

Star axis

When the star axis is selected, the parameters for the "Star axis" are shown in the parameter table and can be edited. The star axis is moved according to the parameters set during the endurance run.

Vacuum system

When the vacuum system is selected, the pressure parameters for "Z axis down (pick action)", "Z axis down" and "Z axis down (placement action)" are shown in the parameter table and can be edited. The endurance run for the Z axis is performed using the pressure parameters set.

DP axis

When the DP axis is selected, the parameters for the "DP axis" are shown in the parameter table and can be edited. The DP axis is moved according to the parameters set during the endurance run.
