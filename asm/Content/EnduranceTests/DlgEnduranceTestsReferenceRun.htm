Machine endurance runs

Reference run

Execute endurance run

The dialog box for selecting the start time for the endurance run is opened.

During the endurance run, the reference runs for all axes and the nozzle scanning operations are performed several times as determined by the specified count. This means that the head axes move to their zero point and the gantry axes move the placement head over the reject bin and backwards and forwards between the maximum and minimum positions five times.

Note: During the endurance run, the Execute endurance run button and the buttons are shown on a green background. If the endurance run is halted during execution, they are shown on an orange background.

Parametrize

Parameter Value

Number of reference runs

This is the number of times that reference runs and nozzle scanning are started.

Edit...

A dialog box opens in which you can change the preset values for the number of machine endurance runs to be performed (default: 5).

  1. Enter the required value within the permitted range or select the required option from the drop-down list.

  2. Press the Accept button to confirm your entry or selection or press Cancel to cancel the operation.

Choose subsystems

This allows you to select the subsystems that are to be included in the machine endurance run.

Conveyor

Boards are transported through the machine and back on all lanes.
To do this, insert a board in each conveyor lane.

Tape cutter

Is started after each overall machine reference run:
All tape cutters are actuated once in sequence.

Feeder locking

During every third endurance run cycle (cycle 2, 5, 8, ...):

The locking mechanism for all tracks is engaged once and then released.

Nozzle changer

During every third endurance run cycle (cycle 1, 4, 7, ...):

For all nozzle changers: The individual magazines of each magazine carrier are opened and then closed again in sequence.

Pin picker

Prerequisite: There must be a support pin in the calibration garage of a calibrated support pin magazine for each pin picker.

During every third endurance run cycle (cycle 3, 6, 9, ...) after the associated head reference run:

The pin picker picks up the support pin from the calibration garage and then replaces it.
