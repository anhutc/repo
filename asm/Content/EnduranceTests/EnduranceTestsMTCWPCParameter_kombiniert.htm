Endurance runs for MTC / WPC: Parameters for the combined endurance run

You can provide several different waffle pack tray carriers in sequence or in random order. First select the waffle pack trays you want to provide. All waffle pack trays are selected by default.

Parametrize  

Parameter

Value

Wait time between Cycles [ms]

This is the wait time in the output area after each waffle pack tray carrier has been provided.

Number of cycles

Specified number of repetitions until the endurance run is automatically terminated.

Edit...

A dialog box opens in which you can change the preset values for the selected parameters.

  1. Enter the required value within the permitted range or select the required option from the drop-down list.
  2. Press the Accept button to confirm your entry or selection or press Cancel to cancel the operation.
