Choose start options

 

Start now

The endurance run is started immediately.

When you click OK, the dialog box for executing an endurance run is opened. The endurance run is executed immediately with the parameters set.

Start later

You can set the required start time for executing the endurance run by entering the time directly or by using the arrow keys.

When you click OK, a dialog box showing the time remaining until the set start time is opened. (You can still click Cancel here to cancel the operation.)

When the set start time is reached, the dialog box for executing an endurance run is opened. The endurance run is executed with the parameters set.

Stop at
(can only be selected for the machine endurance run)

Here you have the option of specifying whether the endurance run is to be stopped at a specific time, irrespective of the number of repetition cycles executed up to that time.

OK

The endurance run is executed immediately or at the start time set, depending on the settings made above.  

Cancel

The dialog box is closed and the settings discarded.
