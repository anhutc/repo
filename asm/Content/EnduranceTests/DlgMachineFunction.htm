Endurance run

Execution

This shows the current status of the machine function.

Start: Start time of the endurance run.

End: End time of the endurance run.

The progress of the endurance run is shown by a progress bar.

Moves done: Number of moves performed during the endurance run.

Restart

The endurance run is restarted.

Note: This button is only enabled after completion of the endurance run or when the test is halted.

Pause

The endurance run is halted. The Execute endurance run button and the associated icons are shown on an orance background.

Collapse

The dialog box is hidden during execution. You can show it again if you click the Execute endurance run button on a green or orange background.

Close

Closes the dialog box.

Note: This button is only enabled after completion of the endurance run or when the test is halted.
