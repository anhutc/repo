VHF P&P head endurance runs

Head endurance run

The endurance run is performed with all head axes.

DP axis endurance run / Z axis endurance run

The endurance run is carried out on the corresponding axis.

Z axis APP (APP= Axis Performance Protocol: axis-specific time/travel chart)

The endurance run is carried out on the Y axis. In this case, the axis is moved to the positions defined in the file "PlaceHeadTypes.xml". The definitions can be modified directly in the file, which is stored under "C:\Sirio\Type".

Execute endurance run

The dialog box for selecting the start time for the endurance run is opened.

Note: During the endurance run, the Execute endurance run button and the buttons are shown on a green background. If the endurance run is halted during execution, they are shown on an orange background.

 

Selects an entry above the currently selected entry.

Selects an entry below the currently selected entry.
  1. Select the parameter you wish to change from the table.
  2. Press the Edit... button.

Edit...

A dialog box opens in which you can change the preset values for the selected parameters.

  1. Enter the required value within the permitted range or select the required option from the drop-down list.
  2. Press the Accept button to confirm your entry or selection or press Cancel to cancel the operation.

Choose subsystems

When you select the endurance run option "Head endurance run", those subsystems are offered here that will be included in the endurance run for the selected endurance run option. It is only possible to select the two segments for all the other endurance run options.

Z axis

When the Z axis is selected, the parameters for "Z axis down" and "Z axis up" are shown in the parameter table and can be edited. The Z axis is moved up and down using the parameters set during the endurance run.

Vacuum system

When the vacuum system is selected, the pressure parameters for "Vacuum system before Z axis down", "Vacuum system Z axis down" and "Vacuum system after Z axis down" are shown in the parameter table and can be edited. The endurance run for the Z axis is performed using the pressure parameters set.

DP axis

When the DP axis is selected, the parameters for the "DP axis" are shown in the parameter table and can be edited. The DP axis is moved according to the parameters set during the endurance run.
