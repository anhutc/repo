VHF P&P head > Functions

Functions

Reference head axis

Starts a reference run for all head axes, i.e. all head axes move to their zero point. When so doing, the gantry axes move the placement head above the reject box to discard the component in a defined manner.

Nozzle reference run

References the nozzle on the head.

Reject component

Deposits the component currently on the nozzle in the reject bin.

Head EPROM data

Reads the data from the head EEPROM and shows it in a dialog box:

Head: Product Id, Build Date, Camera type

Z axis: Zero point correction, minimum and maximum position, zero point correction for force sensor, pre-tensioning force for spring.

DP axis: Zero point correction

Move above reference point

Moves the gantry to a position above the edge of the conveyor for subsequent endurance runs or Z axis tests.

Zero correction

Calibrate Z axis

Starts a run to determine the zero correction of the Z axis.

Calibrate DP axis

Starts a run to determine the zero correction of the DP axis.

A dialog box appears showing the precondition that must be met for carrying out zero point correction and you must confirm this dialog box.
