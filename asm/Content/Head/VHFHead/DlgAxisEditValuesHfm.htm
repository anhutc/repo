The zero point correction value can be modified:

Head EEPROM

Shows the parameters that can be changed and the associated values stored in the head EEPROM for the selected axis.

Machine data

Shows the parameters that can be changed and the associated values stored in the machine data for the selected axis.

The zero point correction values stored in the head EEPROM and in the machine data must match.

The machine parameter data for the selected axis is read from the head EEPROM and listed under Machine data (positions).

The machine data that has been determined is written to the head EEPROM after you have confirmed that this is to be done in a dialog box.
