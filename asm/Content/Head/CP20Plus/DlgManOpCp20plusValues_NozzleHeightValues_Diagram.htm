Measured nozzle length

The measured heights of the nozzles on the head are shown in the form of a chart:

Magenta

Length sensor

Length calculated by component sensor.

Critical value for evaluation.

Dark blue

Length Z axis

Offset value determined when the Z axis makes contact with the conveyor edge

This value is only taken into account if the component sensor is not working.

To display the detailed values for one segment, position the mouse pointer over a segment.

Evaluation:

If the relevant nodes are in the range shown in green, the nozzle is okay.
