Manual operations > Head > Segments

Vacuum values

Segment

Number of the segment.

Type

Nozzle type

Open

The measured vacuum value when nozzle is open [mbar].

Closed

The measured vacuum value when nozzle is closed. The nozzle is positioned on the conveyor edge for this measurement [mbar].

Deviation

Calculated value showing the difference between the measured values for vacuum closed and vacuum open [mbar].

Min. difference

The minimum difference between the vacuum closed and vacuum open values [mbar].

Holding circuit

Measured vacuum value of the holding circuit [mbar]. 0 mbar: The value was not measured. (This occurs when the segment 10 positions further on was omitted.) This does not have any impact on placement operation.

Min. Holding

Minimum vacuum in holding circuit [mbar].

Status

Result of the vacuum measurement: OK or error message (marked in red).

Switches to the chart display of the vacuum values.

Measure vacuum values

Starts the nozzle reference run with the aim of updating the vacuum values.
