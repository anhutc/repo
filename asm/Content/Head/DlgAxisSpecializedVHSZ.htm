Light sensor values

Opens the dialog box for executing the Measure light sensor function. The measurement is started automatically. The Z height at which the sensor triggers is measured for each segment. The Results tab shows the values determined for each segment.

Current sensor values

Opens the dialog box for executing the Measure current sensor function. The measurement is started automatically. The Z height at which the current sensor triggers is measured for each segment. The Results tab shows the values determined for each segment.

Spring deflection distance

Opens the dialog box for executing the Measure spring deflection distance sensor function. The measurement is started automatically. The Z height at which the Z axis down light barrier triggers is measured for each segment. The Results tab shows the values determined for each segment.
