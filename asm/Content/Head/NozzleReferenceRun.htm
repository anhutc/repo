Starting a nozzle reference run

Determines the nozzle heights for all segments again. To do this, the gantry moves to a defined position above the board conveyor. The Z axis moves down with all nozzles in turn until the nozzles make contact with the conveyor.

 

The nozzle reference run can be started either separately or as part of the overall reference run.

If the machine is in automatic mode, the nozzle reference run is also started by pressing the Start button.

To start a nozzle reference run:

  1. Switch to the view "Manual operations > Head > Functions" (e.g., for the CPP head)

    and press the Reference nozzles button.

To start an overall reference run:

  1. Switch to the "Manual operations > Entire machine" view: 

     

    and press the Overall reference run button.
