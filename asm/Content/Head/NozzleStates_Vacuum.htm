Vacuum values - Status

The Vacuum values table shows possible vacuum errors as well as the status of the individual segments.

Caution:

Make this column as wide as possible to ensure that all the information is shown.

open vacuum too low

The measured vacuum value for an open nozzle is smaller than the threshold value specified for the nozzle type.

The nozzle is dirty or the nozzle type is incorrect.

Clean or replace the nozzle as specified.

difference too low

The calculated value for the difference between the measured values for vacuum closed and vacuum open is smaller than specified.

Check whether the correct nozzle type is fitted to the specified segment and whether the nozzle has shifted or is damaged.

closed vacuum too low

The measured vacuum value for a closed nozzle is smaller than the threshold value specified for this nozzle type.

Check whether the correct nozzle type is fitted to the specified segment and whether the nozzle has shifted or is damaged.

holding vacuum too low

Insufficient vacuum in the holding circuit.

Check whether the correct nozzle type is fitted to the specified segment and whether the nozzle has shifted or is damaged.

Check the vacuum pump and the hoses.

 

OK

The tests were completed successfully.

Not referenced

First perform the nozzle reference run.

Not used

As a result of problems with the process, this segment is not to be used during the placement process.
- The nozzle is dirty/worn and there is no replacement nozzle available
- The filter disk is missing
- High error quota for a segment
- Hoses are leaking
disabled by user

The segment has been disabled by the operator:
- The segment is faulty
- The DP axis does not move freely
- The segment should be checked the next time regular maintenance is performed
disabled by software
