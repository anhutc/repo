Vacuum values

Type

Nozzle type

Open

The measured vacuum value when nozzle is open [mbar].

Closed

The vacuum value in [mbar] with a closed nozzle measured during the last reference run with the current nozzle. The nozzle is positioned on the conveyor edge for this measurement.

Closed (calibrated)

Vacuum reference value:

The measured vacuum value in [mbar] for nozzle type 518 when nozzle is closed.

Deviation

Calculated value showing the difference between the measured values [mbar] for vacuum open and vacuum closed (calibrated).

Min. difference

The minimum difference between the vacuum open and vacuum closed values (calibrated) [mbar].

Status

Result of the vacuum measurement: OK or error message (marked in red).

Measure open vacuum values

Starts the nozzle reference run with the aim of updating the vacuum values (open and closed).
