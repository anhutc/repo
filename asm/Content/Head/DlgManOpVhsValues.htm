Matching nozzle

If the reference run detects that an incorrect nozzle is on the head, this shows the nozzle types for which the values determined could apply.
Ensure that the nozzle specified under "Type" is on the head.

Nozzle reference run

Starts the nozzle reference run and enters the values that are determined.

Selection

Enable

Activates the selected segment.

Disable

Deactivates the selected segment. No reference run is carried out and no values are determined.
If nozzles are faulty or if they have been removed from the segment, you should deactivate the segment.

 

Vacuum values

Segment

Number of the segment.

Type

Nozzle type

Open

The measured vacuum value when nozzle is open.

Closed

The measured vacuum value when nozzle is closed. The nozzle is positioned on the conveyor edge for this measurement.

Deviation

Calculated value showing the difference between the measured values for vacuum closed and vacuum open.

Min. difference

The minimum difference between the vacuum closed and vacuum open values.

Holding circuit

Measured vacuum value of the holding circuit. 0 mbar: The value was not measured. (This occurs when the segment 11 positions further on was omitted.) This does not have any impact on placement operation.

Min. Holding

Minimum vacuum in holding circuit.

Status

Result of measurement:

OK or error message (red highlighting).

Measure vacuum values

Starts the nozzle reference run with the aim of updating the vacuum values.

 

Height values

Segment

Number of the segment.

Type

Configured nozzle type.

Expected Length

Specified nozzle length.

Length Z axis

Offset value determined when the Z axis makes contact with the conveyor edge

Length sensor

Length calculated by component sensor.

Component sensor up

Nozzle length measured by component sensor when head is traveling up.

Component sensor down

Nozzle length measured by component sensor when head is traveling down.

Nozzle spring

 

Spring compression in mm: Specifies the distance between the nozzle making contact (e.g. with the conveyor edge or with the board) and activation of the Z axis light barrier.
