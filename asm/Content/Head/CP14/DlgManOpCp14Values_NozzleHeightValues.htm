Status

Description of the status of the segment:
Possible states.

Very low force *

 

 

 

 

 

 

When the first board of a new job is moved in, a check is performed to establish whether it is possible to process components with a force of 0.5 N using the segments. Prerequisite: At least one component has been specified that is only permitted to be placed, picked up or dipped at very low force.

OK

The segment can be used to place components for which the placement, pickup or dipping force should be 0.5 N.

Not OK

Components that are to be placed, picked up or dipped at a force of 0.5 N cannot be picked up by this segment.

Not measured

No check as to the suitability of the segment has been performed.

Switches to the chart display of the height values.

Measure height values  

Starts the height reference run for the nozzles. 

 

* The "Very low force" column is only shown if a processing step with a force of 0.5 N is required for at least one component from the setup.
