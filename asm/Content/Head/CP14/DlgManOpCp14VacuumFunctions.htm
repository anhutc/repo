Head > Vacuum

Common functionality

Actions

Vacuum set value [mbar]:

Vacuum value set for checking the vacuum.

Vacuum on

Switches the vacuum on in the placement circuit.

Pressure set value [mbar]:

Nominal pressure at which the pressure regulator is to be set during testing.

Pressure on

Switches the specified pressure on in the placement circuit.

Threshold for pressure measurement [%]:

Threshold value in percent at which the target pressure is to be measured

Ambient pressure

Switches the vacuum system to a nozzle-dependent reference value.

Timeout for target pressure [ms]:

Maximum permitted time to reach the target pressure.
If the required pressure has not been achieved within this time, the function is canceled.

Measure pressure

Measures the pressure in the holding and placement circuit. The measured values are displayed.

Results

Holding circuit [mbar]

Pressure measured in the holding circuit.

Placement circuit [mbar]

Pressure measured in the placement circuit.

Time [ms]

Time required to achieve the nominal pressure.
