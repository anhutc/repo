Selftest air pressure - Results tab

After completion of the self-test for the pressure regulator, a table with the following contents is shown in the Results tab:

Column description:

Target Pressure (mbar)

Pressure at which the pressure regulator was set during testing.

Target Threshold (mbar)

Threshold at which the the system tests whether the specified pressure has been reached.

Measured Pressure (mbar)

Pressure that was measured.

Max. time (ms)

The vacuum system must have reached the target threshold within this time.

Used Time (ms)

Time required to achieve the actual pressure.

Status

Result of measurement

OK -> The measurement was okay.

Not OK -> Permissible values were exceeded.

Restart

Restarts the pressure regulator self-test.

Close

Closes the dialog box.
