Common functionality

Position

Shows the current position of the selected axis.

Axis functions

Servo on

Switches the servo of the axis on.

Servo off

Switches the servo of the axis off.

Axis reference run

A reference run is performed with the axis.

Move axis...

Opens the Move Axis dialog box in which you can specify the target position, travel profile, acceleration, and force used to move the axis.

The axis is moved to the specified target position.

Check motor adjustment

Checks the mechanical alignment of the star axis and shows any deviations detected.
