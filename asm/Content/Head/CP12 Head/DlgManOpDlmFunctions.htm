C&P12 head > Functions

Functions

Reference head axis

Starts a reference run for all head axes, i.e. all head axes move to their zero point. When so doing, the gantry axes move the placement head above the reject box to discard components in a defined manner.

Nozzle reference run

References all nozzles on the head.

Reject components

Deposits all components currently on the nozzles in the reject bin.

Scan nozzles

Determines the position and height of the nozzles on the segments.

Move above reference point

Moves the gantry to a position above the edge of the conveyor for subsequent endurance runs or Z axis tests.

Perform maintenance

Valve plunger

Moves the head to a position at which the maintenance on the valve plunger can be performed.

Steps the head one segment anti-clockwise.

Steps the head one segment clockwise.

Steps the head one half of a segment anti-clockwise.

Steps the head one half of a segment clockwise.

Service position

The following work can be carried out on the segment in this position:

• Inspection of the nozzle

• Manual check of the nozzle seating

This segment can be rotated at the DP axis.
