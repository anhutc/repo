Manual operations > Head > Segments

Vacuum values:

The calculated value showing the difference between the measured values for vacuum closed and vacuum open is shown for each segment in the form of a chart.

To display the detailed values for one segment, position the mouse pointer over a segment.

Evaluation:

If the relevant nodes are in the range shown in green, the nozzle is okay.

If a node is in the red range, the specified minimum distance was exceeded or not reached. The segment is shown in red.

Check that the correct nozzle is on the segment.

Switches to the tabular display of the vacuum values.

Measure vacuum values

Starts the nozzle reference run with the aim of updating the vacuum values.
