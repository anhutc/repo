Measure voltage

This dialog box allows you to check the effectiveness of any cleaning of the component sensor.

Restart

Determines the current voltage at the component sensor and shows the result in the view.

Close

Closes the dialog box.

0 V < measured voltage < bottom error threshold

The component sensor is heavily soiled or the laser is faulty or the laser beam is obstructed.

Bottom error threshold < measured voltage < bottom warning threshold

The component sensor is dirty and must be cleaned.

Bottom warning threshold < measured voltage < upper warning threshold

The component sensor is okay.

Upper warning threshold < measured voltage < upper error threshold

The component sensor should be replaced at the next opportunity (e.g. service).

Upper error threshold < measured voltage < 5 V

The component sensor must be replaced.

Example:

The voltage shown in the example (3.5 V) shows that the component sensor is okay.
