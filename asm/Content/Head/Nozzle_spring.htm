Nozzle spring

If the spring pre-tension value is above or below the value thresholds (0.05 mN and 0.5 mN), the default value of 0.15 mN is entered here.

If a value of 0.15 mm is entered for a nozzle, you should check the nozzle spring and replace it if necessary.

If the value 0.15 mN is entered for all segments and the nozzle lengths determined using the component sensor are entered, this is an indication that the head does not move to the conveyor edge correctly, i.e. that the axis does not move freely.
