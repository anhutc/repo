CPP head > Functions

Functions

Reference head axis

Starts a reference run for all head axes, i.e. all head axes move to their zero point. When so doing, the gantry axes move the placement head above the reject box to discard components in a defined manner.

Nozzle reference run

References all nozzles on the head.

Reject components

Deposits all components currently on the nozzles in the reject bin.

Scan nozzles

Determines the position and height of the nozzles on the segments.

Head EEPROM data

Reads the data from the head EEPROM and shows it in a dialog box.

Head: Product Id, Build Date, Camera type.

Z axis: Zero Point Correction, Minimum Position, Maximum Position.

Star axis: Zero point correction.

Write calibration data to EEPROM (only shown on Service activity level)

The determined zero point correction values are saved to the head EEPROM.

Calibrate component sensor

Determines the angle of the individual segments to the component sensor to improve presence checking with very small components.

Activate previous calibration data for component sensor (Only displayed on the Service activity level)

The data determined the last time the component sensor was calibrated will be used again.

Check nozzle seating

Moves the gantry over the reject position and then moves the Z axis down, firstly with the vacuum switched on and then with a forced air pulse. Each time, the position of the nozzle tip is determined using the component sensor. If the difference between the two measurements is above a defined threshold, the nozzle is marked as worn and must be replaced.

Move above reference point

Moves the gantry to a position above the edge of the conveyor for subsequent endurance runs or Z axis tests.
