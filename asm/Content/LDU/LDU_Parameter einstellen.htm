LDU: Production parameters

Production parameters:

Mode

 

Cavity:

The flux layer thickness is determined by the mechanical impression in the dipping plate.

Variable gap:

 

The flux layer thickness is determined by the freely programmable gap height.

Gap height

Only in "Variable gap" mode: Distance between the bottom of the flux container and the dipping plate [µm].

Appl. Speed

Squeegee speed of the flux container [µm].

Warm-up time

Period during which the squeegee is in operation after the "Warm-up" function is called [ms].

Interm. Cycle

If no movement of the squeegee is detected within the time set here, a squeegee cycle is initiated [s].

Cycles after refill warning

Number of squeegee cycles still possible after a filling level warning has been issued and before an error message is issued.

 

When you confirm your entries with "OK", the system switches to the submenu and the "+" and "-" keys are enabled. "OK" saves the set value.
