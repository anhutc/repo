LDU: Function keys and LEDs

Function keys:

OK

Selection key / acknowledgment key

BACK

Goes back to the previous menu, cancels the function.

SELECT

Selects the previous line.

SELECT

Selects the next line.

Value +

Increments the value of the selected parameter:
Pressing the button once increments the parameter by 1 each time.

Pressing and holding the button constantly increments the parameter by 10.

-

Value -

Decrements the value of the selected parameter:
Pressing the button once decrements the parameter by 1 each time. 

Pressing and holding the button constantly decrements the parameter by 10.

LEDs:

Green

Status OK

Yellow

LDU not ready or only partially ready, LDU in programming mode.

Red

LDU not initialized or LDU in error mode.

blue

Shows the active line if this can be selected.
OK: The function of the line is selected.

The LDU display has the following pages:

Start page
Main menu
Production parameters
Production functions
System setup
System information

The "System setup" and "System information" pages will not be described here.
