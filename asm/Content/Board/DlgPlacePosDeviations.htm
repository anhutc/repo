Board > Display deviations for place positions

 

Board height deviation

2D absolute

Discrepancy between the actual value and specified value for the height at which the component was placed.
If the value is positive, the board is higher than expected at this position.

2D relative

Deviation from the arithmetic mean of the absolute values.

3D scaled

Three-dimensional representation of the height deviations.

Actions

Update board view

If the selected board is in placement mode, you must press this button to obtain an up-to-date image.

Conveyor

Press on a board icon on the conveyor to show the components that have already been placed on this board.

Graphic:

Every placement position on the board is shown as a rectangle whose outline indicates the size of the component.

The color used to show the component indicates the height deviation at the corresponding position as determined by the color scale which is displayed.

Example if "Absolute" is selected:

If the component is shown in green, the component was placed at the expected height (the actual value corresponds to the specified value, i.e. deviation = 0).

If the component is shown in red, the board is approximately 2 mm higher than expected. i.e. the component is placed higher than expected.

Evaluation:

Absolute

Uniform color distribution

The entire board is higher than expected.

The board is flat and is at the expected height.

The entire board is lower than expected.

The components are shown in a different color in certain regions.

The board is curved.

Check whether boards support can be improved.

Individual components of the same type are shown in different colors.

The actual height of individual components differs from the specified height.

Uneven color distribution in particular regions

The board is vibrating.

Check whether boards support can be improved.

Relative

Uniform color distribution
The board is flat and is not vibrating.

Individual components are shown in different colors.

The actual height of individual components differs from the specified height or the nozzle height is incorrectly set.

Tip:

The precise height values are displayed along with the placement position name, the component name and the component shape name if you move the mouse pointer over a placement position.
