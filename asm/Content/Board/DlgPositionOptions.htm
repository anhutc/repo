Vision measurement options

Dump on error

The measurement result is saved for the selected placement step(s) after every measurement step during which an error occurs.

Dump always

The measurement result is saved for the selected placement step(s) after every measurement step.

Stop on error

The machine is stopped if an error occurs during a placement step.

Take an image of the component...

In addition to the standard measurement of a component after it has been picked up, it is possible to halt the placement process, take an image and save the results as a Vision dump for the purposes of analysis. It is possible to analyze placement errors using the Vision dumps saved for a particular placement position.

However, this can impact significantly on placement performance.

Before pickup

Takes an image of the component before picking it up.

Before pick up (component pocket mode)

An image is taken of the component in the component pocket before it is picked up.

After placement

An image is taken of the placed component.

Compare image before placement (can only be selected for certain placement heads)

The placement process is interrupted before the component is actually placed. The component is positioned in front of the relevant camera at reduced speed and an image is captured.

Comparison with other images makes it possible to detect whether the component has slipped between pickup and placement.

OK

The selected options are set for the placement position and the dialog box is closed.

Cancel

Your entries are discarded and the dialog box is closed.

If only one placement position has been selected, the options that previously applied to this position are displayed when this view is called. If several placement positions have been selected, the view shows the default values for the options.
