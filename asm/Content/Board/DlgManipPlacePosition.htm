Functions > Placement position adjustment

Every click of these buttons moves the component one step of the selected step size in the corresponding direction or corrects the coordinates by the appropriate amount.

Specify the step size.

First click this button and then click a point in the camera image. This position is specified as the reference point (center of the body of the component).

Every click of these buttons rotates the component by the set angle in the corresponding direction in the display or corrects the placement angle accordingly.

Specify the angle step size.

Reset position

The initial values are restored.

Clear position offset

The values for "Position relative to placement position" are reset to 0, and the component is again at the absolute placement position.

Select an action to continue placement.

Accept position

Accepts the corrected placement position and stores it in SIPLACE Pro.

Accept position and reject component

For test purposes (e.g. placement without components):

Accepts the corrected placement position, stores it in SIPLACE Pro, but does not place the current component.

Position

X [mm] / Y [mm]

Coordinates of the placement position in X and Y.

Angle [degrees]

Placement angle.

Position relative to placement position

Offset from the original placement position in camera coordinates.

Absolute placement position

Position specified by SIPLACE Pro in board coordinates.

View

Component shape

Shows the contours of the component as per the component shape description.

Component image

Shows the current real image of the component (displayed as a semi-transparent image over the representation of the placement position).

Switches back to the Functions > X-Table > Service data view.
