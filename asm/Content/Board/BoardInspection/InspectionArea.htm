Inspection area

In the programming system, you can specify for individual placement positions that the area in which the component is to be placed is to be inspected.

First, the model of an inspection area has to be learned. During the placement operation, the inspection areas are compared with the corresponding model inspection areas.

There are three different inspection types:

Solder paste

 

 

Inspection of the solder paste is done on the first placement machine in the line so that faulty boards can be identified and discarded immediately.

•

Height of the solder deposit.

•

Volume of the solder deposit (area x height).

•

Delineation of the solder deposit in order to identify solder bridges.

Before placement

 

Immediately before the component is placed:

•

Components below the current component have already been placed

•

There is no lost component in the inspection area

•

The possible placement offset is below the specified threshold.

After placement

 

Immediately after the component is placed:

•

The component has been placed correctly.

•

The possible placement offset is below the specified threshold.
