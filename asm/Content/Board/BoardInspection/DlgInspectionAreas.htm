Inspection areas

List of inspection areas for the selected lane:

Name

Name of the inspection area.

Inspection type

 

 

Solder paste

Violet area in the representation:

Inspection of the solder paste is done on the first placement machine in the line so that faulty boards can be identified and discarded immediately.

•

Height of the solder deposit.

•

Volume of the solder deposit (area x height).

•

Delineation of the solder deposit in order to identify solder bridges.

Before placement

Yellow area:

Immediately before the component is placed:

•

Components below the current component have already been placed

•

Any placement offset that is present for components already placed is below the specified threshold

•

There is no lost component in the inspection area

After placement

Brown area:

Immediately after the component has been placed, e.g. for BGA and QFP components:

•

The component has been placed correctly.

•

The possible placement offset is below the specified threshold.

Reference object

 

Solder paste

Name of a panel: The placement position for which the inspection area has been defined is assigned to this panel.

Before / after placement

Name of the placement position for which the inspection area has been defined.

Status

 

 

 

Protected: The inspection area cannot be changed at the station.

Incomplete: The inspection area has not yet been completely described. Adjustments must be made in the SIPLACE Vision views.

Skip inspection.

Teaching required: An inspection area defined in the programming system has not yet been learned in SIPLACE Vision.

Teaching required: The operator has required that teaching is to be carried out again.

Selected inspection area

Teaching required

The selected area must be measured and saved again in SIPLACE Vision before inspection.

This option can be set, for example, if the model that was initially learned cannot be recognized well.

The machine stops when the placement position or panel is reached.

Skip inspection

Inspection is not performed.

Board representation:

The inspection area selected in the list is indicated by a dashed blue frame in the graphical representation.

The icons for Teach and Skip indicate the areas that either still have to be learned or for which inspection is not to be carried out. If no icon appears, the area has already been learned and will be inspected during placement.

Example: An inspection area that is to be inspected after placement still has to be learned.
