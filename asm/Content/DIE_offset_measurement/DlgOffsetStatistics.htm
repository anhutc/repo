Statistics for front-back offset measurements

Deviation: Measured position of the top-side feature vs. expected position of the top-side feature (as a result of measurement from the bottom).

Valid measurements: Measurements for which the determined deviation is less than the position tolerance specified in SIPLACE Vision.

Specification

Minimum percentage of valid measurements

Number of successful measurements / number of measurements: Value specified in the programming system that must be achieved for determination of the offset to be valid and taken into account when placing the components.

Statistical values

Percentage of valid measurements

Number of successful measurements / number of measurements.

Number of measurements

Total number of measurements performed.

Number of valid measurements

The measurements whose results will be taken into account when determining the offset.

Statistical analysis

Positional tolerance

Maximum permitted deviation for the position of the top-side feature; set in SIPLACE Vision.

Mean offset

Deviation calculated across all successful measurements.

Standard deviation

Scatter of measurement results.
