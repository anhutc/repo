Buttons for the camera image

 You can change the display with the following buttons:

Zoom in: Magnifies the camera image around the center of the image.

Zoom out: Reduces the camera image.

Center:
Centers the image.
This function is only available if the camera image has previously been moved or magnified.

Pseudocolors: Represents grayscale values using a color palette.

Crosshair: Displays a crosshair in the current image to facilitate orientation.

1:1 representation of the camera image.

Fullscreen representation: The entire size of the window is used to show the Vision image.

 

Moves the camera image in the direction of the arrow.
