Functions > Options

Image unit

Coordinate specifications for the camera image are shown in pixels or millimeters.

Unit

All size and coordinate specifications are displayed in the selected units (inches or millimeters) and user entries are expected in the same units.

Save to file

Save camera image...

Opens a dialog box in which you can define the location to which the current camera image is to be saved.

Save SIPLACE Vision dump file...

Opens a dialog box in which you can define the location to which a measurement context is to be saved.

All the data required to reconstruct a measurement job is saved.

Import data

Import SIPLACE Vision data...

Opens a dialog box in which you can select a previously saved Vision model.

After the data has been imported, the currently loaded Vision data will be replaced by the data stored externally.

Import ball description...

Opens a dialog box in which you can select a ball list or column list for SIPLACE Vision.
The ball list or column list must be available as a text file with the following structure if it is to be read into SIPLACE Vision: <X position> separator <Y position>
Spaces, commas or "|" can, for instance, be used as separators.
