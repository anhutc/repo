Glue dot inspection: Selecting illumination

Yellow frame

Target area in which the glue dots is expected.

Green outline

Detected outline of the glue dot. The size and position of the glue dot correspond to the specification.

Green cross

Center of the detected glue dot.

Red outline

Detected outline of the glue dot which has been identified as incorrect.

Red cross

Center of the glue dot which has been identified as incorrect.

Evaluate the camera image:

  • If it is possible to identify the glue dot easily using the component illumination, this illumination should be used. Click Use component illumination, followed by Next.
    The Edit glue dot data view opens.

  • If the outline of the glue dots is not clearly visible, click Use glue dot illumination, followed by Next.
    The Set illumination view opens. In this view, choose a meaningful illumination value for the glue dots.

 

Use glue dot illumination

The component illumination is not sufficient to detect the glue dot correctly. An illumination setting will therefore be chosen with which both the component and the glue dot can be identified successfully.

Use component illumination

The current setting for illumination of the component will be used.

 

Find optimum glue dot illumination

Starts an automatic search for a suitable illumination.

Measurements with different illumination values are carried out for the first glue dot for approximately 15 seconds.

If this optimization process was successful, Use glue dot illumination is selected automatically.

 

Next

Switches to the Set illumination or Edit glue dot data view.

Cancel

Closes the Wizard and rejects the changed data.
