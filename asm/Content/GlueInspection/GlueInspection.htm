Inspecting glue dots

The following aspects are checked for every glue dot:

  • Presence of the glue dot

  • Is the size of the glue dot within tolerance?

  • Is the glue dot completely within the specified target area?

 

Checking is carried out on the basis of a comparison measurement. This means that a camera image is taken both before and after the glue is dispensed. The difference between these images represents the glue dot.

 

The illumination setting is critical when detecting glue dots, because the glue dots are generally semi-transparent. The illumination must be set in such a way that the edge of the glue dot is clearly visible.

 

This inspection can be deactivated for specific glue dots.

This is recommended, for instance, under the following circumstances:

  • If reliable detection is not possible because of the underlying structure of the component.

  • If two or more glue dots overlap, they cannot be detected individually.

  • If the brightness of the components varies considerably within a batch, the illumination will no longer be appropriate.

  • If round structures are present in the target area, these can be confused with glue dots.
