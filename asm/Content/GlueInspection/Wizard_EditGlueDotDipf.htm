Glue dot inspection: Edit glue dot data

The following parameters can be set:

Parameters

Description

Measure glue dot

Yes

Recognition of the selected glue dot has been activated.

No

Disables measurement of the selected glue dot.

Position X / Y [mm]

Shows the coordinates of the center of the glue dot.

Diameter [mm]

Shows the diameter of the glue dot.

Diameter tolerance

Permitted deviation of the diameter.

Length of target area in X / Width of target area in Y [mm]

Length and width of the area in which the system searches for the glue dot.

 

Select target area

Manual specification of the size of the area in which the system searches for the glue dot:

A cross is shown in the image when you click this button:
Draw the region of interest around the glue dot by clicking in the window and then moving your finger over the touchscreen or moving the mouse pointer over the image while holding the mouse button down (the position at which you pressed the left mouse button indicates the top left corner and the position at which you released the left mouse button indicates the bottom left corner).

Next

Switches to the Accept glue dot data view.

Back

Switches back to the Set illumination view.

Cancel

Closes the Wizard and rejects the changed data.
