Glue dot inspection: Accept glue dot data

Yellow frame

Target area in which the glue dots is expected.

Green outline

Detected outline of the glue dot. The size and position of the glue dot correspond to the specification.

Green cross

Center of the detected glue dot.

Red outline

Detected outline of the glue dot which has been identified as incorrect.

Red cross

Center of the glue dot which has been identified as incorrect.

 

The recognition steps for glue dot inspection are displayed and can be selected.

 

Back

Switches back to the Edit glue dot view.

Accept (Only shown if glue dot parameters have been changed.)

The modified glue dot parameters are saved.

OK (Only shown if no glue dot parameters have been changed.)

The current settings are retained and the wizard is closed.

Cancel

Closes the wizard and rejects the changed data.
