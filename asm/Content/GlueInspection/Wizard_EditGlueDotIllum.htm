Glue dot inspection: Set illumination

Yellow frame

Target area in which the glue dots is expected.

Green outline

Detected outline of the glue dot. The size and position of the glue dot correspond to the specification.

Green cross

Center of the detected glue dot.

Red outline

Detected outline of the glue dot which has been identified as incorrect.

Red cross

Center of the glue dot which has been identified as incorrect.

How to adjust the component camera illumination for glue dots:

  1. Select an entry from the list of illumination angles:
    E.g. steep, medium, flat.

  2. Set the required intensity for the relevant illumination type:
    Click the or buttons repeatedly until the required value is reached or click and enter the value in the numeric keypad.

 

A new camera image is acquired and measurement of the component is carried out each time the illumination setting is changed.

Make sure that the outline of the glue dot is clearly visible.

 

Use one illumination setting for components and glue dots

Click this button if both the glue dot and the component can be detected well using the glue dot illumination currently set.

Use two illuminations

Two images are captured. The component is measured using the component illumination, and the glue dots are measured using the glue dot illumination.

This is necessary if the component and the glue dots cannot be measured with any degree of robustness using a single illumination setting.

Caution: Two images will be captured each time, which will impact negatively on placement performance.

 

Hint:

If at all possible, use the same illumination for the component and the glue dots in order to avoid any loss of placement performance.

A second illumination should only be activated if this is not possible.

 

Find optimum glue dot illumination

Starts an automatic search for a suitable illumination.

Measurements with different illumination values are carried out for the first glue dot for approximately 15 seconds.

Use two illuminations is automatically selected because the optimization process does not check whether the component can also be measured reliably using the illumination that was determined.

 

Next

Accepts the illumination currently set as the glue dot illumination and switches to the Edit glue dot data view.

Back

Switches back to the Set illumination view.

Cancel

Closes the Wizard and rejects the changed data.
