Name

Name of the glue parameter set.

Comment

Freely selectable text describing the glue parameter set.

Glue dot size settings

Cycle time [ms]

Time between 2 shots.

Diameter [mm]

Required size of the glue dot.
In order to produce a glue dot of this size, it is necessary to dispense the number of shots specified under Shots at the predefined distance.

If a glue dot of a specific size has been specified in SIPLACE Pro, the values for the next smaller defined setting are taken over from the current glue feeder parameter set.

Distance [mm]

Distance between component and dispenser nozzle.

Shots

Number of dispensing shots per glue dot.

Add diameter

Creates a new definition for the size of a glue dot depending on the distance and number of shots.

Remove diameter

Deletes the settings for the selected diameter.

Dispenser nozzle settings

Dispenser nozzle shape

Set dispenser nozzle type.

Automatic measurement time interval

The dispenser nozzle is automatically checked to see whether it is dirty after the time interval specified here has elapsed.

Automatic measurement shots interval

The dispenser nozzle is automatically checked to see whether it is dirty after the number of shots specified here have been dispensed.

Measurement lower bound [%]

The measured diameter of the dispenser nozzle must be within the limits specified here. If this is not the case, an error message is issued and the dispenser nozzle must be cleaned or replaced.
