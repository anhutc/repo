Setting the specified temperature of the glue

The temperature of the glue can be set in a range 40°C through 70°C.

A specified time period is available for the specified temperature to be reached:

Heating up: 60 seconds.

Cooling down: 180 seconds.

If the temperature is not reached within this time, an error message is issued.

 

  1. Activate the menu for setting the temperature.

  2. Pressing the Forward or Back button raises or lowers the currently specified temperature by 1° respectively.

  3. Press the Set button when the required setting has been reached.
