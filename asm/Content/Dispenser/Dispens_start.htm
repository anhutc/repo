Starting dispensing operation

It may be necessary to start the dispensing operation from the operating panel in order to test or set up the glue feeder. A special dispenser cover must be attached over the dispenser nozzle in order to protect the operator and the machine from becoming dirty.

To start dispensing when the dispenser cover has not yet been fitted, proceed as follows:

  1. Place the dispenser cover in the special holder.
    The temperature regulator for the glue is activated and the specified temperature is set.

  2. When the specified temperature has been reached, press the Forward button.

To start dispensing when the cover has already been fitted and a timeout has occurred, proceed as follows:

  1. Press the Forward button.
    The temperature regulator for the glue is activated and the specified temperature is set.

  2. When the specified temperature has been reached, press the Forward button again.

If you press the Forward button briefly (< 0.75 seconds), one glue dot is dispensed. If you press the button for longer than 0.75 seconds, however, a maximum of 20 glue dots are dispensed until you release the button.

Caution:

The machine axes must not be moved if the dispenser cover of the dispenser module is attached, otherwise it is possible for the placement head to be damaged.
