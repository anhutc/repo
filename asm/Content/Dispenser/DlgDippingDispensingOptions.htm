Teaching options for dipping and glue dispensing

Before the selected component is taught, you can decide whether the preset options should actually be performed.

Dip component

The component is dipped in flux prior to placement.

Do flux inspection

To ensure that all the balls on BGA components are coated with flux after dipping, the balls are inspected with image processing techniques after dipping. The technique used can identify whether flux is present on a ball, but not whether the quantity of flux is sufficient.

Dispense glue on component

The specified number of glue dots are applied to the component.

Do dispense inspection

Image processing is used to check whether the specified glue dots are all present out the right size and in the specified gluing area.

 

Accept

The current settings are applied and the dialog box is closed.
