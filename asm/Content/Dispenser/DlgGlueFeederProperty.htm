Setup > Glue parameter sets

Available glue parameter sets

Active

Identifies the glue parameter set currently active for the station.

Protected

The glue parameter set may not be changed.

You can, however, create a copy of this parameter set and modify it appropriately.

Name

Name of the glue parameter set.

Dispenser nozzle shape

Dispenser nozzle type.

Comment

Freely selectable text describing the glue parameter set.

Enable

Sets the selected glue parameter set as valid for the entire station.

Add copy

Copies the selected glue parameter set, which can then be edited.

Remove

Deletes the selected glue parameter set.

Export...

Opens a dialog box in which you can specify the file (XML format) in which the newly defined glue parameter set is to be stored.

Import...

Opens a dialog box in which you can select an XML file from which you can import a defined glue parameter set.

Edit

Switches to the view Functions > Edit glue parameter set, in which you can change the individual parameters for glue dispensing.
