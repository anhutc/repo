Checking component recognition

 

  1. Click the Stop processing PCB icon in the main view.
    The placement process is stopped.
  2. Click the Test component icon.

  3. Select the component you wish to check.

  4. Start SIPLACE Vision by clicking Test component.
    A component is picked up, placed in front of the camera and measured. The result of this measurement is displayed.
