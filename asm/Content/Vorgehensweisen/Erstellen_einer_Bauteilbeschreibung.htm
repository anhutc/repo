Creating a component description

  1. Definition of a new component in the programming system (name, external dimensions).

  2. Set up the component at the position described in the setup.

  3. Click the Stop processing PCB icon in the main view.
    The placement process is stopped.

  4. Switch to the view Functions > Component shapes .

  5. Select the component which has already been defined in the programming system.

  6. Start SIPLACE Vision by clicking Test component.

  7. Specify the component type.

  8. In the camera image, check that all the component connections are clearly visible.

  9. In the case of special components, it may be that the contrast is too low:
    In this event, optimize the illumination settings    .
     

  10. Switch to the Geometry view .

  11. Describe the geometrical characteristics of the component:
    Component dimensions, description of connections and special geometrical features (e.g. corners).   

  12. Click the New measurement button to test recognition of the component.

  13. Switch to the main view or to the component shape selection view and save the modified data.
