Component cannot be recognized

Component without leads:

  1. Check the component description:
    Orientation and size of the connections.

  2. Check the illumination values:
    The features should be well illuminated and should be clearly distinguishable from the background.

  3. Optimize the algorithm parameters:

  • Minimum quality for symmetry method
    A reduction (of up to 0.65) is recommended for components whose connections are heavily structured.

  • Minimum quality for template method
    A reduction (of up to 0.65) is recommended if structures in the component or in the background affect the quality.

  • Threshold in template for fine position detection
    Hint: The light red, horizontal line should be positioned in such a way that it intersects the comparison curve at two points where the curve has a similar gradient and the gradient does not change or only changes slightly in the immediate vicinity.

Component with leads:

  1. Check the component description:
    Orientation and size of the connections.

  2. Check the illumination values:
    The features should be well illuminated and should be clearly distinguishable from the background.

  3. Optimize the algorithm parameters

  • Contrast threshold
    Reduce the contrast threshold: Features can also be detected that only have a low contrast.
    Problem: Other structures in the image can be flagged as features.

  • Filter size:
    First check the geometrical description of the features.
    Then check whether the filter descriptions are large enough - the outer filter points (yellow) must be further apart than the width of the features.
    If the features are close together, you must make sure that the outer filter points do not lie on neighboring features.

  • Minimum number of pixels in filtered image
    Increase in the value: Suppression of artifacts.
    Reduction in the value: Features can be found (problem: artifacts can also be feature candidates).

  • BGA and CCGA: First check the specified diameter of the features and then the selected filter type.

  • Shield: Edge detection method
    Check the selected edge detection method.
