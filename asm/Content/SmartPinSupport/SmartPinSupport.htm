Smart pin support

In order to prevent large boards in particular from bending in the processing area, it is necessary to support the board with support pins that are placed on the lifting table automatically using a gripper (pin pickup).

The positions at which the support pins are to be placed are specified in the product data by the programming system.

If there are already any support pins on the lifting table when the product is changed, these pins are moved to the new positions. Any missing pins are taken from a special pin magazine and placed on the lifting table as well.

To start with, those support pins that the programming system has identified as "required" are placed on the lifting table. The "optional" pins are then placed in accordance with the specified priority.

There are 4 different pin magazines in the current version:

10 support pins

L10

 
10 support pins

Q10

 
5 support pins

W5

 
5 support pins T5

Supported pin type

Type 101

The support pins can only be placed when there are no longer any boards in the machine. If you're using barcode support for placement, you should therefore try to make sure that the same positions are used wherever possible, in order to avoid excessive changeover times.

All support pins must be returned to the magazines if any conversion work is done on the conveyor (e.g. width adjustment or fixing the fixed conveyor rail). Exception: If the conveyor lane width is increased or reduced by less than 3 mm.
