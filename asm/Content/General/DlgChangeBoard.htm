Select board

In "standalone" mode it is possible to load a board type from the hard disk of the station computer.

Setup

Name of the current setup.

Board for conveyor 1

Name of the current board type for the right-hand conveyor lane.

Board for conveyor 2

Name of the current board type for the left-hand conveyor lane.

Board selection

List of possible board types.

/

Indicates the conveyor lane for which the board type was selected.

Indicates the conveyor lane for which the board type cannot be selected.

Permanent:

The product specification is not deleted when other boards are loaded by the programming system.

If more than 30 boards are present, the oldest must be deleted when new specifications are downloaded. Boards flagged as "permanent" are nevertheless retained.

If a board with the same name is sent, the one that is already present is overwritten.

E.g.: Specifications for verifying the machine or checking placement forces.

 

 

Nonpermanent: Boards for production.

These can be deleted as needed.

Display all boards

All the programs currently stored on the machine are displayed.

Display only permanent boards

Only the product specifications that may not be deleted are displayed.

Display only nonpermanent boards

Only the product specifications that may be deleted are displayed: placement programs.

Remove boards...

Opens the dialog box "Select boards to be removed".

Select all the boards to be removed and complete the operation by pressing Apply.

Cancel all jobs and close

Production of the current board type is halted.

Choose conveyor lane

Lane 1 right

The board type selected in the list is selected for the right-hand conveyor lane.

Lane 2 left

The board type selected in the list is selected for the left-hand conveyor lane.

Both

The board type selected in the list is selected for the both conveyor lanes.

Clear selection

Resets the current selection for the board and conveyor lane.

Adjust lane width automatically

When a new board is specified, the conveyor is set to the required width.

 

Apply

The selected board is activated for the selected lane and the dialog box is closed.

Close

Closes the dialog box.

To cancel a job:

  1. Press the Cancel all jobs and close button.

To select a board type:

  1. Select the relevant board type from the list.

  2. In the case of machines with dual conveyors: Select the conveyor lane for which the new board type is to be loaded: Conveyor lane 2, Conveyor lane 1 or Both.

  3. The Setup and "Board for conveyor 1" or "Board for conveyor 2" fields are updated.

  4. Press Apply.
