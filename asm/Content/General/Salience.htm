Alarms and warnings: Color coding of buttons

It may become necessary for the operator to intervene during production. To make sure that the operator is always kept up to date in respect of such intervention, this information is shown at the user interface, among other things by color-coding the buttons.

Depending on the area involved, the color coding can have different meanings. If a number of different statuses apply for a given area, the status that requires operator intervention most urgently is always indicated in the button in the toolbar. To precisely identify the location at which a problem has arisen, for instance, you must navigate to the relevant views.

Production

Green

The machine is ready for productive operation.

Flashing green

The machine is ready for productive operation, but the machine is not producing boards because no board is present.
 

No highlighting

The machine would be able to produce boards, but has been stopped by the operator.

Component level indicator

Orange

An error has occurred, but production is still possible:

•

The filling level has fallen below the warning threshold for a low filling level on a feeder.

•

An error has occurred on a feeder required for the current setup.

•

A feeder for which the component level indicator has been activated is empty.

•

The component pitch was changed at the feeder.

The machine cannot continue placing components because it is necessary to pick up components from a tray that has not been fully described.

•

LDU 2: The linear dipping unit is not ready for use. Examples:

- LDU 2 is not referenced or is not warmed up

- Flux is empty
