Error messages

Each error message is identified by a 5-digit number. This error number also allows you to find every error message in the Help contents.

Help on error messages:

First select the error line to show all current error messages. Then select the error message for which you want to show more information and press Help.

Text in green in the Help system indicates possible measures to rectify the error.

If, however, the error message is one which forms part of a chain, you must take account of the error messages that make up the error chain to narrow down the cause of the error.

Shows the list of active error messages including the associated error chains. You can open individual error chains by pressing the parent error message.
