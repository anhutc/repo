Input field

Input fields for entering values:

As soon as you select this field by pressing it, you can enter the value using the keyboard or virtual keyboard. Confirm your entry by pressing the confirmation button .

The current value is shown, and the permitted value range is shown below the field if any restrictions apply.

Units:

If you press this icon, a list of possible units appears, from which you can select the one you require.

 

Input fields with defined input options

If an input field has a defined set of possible entries, a pull-down menu appears when you select the input field.

Choose the entry you require.
This is immediately accepted and displayed.
