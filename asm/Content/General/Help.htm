Using Help

 

Provides context-sensitive help on the current view. All the control elements in the view are described briefly.

 

Help on error messages:

First select the error line to show all current error messages. Then select the error message for which you want to show more information and press Help.

 

Buttons in the online Help system:

Moves back to the topic previously viewed.

Moves forward one topic in the history of topics viewed.

Displays the table of contents on the left of the window.
Displays the index on the left of the window.

Switches to full-text searching.

All the topics containing the string you enter appear as a table.
Note: You cannot search for strings that contain only special characters.
Returns to the summary page.
