Checking the component height (component sensor)

1.  Switch to the view Setup -> Check and teach component shape .

2.  Choose the required entry from the list and press Test component height.

3.  In the dialog box that appears, press Pick up component to start the measurement.

4.  If the measured height differs from the specified height, it should be entered in the programming system:
If you have write permission: Press Save and close.
If you do not have write permission: Enter the values directly in the programming system and press Close.
