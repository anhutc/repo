High rejection rate

If the rejection rate for a particular component is too high, the following steps should be taken:

    -   Check whether the specified component height matches the height measured by the component sensor for the components in the current batch.

    -   Check the specified handling data (acceleration) on the programming system if components slip on the nozzle or are lost.

    -   Set the pickup position on the feeder to an optimum value.

    -   Use a spacer if necessary.

    -   Make sure that the nozzle is neither dirty nor damaged and that you are using the correct type.

    -   Check the settings in SIPLACE Vision: Are the tolerances too small or are the dimensions or illumination settings incorrect?
