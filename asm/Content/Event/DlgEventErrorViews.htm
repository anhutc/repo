Messages > Message history

Different lists of error messages are shown depending on the errors you select in the drop-down list :

Show track messages

Only error messages that are issued during placement and hardware errors on a location are listed.

Component

Name of the component.

Position

Location/track/division from which the component is picked up.

Segment

Number of the segment with which the component is picked up.

Counter

Number of errors that occurred.

Message

Text of the error message.

Show conveyor messages

Only the errors that occurred when transporting the boards are listed.

Origin

Issuer of the error message.

Counter

Number of errors that occurred.

Message

Text of the error message.

Time

Time the error message was issued: Year-Month-Day Hour:Minute:Second

Display message history

All error messages are listed in chronological sequence.
Chained error messages can be shown by pressing the icon.

Origin

Issuer of the error message.

Message

Text of the error message.

Time

Time the error message was issued: Year-Month-Day Hour:Minute:Second

 

Delete all

Deletes all the entries from the list displayed.

Show help

Shows additional information on the error message selected in the list. The associated error number is displayed on the button.
