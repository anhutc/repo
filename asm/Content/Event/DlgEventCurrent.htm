Messages > Current events dialog box

List of most recently issued error messages (maximum number of messages: 2000). 

Origin

Issuer of the error message.

Message

Text of the error message.

Time

Time the error message was issued: Month/Day/Year Hours:Minutes:Seconds AM / PM

Chained error messages can be shown by pressing the icon.

Custom message

The text box allows you to enter a message of your choice that will be shown over the current screen when you press Display message as popup.

This message can, for instance, be used to leave messages for the next shift when the shift changes.

Touching the screen in the popup area clears the message.

Change popup text color

Opens a dialog box in which you can set the color in which text messages are shown. Pressing OK accepts the current color.

Change popup background color

Opens a dialog box in which you can set the color of the popup background. Pressing OK accepts the current color.

Display message as popup

The text currently shown in the text box is immediately displayed over the screen.
