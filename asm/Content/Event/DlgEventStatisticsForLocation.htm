Messages > Placement statistics for one location

Shows the number of error messages that have been issued for individual pickup positions at the selected location.

Component

Name of the component.

Position

Position from which the component is picked up

Error percentage

Percentage of errors with reference to the total number of pickup operations.

Pickup errors

Number of errors when picking up a component reported since the last reset for this position.

Vision errors

Number of errors related to component recognition reported since the last reset for this position.

Process errors

Number of errors when placing a component reported since the last reset for this position.
Takes into account all error messages that were issued between the Vision operation and picking up the next component:
- there is no component on the nozzle before placement
- a component is still on the nozzle after placement
- a component is still on the nozzle before pickup

Details for selected item

The bar chart shows the error classes with reference to individual time intervals.

The Error distribution chart shows the proportions of the different error classes for which errors have been displayed for the selected pickup position since the last reset.

Each error class is shown in a different color and individual error messages are shown separately.

The Segment distribution chart shows all error messages for each segment issued for the selected pickup position since the last reset. The number of messages in each case is shown in parentheses.
