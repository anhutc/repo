Messages > External messages

One of the following icons is displayed if there are no current messages:

There is no connection to the message service.

There are currently no messages.

Message sequence:

  • The message that has been sent most recently is displayed, irrespective of whether it is a new message or a changed message.

  • The messages are sorted by the priority sent with the message. The lower the value, the higher the priority and the higher the message will appear in the list.
    The time of the message is also taken into account if messages with the same priority are present.

 

A severity level is passed with each message. These are the severity levels:

None

-

Info

For information purposes.

Warning

Error

Error

If a message is issued with the severity level "Warn" or "Error", this is indicated by the color of the button: .

Shows the message with the next higher priority.

If additional, detailed information is available for a message, it can be displayed by pressing this button.
It is possible to specify in the message itself that the detailed information is to be displayed by default.

Hides the detailed information.

The message sent most recently continues to be displayed until the operator sends a response:

Possible response to the system that sent the message.

If there is a hierarchical response structure, this is implemented in the form of a pull-down menu.

You can switch between the response categories that are available until a response is selected.

Restriction:

Messages and response texts are displayed in the language in which they were sent.
