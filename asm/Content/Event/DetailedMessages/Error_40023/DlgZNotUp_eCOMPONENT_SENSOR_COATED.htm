40023 CAUTION: Do not move gantry! Segment is down

The component sensor is covered or faulty:

1

If the gantry is above the feeder, high tape material can cover the component sensor.

Ensure that the light beam from the component sensor can be directed onto the object to be measured without obstructions.

 

2

If you can see that the light beam is not obstructed, the component sensor is too dirty.

Clean the component sensor and check the voltage of the component sensor.

Note:

The voltage value of the component sensor can be checked in the view C&P20 or CPP head > Z axis :

Switch to the Common functions tab and press Measure voltage... under Component sensor functions.

The lower the measured voltage, the greater the degree of soiling.

If it is no longer possible to clean the sensor:

Replace the component sensor.

3

The light from the component sensor is obscured by a high component on the nozzle. The component has slipped, either because an incorrect nozzle is in use or because the acceleration values have been set incorrectly.

Remove the component from the nozzle.

Check the nozzle configuration and the specified handling data in SIPLACE Pro (acceleration).
4

If the component sensor is neither covered nor dirty, it is faulty.

Replace the component sensor.
