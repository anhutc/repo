40023 CAUTION: Do not move gantry! Segment is down

The return cylinder is set too low (C&P20 head).

  1. Switch to the view Manual operations -> C&P20 head -> Z axis , choose the Common functions tab and press Servo on followed by Servo off.

  2. Carry out the head reference run.

  3. If this is not possible:
    Service engineer: Adjust the return cylinder.
