Messages > Statistics

Top track and segment combinations

Component

Name of the component.

Position

Position from which the component is picked up

Segment

Number of the segment with which the component is picked up.

Error percentage

Percentage of errors with reference to the total number of pickup operations.

If you sort the table by this column you will see the best and worst combinations of position and segment.

Pickup errors

Number of errors when picking up a component reported since the last reset for this combination of position and segment.

Vision errors

Number of errors related to component recognition reported since the last reset for this combination of position and segment.

Process errors

Number of errors when placing a component reported since the last reset for this combination of position and segment.
Takes into account all error messages that were issued between the Vision operation and picking up the next component:
- there is no component on the nozzle before placement
- a component is still on the nozzle after placement
- a component is still on the nozzle before pickup

Display only summary of components

Only the total number of all errors is shown for each component type, irrespective of the pickup location and the segment used.

Overall failure distribution

The chart shows the proportions of the different error classes for which errors have been displayed since the last reset.

Each error class is shown in a different color and the total number of errors issued for each error class is shown in parentheses.

Overall pickup rate

Percentage of successful pickup attempts (no pickup or Vision error) with reference to the total number of pickup attempts.

Overall pick up rate = 1 - (pickup errors + Vision errors) / pickup attempts

Actions

Reset all

Resets all values to zero.

All the values shown relate to the period between the last reset and the current time.

Location navigation / head navigation

Select the location and head for which you wish to view detailed error information.
