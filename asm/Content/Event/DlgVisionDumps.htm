Pictures > Saved SIPLACE Vision dump files

The currently stored measurement contexts (individual measurements) are listed in this view.

Shape name

Measured object (component, fiducial, nozzle or tape pocket).

Component name

Name specified by SIPLACE Pro.

Location

Location/track/division from which the component is picked up

or gantry (for fiducials: the board camera with which the fiducial is measured).

Camera type

The object was measured with this sensor type.

Result

Quality of the measurement.
Green: Measurement was successful.
Red: Measurement was not successful.

Type

Description of the component geometry.

Error number

If an error message was issued as a result of the stored measurement step, the error number is shown here.

Segment

The component was picked up with this segment.

Nozzle type

Only for components: The component was placed with this nozzle type.

Creation time

Time of the measurement.

File name

Name of the file under which the measurement was saved.

Selected objects: (number)

Select all

All the saved measurement contexts in the list are selected.

Deselect all

The entries selected in the list are no longer highlighted and the selection is canceled.

Copy selected files...

This opens a dialog box in which you can specify the folder in which the selected entries are to be stored for subsequent evaluation. Navigate to the required folder and confirm your choice with OK.

Delete selected files

All the currently selected measurement contexts are deleted.

Open dump file

Launches the SIPLACE Vision views in which you can obtain details on the saved measurement step.

All columns

The entire list is displayed.

Default columns

Only the columns proposed by the system are shown.

Custom selection

Opens the Define custom columns dialog box in which you can specify what information you wish to see in the list.

No filter

All measurement contexts are displayed.

Custom filter

Opens the Define custom filter dialog box in which you can specify what entries in the list are to be displayed.
