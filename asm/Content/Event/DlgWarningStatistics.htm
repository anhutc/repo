Messages > Warning/error statistics

Displays the three most frequent error messages or warnings:

Each bar shows how frequently the error/warning has occurred since the last start/reset.

If you select a bar, the information displayed at the bottom is updated for this error/warning.

Chart: Relative occurrence

This view shows the ratio between the number of placed components and the number of occurrences of the selected errors/warnings over a given period.

Table view
