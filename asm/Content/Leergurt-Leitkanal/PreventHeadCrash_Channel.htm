Check for protruding empty tape

The aim of this recognition step is to generate an error message in the event that tapes are not inserted properly in the waste tape chute. This is intended to prevent the placement head from being damaged by protruding empty tapes.

Restriction: Inspection can only be carried out on machines on which the track scale has fiducial structures (cross-shaped fiducials) printed on it.

Tracks with white markings on the scale:

2 blue fiducials

Tracks with blue markings on the scale:

2 white fiducials

As soon as a feeder is attached, the check for protruding empty tapes is enabled and the corresponding fiducials are measured. If a tape is not inserted properly in the waste tape chute, the fiducial is hidden and cannot be measured.
