Twin Head > Functions

Functions

Reference head axis

Starts a reference run for all head axes, i.e. all head axes move to their zero point. When so doing, the gantry axes move the placement head above the reject box to discard components in a defined manner.

Nozzle reference run

References the nozzles on the head.

Reject components

Deposits all components currently on the nozzles in the reject bin.

Head EPROM data

Reads the data from the head EEPROM and shows it for each of the two segments in a dialog box.

Head: Product Id, Build Date, Camera type

Z axis: Zero point correction, minimum and maximum position, zero point correction for force sensor, pre-tensioning force for spring.

DP axis: Zero point correction

Move above reference point

Moves the gantry to a position above the edge of the conveyor for subsequent endurance runs or Z axis tests.

Zero point corrections

Calibrate Z axes

Starts a run to determine the zero point correction of the Z axes.

Calibrate DP axes

Starts a run to determine the zero point correction of the DP axes.

A dialog box appears showing the precondition that must be met for carrying out zero point correction and you must confirm this dialog box.
