Segment

Type

Nozzle type

Vacuum state

Result of the vacuum measurement: OK or error message (marked in red).

Z state

Status of the Z axis.

Reference nozzles

Starts the nozzle reference run and enters the values that are determined.

Selection

Enable (only available as of activity level "Advanced production")

Activates the selected segment again.

Do not use

The selected segment is no longer to be used to pick up components because there have been recurrent pickup problems all because placement accuracy is too low. The segment can be taken into service again after the fault has been rectified.

Disable (only available as of activity level "Advanced production")

Deactivates the selected segment. No reference run is carried out and no values are determined.
It is only recommended that you deactivate a segment if serious faults are present.
