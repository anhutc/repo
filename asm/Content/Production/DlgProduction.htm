Production

Start:

Restarts production if all the preconditions have been met (no alarm, electrical circuit closed).

The Start button is only enabled if the machine has previously been stopped.

Stop:

Stops the current placement process.

The components on the placement head at the time the button is pressed are still placed to prevent them from being dropped.

The Stop button is enabled when the machine is in production mode.
