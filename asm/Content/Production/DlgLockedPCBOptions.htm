Board options for locked boards

The board is locked to prevent continued production. This means that an acknowledgment from one or more external systems (clients) must be received for the board before the board can be transported on or assembly can continue.

Order number

This number can either be entered for a recipe at the programming system or be sent by an MES (manufacturing execution system). It can, for instance, be used as an SAP order number.

Choose board options

Cancel board (only available on Service activity level)

Assembly of the selected board is not continued, irrespective of the current location of the board.

After assembly of the board has been canceled, the board will be highlighted in red in the display.

Here you can choose whether the board is to be removed from the output area of the current station or from the output area of the last station in the line. The board stops at the selected position.

Ignore lock (only available on Service activity level)

Processing of the board continues and the system does not wait for a response from the external systems that have set a lock.

This function can be used, for example, if the external system is no longer responding and is thus preventing production from continuing.
