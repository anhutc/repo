Processing a board

During the placement process, the production view provides an online display of the exact location of boards in the machine. If the gray display area is completely covered by a board, this signifies that the board is clamped in the corresponding section. By contrast, if only a partial area is covered, the board is being loaded. If a machine is in the Stop state (stopped by operator intervention, EMERGENCY STOP, error or switch to manual operations), all boards located in the input section, processing section 1, intermediate section or processing section 2 are represented as buttons.

or with combined PCB:

Clicking on a board causes the Board options dialog box to open in which you can specify that assembly of the board is to be aborted, or that the board is not to be assembled in a particular processing area. It is also possible to display the list of placement positions in the event of placement errors.
