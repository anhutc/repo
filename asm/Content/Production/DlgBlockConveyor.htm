Locking/unlocking conveyor lanes

Input section

Block

No new board will be moved into the corresponding conveyor lane or pair of lanes.

Unblock

Boards can be moved into the corresponding lane again.

Drive board (appears depending on the machine type)

Extends the stopper and moves the board in the input section over the input section sensor.

Note: First click the button and then place the board in the input conveyor.

OK

The current settings will be accepted.

Cancel

Closes the dialog box and the current settings are rejected.
