Random setup

Caution:

If components are not set up in accordance with the optimized required setup, the longer travel paths can lead to reduced placement performance.

For individual components, a specification can be made in the programming system that they need not necessarily be set up at the position specified in the setup sent by SIPLACE Pro (required setup).

These components can additionally be set up at a different position, which must, however, be able to be reached by the same head.

This means that a number of feeder/component combinations not specified in the required setup can be present.

Prerequisite: Component verification.

 

To specify additional alternative tracks, proceed as follows:

  1. Position an X-feeder at the required track or place a waffle pack tray in the required level.

  2. Verify the combination of feeder and component: Scan the component barcode.
