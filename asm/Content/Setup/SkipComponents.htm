Omitting components

When the first board of a new type is loaded, it is possible to omit components that are not properly placed on this current board. This component type will then be ignored on all tracks where it is set up and will not be placed on the subsequent boards. The incompletely assembled boards will subsequently be moved to the output section and must be removed by hand. If the components are to be placed again, the components previously omitted need to be reactivated.

This function can be accessed from "Advanced production" activity level onwards.
