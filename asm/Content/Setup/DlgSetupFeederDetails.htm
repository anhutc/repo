Feeder pitch [mm]

Distance between the centers of the component pockets.

E.g. 1, 2, 4, 8, ...mm or "Automatic detection".

Pickup position in feeder

There are 1, 2, 3 or 4 pickup position markers (notches) depending on the feeder type.

The center of the component pocket in the feed direction should correspond to the required pickup position marker.

Speed

Synchronize with expected settings

The feeder settings specified in the setup data are restored.

Check tape position at pickup position in feeder...

A camera image of the pickup position is shown.

If the center of the component pocket is not positioned at the mark (notch) for the pickup position set in the feeder, press Feed forward (2 mm) to step the tape forwards by the smallest possible increment.

Make sure that the center of the component pocket is located at the mark for the pickup position set in the feeder.

Feed forward (2 mm)

Fine setting of the pickup position on feeders that are at least 16 mm wide: The feeder is stepped forwards by 2 mm and a new camera image is displayed.

Feed forward (current pitch)

The feeder is stepped forward by the current pitch and a new camera image is displayed.

Positions correction

Taught

Offset from the specified pickup position set manually by the user as a result of teaching.

Learnt

Deviation from the taught deviation as determined automatically by the machine in learning cycles.

Note: These values are set to zero after a new teaching operation.

X [mm]

Offset of the determined pickup position in X.

Y [mm]

Offset of the determined pickup position in Y.

Angle [degrees]

Offset of the determined rotation.

Skip recognition

Component pocket recognition is skipped for this feeder.
This should only be activated if errors occur during component pocket recognition that prevent components from being picked up from this feeder.
