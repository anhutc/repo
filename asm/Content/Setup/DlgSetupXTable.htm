Setup > X-Table

X-Table overview

List of all feeders at the selected location

Press this button to switch between the list and the overview.

Feeder selection

Selection

Select all tracks

Selects all feeders and all free tracks.

Select all empty feeders

Selects all empty feeders.

Select all unused feeders

Selects all feeders not used by the setup and all feeders from which the tape reel has been removed (with the status "Reel removed").

Select all dummy feeders

Selects all dummy feeders.

Deselect all feeders

Deselects all elements in the list.

Selected feeder(s)

Enable

The feeder is activated again.

Locking

Unlock

The locking mechanism is released for all selected feeders. The feeders can be removed.

Lock

The locking mechanism is engaged for all selected feeders.

Single feeder

Settings...

Opens a dialog box showing the specifications from the programming system and the data currently valid for the selected track: Feeder type / pitch / pickup position / travel profile / splice detection.

Correct positions...

Opens a dialog box for manually correcting the component pickup position.

Other actions

Actions

All feeders - position recognition

Activates feeder position recognition for all feeders of the location:
The exact position of all feeders and thus the exact pickup position are determined with a board camera.

Move gantry to setup position

Moves the gantry out of the feeding area so that the feeders become freely accessible.

Highlight feeders: SmartFeeder Xi specified, but not set up

The status indicator of the corresponding feeder on the current table briefly flashes orange if:

The components are picked up by a C&P20-P2 head and a standard X-Feeder has been mounted instead of the specified SmartFeeder Xi.

Caution: The use of standard X-feeders together with the C&P20-P2 head means that it may not be possible for the C&P20-P2 head to perform as specified.

Random Setup: Highlight feeders

All feeders that are not at the positions indicated by the optimized setup are marked (orange).
You can adjust the setup and improve placement performance.

No additional marking of the feeders.
