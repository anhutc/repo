Setup > Pickup position correction

The aim is to position the model of the component (shown in green) as precisely as possible over the component that is to be picked up and is shown in the camera image.

Each press of these buttons moves the gantry one step of the selected step size in the corresponding direction.

Specify the pitch and select the coordinate system you wish to use by pressing this button and making your selection from the drop-down menu.

First press this button and then press a position in the camera image. The camera is moved to the corresponding position.

Save

Accepts the corrected pickup position.
Pickup positions changed in this way are valid until the next time the setup is changed.

Reset current changes

Changes that have not yet been saved (Additional column) are not applied.

Reset all changes

All the changes made to date (Total column) are rejected.

Show entire component

The entire component is shown in the camera image.

Transport component

The feeder is stepped on by one component. The current image is displayed. This allows you to check whether the pickup position was described correctly.

Correction values (programming system)

The values specified here refer to the coordinate system of the feeder module. The discrepancies determined under "Total" can be transfered directly to the programming system and do not have to be converted.

Taught

The deviation from the specified values resulting from the teaching cycles that have been performed since the last time a product was specified.

Additional

The deviation from the current pickup position derived from the teaching cycles that have been performed since the last save operation.

Total

The total of all deviations (Taught + Additional) from the specified pickup position.
