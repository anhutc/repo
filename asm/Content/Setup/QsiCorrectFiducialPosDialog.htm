Correcting the fiducial position

Every click of these buttons moves the gantry one step of the selected step size in the corresponding direction.

Specify the step size.

First click this button and then click a position in the camera image. The camera is moved to the corresponding position.

Reset changes

The changes are discarded.

Choose action

Measure fiducial again using the changed position

The measurement operation for the fiducial is repeated at the camera position currently set.

Use camera position as the recognized fiducial position

The camera position currently set matches the fiducial position.

Ignore fiducial for this board

This fiducial is not used for board position recognition.

OK

Accepts the corrected fiducial position.
The fiducial position changed in this way is valid until the next time the setup is changed.

Cancel

Closes the dialog box without saving the data you have changed.
