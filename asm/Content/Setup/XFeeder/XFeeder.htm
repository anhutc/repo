X-Feeder

The term X-Feeders refers to feeders of the X series.

These feeders are powered using a contactless technique in combination with the component tables of the X series, i.e. they need not be attached using physical connectors.

X-Feeders are currently available in the following widths: 8 mm, 12 mm, 16 mm, 24 mm, 32 mm, 44 mm, 56 mm, 72 mm, 88 mm.

1

Operating panel

2

Display: (By default, the track number and pitch are displayed here).

3

FORWARD button - each time you press this, the tape is moved one increment towards the machine.

4

BACK button - each time you press this, the tape is moved one increment backwards. - if this button is pressed for at least 2 seconds the feeder removal process starts (display: "Removed from setup"). The foil must be cut during the next 5 seconds and the foil rocker must be in the up position.

5

FOIL button - the foil is tensioned.

6

SET button - Selects the menu or the component verification button if component verification is activated. If this button is pressed while component verification is activated, the track is selected in the Setup Center and the barcode can be scanned.

7

Status display:

lights up green

The feeder is operational.

lights up orange

Low filling level.

lights up red

An error was reported.

For instance, the removal handle is not latched in place; torn foil; motor temperature is too high;

foil tension is too low; pickup error; Vision system error; track empty error.

Off

The feeder is not part of the current setup or the current board (depends on the setting in SIPLACE Pro Line Control).
