Status indicator on 2x8 mm X-Feeder

Error messages from the control unit of the feeder:

Status display

Pitch indicator

Meaning

Troubleshooting

Operating voltage too low when feeder is switched on.

Check the power supply.

The feeder has logged on to the FCU.

This indicator disappears again after a brief time.

If the indicator continues to be displayed for a longer period:

Unlock the feeder from the user interface.

 

Communication between feeder and FCU interrupted.

Log the feeder off and then on again and check the fiber optic interface (EDIF) and clean it if necessary.

If problem still exists:
Send the feeder in for repair.

Data error.

Send the feeder in for repair.

The feeder has been logged on, but the removal handle has still not been pushed in.

Push in the removal handle.

If problem still exists:
Send the feeder in for repair.

 

If this indicator remains for a longer period:

Check the power supply to the FCU and the fiber optic interface (EDIF).

The feeder software does not start the application.

Reload the feeder software.

Status display

Pitch indicator

Meaning

Troubleshooting

The current function is not permitted when the foil is tensioned.

If you nevertheless wish to execute the function:

Remove the foil from the pair of cogs and cut it off. Then repeat the required function.

 

Transport errors:

Tape reel jammed in the container or component jammed between the tape and the pickup window in the pickup area.

Eliminate the reason for the jam and press the Forward and Back buttons.

Torn foil (the foil was not tensioned within the specified time)

Foil is torn

Insert the foil again and tension it.

Foil container is full

Empty the foil container.

Stuffing wheels cannot exert pressure

Check the stuffing wheels for foreign bodies and dirt, and clean them if necessary.

The foil is not removed.

If necessary, pull the foil out under the pickup window and tension it again.
