Setting the pickup position on the X-Feeder

Proceed as follows to set the pickup position on an X-Feeder:

  1. Press the SET button and hold it down.
  2. Briefly pressed the Foil button and release it.
    The LED for the currently set pitch flashes (quickly) and then shows that setting mode has been activated.
  3. Press the Forward or Back button to move the pinwheel to the next or previous pickup position. Repeat this action until the component is in the required pickup position.
  4. Release the SET button.
    The changed pickup position is stored.
