Setting the feeder pitch

Proceed as follows to set the pitch on an X-Feeder:

  1. On 2x8 mm feeders only: Use the Select division button to select the division for which the pitch is to be set.
  2. Press the SET button and hold it down.
  3. Immediately press the Forward or Back key repeatedly until the LED for the required pitch lights up.
  4. Release the SET button.
    The changed pitch is stored.
