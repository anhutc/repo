Checking the tape position at the pickup position in the feeder

 

Feed forward (2 mm)

If the center of the component pocket is not positioned at the marker for the pickup position set in the feeder, press this button to step the tape forwards by the smallest possible increment.

Feed forward (current pitch)

Press this button to check whether the next component is moved to the pickup position correctly.

Start

Takes a camera image of the pickup area of the selected feeder.

Start / Restart

Takes a new camera image of the pickup area of the selected feeder.

Close

Closes the dialog box.

How to check the tape position:

1.  Switch to the view Setup > X-Table .

2.  Select the relevant feeder and press Settings...

3.  Press Check tape position at pickup position in feeder....

4.  A red, dashed crosshair is shown at the pickup position currently set.

5.  For the 8 mm feeder, the crosshair must coincide with the selected pickup position mark (notch) for the pickup position set.

6.  If this is not the case, you must select a different pickup position.
