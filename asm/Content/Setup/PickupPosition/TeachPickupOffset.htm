Correcting the pickup offset

1.  Switch to the view Setup > X-Table .

2.  Select an entry and press Correct positions...

3.  Choose Edit pickup offset from the drop-down menu.

4.  Use the rotary controls to move the component model (shown in green) in such a way that it coincides with the camera image of the component as accurately as possible.

5.  Press Save changes to save the overall deviations specified.

6.  Choose the location in which the corrections are to be stored.
