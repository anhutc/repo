Correcting the pickup point

1.  Switch to the view Setup > X-Table .

2.  Select an entry and press Correct positions...

3.  Choose Edit pickup point from the drop-down menu.
The cross section of the nozzle is shown in a dashed blue rectangle in the camera image.

4.  Use the rotary controls to move the nozzle (shown in magenta) in such a way that it is shown at the point at which the component is to be picked up.

5.  Press Save changes to save the overall deviations specified.

6.  Choose the location in which the corrections are to be stored.
