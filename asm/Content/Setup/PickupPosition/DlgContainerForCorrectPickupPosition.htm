Positions correction

The aim is to determine the exact pickup position of the component. The crucial aspects here are the exact position of the component in the component pocket and the point at which the nozzle is positioned over the component during pickup, the pickup point.

The values specified here refer to the coordinate system of the feeder module. The discrepancies determined under "Total" can be transfered directly to the programming system and do not have to be converted.

You can measure and save either the current pickup offset or the pickup point for a component. The current selection is indicated in the camera image by a blue dashed frame. All activities refer to this selection.

Edit pickup offset: Use the rotary controls to move the green component description in such a way that it coincides with the camera image of the component as accurately as possible.

Edit pickup point: The point at which the nozzle picks up the component.

Select the nozzle for which this setting is to apply. The nozzle cross section is indicated in the image. Only those nozzles can be selected that are present in the setup and assigned to the component shape.

Use the rotary controls to move the nozzle (shown in magenta) in such a way that the contact area of the nozzle makes contact with the component at a point that is as flat as possible and as close as possible to its center of gravity.

Taught

The deviation from the specified values resulting from the teaching cycles that have been performed since the last time a product was specified.

Additional

The deviation from the current pickup offset derived from the teaching cycles that have been performed since the last save operation.

Total

The total of all deviations (Taught + Additional) from the specified pickup offset.

The feeder is stepped on by one component. The current image is displayed. This allows you to check whether the pickup offset was described correctly.

Revert: The most recent change or entry is reverted.

Saving changes

Accepts the corrected pickup position.
Pickup positions changed in this way are valid until the next time the setup is changed.

Reset to downloaded values

The values specified by the programming system are restored. All changes are reverted.
