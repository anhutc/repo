Machine requests: Teaching fiducials for feeder modules

If the system detects that an object still needs to be taught during production, production is halted and you have to use the SIPLACE Vision views to measure the elements shown.

Fiducials for feeder modules:

This view also shows fiducials for recognizing the position of trays.

Name

Name of the component pocket or fiducial

Type

Component pocket

The fiducial has not been described completely. Adjustments must be made in the SIPLACE Vision views.

Tray fiducial

Barcode

A component-specific barcode is attached to the top of the component or directly next to the component pocket.

Description

Freely selectable comment.

Protected

 The fiducial cannot be changed at the station.

Incomplete

The fiducial has not been described completely. Adjustments must be made in the SIPLACE Vision views.

Position

Position at which the component is set up.

 

Selected fiducial

Test

Starts SIPLACE Vision to check and if necessary change the description of the selected feeder fiducial.

Remove from list

The element is removed from the list and the request for teaching is rejected.
