Actions

Measure pickup offset before pick

Takes an image of the component in the feeder module with the board camera and starts SIPLACE Vision to determine the exact pickup position.

Align component using top surface feature

Starts SIPLACE Vision to check and, if necessary, change the pickup position of the selected component on the basis of the structures on the top side of the component.

Teach component barcode

Starts SIPLACE Vision to check and if necessary change the component barcode on the top of the selected component. The barcode type, barcode filter, size and illumination can be set in order to obtain the best possible results.

Measure front/back offset

Starts SIPLACE Vision to determine the precise placement position of the selected component on the basis of the structures on the top side of the component.

Test height (can only be selected if the selected component is able to be measured using the component sensor)

Measures the height of the selected component using the component sensor and outputs the specified and measured heights in the Test component height dialog box.

Teach acceleration values

Opens a dialog box in which you can determine the greatest possible gantry and head axis acceleration at which the placement process can still be performed reliably.

Test acceleration values

Opens a dialog box in which you can check specified or taught axis acceleration values and transfer them to the programming system as necessary.

Change component shape

Opens a dialog box in which you can assign a different component shape from the one currently selected to the selected component.

This is only possible if alternative component shapes have been stored in the programming system.

Select coplanarity sensor parameter
(Only appears if a component is selected that can be measured with the coplanarity sensor and camera type 37 is selected.)

Soldering paste thickness (coplanarity tolerance)

Determines the maximum possible difference between the height of the lead and the placement plane for the current board. This value is only used by the manipulator if the coplanarity tolerance was set to 0 for the component shape in SIPLACE Pro (this corresponds to the entry "Default" in the SIPLACE Vision Manipulator). It is not stored.

If, for instance, greater coplanarity discrepancies are to be compensated for with BGA components, a concrete threshold must be entered either in SIPLACE Pro or in the component shape manipulator.

Manual inspection required (Appears after Vision errors occur.)

Start manual inspection

Starts the manipulation views in SIPLACE Vision to check the component shape.

Skip manual inspection

Cancels inspection of the specified component shape.
