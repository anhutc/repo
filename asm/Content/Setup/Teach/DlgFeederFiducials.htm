Setup > Fiducials for feeder modules

"Fiducials" can be specified for specific feeder modules in SIPLACE Pro in order to precisely determine the position of the components being provided. These could, for instance, be component pockets or fiducials on a waffle pack tray.

Before a fiducial for feeder modules can be tested or its settings be changed, it must be loaded so that the system knows which fiducial it is to use.

If changes were made to the settings for the selected fiducial, the possibility exists to save these changes permanently in the programming system.

This view also shows fiducials for recognizing the position of trays.
