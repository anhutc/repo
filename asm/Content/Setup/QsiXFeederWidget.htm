Feeder pitch [mm]

Distance between the centers of the component pockets.

Pickup position in feeder

There are 1, 2, 3 or 4 pickup position markers (notches) depending on the feeder type.

The center of the component pocket in the feed direction should correspond to the required pickup position marker.

Speed

Speed with which the components are transported in the feeder:

> Slow

>> Medium

>>> Fast

Submit changes

Sends the changed values to the programming system.
The values for the pitch, pickup position and speed can be adjusted.

Position correction

X [mm]

Deviation of the manually determined pickup position in X.

Y [mm]

Deviation of the manually determined pickup position in Y.

Angle [degrees]

Deviation of the manually determined rotation.

Reset

Discards the position correction values determined manually and resets the values specified by the programming system.

The representation shown at the bottom indicates the exact position of the feeder shown in the location.
