Coplanarity threshold

The permitted coplanarity threshold of a component shape is by default derived from the thickness of the solder paste on the board. To do this, the "Tolerance" parameter must be set to 0 mm in the component shape editor in SIPLACE Pro (under Basic Handling -> Coplanarity -> 3D Coplanarity) or the "Coplanarity tolerance" parameter must be set to "Default" in the SIPLACE Vision Manipulator (under Recognition parameters: 3D coplanarity sensor).

Alternatively, you can specify the threshold for a component shape individually. This makes sense, for instance, for BGAs with "collapsing balls", as these melt during the soldering process, thus compensating for greater discrepancies in coplanarity than would be permitted by the thickness of the solder paste. In this case, you must enter a concrete threshold value for the coplanarity tolerance in SIPLACE Pro or in the SIPLACE Vision Manipulator. This value overrides the value for the solder thickness.
