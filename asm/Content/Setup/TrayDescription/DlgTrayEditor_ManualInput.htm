Tray editor: Manual entries

Number of columns

Number of pockets in horizontal direction.

Number of rows

Number of pockets in vertical direction.

Height of component seating

The difference between the bottom of the tray and the surface on which the component rests in the pocket.

Pickup height = Height of component seating + height of component.
