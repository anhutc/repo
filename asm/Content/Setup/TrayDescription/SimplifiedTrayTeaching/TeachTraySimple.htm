Teach tray

1.  Switch to the view Setup > Tray descriptions .

2.  Select the tray and press Edit tray description.

3.  Select the pocket you want to edit first: Selected pocket (Example: Edit first pocket).

4.  Use the rotary controls to move the component model (shown in green) in such a way that it coincides with the camera image of the component as accurately as possible.

5.  Now select the pocket located opposite: Selected pocket (Example: Edit last pocket).

6.  Use the rotary controls to move the component model (shown in green) in such a way that it coincides with the camera image of the component as accurately as possible.

7.  Close this view and press Save changes and leave dialog in the confirmation prompt.
