Setup > Tray editor

The aim is to accurately describe the position of the components in the pockets of the tray to allow them to be picked up reliably.

To do this, move the component descriptions (shown in green) for the first and last pocket in such a way that they coincide with the camera images of the corresponding components as closely as possible.

•   First pocket: The pocket specified as the first pocket from which components will be picked up in the programming system.

•   Last pocket: The pocket located opposite the first pocket.

•   Positioning the component description:

    -   By dragging and dropping: Move the marked area directly on the screen

    -   Using rotary controls

    -   By entering the X and Y coordinates in the text boxes

 

Basic description

These values are specified in the programming system and do not generally need to be changed.

Columns

Number of pockets in horizontal direction.

Rows

Number of pockets in vertical direction.

Height of component seating

The difference between the bottom of the tray and the surface on which the component rests in the pocket.

Pickup height = Height of component seating + height of component.
