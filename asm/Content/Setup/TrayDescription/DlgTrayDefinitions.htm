Setup > Tray descriptions

Shows all the waffle pack trays present in the current setup.

Tray

Name of the tray.

Grid

Rows x columns.
(number of pockets in Y x number of pockets in X)

Protected

The component cannot be changed at the station.

Incomplete

The component has not been described completely. Changes still need to be made in the tray editor.
Components cannot yet be picked up from this tray.

Component

Name of the component.

Component shape

Name of the assigned component shape.

Position

Position at which the component is set up.

 

All columns

The entire list is displayed.

Custom selection...

Opens the Define custom columns dialog box in which you can specify what information you wish to see in the list.

No filter

All trays are shown.

Custom filter...

Opens the Define custom filter dialog box in which you can specify what entries in the list are to be displayed.

Selected tray

Edit tray description

Starts the tray editor, in which you can specify the number and location of the pockets.

Advanced: Test and modify tray description

Starts the tray editor (advanced), which allows you to check and, if necessary, change the description for the selected tray.

The position and size of the tray, the size of the pickup pockets and the pickup direction can also be edited.
