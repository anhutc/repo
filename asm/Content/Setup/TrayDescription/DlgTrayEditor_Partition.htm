Tray editor: Partition

Specify the offsets between the pockets:

When this button is pressed, use the dial on the right-hand side to move a horizontal comparison line and use the dial below the image to move a vertical comparison line.

Choose these comparison lines in such a way that a corresponding line can be clearly identified in each pocket in the underlying image. This can, for instance, be the boundary of a pocket.

When this button is pressed, use the corresponding dial to increase or decrease the offset between two adjoining pockets.

To do this, you should mark the corresponding lines in the neighboring pocket (e.g. the top and right-hand edge of the pocket).

 

In order to ensure that the specification of the offsets is as accurate as possible, you should then specify comparison lines at the opposite corner of the tray in order to set the appropriate offsets.

You can also enter the values manually:

1

Column pitch

E.g. offset between the left-hand boundary of a pocket and the left-hand boundary of the adjoining pocket in the horizontal (X) direction.

2

Row pitch

E.g. offset between the top boundary of a pocket and the top boundary of the adjoining pocket in the vertical (Y) direction.

Start pickup

Specifies the start point and direction for defining the offset between the pockets.
Example: If the top-left pocket is selected, the offset between the columns in X is shown as a positive value and the offset between the rows in Y is shown as a negative value.
