Tray editor: Capturing images

This allows you to capture images to ensure that the tray description matches the real tray as closely as possible.

The regions that you select are captured using the board camera and are used as the background for the diagram.

In the editor, choose a region to be scanned.

Press the button and first choose a region in the top right part of the image.

Press the button again and then choose a region as close as possible to the opposite corner.

False colors

If the system does not identify the outlines, you should switch to pseudo-color (false color) mode.

Caution:

The larger the region you choose, the longer the capture process will take, and the longer the system will need to display the image.
