Clear Random Setup

The original setup specified by the programming system is intended to achieve the highest possible placement performance. If this optimized setup is no longer available as a result of the use of Random Setup, this can lead to a decrease in performance.

Restoring the original setup for one feeder:

  1. Switch to the view Setup > X-Table .

  2. Click Random Setup: Highlight feeders.
    All feeders which are not at the specified positions will be highlighted in orange:

  3. If you use the arrow keys to select a feeder that is highlighted in this way in the "Current" chart, this feeder is marked as "Selected" at the specified track in the "Expected" chart.

  4. If the specified track is free, you can locate the feeder at this position.

    Note: If, however, this track is occupied, you must first clear it.
