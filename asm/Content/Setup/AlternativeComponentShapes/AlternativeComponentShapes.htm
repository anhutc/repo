Alternative component shapes for a component

If a component is procured from different suppliers, or if a supplier has different component shapes for the same component, it must be possible to take account of these different component shapes when specifying the program.

Although the components have the same electronic parameters, the dimensions may well be different. It is therefore necessary to provide different descriptions for SIPLACE Vision.

The programming system provides the possibility of creating and managing different component shape descriptions for these components.

To allow this to be shown at the station as well, the "Enable alternative component shapes" option must be enabled for the component in the programming system.

In the view Setup > Component shapes , you can assign a different component shape description to a component by choosing Change component shape.
