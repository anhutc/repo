Changing the component shape for the selected component

All the component shapes that have been described for the selected component in the programming system are displayed.

Available component shapes

Component shape

Names of the component shapes that have been assigned to the selected component.

Incomplete

The component has not been described completely. Adjustments must be made in the SIPLACE Vision views.

Category

 

 

Supplier

Assignments set in the programming system.

E.g. category = Manufacturer

Attribute = Company name

A number of different categories can be defined for each component shape.

Manufacturer

Manufacturer Component Name (manufacturer's component name or part number)

Extra 1-5 (freely definable)

Attribute

Freely selectable text for describing the selected category as defined in the programming system.

Default

Indicates the preset component shape in the programming system.

Current

Indicates the component shape that is currently set.

Do not test component shape after applying selection

Default: As soon as Apply is pressed, SIPLACE Vision is started in order to inspect the component.

Inspection in SIPLACE Vision will not be started.

Apply

Assigns the selected component shape to the component and closes the dialog box.

Close

Closes the dialog box without changing the assignment.
