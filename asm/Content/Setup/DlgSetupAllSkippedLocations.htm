Setup > Omitted tracks

Empty tracks

Position

This is where the component is set up (component setup position).

Reason

Reason that this position has been flagged as "empty":

Component

Name of the component.

Component shape

Name of the component shape.

Omit

Changes the status of the position from "empty track" to "omitted track".

All tracks occupied by the selected component will be excluded from placement and will be entered in the list of the omitted components along with alternate tracks.

Omitted tracks

Position

Position at which the component is set up.

Reason

Reason that this position has been flagged as "omitted":

Component

Name of the component.

Component shape

Name of the component shape.

Reset selected

The selected position is no longer omitted.

If the reason for deactivation of the feeder has been eliminated in the meantime, the position can be used again. Otherwise, the position is again listed under "Empty tracks".

Reset all

All omitted components and alternative tracks will be used again.
