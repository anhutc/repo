Selecting an alternative component

If it is no longer possible to refill a track with the component that was set up previously, a different component specified for this track must be selected:

  1. Switch to the view Setup > X-table , Functions > Locations > WPC x or Functions > Locations > MTC x .

  2. Select the corresponding pickup location from the list displayed.

  3. Click in the Component field.
    All possible predefined components are displayed.

  4. Refill the components that are to be placed next and scan the barcode of the refilling unit.
