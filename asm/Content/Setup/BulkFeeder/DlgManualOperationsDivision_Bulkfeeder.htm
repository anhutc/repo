Container shutter

Cover for the component container.

Open / Close

Actuates the container closing mechanism.

Pickup plate contamination

Measure

Determines the current degree of contamination of the pickup plate.

Degree of contamination: Current measured value for the degree of contamination of the pickup plate.

If the value for the current degree of contamination is in the red range, SIPLACE Vision can no longer deliver reliable measurements and the pickup plate must be cleaned.

BulkFeeder X data

SIPLACE Vision live image

Flash camera

A new image of the components on the pickup plate is taken and displayed.

Settings

Optimal number of components

Number of components on the pickup plate that allows the components to be picked up most efficiently.

Default: 120.

Contactless offset

The nozzle does not make contact with the component. The downward movement is shortened by this distance to prevent the nozzle making contact with the component.

 

Minimal component distance

Minimum distance between the components on the pickup plate to allow them to be picked up.

Edit...

Pressing this button or pressing twice in the appropriate value field allows you to edit the parameter by making an entry within the range shown.

Confirm your entry by pressing Return.
