Errors shown on the SIPLACE BulkFeeder X (7-segment display)

The status indicator on the BulkFeeder shows red, i.e. an error has occurred:

The following error numbers can appear on the 7-segment display:

7-segment display 2:

01

Removal handle not pushed in.

02

The embedded software has detected an unsupported hardware version.

07

The status of the CAN controller is errored (BUS PASSIVE or BUS OFF)

08

The maximum permitted processor temperature has been exceeded.

17

The maximum permitted temperature of the vibration actuator has been exceeded.

25

Timeout on plate shutter (embedded software error)

26

The plate shutter has not reached the target position.

27

Error on referencing the plate shutter.

29

Timeout on the container shutter.

30

The container shutter has not reached the target position.

31

Error on referencing the container shutter.
