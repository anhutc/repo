Feeder information from an X-table (table)

Track

Position at which the component is set up.

 

Random Setup: In the programming system it was specified that the component must not necessarily be set up at the position specified in the required setup.
It is also possible to set up additional alternative tracks.

Prerequisite: Component verification.
 

The component must be set up at the position specified in the required setup.

Feeder type

Feeder currently set up.

Required type

The feeder type specified in the programming system.
If the programming system does not require a specific feeder, this field is empty.

Component

Name of the component which is set up.
This is only shown if the information is provided by the programming system and if the component is required for assembling the current product.

At least one alternative component or an alternative component shape has been specified for this pickup location.
First select the corresponding row in the table and press the field to display all possible alternatives.

The component that is currently being used is shown in the first row of the field.
If, however, no component supply has yet been scanned in, the top row is empty.

Currently, all alternative components and alternative component shapes are shown.
Press the field to hide the alternatives.

Component shape

Name of the component shape.
This is only shown if the information is provided by the programming system and if the component is required for assembling the current product.

 

Alternative component shapes:

If "..." appears for alternative component shapes in this column, you should adjust the column width to show the other possible component shapes.
The component shape that is currently being used is shown in the first row of the field.

Component pocket recognition has been deactivated.

Activated if the pickup position has been corrected manually and differs from the specified component position.

At least one of the following feeder parameters does not match the specification from the programming system:
Feeder pitch, speed or pickup position in the feeder or, in the case of a SIPLACE BulkFeeder X, the vibration profile.

Status

Status of the feeder, filling level of the feeder, other warnings.

If a feeder is not required for assembling the current product, even a serious problem will not cause an alarm to be issued.
