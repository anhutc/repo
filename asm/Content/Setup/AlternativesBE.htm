Alternative components

In order to avoid supply bottlenecks or in order to react better to the demands of the market, components are not always ordered from the same manufacturer. Consequently, it is possible that different components are kept in stock for a single component type.

In order to avoid standstills if other components are used for refilling during production, it is possible to specify different components with disparate component shape descriptions for a single pickup location in the setup. If this is done, the next component to be used for refilling at the pickup location can be selected from a list. As soon as the new refilling unit has been scanned, the component is selected.
