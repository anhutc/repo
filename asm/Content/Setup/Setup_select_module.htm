Selecting a module at a location

The representation shown at the bottom indicates the exact position of the feeder shown in the location.

You can show a different track by pressing the arrow buttons:

Selects the feeder module set up on track 1 or the furthest feeder to the left.

Selects the feeder module set up to the left of the currently selected feeder module.

Selects the feeder module set up to the right of the currently selected feeder module.

Selects the feeder module set up on track 40 or the furthest feeder to the right.
