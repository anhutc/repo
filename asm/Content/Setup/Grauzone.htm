Grayzone

  • If a track is being monitored by the splice detection system and if two barcodes have already been scanned, the system switches to for only a few components, i.e. the batch information for the current batch and the subsequent batch is stored for these components (gray zone).
    The status indicator then switches to and the batches can be assigned uniquely again.

  • If the splice detection system is not active, both sets of batch information are recorded for the components immediately the second barcode has been scanned.
