Position recognition for trays

Fiducials can be defined in SIPLACE Pro for individual trays in order to improve specification of the positions of the components.

The fiducials are measured after any movement of the tray carrier on which the tray is located.

These fiducials are shown in the view Setup > Fiducials for feeder modules .

It is possible to teach a fiducial in the SIPLACE Vision views if you select it from the list that is displayed and press Test fiducial.
