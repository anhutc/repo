Service data for SIPLACE PowerConnector X

Shows the data for the feeders set up on the SIPLACE PowerConnector X.

If the feeder has multiple divisions, the data is split over several tabs.

Settings

Different parameters appear here, depending on the feeders set up on the PowerConnector X. These can be edited as required.

To do this, press on the relevant text box to open a drop-down menu, where you can select other settings.

 

The representation at the bottom indicates the exact position of the feeder module shown.
