Component orientation check

Components which are rotated in the feeder cannot be measured and therefore also not be placed, as their actual position deviates too far from the expected orientation. Such components are rejected. When placing expensive components, this should, however, be avoided wherever possible.

If the arrangement of the connections makes it possible to identify the orientation, i.e. if the connections are arranged asymmetrically, the orientation detection inspection mode can be activated in order to identify whether a component is rotated.

If the connections are arranged symmetrically, an additional orientation mark (Pin 1) must be present.

It is possible to specify the appearance of this orientation mark and its location for certain components in SIPLACE Pro.

If a new feeder containing this component is inserted in the machine, the operator must first check whether the components are located in the tape or on the tray as described.

Starting the component orientation check manually:

  1. Select the component you wish to check (either in the view Setup > X-Feeder or in the view Setup > Waffle pack tray detail view and click Correct position...

  2. Use the camera image to check whether the component is at the expected position in the tape or on the waffle pack tray. If necessary, correct the specified orientation by clicking the +90° / -90° buttons.

  3. Then click Confirm component orientation.
