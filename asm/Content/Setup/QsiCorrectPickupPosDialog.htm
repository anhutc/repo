Pickup position correction

Every click of these buttons moves the gantry one step of the selected step size in the corresponding direction.

Specify the step size.

First click this button and then click a position in the camera image. The camera is moved to the corresponding position.

Reset changes

The changes are discarded.

Reset all changes

All the changes made to date (Total column) are rejected.

Show entire component

The entire component is shown in the camera image.

Transport component

The feeder is stepped on by one component. The current image is displayed. This allows you to check whether the pickup position was described correctly.

Correction values

Taught

The deviation from the specified values resulting from the teaching cycles that have been performed.

Additional

The deviation from the current pickup position as determined by the current teaching cycle.

Total

The total of all deviations from the specified pickup position.

Upload changes to SIPLACE Pro

The corrected values are also sent to the programming system when they are accepted.

OK

Accepts the corrected pickup position.
The pickup position changed in this way is valid until the next time the setup is changed.

Cancel

Closes the dialog box without saving the data you have changed.
