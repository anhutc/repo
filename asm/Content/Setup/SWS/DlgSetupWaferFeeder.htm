SIPLACE Wafer System

The use of the SIPLACE Wafer System in the SIPLACE CA placement machine allows the placement of dies directly from wafers of sizes between 4" and 12". It is possible to switch automatically between multiple wafers in one magazine, which makes it possible to place different dies.

Both the flip chip and the die attach process are supported during die handling.
