The details refer to the track marked below:

Position: Track - division (the value for the division is always 1 for the LDU)

Settings:

Values currently set or values specified by the programming system

Type

LDU (linear dipping unit)

Submit changes

Transfers the changed values to the selected linear dipping unit.

 Dipping area

Refresh time [s]

Duration of a squeegee cycle (squeegee travels over the dipping area once).

Cycles

Number of squeegee operations performed after the operator initiates a squeegee cycle. Default: 1.

Refresh dipping medium

The squeegee is moved back and forward once to smooth the surface of the flux or to apply new flux.

Initial activation

Activates the linear dipping unit.

Planarity

Caution:

There must be no flux in the dipping area during the measurement, otherwise the nozzle will be rendered useless.

In order to ensure that the components can be dipped into the medium throughout the entire dipping area, the dipping area must not be tilted.

During measurement, a nozzle is moved to various measuring points in the dipping area in order to determine the height of each point. These values are compared with the height of a reference point.

ID

1: Reference point

2: Measure point

3: Measure point

Absolute

The measured height of the relevant measuring point.

Relative

The deviation of the absolute height of measurement point 2 or 3 from the reference point. For reference point 1, this value is always 0.

Check planarity

Starts measurement of the dipping area.
