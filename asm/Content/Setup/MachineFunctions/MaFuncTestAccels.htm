Test acceleration values

The aim is to check that the maximum permitted deviations of the component position in X and Y and the maximum permitted rotational deviation are not exceeded over several runs. This is done using acceleration values for the individual axes (X/Y axes and, depending on the head type, the star axis and the DP axis) that have either been specified or determined by teaching.

If the values are not exceeded, they can be transfered to the programming system.

Configuration

Options

Nozzle

Press the edit box and choose the nozzle to be used to check the acceleration values from the drop-down menu.

All nozzles are shown that are assigned to the component shape, are available in the setup and can be used on the placement head that is performing the operation.

Components must be removed manually (can only be set for certain placement heads)

Default: If the function fails or is canceled, the component is rejected automatically.

The component must be removed from the nozzle manually when measurement is complete or if an error occurs. This is particularly recommended for expensive components, but is not possible for all head types.

Number of cycles

The number of times the combined movement of all axes is repeated.

Acceleration values

Current values (programming system)

The acceleration values for the X/Y axes, the star axis and the DP axis as specified for the component shape in the programming system. The values are expressed as percentages of the maximum possible acceleration.

Values for test

Enter the values for which you want to perform a test run.

Execution

The progress bar shows the percentage of the specified repetitions already completed.

The current camera image is shown in each case.

Results

#

Repetition.

Deviation X / Deviation Y [µm]

The deviation of the component position in X and Y in [µm] determined during the specified repetition using the specified acceleration values.

Angle deviation [°]

The rotational deviation of the component in [°] determined during the specified repetition using the specified acceleration values.

Status

 

The deviations that have been determined are less than the maximum permitted deviations.

At least one of the deviations that have been determined is greater than the maximum permitted deviation.

The acceleration values used cannot be transfered to the programming system.

Upload acceleration values to programming system

The acceleration values for the individual axes as used for the test run are transferred to the programming system when the test run has been completed successfully. The component shape is changed accordingly and these values are then used for placement operations.

 

Start / Restart

Starts the measurement operation.

Stop

Interrupts the measurement operation.

Close

Stops the measurement operation without saving the determined values.
