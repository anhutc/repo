Teach acceleration values

The aim is to determine the greatest possible gantry and head axis acceleration at which the placement process can still be performed reliably for a given component shape, i.e. at which the component does not slip more than the maximum permitted distance.

To do this, a component is picked up from the feeder module, the relevant axis movement is performed and a measurement is carried out.

The measurement process determines any deviations in the position of the component in X and Y and any rotational deviation at 25 %, 50 %, 75 % and 100 % of the maximum acceleration for the specified axes. This will depend on the placement head used.

The acceleration value at which the determined deviations are just below the permitted threshold is then recommended.

Configuration

Options

Nozzle

Press the edit box and choose the nozzle to be used to determine the acceleration values from the drop-down menu.

All nozzles are shown that are assigned to the component shape, are available in the setup and can be used on the placement head that is performing the operation.

Components must be removed manually (can only be set for certain placement heads)

Default: If the function fails, the component is rejected automatically.

The component must be removed from the nozzle manually when measurement is complete or if an error occurs. This is particularly recommended for expensive components, but is not possible for all head types.

Allowed deviations

X / Y [µm]

The maximum permitted deviations in the component position in X or Y in [µm].

Default: 25 µm.

Caution: The larger the permitted deviations chosen here, the larger the potential placement offset.

Angle deviation [°]

The maximum permitted rotational deviation of the component in [°].

Default: 1 °.

Execution

The progress bar shows the percentage of repetitions already completed.

The image on the left shows the last camera image as a reference. The image on the right is the current camera image.

The current acceleration values for the axes and the associated deviations determined for the component are shown.

Results

Shows the measurement results

X/Y [%]

Acceleration used (percentage of maximum possible acceleration) for movements in X and Y.

Star [%]

Acceleration used (percentage of maximum possible acceleration) for movements of the star axis.

DP [%]

Acceleration used (percentage of maximum possible acceleration) for movements of the DP axis.

Deviation X / Deviation Y [µm]

The deviation of the component position in X and Y in [µm] determined during the specified repetition using the specified acceleration.

Angle deviation [°]

The rotational deviation of the component in [°] determined during the specified repetition using the specified acceleration.

Current values (programming system)

The acceleration values for the X/Y axes, the star axis and the DP axis as specified for the component shape in the programming system. The values are expressed as percentages of the maximum possible acceleration.

Recommended values

The optimum acceleration values determined by the measurement operation.

After the individual measurements have been completed successfully, the result that has been determined has to be checked. To start the operation, press Test recommended acceleration values. The movements of the individual axes are performed in combination and a measurement is taken following an entire operation.

Start / Restart

Starts the measurement operation.

Pause

Interrupts the measurement operation.

Close

Stops the measurement operation without saving the determined values.
