Safety circuit

It can occur that the machine's safety circuit is interrupted during placement or when starting placement. If this happens, a window appears in the foreground of the user interface. This window shows the cause of the interruption.

After the problem which caused the window to appear has been eliminated, press Cancel and then press the Start button to allow the machine to continue placement.

Move in all component tables

All component tables that have been undocked from the location, for instance in order to carry out maintenance, are docked at the relevant locations again.

Cancel

Closes this window.

Note:

If the dialog box does not display an icon and the machine can still not be started, it could be because:

• a WPC or MTC is not connected or is not connected correctly,

• the safety circuit is interrupted at a position not displayed here,

• the signaling input for the icon which ought to be displayed is faulty.
