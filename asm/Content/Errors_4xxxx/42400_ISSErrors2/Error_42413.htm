42413 Reference move is not allowed due to possible collision

The placement head or individual axes of the placement head may not be moved.

If the error occurs when placing high components (the board is flagged with the following symbol:), it is not possible to reference the gantry axes.

Assembly of the board cannot proceed, and it must be transported out of the machine.

Switch to the view Manual operations > Subsystems > Lane > Functions and transport the canceled board out of the machine or the line.

If a component to be placed is very long or wide, it is not possible to rotate the DP axis or move the Z axis due to the risk of brushing off the component.

Service engineer:

  • Remove the component by hand.

  • Switch to the view Setup > Nozzle changer - Head functions or .

  • Press Set another nozzle... and first select "None".

  • Then press Set another nozzle... again and select the nozzle that was in use previously.

  • Then start the nozzle reference run.
