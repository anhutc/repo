45603 Tape end

The tape in the specified tape duct is too short to be moved on. It is moved back such that it is still engaged in the rear pinwheel. The tape must not be inserted again.

Eject the tape and insert a new tape as specified.

 

If a tape that is too short has been inserted:

Select the tape duct by pressing the button briefly, and press and hold the Back button on the feeder until the tape moves and can then be removed.
