40071 IPC-HERMES-9852: Handover of board to next machine failed, message is missing

"The Hermes Standard" (IPC-HERMES-9852): The message from the downstream machine that the machine is ready was not received.

Visually check that the board is already located in the downstream machine and confirm that you have done so by pressing the relevant button:

Board is in the next machine: Starts production of the board.

Board removed: The board has been removed by hand.
