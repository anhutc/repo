40047 Component shape matching failed (no matches)

SIPLACE Vision could not identify the component shape that is set up for the component to be measured. Either the component shape must be taught or the camera image shows that the current component is faulty and must be rejected.

 

The component appears to be OK:

The list shows all component shapes that have been stored for the component.

 

Select a component shape and press Assign component shape.

If the selected component shape has already been completely described, the next component is picked up and a new measurement is performed. If component shape matching fails again, SIPLACE Vision is started and you will have to teach the component.

If the selected component shape has not been completely described, SIPLACE Vision is started immediately and you will have to teach the component. A new measurement is then performed.

 

The component is faulty and must be rejected. After the component has been rejected, a new one is picked up and measured: Reject component.

Component shape

Names of the component shapes that have been assigned to the selected component.

Incomplete

The component has not been described completely. Adjustments must be made in the SIPLACE Vision views.
