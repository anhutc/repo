40051 Board did not arrive

No acknowledgment was received that the board has arrived in the Shuttle Extension, even though the board has left the upstream machine.

Visually check the position in the machine at which the board is located and choose this position from the drop-down menu . Then continue placement by confirming the relevant action.

Board removed: The board has been removed by hand.

Board is completely in Shuttle Extension: Although the board is located completely in the Shuttle Extension, the sensor has not triggered.

Board is in previous machine: The board is still in the upstream machine or in an intermediate position between the machine and the Shuttle Extension. An attempt is made to move it into the Shuttle Extension.

First make sure that the board can be moved freely in the conveyor.

Note: If the board does not move from the current position, open the protective cover of the Shuttle Extension and push the board up to the sensor.

Board is in machine: The board is still in the machine to which the Shuttle Extension is assigned or in an intermediate position between the two machines. An attempt is made to move it into the Shuttle Extension.

First make sure that the board can be moved freely in the conveyor.

If problem still exists:

Check that the laser light barrier is set correctly and is fully functional. Replace it if necessary.
