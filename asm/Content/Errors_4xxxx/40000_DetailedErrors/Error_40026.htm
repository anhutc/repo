40026 Detected unsteady nozzle seating

Either the nozzle seating is dirty or there is too much play in the bearing.

Nozzle length (Comp. sensor up)

 Nozzle length measured by component sensor when head is traveling up [mm].
Nozzle length (Comp. sensor down) Nozzle length measured by component sensor when head is traveling down [mm].
Deviation The measured difference between the nozzle lengths that have been determined [mm].
Maximum allowed deviation This error message is issued if the measured difference is above this nozzle-specific threshold [mm].

Repeat measurement: Incorrect measurements are possible, particularly for nozzle type 1004. You should therefore first attempt to perform a correct measurement.

Reject nozzle and pick up nozzle: If the error could not be rectified by repeating the measurement, and if there is another nozzle of the same type in the nozzle changer, the nozzle that has been measured incorrectly should be replaced.

Do not use segment: If the nozzle cannot be replaced because the nozzle changer does not contain any more nozzles of the same type, the segment must be deactivated to allow placement to continue with the remaining segments.

Caution: This reduces placement performance!

Make a replacement nozzle available as soon as possible and then reactivate the segment.
