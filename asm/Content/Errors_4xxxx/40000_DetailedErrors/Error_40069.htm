40069 The board did not leave the output section

No acknowledgment was received that the board reached the input section of the downstream machine, even though the board has left this machine.

Visually check where the board is located:

Board is in output section: The board is still in the output section or in an intermediate position between the two machines.

Board removed: The board has been removed by hand.

Board is completely in the next machine: The board is already completely in the downstream machine.

First make sure that the board can be moved freely in the conveyor.

If problem still exists:

Check that the laser light barrier of the output section is set correctly and is fully functional. Replace it if necessary.
