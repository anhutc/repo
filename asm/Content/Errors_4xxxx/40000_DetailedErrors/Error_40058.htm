40058 Component touched board despite of expected touchless placement

Touchless placement has been activated for the component in the programming system.

Actuate clamping: Closes or releases board clamping.

After releasing board clamping, try to align the board correctly and then actuate clamping again.

The board is clamped.

Board clamping is released.

If there is an error and if assembly of the whole of the rest of the board is to be canceled: Cancel board.

From the drop-down menu, specify the position at which the board is to be removed.

If the last component to be placed appears to be intact and if assembly of the board is to be completed: Complete placement.

The system then moves to the measuring points for height inspection again and measures them in order to check planarity of the board.
