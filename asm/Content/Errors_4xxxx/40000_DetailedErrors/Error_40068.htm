40068 An invalid board id has been signaled by the previous machine

"The Hermes Standard" (IPC-HERMES-9852): The upstream machine reports that a board has been passed. However, a different board with the same board ID is already in this machine..

The current board cannot be assembled, otherwise this would lead to data inconsistency.

Remove the board from the upstream machine or cancel the board by pressing Move board into machine and cancel board and use the drop-down menu to specify where the board is to be removed.
