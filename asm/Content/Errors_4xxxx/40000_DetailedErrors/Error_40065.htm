40065 IPC-HERMES-9852: Handover of board from previous machine failed, message is missing

"The Hermes Standard" (IPC-HERMES-9852): The message from the upstream machine that transport of the board has been completed was not received.

Visually check that the board is fully located in the input section and confirm that you have done so by pressing the relevant button:

Board is completely in the input section: Starts production of the board.

Board is not completely in the input section: Moves the board into the input section and starts production.
