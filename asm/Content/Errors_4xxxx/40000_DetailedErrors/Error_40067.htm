40067 A board without a signal from the previous machine has been detected

"The Hermes Standard" (IPC-HERMES-9852): Confirmation from the upstream machine that a board has been passed has not been received. However, there is a board in the input section for which no data is available.

Choose whether the board is to be assembled despite the missing data or remove the board and confirm that you have done so:

Start processing of board: Starts production of the board even though the board data was not sent. This data must be entered by hand if necessary.

Board removed by hand: Confirms that you have removed the board from the machine.
