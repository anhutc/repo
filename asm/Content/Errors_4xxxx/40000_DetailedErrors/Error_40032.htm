40032 Incompatible eSW version detected

The embedded software for the subsystems listed is not up to date.

Update subsystem: Press this button to download the appropriate embedded software.

Do not switch off the machine during the update process.

If the machine needs to be rebooted after the update, a message box appears prompting you to do so.

In this event, press Shut down, then switch the machine off and on again.
