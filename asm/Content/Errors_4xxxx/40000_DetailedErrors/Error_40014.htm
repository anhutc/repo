40014 Support pin is missing on lifting table

No support pin has been detected at the expected position on the lifting table (marked in red). Either it has already been removed by hand or it has slipped and should therefore be removed.

Check the corresponding area on the lifting table:
Has the support pin already been removed manually?
If the support pin has not been detected because it has slipped, remove it manually.

Then confirm that there is no longer a support pin at the corresponding position: Support pin removed.

Multiple support pins have been removed manually or have slipped and cannot be detected.

Check the corresponding area on the lifting table:
Have multiple support pins already been removed manually?
Are support pins not detected because they have slipped? Remove these support pins by hand.

Then confirm that there are no longer any support pins at the corresponding position: Support pins removed.

All the support pins that were not detected during scanning are flagged as not being present.
