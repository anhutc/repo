40052 Snap-in confirmation required

You must confirm that placement was carried out correctly for the placement positions shown in the list.

Check that the board locks have been fully inserted through the board and that the locking mechanism is engaged.

Select each entry in the list and choose an option from the drop-down menu :

Component was placed correctly

The Z height calculated during placement of this component is used as the Z reference height for snap-in detection.

Repeat measurement

For the next component, a decision has to be made whether it has engaged correctly and that the Z height is thus to be used as the Z reference height.

 

Then press All placement positions confirmed.
