40037 Individual barcode of component cannot be read

Try measuring again: Try to read the barcode again.

If the error recurs:

Correct the position at which the barcode is to be read:

To do this, switch to the view Setup > Fiducials for feeder modules.Choose the required entry (barcode type) and press Test.

Confirm that this has been done: Barcode position taught.

If this does not rectify the problem:

Do not stop machine if barcode for this track cannot be read: Placement continues without storing the barcode for each individual component. The machine will no longer be stopped if the barcode cannot be read.
