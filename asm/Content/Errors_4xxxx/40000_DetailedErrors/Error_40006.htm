40006 Board cannot reach target position

It was not possible to continue transporting a board at the section marked in red.

Example: Right-hand conveyor lane, placement section 2.

Make sure that the board can be moved freely in the conveyor and confirm that you have checked this: Board checked.

If the board has been removed from the conveyor manually:

Board has been removed manually.

If assembly of the board is not to be continued:

Cancel board: First specify the point at which you wish to remove the board by selecting it from the drop-down menu .
