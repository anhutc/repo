Correct fiducial position

The aim is to position the board camera in such a way that the fiducial can be clearly identified in the camera image.

Each press of these buttons moves the gantry one step of the selected step size in the corresponding direction.

Specify the pitch and select the coordinate system you wish to use by pressing this button and making your selection from the drop-down menu.

First press this button and then press a position in the camera image. The camera is moved to the corresponding position.

Reset changes

Changes that have not yet been saved are not applied.

Choose action

Measure fiducial again using the changed position

The current position is used as the basis for a repeat measurement.

Use camera position as the recognized fiducial position

The current position is used as the fiducial position without any further measurement.
