Processing detailed error messages

If an error has a number of different possible causes and the system cannot distinguish between them, several different measures to rectify the error are shown at the same time.

If you cannot identify the measure that will rectify the error and allow the machine to resume production, work through the measures in the sequence shown (from top to bottom) as the solution that applies most frequently always appears at the top.

A distinction is made between actions performed by the machine and actions that you must perform and then confirm .

Execute action

The system performs the selected action and closes the detailed view.

Confirm action

By pressing this button, you confirm that the corresponding instructions have been followed. The detailed view is closed.

Make sure that you have followed all the instructions before pressing this button.

 

Close

Closes the detailed view. The error is still active.

Opens the online Help system with information on the current error.
