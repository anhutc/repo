40005 Cover foil torn

 

Location

X-Table

Track number

Track number - division number

Visually check that the cover foil is not torn and press Try to tension cover foil.

The system attempts to pull the cover foil taut for a specified time.

If it has been specified that the feeder is to run until empty:

Activate short tape handling: Carefully pull off all the cover foil and continue.

Otherwise:

Attach the cover foil to the specified feeder and press Cover foil fixed.

Press to see a video showing how to secure the cover foil.
