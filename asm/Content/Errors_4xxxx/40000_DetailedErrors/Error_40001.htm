40001 Pickup failure

You can switch the camera image to the pickup position, the component, or the placement position by pressing the appropriate tab.

The pickup position is described inaccurately.

Check the pickup position (pickup offset and pickup point) and correct it if necessary:

Correct positions...

Position the component as exactly as possible in the view Position correction and press Save changes.

Taught

Offset from the specified pickup position set manually by the user.

Learnt

Offset determined by the machine in learning cycles.

Inspect pickup window...

If the center of the component pocket is not positioned at the marker for the pickup position set in the feeder, press Feed forward (2 mm) to step the tape forwards by 2 mm. If the pitch is set to less than 2 mm, the tape is not moved.

Press Feed forward (current pitch) to check whether the next component is moved to the pickup position correctly.

Feeder values are inconsistent, send SIPLACE Pro data to feeder

The values currently set for the pickup position or the speed do not match the expected values. Use the values from SIPLACE Pro for the feeder.

The hardware may need to be checked: A number of different videos are available showing you how to perform the relevant steps.

- Insert the tape in the feeder correctly.

- Close the pickup window.

- Use a pin for the tape reel.

-Check that the camera is fully functional.

Then confirm that this has been done: Hardware checked.

A tray must be refilled, as no more components are available to be picked up.

Refill the tray as specified and press Tray refilled when this has been done.

If the unit is an AutoRefillFeeder 8mm X:

Make sure that the other tape duct is loaded and ready and press the button Switch to tape duct 1 or Switch to tape duct 2.
