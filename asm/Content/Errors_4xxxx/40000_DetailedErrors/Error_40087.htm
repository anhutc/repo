40087 The allowed placement position deviation of a component has been exceeded

If component fiducials have been described for a component, the exact position of the component is determined after placement by measuring these fiducials and comparing the results with the expected position.

The component fiducials are named as follows: CAC_name.

(CAC: component accuracy check).

At least one value at the placement position shown exceeds the maximum permitted deviation from the specified position: exact position in X or Y or placement angle.

If the permitted deviation for X, Y or the angle is exceeded, the corresponding row in the table is marked in red.

Use the values shown to check whether the calculated placement accuracy is adequate.

If there is a genuine error and if assembly of this panel only is to be canceled: Skip placement of panel.

If the area to be inspected does not appear to have any errors: Ignore result and continue with placement in panel.

If there is an error and if assembly of the whole of the rest of the board is to be canceled: Cancel board.

From the drop-down menu, specify the position at which the board is to be removed.

If it is not clear whether there is a fault: Mark board for inspection.

From the drop-down menu, specify the position at which the board is to be inspected.
