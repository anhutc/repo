40079 Front-back offset measurement check failed

The required minimum percentage of valid measurements was not achieved. No further selected components are specified for offset measurements.

Valid measurements: Measurements for which the determined deviation is less than the position tolerance specified.

Deviation: Measured position of the top-side feature vs. expected position of the top-side feature (as a result of measurement from the bottom).

Specification

Minimum percentage of valid measurements

Number of successful measurements / number of measurements: Value specified in the programming system that must be achieved for determination of the offset to be valid and taken into account when placing the components.

Statistical values

Percentage of valid measurements

Number of successful measurements / number of measurements.

Number of measurements

Total number of measurements performed.

Number of valid measurements

The measurements whose results will be taken into account when determining the offset.

Statistical analysis

Positional tolerance

Maximum permitted deviation for the position of the top-side feature; set in SIPLACE Vision.

Mean offset

Deviation calculated across all successful measurements.

Standard deviation

Scatter of measurement results.

The dies to be used for front/back offset measurement are marked with bin codes:

Close the error view by pressing Close and revise the wafer map for the current wafer. Then start the measurement again.

The dies to be used for front/back offset measurement are set by the SIPLACE Wafer System:

Press Start measurement again. The number of dies required for front/back offset measurement as specified in the programming system are selected by the SIPLACE Wafer System and the measurement is repeated.

The measurement results already available are sufficient and no more measurements are to be carried out:

Press Use calculated offset and continue.
