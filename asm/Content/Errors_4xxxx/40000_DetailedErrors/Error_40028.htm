40028 Component orientation check needed

"Orientation check by operator" has been enabled for the specified component in SIPLACE Pro.

The machine can stop if

- a new feeder with this component has been used

- a new batch has been started

- a splice has been detected

- a new filling level for trays has been entered

 

Use the camera image to check whether the component is at the expected position in the tape or on the waffle pack tray. If necessary, correct the specified orientation by pressing the +90° / -90° buttons.

 

Then confirm that the check has been carried out: Component orientation checked.

Tip:

If multiple camera images are available, you can call them up using the and buttons.

Find the view in which the upper side of the component is most clearly visible in order to determine the location of the mark for component orientation (Pin1 detection). You can also select the pseudocolor representation for each image.
