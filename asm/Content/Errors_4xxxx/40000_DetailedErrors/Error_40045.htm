40045 Component shape must be assigned

A number of different component shape descriptions have been specified for the specified component in the programming system.

Select the component shape that is set up on the specified track and press Assign component shape.

Component shape

Names of the component shapes that have been assigned to the selected component.

Incomplete

The component has not been described completely. Adjustments must be made in the SIPLACE Vision views.

Category

 

 

 

Supplier

Assignments set in the programming system.

E.g. category = Manufacturer

Attribute = Company name

A number of different categories can be defined for each component shape.

Attribute

Position
