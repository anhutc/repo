40080 Pin-in-paste height check, confirmation required

You must confirm that placement was carried out correctly for the placement positions shown in the list.

Check whether the centering pins have been fully passed through the board.

If this is not the case, it is necessary to perform a new measurement with additional components until a correct Z reference height is obtained.

Select each entry in the list and choose an option from the drop-down menu :

Component was placed correctly

The Z height calculated during placement of this component is used as the Z reference height for the pin-in-paste height check.

Repeat measurement

For the next component, a decision has to be made whether it is placed correctly and that the Z height is thus to be used as the Z reference height.

 

Then press All placement positions confirmed.
