43000 Calibration is not up to date

A component (e.g. an axis) has not been completely calibrated.

Repeat the specified calibration step:

Switch to the view Service > Automatic calibration , select the relevant calibration step and start calibration.

 

Some calibration steps have to be invoked via Service > Single calibration or via the manual operations of the placement head.
