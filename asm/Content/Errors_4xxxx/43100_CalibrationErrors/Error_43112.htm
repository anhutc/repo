43112 Verification of vacuum tooling failed, because value is outside valid range. Probably a wrong correction file was applied.

 

Use the correction file from the measurement machine for calibrating vacuum tooling. Save the calibration results from the measurement machine on a USB stick and read the data in on the current machine:

Switch to the view Service > Conveyor configuration > Conveyor lane x > Vacuum tooling configuration , enter the appropriate ID and press Edit...

In the dialog box that opens, navigate to the location at which the correction file (ChuckCorrection-*.xml or VacuumToolingCorrection-*.xml) is stored and press Start.
