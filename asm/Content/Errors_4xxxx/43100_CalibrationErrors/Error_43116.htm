43116 Calibration (partly) skipped. Ensure that all required calibration tools are located within their pockets

A reference run was aborted during the calibration process. Depending on the calibration settings that have been made, it may be possible for other calibration steps to be performed.

The unsuccessful calibration steps are marked in red.

Use the error chain to check whether this is caused by a hardware problem and rectify it.

Then make sure that all the required calibration tools are in the appropriate calibration tool pockets and start the calibration steps marked in red again.
