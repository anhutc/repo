Describing a substrate

The following parameters are used to describe substrates (wafers / carriers):

Parameter

Description

Global properties

Name

Name of the substrate.

User text

Comment specified in the programming system.

Diameter

Outer dimension of the substrate.

Diameter tolerance

Maximum permitted deviation

Substrate is bright

Yes

The substrate is brighter than the background (substrate carrier / carrier material).

No

Default: The substrate is darker than the background.

Start angle

Position of the first evaluation point with reference to the described position of the substrate.

Number of X/Y evaluation positions

 

Number of points along the edge that will be used to exactly determine the X/Y position of the center of the circle and the radius of the circle.

The more positions are used, the longer the measurement will take.

Default: 3

 

Evaluation of position

Specified by the programming system:

X and Y

Notch detection will not be performed. Only the X/Y position will be determined.

X, Y and angle

In addition to determination of the X/Y position, notch detection is also performed. This allows the orientation of the substrate to be determined.

Special measurement options

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.

No

The check is carried out.

Notch

Notch detection

Yes

The orientation of the substrate is determined on the basis of the detected notch.

No

Notch detection is not performed.

Notch width

Extent of the notch along the edge of the substrate.

1

Notch depth

Maximum depth of the notch.

2

Notch radius

Describes the curvature of both sides of the notch.

3
