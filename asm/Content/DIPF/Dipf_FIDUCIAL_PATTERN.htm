Taught fiducial

The following parameters describe a fiducial that has been learned as a pattern:

Parameter

Description

Name

Name of the fiducial

User text

Comment specified in the programming system.

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

No

The check is carried out.

Rotation tolerance

The angle entered here is the maximum rotation, relative to the training image, that the fiducial may have during recognition.
Default: 2Â°

Size tolerance

Maximum deviation of the fiducial size from training image in % of the region of interest.
Default: 5%.

Structure information

The length of the line segments that are determined in the region of interest during teaching:
The finer the structures, the shorter the line segments.

Detailed: very small,
standard: small,
Coarse: medium length,
Very coarse: long line segments.

Reference point horizontal / vertical position

This corresponds to the position of the fiducial as set in the board description in the programming system.

Search area: horizontal / vertical position

Marks the top left corner of the area in which the system looks for the fiducial.

Search area: horizontal / vertical size

The extension of this area in the X and Y direction.

 

Fiducial wizard

Switches to the view Choose fiducial type, where you can specify whether the fiducial has been described as a geometrical shape or is a fiducial that has been taught as a pattern.
