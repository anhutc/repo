Synthetic ink spot

The model comprises the geometrical description of the structure. If the structure is found, the machine assembles the board/panel, otherwise it will be omitted during placement.

 

This procedure is particularly useful when a fiducial is used as an ink spot and this fiducial has, if the board is bad, been covered with a sticker or by a black pen marking. Make sure that the fiducial is completely covered and cannot shine through, otherwise it may be detected.

The following parameters describe a synthetic ink spot:

Parameters

Description

Name

Name of the ink spot

User text

Comment specified in the programming system.

Suppress checking of threshold values

 

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

No

The check is carried out.

Fiducial is light

Yes

The ink spot is lighter than the background.

No

The ink spot is darker than the background.

Place if recognized

Yes

Default: The panel is assembled if the ink spot is detected, i.e. if a fiducial matching the described geometric model was found in the region of interest.

No

The panel is assembled if the ink spot is not detected.
