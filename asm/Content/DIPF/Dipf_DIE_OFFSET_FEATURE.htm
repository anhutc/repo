Parameters for offset measurement

Copper studs can be described as upper side features of die components (connection type: Columns).

The purpose of offset measurement is to use the board camera to determine the exact position of the copper studs (position and angle) with reference to the position of the component as measured from below by the stationary component camera (SST 40).

Parameter

Description

Global properties

Name

Name of the component shape.

User text

Comment specified in the programming system.

Positional tolerance X/Y [µm]

Maximum permitted deviation of the position of the upper side feature in X or Y*, preset value: 50 µm.

Radial positional tolerance [µm]

Maximum permitted overall deviation of the position of the upper side feature*, preset value: 70 µm.

Angular positional tolerance [°]

Maximum permitted rotational deviation of the upper side feature with reference to the component, preset value: 5°.

Special measurement options

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.

No

The check is carried out.

Connection groups

Connection type

Column

Feature is light

Yes

The columns are lighter than the body of the component.

No

The columns are darker than the body of the component.

Diameter

3

Diameter of a column

Group (n):

The following parameters apply to the specified group:

1

Pitch X:
The distance between the column center points of two neighboring columns in the X direction.

2

Pitch Y

4

Offset X:
The distance between the center point of the group and the center point of the component in X direction.

5

Offset Y:
The distance between the center point of the group and the center point of the component in Y direction.

 

Pitch error threshold

Maximum permitted deviation from the predefined pitch.

Angle

Rotation of group in relation to component.

Number of leads X or Y

Number of columns in X or Y direction

* The tolerances are specified with reference to the coordinate system of the component.
