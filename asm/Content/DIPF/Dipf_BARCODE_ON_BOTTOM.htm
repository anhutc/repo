Barcode conditions

All the defined requirements must be met in order to recognize a barcode.

Barcode filter

This allows you to specify whether only a partial section of the scanned barcode is to be used for traceability.

Start text

Enter a string here.

The barcode is evaluated after the string you enter.

Start offset

The specified number of characters of the barcode are omitted.

End text

The barcode is evaluated up to this string.

Text length

Number of characters evaluated.

Add filter

You can specify a partial section of the raw barcode data for evaluation.

Remove filter

Deletes the currently selected filter and selects the previously defined filter.

To use the defined filter, you must choose Barcode filter and specify the filters to be used in the Format field.

You can combine the predefined filters or supplement them with freely selectable text.

Filter 1 corresponds to %1, filter 2 to %2 etc.
