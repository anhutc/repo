QR barcode

A QR barcode is a 2D matrix barcode made up of square modules. These modules are arranged in a square array.

The QR barcode symbol is made up of:

•

function patterns:

 

-

three finder patterns,

 

-

timing patterns,

 

-

alignment patterns,

 

-

separators: Free space around the barcode,

•

encoding region.

40 symbol sizes are possible: 21x21 through 177x177 modules

Type

QR Code Model2, ISO/IEC 18004

 

Requirements for recognition of a QR barcode:

•

The difference in contrast between bits that are set and not set must be at least 20 grayscale units.

•

The barcode symbol must be rectangular.

•

Ratio of column width / row height:

0.5 ≤ column width / row height ≤ 2

•

The mark must not be applied too thickly or thinly:

1/3 of the matrix cell ≤ mark width ≤ 5/3 of the matrix cell
