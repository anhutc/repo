Feature type "Pattern"

If it is not possible to fully describe odd-shaped components using features such as leads, balls, columns or corners, the component shape wizard provides the option of describing their characteristic shapes (e.g. locking mechanisms , chamfered leads ) or non-rectangular packages (e.g. lenses ) using the feature type "Pattern".

This feature type can only be set for component shapes of the type Nonstandard or Connector if the "Odd-shaped component" measurement option has been set. This option is subject to a license fee.

 

When this feature is used, an area is selected in the camera image in which contours of any shape can be learned:

Regions of high contrast, i.e. where the transition between bright and dark is sufficiently strong, are found (edge elements), and the associated points are learned as a pattern.

If the region of interest contains artifacts, individual regions can also be removed for recognition purposes.

 

The contours that have been learned can be used for detecting the position of the component during the placement process.
