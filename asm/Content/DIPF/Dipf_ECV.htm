Component shape: ECV

The following parameters describe a component shape of the type "ECV":

Vertical, cylindrical body, 2 lead groups each containing one lead. The leads must be identical.

Parameter

Description

Global properties

Name

Name of the component shape.

User text

Comment specified in the programming system.

1

Body diameter of the component (X) = Overall length

2

Height of the component (Z) = overall height

Separate ICOS group description

Recommendations

Yes

The component shape has separate group descriptions for ICOS that cannot be changed in the component shape manipulator.

No

No separate group descriptions are available.

Special measurement options

Check connection widths

Yes

The system checks the extent to which the width of a connection deviates from the nominal width.

No

This inspection is not carried out.

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.

No

The check is carried out.

Connection groups (2):

2 lead groups, each with 1 lead.

 

3

Length of the connecting lead

4

Width of the connecting lead <= body diameter

Contact length (less than the lead length)

 

Offset X/Y: The offset from the center point of the lead to the center point of the component in the X or Y direction.
