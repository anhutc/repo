Describing the "Pattern" feature type

This feature type allows you to describe unusual shapes in the component shape wizard:

  1. Open the Component Shape Wizard.

  2. Confirm the component shape type as Nonstandard or Connector by pressing Next.

  3. In the Other features tab, press Pattern.

  4. In the camera image, select the area in which the contours are to be found by drag-selecting it.

  5. The contours detected in the area are shown in green.

  6. If the area contains items that are not relevant for detection of the feature, you can remove them:
    To do this, press Delete pattern and select the area to be ignored in the camera image.

  7. Press the Next button.

  8. Confirm the position of the component model shown in green and of the reference point or set new values.

  9. After a final check, press Finish.

All the contours detected in the selected area have been learned and can be used for detecting the position of the component during the placement process.
