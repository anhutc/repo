Component pocket

The following parameters describe a component pocket:

Parameter

Description

Global properties

Name

Name of the component pocket.

User text

Comment specified in the programming system.

Length X

Size (in mm) of the component pocket perpendicular to the feed direction.

Width Y

Size (in mm) of the component pocket in the feed direction.

Special measurement options

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

No

The check is carried out.

A component may need to be taught in individual cases:

If components with a length less than or equal to 1.0 mm are to be picked up from black blister belts in 8 mm X-Feeders. In the case of light-colored belt material, optical recognition of the component pocket as calculated automatically from the component shape will generally not present a problem.
