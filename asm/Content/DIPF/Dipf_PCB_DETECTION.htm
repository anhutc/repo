Synthetic ink spot for PCB recognition

 

The following parameters describe a synthetic ink spot:

Parameters

Description

Name

Name of the ink spot

User text

Comment specified in the programming system.

Feature is light

Yes

The ink spot is lighter than the background.

No

The ink spot is darker than the background.
