Positional tolerance

Hint: 

The selected positional tolerance should approximate to the distance between the pin and the edge of the hole.

-

If too many placeable components are rejected, you should increase the positional tolerance.
-

If components are placed on the board even if they do not match the holes or have bent pins, you should reduce the positional tolerance.

Symmetrical tolerance

On
(Default)

1: Positional tolerance

The system checks whether the detected positional deviation of the tip of the pin is within a circular tolerance region with a radius equal to the positional tolerance.
 

Off

1: Positional tolerance in Y

2: Positional tolerance in X

The system checks whether the detected positional deviation of the tip of the pin is within this area.

Width = double the positional tolerance in X

Length = double the positional tolerance in Y

 

Application cases:

• Rectangular pins must fit into round holes

• Oval holes
