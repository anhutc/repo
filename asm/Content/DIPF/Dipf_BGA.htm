Component shape: Ball grid array (BGA)

The following parameters describe a component shape of the type "Ball Grid Array":

Parameter

Description

Name

Name of the component shape.

User text

Comment specified in the programming system.

1

Body length (without leads) = Length of the component = overall length (with leads)

2

Body width (without leads) = Width of the component = overall width (with leads)

3

Body height (without leads)

4

Overall height: Body height + lead height

Number of connections

The total number of all described balls that have to be recognized by SIPLACE Vision.

Separate ICOS group description

Recommendations

Yes

The component shape has separate group descriptions for ICOS that cannot be changed in the component shape manipulator.

No

No separate group descriptions are available.

Special measurement options

Inspection

Yes

All features are checked for presence, positional tolerance and to a certain extent for their properties. The result of this process is used to determine the exact position of the component.
If dipping has been specified for a component, an additional check is carried out to ensure that all connections are adequately coated with flux.

No

This inspection is not carried out.

Orientation

 

Checking the orientation with the stationary camera degrades placement performance.

1

Default:
The system does not check whether a component is rotated by 180°, 90°, 60° or 45° in the feeder.
If a component has been picked up at an incorrect orientation, and its position cannot be measured due to asymmetrically arranged connections, it is rejected.

Caution: The system cannot detect rotation on components with symmetrically arranged connections.

If components which are rotated in the feeder are to be placed, you must specify how many different positions the component can have in the feeder.

2

Rotation through 180° can be detected (2 different positions).

4, 6, 8

4, 6 or 8 different positions for the component can be distinguished.

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.

No

The check is carried out.

Chipping inspection

Yes

The edges of the component are inspected for signs of splintering or chipping.

No

This inspection is not carried out.

Group (n):

The following parameters apply to the individual groups

1

Pitch X:
The distance between the ball center points of two neighboring balls in X direction.

2

Pitch Y:
The distance between the ball center points of two neighboring balls in Y direction.

4

Offset X:
The distance between the center point of the group and the center point of the component in X direction.

5

Offset Y:
The distance between the center point of the group and the center point of the component in Y direction.

 

 

Pitch error threshold

Maximum permitted deviation from the predefined pitch.

Angle

Rotation of group in relation to component.

Number of leads X or Y

Number of balls in X or Y direction
