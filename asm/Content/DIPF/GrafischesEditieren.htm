Graphical editing in the geometry editor

When working in the geometry editor and selecting properties, various settings for components can be made directly in the graphical view:

  1. First select an object from the Properties list.
    The selection is marked in cyan.

  2. Within this selection, tap a point, hold your finger down and drag the pointer icon to the required position.

Dragging in the graphical interface yields different results depending what object is selected and where it is selected. Different pointer icons indicate different changes.
