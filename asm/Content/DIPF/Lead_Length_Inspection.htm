Checking the connection length

With small components in particular that are provided in tapes, it is possible for the component to be sucked onto the nozzle with the underside facing up when it is picked up from the feeder. If this is not detected, it is placed on the board with the underside facing up.

To avoid this, detection of flipped components can be enabled. This works by comparing the brightness of the top of the component with the brightness of the underside.

 

If the two brightness values do not differ significantly, but the lengths of the connections differ significantly between the top of the component and the underside (e.g. with chip or molded component types), the lead lengths can be measured to identify whether the components have been picked up correctly.

 

Components which have been picked up on edge can also be recognized:

For instance, the connection metalization of resistors is generally only present on the top and bottom, but not on the sides (see the right-hand component in example 1).

 

If the lengths of the two connections differ considerably on the same side, it is also possible to check polarity.

 

If a component is detected as having been picked up upside down (flipped) or on edge, it is rejected.

 

Conditions for using this inspection step:

-

The connection lengths on the top and underside must differ by more than the tolerance.

-

There must be no unbroken change in brightness on the top side in the tolerance region of the inner end of the connection.
