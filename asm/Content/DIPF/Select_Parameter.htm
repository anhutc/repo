Select parameters

Select the entry directly from the parameter list or select the entry by clicking the following buttons:

 

Selects the parameter above the selected entry.

Selects the parameter below the selected entry.

 

 

Alternatively, you can also select an entry in the list and navigate through the list using the cursor up and cursor down keys.

 

 

Hint:

In the Geometry view or you can also select the parameters in the graphic at the bottom left of the screen.
