Taught ink spot

The model is derived automatically by the algorithm using example images, thus the system teaches itself.

The following parameters describe a taught ink spot:

Parameters

Description

Name

Name of the ink spot

User text

Comment specified in the programming system.

Search area: Horizontal size

Extent of the search area in X.

Search area: Vertical size

Extent of the search area in Y.

Method

Determines the evaluation method: Recommendation

Template-based

Grayscale histogram

Mean grayscale value

Subsampling

Specifies whether the resolution of the image is reduced.

Example: 4 -> Areas of 4x4 pixels in size, are grouped as a single pixel. The accelerates the teaching and evaluation processes.

If the subsampling specification is changed, all example images previously created must be discarded.

Special parameters that are set when using the "Template-based" method:

0/1 Class 0 template is evaluated

Yes: For each evaluated image, the correspondence with the no-go sample is evaluated.

No: The correspondence with the no-go sample is not evaluated.

0/1 Class 1 template is evaluated

Yes: For each evaluated image, the correspondence with the go sample is evaluated.

No: The correspondence with the go sample is not evaluated.

 

Special case:

0/1 Class 0 template is evaluated   Yes
0/1 Class 1 template is evaluated   Yes

For each image to be evaluated, the difference between the similarities with the go and no-go samples are calculated.

The difference is calculated as follows: Correlation value for "go" template - correlation value for "no-go" template.

Special parameters that are set when using the "Grayscale histogram" method:

Histogram resolution

The number of bars on a histogram  (the smaller the value selected, the more grayscale values will be summarized as a single bar).
Recommendation: 15-50

Select region of interest

The size of the area to be used for deriving the model can be selected when the button is clicked (the cursor in the camera image is shown as a cross).

Previous images

The images taught to date are listed together with their classification.

Delete

The relevant image is deleted.

Ink spot pattern

Place

The sample in the camera image is defined as a "go" sample.
The machine assembles the board if this ink spot is detected.

Do not place

The sample in the camera image is defined as a "no-go" sample.

Automatic

Concludes the teaching process. After this, ink spot measurements can be carried out.

Warn if in doubt

Button pressed: If the ink spot cannot be uniquely assigned during placement, a warning is output. This is used for troubleshooting and testing.

Place if in doubt

Button pressed:
If the ink spot cannot be uniquely assigned during placement, the board/panel is assembled. This optimizes board throughput.
This button should not be pressed when you are using expensive components and wish to save components.

 

Button not pressed:
No components are placed on the panel or the panel.
You can achieve maximum component savings, for example, when using very expensive components.
