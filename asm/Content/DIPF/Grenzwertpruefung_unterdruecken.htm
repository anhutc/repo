Suppress checking of threshold values

Certain minimum or maximum dimensions (e.g. for lead widths, ball diameter, etc.) are defined for each type of sensor system. Adhering to these dimensions guarantees correct measurement. If these thresholds are not met, an error message is issued and the component must be measured with a different type of sensor system.

 

If a component is to be measured with an unsuitable type of sensor system, checking of the threshold values must be disabled (Suppress checking of threshold values = YES).

 

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.
