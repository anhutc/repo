Component shape: Connectors

The following parameters describe a component shape of the type "Connector":

 

 

Parameter

Description

Global properties

Name

Name of the component shape.

User text

Comment specified in the programming system.

Body type

Rectangle

1

Body length (without leads) = Length of the component (X)

2

Body width (without leads) = Width of the component (Y)

3

Body height (without leads) (Z)

4

Overall height: Body height + lead height

5

Overall length: Body length + lead length

Overall width: Body width + lead length

Number of connections

The total number of all described connections (leads, balls, columns, features) that have to be recognized by SIPLACE Vision.

Separate ICOS group description

Recommendations

Yes

The component shape has separate group descriptions for ICOS that cannot be changed in the component shape manipulator.

No

No separate group descriptions are available.

Special measurement options

Orientation

 

Checking the orientation with the stationary camera degrades placement performance.

1

Default:
The system does not check whether a component is rotated by 180°, 90°, 60° or 45° in the feeder.
If a component has been picked up at an incorrect orientation, and its position cannot be measured due to asymmetrically arranged connections, it is rejected.

Caution: The system cannot detect rotation on components with symmetrically arranged connections.

If components which are rotated in the feeder are to be placed, you must specify how many different positions the component can have in the feeder.

2

Rotation through 180° can be detected (2 different positions).

4, 6, 8

4, 6 or 8 different positions for the component can be distinguished.

Detection of flipped components

Yes

Various methods are available to identify whether a component has been picked up correctly or with the bottom side facing up (flipped).

No

This inspection is not carried out.

Check connection widths

Yes

The system checks the extent to which the width of a connection deviates from the nominal width.

No

This inspection is not carried out.

Check connection collinearity

Yes

The system checks the position of the ends of the leads.

No

This inspection is not carried out.

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.

No

The check is carried out.

Odd shape component

 

Measurement option subject to license fee:

Yes

The component is a component with a special shape for which a number of special processes and settings are possible.

No

Standard component.

 

 

 

Connection groups

Group (n):

The following parameters apply to the specified lead group:

Connection type

JLead, Gullwing, Wraparound, Ballcircle, Column, Blob, THT square pin, THT round pin

1

Pitch:

The distance between the lead center points of two neighboring leads.

2

Width of the lead

3

Offset Y: The offset from the center point of the group to the center point of the component in Y direction.

4

Offset X: The offset from the center point of the group to the center point of the component in X direction.

Notch width

Width of notch in leads: Gullwing with notch.

Number of leads

Number of leads per group.

Pitch error threshold

Maximum permitted deviation from the predefined pitch.

Angle

Rotation of group relative to component.

Collinearity tolerance

Maximum permitted discrepancy between the end of the lead and the calculated position.

Evaluation of position

(only shown if "Odd shape component" is enabled for the component shape in the programming system)

The description of the connection group is only used for determining the selected aspects of the component position: Position in X, position in Y or angle of rotation.

 

Connection type

THT round pin

Feature is light

Yes

The tips of the pins are lighter than the body of the component.

No

The tips of the pins are darker than the body of the component.

1

Pitch X

The distance between the centers of two neighboring round THT pins in X.
This value is not relevant if there is only one round THT pin.

2

Pitch Y

The distance between the centers of two neighboring round THT pins in Y.

3

Diameter of a round THT pin.

4

Offset X
The distance between the center point of the round THT pin group and the center point of the component in X.

5

Offset Y
The distance between the center point of the round THT pin group and the center point of the component in Y.
   

 

Positional tolerance

Maximum permitted deviation of the position of the tip of the round THT pin.

Symm. positional tolerance

Yes

The positional tolerance is the same in X and Y.

No

Different positional tolerances can be given for X and Y.

Number of leads X or Y

Number of round THT pins in X or Y that belong to this group.

Connection type

THT square pin

Feature is light

Yes

The tips of the pins are lighter than the body of the component.

No

The tips of the pins are darker than the body of the component.

1

Pitch X

The distance between the centers of two neighboring rectangular THT pins in X.
This value is not relevant if there is only one rectangular THT pin.

2

Pitch Y

The distance between the centers of two neighboring rectangular THT pins in Y.

3

Width of the rectangular THT pin.

4

Length of the rectangular THT pin.

5

Offset Y
The distance between the center point of the rectangular THT pin group and the center point of the component in Y.

6

Offset X
The distance between the center point of the rectangular THT pin group and the center point of the component in X.

 

Positional tolerance

Maximum permitted deviation of the position of the tip of the rectangular THT pin.

Symm. positional tolerance

Yes

The positional tolerance is the same in X and Y. Only one value is entered.

No

Different positional tolerances can be given for X and Y.

Number of leads X or Y

Number of rectangular THT pins in X or Y that belong to this group.

 

Pattern (contours)

X/Y

Coordinates of the center of the rectangle that encloses the complete contour.

Length X / Width Y

Size of the smallest rectangle that encloses the contour..

Evaluation of position

(only shown if "Odd shape component" is enabled for the component shape in the programming system)

The description of the feature is only used for determining the selected aspects of the component position: Position in X, position in Y or angle of rotation.

 

 

 

Centering pins

Connection type

Centering pin

Centering pin n:

The following parameters apply to the specified centering pin or the centering pin group:

1

Length

Length of the centering pin in X.

2

Width

Length of the centering pin in Y.

3

Offset Y
The distance between the center point of the centering pin group and the center point of the component in Y direction.

4

Offset X
The distance between the center point of the centering pin group and the center point of the component in X direction.

5

Height
Height of the centering pin

6

Pitch:
The distance between the pin center points of 2 neighboring centering pins. This value is only relevant for centering pins.

Number of leads

Number of centering pins per centering pin group.
