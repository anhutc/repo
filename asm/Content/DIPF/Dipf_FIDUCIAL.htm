Synthetic fiducial

The following parameters describe a fiducial defined by its geometrical shape:

Parameter

Description

Name

Name of the fiducial

User text

Comment specified in the programming system.

Suppress checking of threshold values

 

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

No

The check is carried out.

Fiducial is light

Yes

The fiducial is lighter than the background.

No

The fiducial is darker than the background.

Evaluation of position

Specified by the programming system:

X and Y

The X and Y positions are both used for detection of the fiducial.

Y

Only the Y position of the fiducial is relevant.
(Note: The Y axis of the coordinate system is highlighted accordingly in the representation).
