Component shape: Moulded

The following parameters describe a component shape of the type "Moulded":

Rectangular body, 2 lead groups each with one lead. The leads are identical.

Parameter

Description

Global properties

Name

Name of the component shape.

User text

Comment specified in the programming system.

1

Length of the component (X) = Overall length

2

Width of the component (Y) = Overall width

3

Height of the component (Z) = overall height

 

Separate ICOS group description

Recommendations

Yes

The component shape has separate group descriptions for ICOS that cannot be changed in the component shape manipulator.

No

No separate group descriptions are available.

Special measurement options

Inspection

Yes

All features are checked for presence, positional tolerance and to a certain extent for their properties. The result of this process is used to determine the exact position of the component.

No

This inspection is not carried out.

Detection of flipped components

Yes

Various methods are available to identify whether a component has been picked up correctly or with the bottom side facing up (flipped).

No

This inspection is not carried out.

Check connection lengths

Yes

The connection lengths are to be determined in order to identify whether a component has been picked up correctly or with the bottom side facing up (flipped).

No

This inspection is not carried out.

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.

No

The check is carried out.

Chipping inspection

Yes

The edges of the component are inspected for signs of splintering or chipping.

No

This inspection is not carried out.

Connection groups (2)

2 lead groups, each with 1 lead.

Connection type

Wraparound

See figure above.

4

Length of the connecting lead

5

Width of the connecting lead

6

Offset X: The offset from the center point of the lead to the center point of the component in the X direction.

 

Note: If "Check connection lengths" has been enabled, different connection lengths can be specified for the two connection groups.


If the tolerance values are set too large, it is possible that components which have been picked up on edge will not be recognized.
