Taught ink spot for  PCB recognition

The model is derived automatically by the algorithm using example images, thus the system teaches itself.

The following parameters describe a taught ink spot for PCB recognition.

Parameters

Description

Name

Name of the ink spot

User text

Comment specified in the programming system.

Search area: Horizontal size

1 mm

Search area: Vertical size

1 mm

Method

Mean grayscale value

Subsampling

1

 

Previous images

The images taught to date are listed together with their classification.

Delete

The relevant image is deleted.

Ink spot pattern

Placement

The sample in the camera image is defined as a "go" sample.
The machine assembles the second board as well (combined PCB) if this ink spot is detected.

Do not place

The sample in the camera image is defined as a "no-go" sample; the second board is not assembled.

Automatic

Concludes the teaching process. After this, ink spot measurements can be carried out.

Warn if in doubt

Button pressed: If the ink spot cannot be uniquely assigned during placement, a warning is output. This is used for troubleshooting and testing.

Place if in doubt

Button pressed:
If the ink spot cannot be uniquely assigned during placement, the board is assembled.

 

Button not pressed:
No components are placed on the panel or the panel.
