Separate group description for SIPLACE Vision and ICOS

In SIPLACE Pro, it is possible to store descriptions of feature groups separately for SIPLACE Vision and ICOS.

The improved illumination technology used on the SIPLACE Vision cameras allows connector features that were only indistinctly visible or not visible at all with ICOS cameras to be made visible and evaluated reliably.

Special arrangements such as alternating short and long leads can be evaluated with the improved algorithms in SIPLACE Vision whereas this was not possible in ICOS.

Leads with notches can also make it necessary to choose this setting.

 

This separate programming is necessary for the following connector types and is recommended to provide you with an overview:

Corners, polygon circles, blobs and rectangles.

 

Restrictions:

  • If separate descriptions were enabled in SIPLACE Pro, they cannot be disabled in SIPLACE Vision.
  • If separate descriptions were enabled in SIPLACE Vision, this can only be reset in SIPLACE Pro after the component shape has been stored.

 

Separate ICOS group description:

Yes

The component shape has separate group descriptions for ICOS that cannot be changed in the component shape manipulator.

Recommended setting if:
•

the improved illumination technology allows SIPLACE Vision to detect features that can only be detected on camera images indistinctly or not at all by the ICOS image processing system,
•

the ICOS image processing system cannot process the lead type set (e.g. corner, blob lead, etc.),
•

the improved evaluation algorithms in SIPLACE Vision allow connectors to be measured that cannot be measured with the ICOS image processing system

No

No separate group descriptions are available.

Recommended setting if:
•

the connector features can be revealed clearly in the camera image using the illumination technology of both systems (default).
