Describing a board edge

The described board edge is shown in light blue in the representation. The hatched side of the line indicates the side facing the board.

The following parameters describe a board edge:

Parameter

Description

Global properties

Name

Name of the board edge.

User text

Comment specified in the programming system.

Board edge length

Length of the edge in X.

Board is bright

Yes

The board is lighter than the background.

No

The board is darker than the background.

Evaluation of position

Specified by the programming system:

Y

Only the Y position of the board edge is relevant.
(Note: The Y axis of the coordinate system is highlighted accordingly in the representation).

Special measurement options

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.

No

The check is carried out.
