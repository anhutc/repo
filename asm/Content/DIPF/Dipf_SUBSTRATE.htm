Describing a substrate

The following parameters are used to describe substrates (wafers, carriers):

Parameter

Description

Global properties

Name

Name of the component shape.

User text

Comment specified in the programming system.

Diameter

Diameter tolerance

Size of the substrate.

Maximum permitted size deviation.

Substrate is bright

Yes

The substrate is brighter than the carrier material.

No

The substrate is darker than the carrier material.

Start angle

Position of the first camera image for determining the X/Y position with reference to the described position of the substrate.

Tip: Choose the start angle in such a way that the notch cannot be seen in any of the camera images of the evaluation positions.

Number of X/Y evaluation positions

Specifies the number of positions at the edge of the substrate at which a camera image will be captured in order to determine the X/Y position of the center of the substrate.

These evaluation positions are distributed evenly.

Default: 3

Evaluation of position

Specified by the programming system:

X and Y

Notch detection will not be performed. Only the X/Y position will be determined.

X, Y and angle

Notch detection is performed to allow the orientation of the substrate to be determined.

Special measurement options

Suppress checking of threshold values

Yes

Checking of the minimum and maximum values specific to a given sensor system type is not carried out.

If checking of the threshold values is suppressed, this may negatively impact on the recognition rate and the accuracy. This may cause unexpected errors.

No

The check is carried out.

Substrate notch

Notch detection

Specified by the programming system:

Yes

The orientation of the substrate is determined on the basis of the detected notch.

(Evaluation of position: X, Y and angle)

No

Notch detection is not performed.

(Evaluation of position: X and Y)

Notch width (1)

Notch depth (2)

Notch radius (3)
