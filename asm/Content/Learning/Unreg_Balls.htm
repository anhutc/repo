As soon as at least one irregular group has been learned for a component, all other group positions are no longer rounded or aligned with each other.

 

  1. First select a region in which a group of balls is to be learned.
    All the features detected are added to the component shape data.
    If the positions of several of the features lie exactly on a grid, they are combined to form a single group.

  2. Check whether any structures which are not connectors have been identified as features and delete them manually.
