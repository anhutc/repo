Inspection area: Results and parameters

In this view you can either learn a new model or adjust the parameters for an existing model.

A new model is learned:

  • If no model is yet available for inspection. In the recognition step Learned solder deposits / components you can check whether all the solder deposits and components have been found.
  • If a setting has been changed, e.g. if the illumination has been adjusted, or a new teaching area or exclusion area has been defined or an existing one has been edited.

A test inspection is then carried out using the model.

Previously learned model:

Those recognition steps are shown that already existed before the wizard was started. The inspection process is restarted after the parameters have been adjusted.

Parameter

Description

Inspection type

(This is specified in the programming system)

 

Solder paste

The quality of the solder paste is checked:

  • Height of the solder deposit
  • Volume of the solder deposit (area x height)
  • Delineation of the solder deposit in order to identify solder bridges

Before placement

Before the component is placed, the following items are checked:

  • whether components below the current component have already been placed
  • that the placed components have not been placed significantly out of position
  • that there is no lost component in the inspection area

After placement

The following items are checked immediately after placement:

  • that the component is present
  • that the placed component has not been placed significantly out of position

Maximum height difference for solder deposits [mm]

If the average height value of a solder deposit as currently measured differs from the height value in the model by more than the difference specified here, the solder deposit is marked as faulty (red).

Depending on the setting, the currently measured height value is compared either with the height of the corresponding solder deposit or with the average height of all solder deposits in the model.

Average solder height as reference

Yes

An average height for the solder deposits is calculated across all solder deposits in the model.

During evaluation, the height of each measured solder deposit is compared with this average height.

No

Default: During evaluation, the height of each measured solder deposit is compared with the height of the corresponding solder deposit in the model.

Maximum volume difference for solder deposits [%]

Solder volume = height x area

If the calculated volume of the current solder deposit exceeds the volume in the model by more than this percentage, the solder deposit is marked as faulty (red).

Minimum volume difference for solder deposits [%]

Solder volume = height x area

If the calculated volume of the current solder deposit is smaller than the volume in the model by more than this percentage, the solder deposit is marked as faulty (red).

Component inspection sensitivity

1..7

The smaller the value set here, the more sensitive the measurement will be.
If there are a large number of incorrect identifications, this value should be increased.

Minimum length in X for detection of lost components [mm]

Detection of lost components:

If components with a length (in X) less than this value are lost, they cannot be identified.

Minimum width in Y for detection of lost components [mm]

Detection of lost components:

If components with a width (in Y) less than this value are lost, they cannot be identified.

If it was possible to learn the model, but the results are subject to restrictions, one of the following messages is displayed:

Message

Suggestion for rectifying the error

Blue illumination is weak

Switch back to the Blue illumination view and increase the illumination intensity.

Significant height differences in one region

In the case of inspection after placement:

Switch back to the Teaching areas view and redefine the teaching areas in such a way that each one only contains components of the same height.

Determined solder heights differ considerably

During solder paste inspection:

Restart the teaching operation using a model board on which the solder paste has been applied more evenly.

If necessary, enable the "Average solder height as reference" option.

Back

Switches back to the previous view.

OK

(Only shown if no parameters have been changed.)

The current settings are retained and the wizard is closed.

Apply data for board inspection

(Only shown if parameters have been changed.)

The wizard is closed and the changed settings are accepted.

Cancel

Closes the wizard and rejects the changed data.
