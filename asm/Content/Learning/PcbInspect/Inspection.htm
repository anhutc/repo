Inspection areas (board inspection)

In the programming system, you can specify for individual placement positions that the area in which the component is to be placed is to be inspected.

First, the model of an inspection area has to be learned. During the placement operation, the inspection areas are compared with the corresponding model inspection areas.

There are three different inspection types:

Solder paste

 

 

Inspection of the solder paste is done on the first placement machine in the line so that faulty boards can be identified and discarded immediately.

•

Height of the solder deposit.

•

Volume of the solder deposit (area x height).

•

Delineation of the solder deposit in order to identify solder bridges.

Before placement

Immediately before the component is placed:

•

All components that are below the component to be placed have already been placed (e.g. in the case of shields).

•

All the components in the inspection area are checked to ensure that they are not significantly out of position.

•

There is no additional component in the inspection area (lost component).

After placement

 

Immediately after a component has been placed:

•

The component was placed correctly.

•

The position is checked to ensure that the component is not significantly out of position.

 

Proceed as follows to teach the inspection area for solder paste inspection using the wizard in SIPLACE Vision :

 

1.  Specify the values for blue illumination: If the contrast is too low, increase the intensity.

2.  Specify the red illumination if blue illumination does not deliver good results.

3.  Specify the teaching areas (particularly if the components are of different heights).

4.  Check the results and adjust the parameters if necessary.

 

Proceed as follows to teach the inspection area for inspection before placement using the wizard in SIPLACE Vision :

 

1.  Specify the values for blue illumination: If the contrast is too low, increase the intensity.

2.  Specify the teaching areas (particularly if the components are of different heights).

3.  Specify the exclusion areas.

4.  Check the results and adjust the parameters if necessary.
