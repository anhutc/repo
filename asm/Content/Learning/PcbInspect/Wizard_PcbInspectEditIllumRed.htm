Inspection area: Red illumination

Red illumination can only be used for solder paste inspection (SPI).

Tip:

Only activate red illumination in addition if it is difficult to learn the solder deposits well using blue illumination.

Caution:

The use of red illumination increases the time required for capturing and evaluating the images.

 

Example of use:

Red illumination is used to improve detection of solder deposits that are offset on the solder pads to avoid solder bridges:

Tip:

Choose the intensity of the red illumination in such a way that the parts of the pads that are not covered with solder appear white.

 

Use red illumination

No red illumination

Default: Only blue illumination is used.
