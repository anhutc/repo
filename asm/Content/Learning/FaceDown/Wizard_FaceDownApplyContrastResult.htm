Detection of flipped components: Results if contrast method is selected

 

Teach current component:

Specify whether the component currently shown is in the correct orientation or is flipped:

OK

 

The component is in the correct orientation and can be placed.

Flipped

 

The component is flipped and cannot be placed.

 

Trained components

Two components are shown, one in the correct orientation and one flipped.

Example

OK

 

Low contrast: There is a homogeneous range of brightness in the region of interest.

Flipped

 

High contrast: A wide variety of intensities occur in the region of interest.

Region of interest

Yellow frame

Regions of interest: The areas in which the system determines whether different intensity levels occur.

 

Histogram and classification (example)

Blue lines

Brightness distribution in the region of interest.

Pale blue frame

Standard deviation of the brightness distribution.

Yellow frame

Fixed, specified threshold for the standard deviation.

Green horizontal bar

If the standard deviation is in this region, the component is detected as having been picked up correctly.

Red horizontal bar

This indicates the region in which the component is detected as having been picked up with the underside facing up.

 

Result

Representation of the position detected and the orientation of the component.

Green frame

Representation of component.

Blue frame

Comparison area.

Yellow frame

Region of interest for the component.

Green coordinate system

The component is identified as being in the correct orientation and can be placed.

Red coordinate system

The component is identified as being flipped and cannot be placed.

 

Back

Switches back to the Define region of interest view.

Accept (Only shown if parameters have been changed.)

Parameters that have been changed are saved and the wizard is closed.

OK (Only shown if no parameters have been changed.)

The current settings are retained and the wizard is closed.

Cancel

The wizard is closed and any changes made are rejected.
