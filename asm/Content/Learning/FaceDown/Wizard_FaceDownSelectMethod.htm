Detection of flipped components: Selecting the method

When components are picked up, it is possible that individual components are picked up on the bottom side by the nozzle. If the vision system detects this, the component is rejected in order to avoid incorrect placement.

You can choose the method to be used to detect flipped components:

Brightness

To allow the detection of flipped components, the mean brightness of a rectangle in the center of a component is determined. The result is given in brightness levels (range: 0 - 255).

This method is suitable for components where the brightness of the top and bottom differs considerably (e.g. the top is black and the bottom white).

Contrast

Evaluates the contrast in the ROI (region of interest).

This method can be used if, for instance, the top is printed.

  • High contrast: A wide variety of intensities occur in the region of interest.
    The top of a component is generally printed, and the printing contrasts considerably with the background.
  • Low contrast: There is a homogeneous range of brightness in the region of interest.

Detection using gullwing connections

The Vision system determines the average brightness (possible values: 0 = dark through 255 = light) in a rectangle at the end of gullwing leads.

Detection using gullwing connections with illumination from one side

When illumination from one side is used, the gullwing leads that lie opposite each other appear with different brightnesses.
