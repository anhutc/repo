Detection of flipped components: Setting the illumination for detection using gullwing connections

Flat-angle illumination is advantageous if you wish to use the gullwing leads to determine whether a component has been picked up the right way up or not:

Correct:

The component was picked up and positioned in front of the camera correctly:

The region of interest marked in yellow on screen is obscured by the lead. The light does not reach the region of interest. It appears dark in the camera image.

Flipped:

The component was picked up and positioned in front of the camera upside down:

The region of interest is illuminated directly and appears light in the camera image.

 

This setting is only of limited use for detecting the position of a component.

It is therefore recommended that different illumination settings should be used for detecting the position of the component and for detecting flipped components.

 

Create gullwing illumination

Default:

Special illumination settings are used for detecting flipped components. This illumination can be adjusted so that the ends of the leads are illuminated as well as possible, allowing them to be distinguished easily.

No additional illumination

The same illumination is used to detect the position of the component and whether it has been flipped.

It is recommended that a component is taught for each of the cases (correct and flipped).

Only one image is captured.

If detection of the component position fails using the illumination set for detecting flipped components, the change cannot be saved. In this event, you should always use a gullwing illumination.

Recommended illumination values

Sets a flat-angle illumination that is suitable for most components.

 

However, because the geometry of components with gullwing leads can vary considerably, it may sometimes be necessary to use a different illumination from the default.

 

Proceed as follows to set the illumination on the component camera for detecting flipped components if you want to use different illumination from that recommended:

  1. Select an entry from the list of illumination angles:
    E.g.: Flat.

  2. Set the required intensity (brightness value) for the selected illumination type using the rotary control:

 

A new camera image is acquired and measurement of the component is carried out each time the illumination setting is changed.
