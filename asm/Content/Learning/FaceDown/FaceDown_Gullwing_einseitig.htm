Flipped components: Detection using gullwing connections with illumination from one side

When illumination from one side is used, the gullwing leads that lie opposite each other appear with different brightnesses.

Whether the component is flipped or not depends on which side (the row of leads nearer to the illumination or the row further away) appears brighter:

If the component has been picked up correctly (face up), the rows of leads opposite the active illumination segment (light source) will appear bright.

 

Prerequisite for detection using illumination from one side:

•

Component type: SOT, SOXX, QFP, CONNECTOR, NONSTANDARD or SOCKETS.
•

Camera type 48 or 49:

The 60° illumination is split into three segments, each of which can be controlled separately.
 

The segment to be used is selected automatically depending on the presentation position for the measurement.
•

At least two rows of gullwing leads that lie at 180° to each other have been specified.

These two rows of leads are illuminated from one illumination segment only and appear with different brightnesses.
