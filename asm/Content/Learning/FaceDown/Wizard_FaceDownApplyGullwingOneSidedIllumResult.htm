Detection of flipped components: Results if gullwing connections are evaluated using illumination from one side

Yellow bar

The brightness of the leads closer to the light source (active illumination segment).

  

Green column

The brightness of the leads opposite the light source if these are brighter than those nearer to the light source.

 The component is not flipped:

Red column

The brightness of the leads opposite the light source if these are darker than those nearer to the light source.

The component is flipped.

The greater the difference between the yellow and green or yellow and red bars, the more reliable the result.

Example:

OK

The component is not flipped:

The row of leads opposite the light source is brighter than the row nearer to the light source.

 

Flipped

The component is flipped:

The row of leads opposite the light source is darker than the row nearer to the light source.

 

Back

Switches back to the view Detection of flipped components: Selecting the method.

OK

Closes the wizard.

Cancel

The wizard is closed and any changes made are rejected.
