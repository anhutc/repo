Flipped components

When components are picked up, it is possible that individual components are picked up on the underside by the nozzle. If the vision system detects this, the component is rejected in order to avoid incorrect placement.

 

Different methods are used to detect flipped components:

Evaluation of the brightness values of the top and underside of a component: This is only possible if the colors of the top and underside differ significantly (e.g. with certain resistors).

Evaluation of the contrast of a specified area, the ROI (region of interest): This can only be done if the inner structure of the top and underside differs significantly (e.g. SOT labeled on one side).

Detection using gullwing connections for component with gullwing leads.

 

 

 

To start the wizard:

  1. Switch to the view "List of measurement options" .

  2. Choose the measurement option Detection of flipped components.

  3. Click Edit parameters.

The wizard starts, allowing you to select the detection method.
