Activating and deactivating detection of flipped components

To activate detection of flipped components:

  1. Start the Geometry view with the component for which you want recognition to be performed.

  2. Choose Properties -> Special measurement options.

  3. Check the parameter "Detection of flipped components".

  4. Set the value to "Yes".

 

To deactivate detection of flipped components:

  1. Start the Geometry view with the component for which you want recognition to be deactivated.

  2. Choose Properties -> Special measurement options.

  3. Check the parameter "Detection of flipped components".

  4. Set the value to "No".
