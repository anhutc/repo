Stereo measurement: Setting parameters and results

Use the stereo measurement views to check whether the parameters are set correctly, i.e. whether the positions of the connections can be easily recognized in the camera image.

Parameter

Description

THT pin length

Enter the length of the shortest THT pin in [mm].

Traverse path

The total distance in [mm] by which a component is moved.

As a rule, the traverse path does not have to be changed and the default value can be used.

Tip:

The necessary traverse path in X depends on the height of the connections relative to the background:

The shorter the connections, the greater the traverse path needs to be.

Default traverse path for camera type (SST25): 8 mm

Default traverse path for camera type (SST33): 18 mm

Maximum traverse path: Half the field of view of the camera in X.

Vertical resolution

This value is calculated on the basis of the specified traverse path and cannot be edited.

The greater the value entered for the traverse path, the greater the vertical resolution will be.

Default: 1/3 of the height of the connection.

Coarse vertical resolution: Stereo height view

Fine vertical resolution: Fine stereo height view

4x image capture

Yes

This setting creates a high-resolution image using individual offset images.

Recommendation:

Only use this option if a component cannot be recognized
(e.g. with low components).

 

Caution: Use of this setting reduces placement performance!

No

Default.

 

Back

Switches back to the Set illumination view.

OK

(Only shown if no parameters have been changed.)

The current settings are retained and the wizard is closed.

Accept

(Only shown if parameters have been changed.)

The wizard is closed and the changed settings are accepted.

Cancel

Closes the wizard and rejects the changed data.
