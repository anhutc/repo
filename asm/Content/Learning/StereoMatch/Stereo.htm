Stereo measurement of components

The aim of stereo measurement is to use a topographic image to determine the exact position of the tips of THT pins, gullwing connections and pattern features (e.g. boardlocks).

Two images of a component are captured from different perspectives to allow a topographic image of the component to be created:

Initially, the component is positioned centrally in front of the camera. It is then moved left (in a negative X direction) by half the traverse path, and an image is captured. The component is then moved to the right (in a positive X direction) by the entire traverse path, and the second image is captured. These images are used to calculate a height map.

 

Stereo measurements are only possible with type 25 and 33 cameras and only for components of the type Nonstandard or Connector.

Prerequisite: The "Vision OSC Package - Machine Connection" license must be available on the programming system.

Enable 4x image capture:

  • Switch to the Measurement options view.
  • Choose Stereo measurement and press Edit parameters.
  • Press Next under 'Set illumination'.
  • Set the 4x image capture parameter to Yes and press OK to complete entry.
