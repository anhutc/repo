Detection of the top side of the component: Variants

Because components that have the same component shape description can occur in different variants, it may be necessary to have several taught models to determine the pickup position of components in the tape. This can be the case, for instance, if lettering is used as a recognition feature and this changes as a result of switching to a different supplier or due to a new batch designation.

If the current component differs from the one that has been taught, you should create a new model, i.e. a new variant.

Yellow dots (circles)

These represent the currently selected model.

If these yellow dots are located on pixels in the vicinity of which the brightness changes significantly, the model matches.

 

Add variant

Creates a new model for the current component.
Specify a unique variant name. The name should not be too long, because it is displayed in the result view.

Remove variant

Removes the model selected in the list, so that it is not used for recognition.

 

If several taught models exist, they are used for recognition one after another until one of the variants matches.

The variant that was successfully used for recognition is then the first one used for the next component, because it is probable that this will come from the same batch.

 

Next

Switches to the Set illumination view.

Cancel

Closes the wizard and rejects the changed data.
