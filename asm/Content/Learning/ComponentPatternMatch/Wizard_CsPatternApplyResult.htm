Detection of the top side of the component: Results

The recognition steps for detection of the top side of the component are displayed and can be selected.

Position: Level x (variant)

Green cross

The position detected.

If this cross is not displayed, it was not possible to determine the position.

Yellow rectangle

Search area in which the component must be located.

Green dots (circles)

Points in the model at which marked transitions between dark and light were found.

 

Gradient: Level x (variant)

The lighter a point is displayed in the gradient image, the greater the brightness change is at this location in the original image.

 

Result

Representation of the position detected and the orientation of the component.

Minimum grayscale contrast

Measure of the minimum change in brightness.

The greater this value, the greater the change in brightness must be for the algorithm to learn a model point at this position.

Tile model

(determined during teaching)

Yes

The model is divided into a number of different subareas. Each of these subareas is measured separately in the search area.

Advantage: Shorter measurement times
No

The entire camera image is compared with the model in a single step.

Advantage: More robust measurement

Quality threshold

70 %

The displayed quality value with which a component is identified must be greater than the value defined here for the measurement to be flagged as successful.

If several results are shown, you should increase the threshold so that only the best result is shown.

Pyramid level: Start

 

 (set by the system)

Lowest resolution used to start searching.

The higher this value, the quicker measurement will be.

If measurement fails at the topmost level, this value should be reduced to make measurement more robust.
Pyramid level: Stop Default: 1

Highest resolution used to finish measurement and finally determine the position of the component.

The lower this value, the more accurate the results of measurement will generally be.

If the image is not sharp or if it is possible to identify structures of the component body, this value should be increased to make measurement more robust.
