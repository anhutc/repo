Detection: Top side of component

If this measurement option is enabled for a given track, every component is measured by the board camera before it is picked up.

This is necessary, for instance, if the positional tolerance within the component pocket is so large that reliable pickup is no longer possible (e.g. in the case of special components that have been inserted into tapes by hand).

A pattern recognition process is used in which easily recognized sections of the top side of the component are learned. Before each component is picked up, it is compared with the specified model to determine the exact position of the component in the tape.

 

Start the wizard for detection of the top side of the component as follows:

  1. Switch to the view "List of measurement options" .

  2. Choose the measurement option Detection of top side of component.

  3. Press Edit parameters.

 

The wizard is opened showing the Variants view.
