Detection of the top side of the component: Setting the illumination

How to adjust the component camera illumination to detect the top side of the component:

  1. Select an entry from the list of illumination types:
    Red steep, Red diffuse, Red diffuse + steep and Blue.

  2. Set the required intensity (brightness value) for the selected illumination type using the rotary control:

 

A new camera image is acquired and measurement of the component is carried out each time the illumination setting is changed.

When making this setting, make sure that the structure of the surface is clearly recognizable.

 

Note:

If cracked die inspection is performed in addition to component position recognition, the following illumination is set by default: red, steep, intensity 250.
