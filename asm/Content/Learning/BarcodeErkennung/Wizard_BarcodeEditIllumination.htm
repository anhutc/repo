Barcode recognition on the underside of components: Setting the illumination

In order to read a barcode attached to the underside of a component, as is the case with LEDs, for instance, the component is positioned in front of the component camera after it has been picked up. Barcode recognition is performed while the component is being recognized by SIPLACE Vision.

 

Create barcode illumination

If the barcode cannot be recognized clearly using default illumination, you should try to improve the results with an illumination that is only used to recognize the barcode.

Caution: Capture of the additional image reduces placement performance!

No additional illumination

Default: The same illumination is used for recognizing the barcode as for component position recognition.

 

How to adjust the component camera illumination to recognize the barcode on the underside of the component:

  1. Select the illumination type which delivers the best contrast in the camera image.

  2. Set the required intensity for the selected illumination type using the rotary control:

 

A new camera image is acquired and measurement of the barcode is carried out each time the illumination setting is changed.

 

When making this setting, make sure that the barcode is clearly visible.
