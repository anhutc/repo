Chipping inspection: Setting illumination

A new camera image is acquired and measurement of the component is carried out each time the illumination setting is changed.

Create illumination for chipping inspection

Set the illumination in such a way that the edges of the component are clearly visible.

Caution: Capture of the additional image reduces placement performance!

No additional illumination

Default: The same illumination is used for detecting chipped areas along the component edges as for component position recognition.

Tip:

As a rule, steep illumination is used for chipping inspection.

It is recommended that an additional illumination should be created for chipping inspection if the component position is measured with flat illumination (e.g. in the case of dies with ball-shaped connections).
