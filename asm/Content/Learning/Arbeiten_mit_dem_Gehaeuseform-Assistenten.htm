Working with the Component Shape Wizard

 

The Component Shape Wizard provides you with a simple method of describing components whose geometry data needs to be recorded. This does not involve entering the data for individual features or feature groups manually; instead, these can be detected automatically using the camera.

 

The following recognition options are currently supported: components of the types chip, MELF, bare die, shield and molded; corner features, leads with notches, holes, dark features on light backgrounds and particularly narrow leads.

 

Proceed as follows:

  1. Position the component in front of the camera.
    On the Test Center, make sure that the distance from the component to the camera is correct (spacer plate) and that the deviation from the 0°-/90°-/180° or 270° rotational position is not greater than 5°.

  2. Start the Component Shape Wizard.

  3. Specify the component shape type for the component.

  4. Where necessary, select the feature type.

  5. In the camera image, specify the region in which the features are to be detected.

  6. Specify the pitch value.

  7. Confirm the calculated reference point and the body size or set new values.

  8. Check the geometry data and specify appropriate tolerances for high placement quality for the current component batches.
