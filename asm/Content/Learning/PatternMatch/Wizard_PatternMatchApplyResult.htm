Pattern matching: Results

If measurement is successful, the result of the measurement is shown (in green).

 

If measurement is unsuccessful, an error message to this effect appears:

Teaching failed: 

• Check the illumination settings.

• Make sure that there are sufficient structures in the selected search area.

Measurement failed:

If the position of the component that has been detected is outside the presentation tolerances or if the component is only partially visible in the camera image:

• Check whether the presentation tolerance should be increased or whether the pickup position is incorrect.
