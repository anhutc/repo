Pattern matching: Change reference point

Default: Center of the region of interest.

Objective: The reference point should correspond to the specified placement position in SIPLACE Pro.

Yellow rectangle

Region of interest for the pattern.

Yellow cross

Center of the pattern.

Green coordinate system

The origin corresponds to the position of the reference point in the camera image.

 

Proceed as follows to determine the reference point:

  1. Carry out a test placement operation using the preliminary reference point.
  2. Start measurement of the components outside the machine in order to determine the offset (placement error).
  3. Start the pattern matching wizard again.
  4. Change the coordinates of the reference point on the basis of the offset that has been determined (X, Y, angle).

    The orientation of the coordinate system for the reference point is identical with that in the component shape description in SIPLACE Pro.

 

The region of interest is moved by one increment to the left, right, up or down.

Select the point in the image that you wish to define as the reference point.

The coordinate system of the region of interest is rotated one increment to the left or right.

The width or height is increased or decreased by 1 increment.

Step size

The value by which the specified parameter is moved, rotated or resized by clicking one of the buttons shown above.

The increment is increased or decreased.

You can also enter any of these values directly from the keyboard by clicking the corresponding field and entering the value you require.

Next

The model is learned on the basis of the modified data and a measurement is started.
The system then switches to the Result view.

Back

Switches back to the Exclusion areas for pattern matching view.

Cancel

Closes the wizard and rejects the changed data.
