The origin corresponds to the presentation position of the component.

How to adjust the component camera illumination for glue dots:

  1. Select an entry from the list of illumination angles:
    E.g. steep, medium, flat.
  2. Set the required intensity for the relevant illumination type:
    Press the or keys repeatedly until the required value is reached or click and enter the value in the numeric keypad.

 

A new camera image is acquired and measurement of the component is carried out each time the illumination setting is changed.

Make sure that the structures are clearly visible. A high-contrast image is therefore recommended.

 

Hint:

In most cases, the structures of the dies are detected best using the steep illumination levels.
