Buttons of the wizard

 

Back

Shows the previous view. Any entries already made are retained.

Next

Shows the next view.

(This button is only enabled when all the necessary entries have been made in the current view.)

Finish

Closes the Wizard and saves your entries.

Reset

All changes made in a wizard are reset to their initial status when SIPLACE Vision was started.

Undo

Cancels the entries most recently made.

If the button is pressed after closing a wizard, all the changes made in the wizard are reset.

Redo

Restores an entry which had been canceled with Undo.

If changes made in a wizard are reset, all the changes are redone.

Cancel

The Wizard is closed and all changes made are rejected.
