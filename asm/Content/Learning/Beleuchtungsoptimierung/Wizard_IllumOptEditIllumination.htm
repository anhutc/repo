Illumination optimization - Set component position for optimization

If the measurement is not successful, i.e. if the component position is not recognized correctly, move the component model (green) so that it covers the camera image.



The component position is moved by one increment to the left, right, up or down.

Mark the position of the component in the image.

The component position is rotated one rotational increment to the left or right.

Step size

The value by which the specified parameter is moved, rotated or resized by clicking one of the buttons shown above.

The increment is increased or decreased.

Reset

Cancels the position changes that have been made.

 

Start optimization

Starts an optimization run: During optimization, the measurement with the best setting to date is shown.

Redo

Restores the canceled intensity value.

Undo

Resets the intensity value most recently set to the value previously set.

Cancel

Closes illumination optimization.

Next

Switches to the Illumination optimization - Result view.
