Checking the reference point and the body size

The reference point which has been determined is indicated in the image.

If this point or the angle of the coordinate system displayed does not correspond to the placement position programmed in SIPLACE Pro, or if the body size is not correct, you must change the values:

  1. Click the No, change button.

The Move reference point view is displayed.

 

If the reference point and the body size are okay, you can move to the next view:

  1. Click the Yes, accept button.

The Check result view is displayed.
