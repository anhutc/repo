Back

Switches back to the previous view.

Finish

Closes the wizard and saves the information. You should now check the saved data in the Geometry data view.

  • You should in particular check the tolerances for the lead width and lead length and the pitch error threshold.
  • For symmetrical components: Are the group offsets correspondingly symmetrical?
  • For ball or column grids: Are all the available features marked in the model and are any missing features indicated?
     

Cancel

Closes the Wizard. The geometry data for the component must be entered manually. Any changes made are rejected.
