Pitch: Checking the value

This query only appears if the measured value does not exactly match an internally defined typical value.

The Measured pitch corresponds to the pitch measured for the largest group.

The additional selection options correspond to the closest matching internally defined typical value.

If none of the specified values match on the Test Center, the distance to the camera may be set incorrectly:

The values are too small:

The measured object must be moved closer to the camera.

The values are too large:

The measured object must be moved away from the camera.

If none of the specified values match on the machine, it may be that the component height has been specified incorrectly. Check the settings in the programming system.
