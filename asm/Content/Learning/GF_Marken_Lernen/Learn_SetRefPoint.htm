Move reference point

Move the reference point so that it corresponds to the placement position defined in the programming system.

Move reference point:



The reference point is moved by one increment to the left, right, up or down.

Rotating the reference point coordinate system:

The reference point coordinate system is rotated one increment to the left or right.
Width and height of body

The width or length is increased or decreased by one increment.

Step size

The value by which the specified parameter is moved, rotated or resized by clicking one of the buttons shown above.

Increases or decreases the increment.

 

You can also enter the coordinate specifications for the reference point or the length and width specifications for the body size using the keyboard.
