Defining a connection group (stereo measurement)

First select the relevant connections:

Leads:

Gullwing

THT square pin:

THT round pin:

Define a connection group:

  1. Select the lead type.

  2. If you select gullwing leads, specify the orientation of the leads by selecting the angle of the leads in relation to the component.

  3. In the camera image, click and hold down the left mouse button and drag to mark a region in which the connection group should be detected.
    The detected connections are highlighted in green after you have drag-selected the area.

  4. Repeat the process of selecting the connections until all the connection groups are selected.

Other features:

Pattern:

  1. Press Pattern.

  2. In the camera image, drag-select the area in which the pattern is to be detected.

    The system attempts to learn all the selected edges in the marked area.

Delete pattern:

If features of the type Pattern have been described previously, specific regions of these can be explicitly deactivated for the purposes of detection.

  1. Press Delete pattern.

  2. In the camera image, drag-select the area in which no pattern is to be detected.
    All the contours already detected in this region are removed.

 

Show height map

The brighter a point appears in the representation, the nearer it is to the camera.

Switching to this view improves visibility of the connections.

Because the height map is used to identify the location of the component, the learning step is also carried out using the height map.
