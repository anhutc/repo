Working with the Fiducial Wizard for synthetic fiducials

 

The Fiducial Wizard provides you with a simple method of describing fiducials whose geometry data needs to be recorded. You do not need to describe the structure manually. Instead, it can be detected automatically using the camera.

 

Proceed as follows:

  1. Move the PCB camera over the PCB so that the fiducial is clearly visible in the camera image.

  2. Adjust the illumination if necessary. The fiducial should be well illuminated.

  3. Select the form which best describes the fiducial or ink spot:
    Different parameters are displayed depending on this selection.

  4. Specify whether the fiducial or the background is lighter.

  5. Start the Fiducial Wizard.

  6. In the camera image, select the area in which the fiducial is located.

  7. Press Next.

  8. Close the Wizard by pressing Accept.

  9. Check the geometry data.
