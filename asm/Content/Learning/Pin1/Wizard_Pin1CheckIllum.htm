Pin1 detection: Selecting illumination

 

Create Pin1 illumination

Component illumination is not sufficient to perform Pin1 detection successfully. It is therefore necessary to set an illumination that allows clear recognition of the Pin1 mark.

No additional illumination

The current setting for illumination of the component will be used.

 

 

Next

Switches to the Pin1 region of interest view or, if a custom Pin1 illumination is chosen, to the view for setting this illumination.

Cancel

Closes the wizard and rejects the changed data.

 

Tip:

Because the use of different illumination settings can lead to a loss in performance, this should only be done if Pin1 cannot otherwise be detected reliably.
