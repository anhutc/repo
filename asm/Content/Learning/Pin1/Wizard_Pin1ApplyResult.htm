Pin1 detection: Results

The recognition steps for Pin1 detection are displayed and can be selected.

Pin1: Result

Inspection to establish whether there is a significant difference between the Pin1 region of interest (the area in which Pin1 is expected) and the comparison areas.

Brightness: Comparison of the average brightness (2 or 4 areas are compared).

Contrast: Comparison of the grayscale distribution (histogram), (4 areas are compared).

Correlation: Pattern comparison (4 areas are compared).

Green frame

Pin1 was found at the expected position.

Blue frame

Comparison area.

Yellow frame

Pin1 region of interest: The area in which Pin1 is expected
(is only shown if Pin1 was not found here).

Red frame

Pin1 was found, but not in the area expected.

The associated values indicate the brightness.

 

Pin1: Compare regions

Different methods are used one after another to establish whether one of the compared areas is significantly different from the others.

Methods: Comparison of mean brightness, comparison of contrast and pattern matching.

Yellow frame

Pin1 region of interest.

Blue frame

Comparison area.

Light blue highlighting

Represents the threshold value for confirmed detection. If the green column is higher than the threshold value, the Pin1 region is deemed to have been detected.

Green column

Specifies the confidence level for detection of the Pin1 region. The higher the value, the more reliable detection is.

Red column

If the Pin1 region is detected at a position other than that expected, the column is shown in red. Measurement is then restarted with a different presentation angle.

Pink column

It was not possible to find a Pin1 region. The confidence level is therefore below the threshold even after all methods have been applied.

 

Result

Representation of the position detected and the orientation of the component.

 

Parameter

Default

Description

Threshold for unambiguous recognition

10.0

The higher this value, the greater the difference between the Pin1 region and the comparison regions must be before it is detected.

 

Back

Switches back to the Specify orientation value view.

Accept (Only shown if parameters have been changed.)

Parameters that have been changed are saved and the wizard is closed.

OK (Only shown if no parameters have been changed.)

The current settings are retained and the wizard is closed.

Cancel

The wizard is closed and any changes made are rejected.
