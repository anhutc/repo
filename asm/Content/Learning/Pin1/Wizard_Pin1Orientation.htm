Pin1 detection: Specifying the orientation value

If the Pin1 mark was not detected, you can specify how the component is to be treated here:

1

The component is rejected.

2 (default)

The component is rotated by 360°/2 = 180° and measured again.

4

The component is rotated up to 3 times by 360°/4 = 90 ° each time and measured again.

6

Up to 5 times by 360°/6=60° each time

8

Up to 7 times by 360°/8=45° each time

The following applies to the settings 4/6/8: The component is no longer rotated when Pin1 is detected.

 

Back

Switches back to the Region of interest view.

Next

Switches to the Results view.

Cancel

Closes the wizard and rejects the changed data.
