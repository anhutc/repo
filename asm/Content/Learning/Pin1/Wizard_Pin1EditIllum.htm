Pin1 detection: Set illumination

 

How to adjust the component camera illumination for Pin1 detection:

  1. Select an entry from the list of illumination angles:

    E.g. steep, medium, flat.



  2. Set the required intensity (brightness value) for the selected illumination type using the rotary control:

 

A new camera image is acquired and measurement of the component is carried out each time the illumination setting is changed.

When making this setting, make sure that the Pin1 mark is clearly visible, i.e. that it is as distinct as possible from the other regions.

 

Optimize illumination

Automatically sets the illumination in such a way that the Pin1 mark is as clearly visible as possible.
