Buttons of the wizard for importing ball or column lists

 

Back

Shows the previous view. Any entries already made are retained.

Next

Shows the next view.

(This button is only enabled when all the necessary entries have been made in the current view.)

Finish

Closes the wizard.

The ball or column list is transferred to the component shape. All existing groups are deleted and replaced by the new groups.

The system switches to the component shape editor and starts a new measurement.

Cancel

The Wizard is closed and all changes made are rejected.
