Importing a ball or column list

Some manufacturers offer component shape data in electronic form containing the description of irregular ball or column groups in the form of a list. These files can usually be converted to simple text files without any great effort.

 

The ball or column list must be available as a text file with the following structure to allow it to be read by SIPLACE Vision:

<X position> separator <Y position>

Spaces, commas or "|" can, for instance, be used as separators.

Because the text file does not contain details on the unit used or on the defined reference point to which the coordinates of the balls or columns refer, you have to supplement these details after reading in the list.

 

  1. Position the required component in front of the camera.

  2. Open the ball or column list at the station computer GUI or at the Vision Teach Station.

  3. Specify the unit used to describe the balls or columns.
  4. Edit the position list: Define the reference point, the ball size, etc.

  5. Under Representation, check whether the camera image of the component matches the current model.

  6. Close the Wizard by clicking the Finish button.
    The ball list is imported into the component shape. All existing groups are deleted and replaced by the new groups.
