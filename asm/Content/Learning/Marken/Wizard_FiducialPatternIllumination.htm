Pattern matching for fiducials: Setting the illumination

 

How to adjust the component camera illumination for fiducial pattern matching:

  1. Select the illumination type which delivers the best contrast in the camera image:

    Red steep, Red diffuse, Red diffuse and steep or Blue.

  2. Set the required intensity for the relevant illumination type:

    Click the or buttons repeatedly until the required value is reached or click and enter the value using the numeric keypad.

 

A new camera image is acquired and measurement of the fiducial is carried out each time the illumination setting is changed.

 

When making this setting, make sure that the fiducial is clearly visible.
