Pattern matching for fiducials: Define reference point

The reference point must lie within the region of interest and corresponds to the center of the fiducial.

 

Yellow rectangle

Region of interest for the pattern.

Yellow cross

Center of the pattern.

 

Set reference point automatically

Calculates the reference point on the basis of the geometry.
It is not possible to input any data manually.

 

Set reference point manually

You can specify the reference point yourself: Select a point within the specified region of interest - as close to the center of the fiducial as possible.

If possible, you should allow the reference point to be specified automatically in order to be able to achieve the highest possible degree of accuracy.

The reference point must be set as accurately as possible. It must correspond to the position of the fiducial as taken from the board description in the programming system.

 

The region of interest is moved by one increment to the left, right, up or down.

Select the point in the image that you wish to define as the reference point.

The coordinate system of the region of interest is rotated one increment to the left or right.

The width or height is increased or decreased by 1 increment.

Step size

The value by which the specified parameter is moved, rotated or resized by clicking one of the buttons shown above.

The step size is decreased or increased.

Angle increment

The value by which the specified parameter is moved, rotated or resized by clicking one of the buttons shown above.

The step size is decreased or increased.

You can also enter any of these values directly from the keyboard by clicking the corresponding field and entering the value you require.

Parameter

Description

 

Reference point horizontal / vertical position

This corresponds to the position of the fiducial as set in the board description in the programming system.

 

Show component shape (is only shown if there is a component at the fiducial position.)

Solder pads can be used as fiducials for high-precision placement.

Click this button to show the representation of the component shape.

The solid or broken red lines shown in the camera image are auxiliary lines.

If a component placement position is used as a fiducial, the component shape representation can be moved exactly to the placement position on the board. The reference point of the component shape then corresponds to the reference point of the taught fiducial.

 

Next

The model is learned on the basis of the modified data and a measurement is started.
The system then switches to the Result view.

Back

Switches back to the Define region of interest view.

Cancel

Closes the wizard and rejects the changed data.
