Change parameters

  1. Choose the required parameter from the list and click Edit...
  2. Enter a value in the specified range and click Accept.
  3. Repeat the steps above to change further parameters as required.
  4. When you have made all your changes, click Start. The machine function for writing the changed values to the conveyor firmware is executed.
  5. To be able to access the changed values even after the machine has been restarted, clickSave parameters to EEPROM. This writes the data to the EEPROM.
