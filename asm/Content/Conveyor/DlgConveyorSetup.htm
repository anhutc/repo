Setup > Lane x > Width

 

Measure width

Conveyor lane width [mm]:

Shows the most recently measured width of the conveyor lane in mm.

If a "?" appears here, the width is unknown and must be determined again (e.g. if the emergency STOP button was pressed when moving the conveyor rails).

Measure width

The adjusted width of the conveyor lane is measured and displayed.

Adjust width

Adjust width...

Opens a dialog box which allows you to enter the width of the conveyor lane in [mm] again.

1. Enter the required width in the New width [mm]: field.

2. Click Start to start the machine functions for setting the new width.

Each time you click on the following buttons, the width is adjusted by a small or a large step.

Increase (small step)

The conveyor lane width is increased by 0.1 mm by pressing the button.

Increase (big step)

The conveyor lane width is increased by 1 mm by pressing the button.

Decrease (small step)

The conveyor lane width is decreased by 0.1 mm by pressing the button.

Decrease (big step)

The conveyor lane width is decreased by 1 mm by pressing the button.

Take into account the board width sent by IPC-HERMES-9852

Specify whether the board width is to be accepted if it was transfered in an IPC-HERMES-9852 message:

Only if barcode mode is active (default)

The conveyor is only set to the board width given in the message if the machine is in barcode mode.

Always

As soon as a board width is transfered in an IPC-HERMES-9852 message, it is taken into account.

Never

A board width transfered in an IPC-HERMES-9852 message is always ignored.

Caution!

Make sure that there is no board in the conveyor before you change the conveyor lane width.

 

Select conveyor lane. 1: right lane, 2: left lane.
