Conveyor rail configuration

Name

Value

Fixed conveyor rail

Location of the fixed conveyor rail as set in the Conveyor fixed rail options view.

Fixed conveyor rail position

Position of the fixed conveyor rail as set in the Conveyor fixed rail options view.

Fixed conveyor rail position lane 1 [mm]

User-defined position of the fixed rail set for the right-hand lane in the Conveyor fixed rail options view.

Fixed conveyor rail position lane 2 [mm]

User-defined position of the fixed rail set for the left-hand lane in the Conveyor fixed rail options view.

Distance of fixed rails [mm]

Distance between the fixed rails calculated from the current positions of the fixed rails for conveyor lane 1 and conveyor lane 2.

Conveyor widening

Option (Dual or Dual as single) set for conveyor widening in the Conveyor fixed rail options view.

Reference position of fixed conveyor rail
(not shown for all machines)

Teach left conveyor rail / Teach right conveyor rail

Opens the dialog box for teaching the corresponding conveyor rail. This automatically activates the teaching functions. The activated conveyor rail can be moved to the right and the left to teach the position.

Store reference position of fixed conveyor rail

Stores the position of the fixed conveyor rail determined during teaching.

Machine configuration

Parameters

(Only those parameters that can be changed are shown here)

Value

Use barcode position for barcode reading

The position at which the barcode is located was specified by SIPLACE Pro. The barcode scanner is only activated at this position

Thick board conveyor

Setup engineer: If the conveyor has been adapted in such a way as to allow particularly thick boards to be assembled, "installed" should be specified here.

Use vacuum tooling

If the system has detected that vacuum tooling is present, but you do not wish to use it, you can deactivate it here (-).

Use Vario Grid

If the system has detected that Vario Grid is present, but you do not wish to use it, you can deactivate it here (-).

Conveyor extension, input

Setup engineer: Set this parameter and shut the machine down before you install or remove the conveyor extension at the input section.

Conveyor extension, output

Setup engineer: Set this parameter and shut the machine down before you install or remove the conveyor extension at the output section.

Extended conveyor

Setup engineer: Set this parameter and shut the machine down before you install or remove the conveyor extensions at the input and output sections.

Use optional stopper

If an additional stopper has been installed, you can specify whether it is to be used.

Use virtual forward stopper / Use virtual backward stopper

If no additional forward or backward stopper has been installed for assembling long boards, it is possible to specify that a virtual stopper is to be used. This means that the belt motor will be used to transport the board to a specific position without the use of real stoppers.

Use laser light barrier

Yes

Default. The board is positioned using the laser light barrier.

No

The laser light barrier is ignored when positioning the board.
Caution: This option may only be disabled if it is certain that there are no cut-outs at the front edge of the boards.

Use optional laser light barrier

If an additional laser light barrier has been installed, you can specify whether it is also to be used.

SMEMA: Pass/Fail

This parameter should be set if the information whether a board is OK or NOK can be received from the upstream machine or sent to the downstream machine over the SMEMA interface.

Protruding board in output section

For test purposes on standalone machines: A board in the output section is permitted to protrude from the machine.
This increases the maximum board length in the output section, which means that the placement section is no longer occupied.
