Transport board to

Different buttons appear here depending on the machine type. E.g.:

Output section

Moves a board from placement section 2 to the output section.

Placement section 2

Moves a board from the intermediate section to placement section 2.

Intermediate section

Moves a board from placement section 1 to the intermediate section.

Placement section 1

Moves a board from the input section to placement section 1.

Next stop position (only with "long board" option)

Moves a board to the next stop position within the placement section.

Signals

Update

This shows the current status of the sensors (laser light barriers). The LED icons show the section of the conveyor or the stop position a board is located in and what sensor (light barrier) has been triggered.

Triggered (board in section x or stop position x of the conveyor)

Not triggered

 

The conveyor mode currently set is also shown.

Transport mode: normal board / long board 

Board data

Board length

Specified length of the board.

Refresh board status

Checks whether a board has been removed during transport. If this is the case, it is removed from the displayed list and removed from the stored data.

Temperature of vacuum tooling, PA 1 / 2 (only shown if a vacuum tooling type is present)

If the measured value shown in the bar is in the green area, the temperature is OK and the board or wafer can be processed.
If the temperature is too high or too low, the system waits until the temperature has been adjusted.

However, if the temperature is too high or too low for an extended period, you should check the temperature set at the chiller.

 

Select conveyor lane; 1: right lane, 2: left lane.
