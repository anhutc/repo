Service > Conveyor configuration > Lane x > Speed parameters

 

Set speed parameter

All conveyor belts

[mm/s]

Maximum conveyor speed with which a board is transported through the entire conveyor.

Drive board

[mm/s]

Speed of the conveyor when loading a board.

Unloading board

[mm/s]

Speed of the conveyor when unloading a board.

Maximum acceleration

[m/s2]

Maximum acceleration at which a board is transported through the entire conveyor.

Velocity split

[%]

For combined PCB on X-series machines:

Specifies as a percentage the amount the conveyor from which the board is being unloaded is to travel slower than the conveyor in which the board is being loaded. This value should only be specified for very thin boards to prevent boards from overlapping. The lower the value, the less time is lost when transporting boards.

Time combined PCB

[ms]

For combined PCB on X-series machines:

Specifies the lag time of the conveyor for the processing section necessary to ensure that the second board of the "Combined PCB" is moved tight up against the first board.

Edit...

Opens a dialog box in which you can change the parameters shown.

Test speed parameter

Opens a dialog box showing the start time.

The selected conveyor or all conveyors move using the set parameters until you press the Stop button.

All conveyor belts

Tests the maximum conveyor speed with which a board is transported through the entire conveyor.

Loading board

The speed of the conveyor when loading a board in the machine is tested.

Unloading board

The speed of the conveyor when passing the board to the downstream machine is tested.

Save parameters to EEPROM

Saves the conveyor parameters that have been set to the EEPROM of the conveyor controller.
