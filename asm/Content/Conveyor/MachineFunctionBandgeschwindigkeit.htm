Testing selected conveyor speed

Execution

This shows the current status of the machine function.

Start: Start time of the test.

End: End time of the test.

 

Restart

The test is restarted.

Note: This button is only enabled after completion of the test or when the test is halted.

Start

The selected conveyors move continuously at the selected speed.

Stop

The conveyors are stopped.

Close

Closes the dialog box. 

Note: This button is only enabled after completion of the test or when the test is halted.
