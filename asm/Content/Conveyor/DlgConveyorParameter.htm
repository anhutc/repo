Vacuum tooling: Offset of backward board sensor

[mm]

The offset of the backward board sensor from the default stop position in the placement section.

Height of board on vacuum tooling

[mm]

The maximum permitted height of the board when using vacuum tooling.

Increased bumper lifting table

[mm]

This value must be set to 15 mm for the SIPLACE X4I
in order to reduce the space below the board.

Convoy mode: Maximum board length

[mm]

Boards up to this length will be handled in convoy mode.

If 0 mm is entered here, convoy mode is disabled.

Convoy mode: Maximum delay before mode will be stopped

[mm]

Maximum permitted delay before a pair of boards in the placement section are separated.
Convoy mode is terminated after this period has elapsed.

Convoy mode: Waiting time for 'Board available' from previous machine

[mm]

Amount of time the system waits for a board from the upstream machine to allow the boards to be transported in pairs.

This mode is not used if this value is exceeded.

Vacuum tooling: Lifting table, processing area 1, up position

[mm]

The highest position of the lifting table that can be reached if vacuum tooling is installed.

Maximum board height for special sequence 'Thin board'

[µm]

If particularly thin boards are being processed, it is necessary to start a special process for positioning the board. This process is used up to the maximum board thickness specified here. If the value is set to 0, the boards are processed using the default process.

Offset of the board sensor from the default stop position

[µm]

The offset between the board sensor of the placement section and the default stop position of the board at the laser light barrier. This value is used if the laser light barrier cannot be used for positioning the board.
