Service > Conveyor configuration > Conveyor initiation

Only shown for certain machine types.

All belt motors

Check motor current

The motor current is determined and stored and displayed in the parameters "Motor current forward" and "Motor current backward". This function can be used to check that the conveyor is moving uniformly.

All lifting tables

Calibrate motor

The reference point of the motor is determined.

Check motor current

The lifting table is moved slowly up and down. The motor current is determined and stored and displayed in the parameters "Motor current for lifting table, weight dependent" and "Motor current for lifting table, friction dependent".

Reference run

Starts a single reference run for the lifting table (for test purposes).

Adjust width

Calibrate motor

Determines the zero point for the width adjustment motor and saves the zero point correction value.

Check motor current

The motor current during width adjustment is determined and stored and displayed in the parameters "Motor current for width adjustment facility for right-hand conveyor rail" and "Motor current for width adjustment facility for left-hand conveyor rail".

Reference run

Starts a single reference run for the width adjustment facility (for test purposes).
