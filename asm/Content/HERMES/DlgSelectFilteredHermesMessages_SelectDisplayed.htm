"The Hermes Standard" (IPC-HERMES-9852): Selection of message types to be displayed

Select the messages to be displayed. This setting applies to both conveyor lanes and to the input and output sections.

The message is displayed.

Messages (input/output section)

BoardAvailable

The machine is ready to pass a board, i.e. a board is ready for transport.

BoardForecast

Sent to the downstream machine: Either changes are required or advance information on the next board is being sent.

CheckAlive

Checks whether the communication partner can be reached.


MachineReady

The machine is ready to receive a board.

Notification

 

NotificationCodes

Notification sent before the connection is shut down (e.g. after a protocol error or after a shutdown call).


1: Protocol error

2: Connection request refused because a connection is already present

3: Connection reset due to changed configuration

4: Configuration error

5: Machine shut down

6: Error on BoardForecast

 

QueryBoardInfo

Queries information on one of the most recent boards.

RevokeBoardAvailable

The machine is no longer ready to pass a board.

RevokeMachineReady

The machine is no longer ready to receive a board.

SendBoardInfo

 

Sends information on one of the most recent boards.

ServiceDescription

Exchange of information (machine ID, lane ID, ...).
