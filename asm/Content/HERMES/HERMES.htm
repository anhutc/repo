"The Hermes Standard" (IPC-HERMES-9852)

Vendor-independent, open standard for communication between different stations of an SMT placement line (printers, placement machines, conveyor systems, AOI systems, soldering systems, etc.).

This communication interface provides a TCP/IP and XML-based protocol that allows uninterrupted traceability of boards through the entire SMT line.

Standardized machine-to-machine communication using IPC-HERMES-9852.

It is only necessary to enter the board ID at the beginning of a line, provided that the board data is passed from machine to machine using Standardized machine-to-machine communication using IPC-HERMES-9852.

Standardized communication between a station and a higher-level system using IPC-HERMES-9852. This communication takes place over a vertical channel.

The upstream system from which the data is received must be defined for each station in the line. In other words, the upstream system must be defined in terms of the IP address or the host name and port number.

If this is not specified by the line controller, the data must be entered separately for each participating system.
