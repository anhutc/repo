"The Hermes Standard" (IPC-HERMES-9852): Events (vertical channel)

Display showing the messages sent between this machine and a higher-level system (client system) using the IPC-HERMES-9852 protocol (max: the last 100 messages are shown).

Vertical channel
(communication with higher-level system)

Selected message

The message selected in the list.

Allows you to search for a text string in the displayed message.

Delete all

Deletes all messages, irrespective of whether they are currently being displayed or not.

Display all messages

All the sent messages are shown in the list, irrespective of message type.

Display only selected message types

Opens a dialog box in which you can select which messages are shown in the list.

Hide selected message types

Opens a dialog box in which you can select which messages are not to be shown in the list.
