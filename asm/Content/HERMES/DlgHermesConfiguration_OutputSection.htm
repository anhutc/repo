IPC-HERMES-9852 - Configuration for the output section

A peer on the downstream machine can be defined for each conveyor belt of the output section to allow information to be exchanged using IPC-HERMES-9852.

Peer configuration

Optional: Specify the end point for communication with the downstream machine.

Enable

Permits communication with the communication partner on the downstream machine using IPC-HERMES-9852.

Allowed client address

IP address or host name of the communication partner on the downstream machine.

Port

Note: This value cannot be configured.

Apply

Applies the currently entered settings for the permitted client or port.

 

Ignore configuration requests

The configuration specified by the programming system is overwritten by the settings made here.

Default: The configuration specified by the programming system is applied irrespective of the settings made here.
