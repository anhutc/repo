Service > Shuttle configuration > Speed parameters

Sets the speed of the conveyor belt for loading and unloading the boards.

Set speed parameter

Speed while loading board

[mm/s]

Speed of the conveyor belt while a board is being loaded into the Shuttle Extension.

Speed while unloading board

[mm/s]

Speed of the conveyor belt while a board is being unloaded from the Shuttle Extension.

Acceleration of conveyor

[m/s2]

Maximum acceleration when transporting a board.

Edit...

The selected parameter value can be changed within the specified limits.

Enter the required value and terminate your entry by pressing the Return key.

Test speed parameter

Opens a dialog box showing the start time.
The conveyor belt moves using the set parameters until you press the Stop button.

Conveyor belt

Tests the maximum belt speed with which a board is transported through the Shuttle Extension.

Drive board

The speed of the conveyor belt when loading a board into the Shuttle Extension is tested.

Unloading board

The speed of the conveyor when passing the board to the downstream machine is checked.
