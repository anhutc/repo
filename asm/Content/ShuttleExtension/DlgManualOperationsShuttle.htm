Barcode reader

Read barcode... (Only enabled if at least one barcode reader has been configured)

Opens a dialog box showing all configured barcode readers.

Select the barcode reader you wish to test and press Start.

The system then attempts to read a barcode for a defined period of time. To allow this to happen, hold a board in front of the selected barcode reader. You can interrupt the reading process at any time by pressing Pause.

The purpose of this is to check that the barcode reader is connected to the multiplexer correctly and is working as expected.

Actions

Initiation

The conveyor belt moves until it is possible to reliably detect whether a board is present or not.
The system first tries to move a board to the stop position. If this attempt does not succeed within a given time, it is assumed that no board is present.

Reference run

The following movements are performed, depending on whether a board is present in the Shuttle Extension or not:

If no board is present: The shuttle moves in both directions, the conveyor belt moves, the two rails move.

If a board is present: The shuttle moves in both directions.

All the movements are performed until all the positions in the Shuttle Extension are known.
