Shuttle Extension - inputs / outputs

The outputs are shown as buttons. On the left of each function name for an input or output, an LED icon is shown whose color changes from gray to red when the status of the relevant input or output is read. If an endurance run is being performed, the LED for the corresponding output flashes.

The input/output is active.
The input/output is not active.

When you press the button for a particular output, a dialog box opens in which you can activate or deactivate this output.
