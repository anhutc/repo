Service > Shuttle configuration > Shuttle initiation

Checks the motors of the Shuttle Extension.

Belt motor

Check motor current

The motor current value is determined when the belt is moved forwards and backwards. This value is stored in the parameters and displayed. This function can be used to check that the conveyor is moving uniformly.

Shuttle motor

Calibrate motor

The reference point of the motor is determined.

Check motor current

The shuttle is moved slowly to the left and right. The motor current value is determined, stored in the parameters and displayed.

Reference run

Starts a single reference run for the shuttle motor (for test purposes).

Move to position...

Checks whether the motor can be addressed.
Specify the desired absolute position of the shuttle relative to the center point of the Shuttle Extension in Y.

Width adjustment motor

Calibrate motor

Determines the zero point for the width adjustment motor and saves the zero point correction value.

Check motor current

The motor current is determined during width adjustment. This value is stored and displayed.

Reference run

Starts a single reference run for the width adjustment facility (for test purposes).

Move to position...

Checks whether the motor can be addressed.
Enter the desired absolute position of the right-hand conveyor rail relative to the left-hand conveyor rail.

If one of the motor current values is above normal, this is an indication that the shuttle is not moving freely or that there is an obstacle in the travel range.
