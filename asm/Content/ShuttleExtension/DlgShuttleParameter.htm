Service > Shuttle configuration > Parameters

Examples of settings that can be changed by the user.

Speed while loading board

Speed of the conveyor belt while a board is being loaded into the Shuttle Extension.

Speed while unloading board

Speed of the conveyor belt while a board is being unloaded from the Shuttle Extension.

Acceleration of conveyor

Maximum acceleration for transporting a board in the Shuttle Extension.

Timeout on moving board to next machine

When the timeout specified here has been exceeded, an error message is issued to the effect that the board has not reached the downstream machine.

Timeout on moving board from previous machine

When the timeout specified here has been exceeded, an error message is issued to the effect that the board has not arrived in the Shuttle Extension from the upstream machine.

Minimum conveyor width

Minimum board width that can be set.

Speed of changing width and shuttle position

This is the speed at which the conveyor rails of the Shuttle Extension travel when decreasing or increasing the conveyor width and at which both conveyor rails move when moving to a particular shuttle position.

Jolt of shuttle and width adjustment

Maximum jolt when the shuttle motor or width adjustment motor moves.

Maximum current at shuttle motor

If this value is exceeded, the shuttle motor is stopped and an error message is issued.

Maximum current at width adjustment motor

If this value is exceeded, width adjustment is stopped and an error message is issued.

Speed for testing conveyor

Speed at which the Shuttle Extension conveyor belt is moved when the test function is called (view: Test speed parameters -> Conveyor belt).

Speed for testing shuttle

Speed at which the shuttle is moved to the left or right.

Speed for testing width adjustment

Speed at which the conveyor rails are moved apart or together during the test functions.
