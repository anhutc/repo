Service > Single calibration > Shuttle Extension > Width calibration

The conveyor belt of the Shuttle Extension must be precisely adjusted to the width of a board to allow it to be taken from the upstream machine and transported to the downstream machine accurately. Make sure that the board is free to move, but does not have too much play.

Board width

The currently set width of the board to which the Shuttle Extension is to be adjusted.

Adjust width...

Opens the "Adjust conveyor width" dialog box: In the text box, enter the width of the board and then press Start.

Conveyor width

Each time you click on the following buttons, the width is adjusted by a small or a large step.

Increase (small step)

The conveyor width is increased by 0.1 mm by pressing the button.

Increase (big step)

The conveyor width is increased by 1 mm by pressing the button.

Decrease (small step)

The conveyor width is decreased by 0.1 mm by pressing the button.

Decrease (big step)

The conveyor width is decreased by 1 mm by pressing the button.

Accept

The width set for the conveyor is saved.

 

Calibrate board sensor

First make sure that there are no more boards in the Shuttle Extension.
The sensors are adjusted during calibration.

 

To calibrate the conveyor width of the Shuttle Extension:

1.  Have a board to hand.

2.  Press Adjust width..., enter the width of the current board and press Start.

3.  Place the board in the Shuttle Extension and adjust the conveyor width by pressing Decrease / Increase so that the board is free to move in the conveyor.

4.  Press Apply.

 

To calibrate the board sensor:

1.  Make sure that there are no boards in the Shuttle Extension.

2.  Press Adjust width..., enter the required width and press Start.

3.  Press Calibrate board sensor.
