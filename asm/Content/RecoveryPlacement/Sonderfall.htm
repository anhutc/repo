Recovery placement: Special case

 

In the software options, it was specified for a job that no placement should take place in one or both processing areas (skip processing area).

If there is then a power failure, for example, when a board is located on the intermediate section for which "Skip processing area 1" has been activated, the "Placed" status is also output for these placement positions despite the fact that no component has been placed in the corresponding position.

However, processing has been completed correctly since the "Skip processing area 1" option means that the placement positions were omitted and therefore there is no need to take them into account when placement continues either.
