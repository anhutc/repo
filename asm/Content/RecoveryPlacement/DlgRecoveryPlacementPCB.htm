Recovery placement - Details

The placement data for the selected board is shown here.

Name

Name of the placement position.

PA

Processing area in which the placement position is placed.

Head

The head which places the placement position.

Cycle

Head cycle.

Placement status

Placed 

There is a component at the placement position.

Special case

Not placed

The placement position is not occupied.

?

It is not known whether there is a component in the placement position or not.

In this case, the line can be edited. The Placed and Not placed buttons can be selected. You must decide whether the placement position is occupied or not (either by a visual inspection or by looking at the camera image).

 

Selected position

Placed

The selected placement position has already been placed.

Not placed

There is no component placed at the selected placement position.

Show image

When you switch to the camera image, the head is positioned at the selected placement position and a camera image is captured. You can see whether there is a component at the placement position or not.

Reset all positions

All changes made to the current board are discarded.

Board

Finish board

The current board should be assembled.

Cancel board

All changes made to the current board are to be discarded.

Mark for inspection

Depending on the selection made under , the board stops in the output area (output section or output shuttle) of the current station or in the output area of the last station in the line, where it can be inspected.

Canceled boards

Specify what is to be done with boards that have been canceled.

Here you can choose whether the board is to be removed from the output area (output section or output shuttle) of the current station or from the output area of the last station in the line. The board stops at the selected position.
