Recovery placement

If there is a power failure during a placement operation or if a serious machine fault makes it necessary to switch off the machine, partially assembled boards will remain in the machine.

Data about the boards located in the machine is automatically stored during placement.

When the machine is restarted after a power failure or after being switched off, the user is asked whether assembly of the boards in the machine should be resumed.

The machine then operates in recovery placement mode.
