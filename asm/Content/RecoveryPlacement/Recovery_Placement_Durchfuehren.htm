Carrying out recovery placement

 

Proceed as follows if you wish to carry out recovery placement for partially assembled boards after a power failure or if the machine has been switched off:

  1. Check whether components have fallen off the head and are lying on the boards and remove them.
  2. Click YES to acknowledge the dialog box which is displayed.
    Recovery placement is started and the icon appears in the main view.
  3. Check that all boards selected for recovery placement should actually be assembled.
  4. In turn, select the boards from the display that are marked with ?: .
  5. For each of the placement positions that has an undefined status, decide whether the relevant component has already been placed. Set the status to "Placed" or "Not placed" by selecting the position from the list and clicking the corresponding button.
  6. Choose the Finish board option.

Choose the Cancel board option if you cannot define all the placement positions for a board or if a large number of positions are undefined.

  1. Work through steps 4 - 6 until no more boards have the status ?.
  2. Activate the machine options which were set before the power failure / power down.
  3. Continue placement.

Once all boards processed for recovery placement have left the machine, recovery placement mode comes to an end and the icon disappears. "Normal placement" continues after this.

 

No boards may be removed or inserted while the machine is switched off.
