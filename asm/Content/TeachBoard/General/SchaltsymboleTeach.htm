Buttons in the teaching views

 

Reset changes

All changes made since the last save operation are rejected.

This action cannot be undone.

Revert

The most recent change or entry is reverted.

Recover

The change most recently rejected or the entry most recently made is repeated.

Retry

The most recent action is repeated, either for the same element or for a newly selected element.

The camera image of the next object is displayed and you can move and rotate the model of this object until it matches the camera image.

The previous object is selected.

Caution: The previous/next object is chosen according to the list in the table on the Details tab. Any sorting and selection that have been performed are taken into account.

Saving changes

Saves the changes made since the last save operation.
