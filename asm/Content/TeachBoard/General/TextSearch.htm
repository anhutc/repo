Searching for text

If you enter any string in the text box, the first entry containing this text that is found is selected. Pressing the Return key in the text box highlights the next matching entry.

  • The search function is case-sensitive.
  • You can restrict the search by selecting a particular column. In this case, the text you enter is only searched for in the selected column.
    Press the Find button to display a list of available columns, from which you can select the one you require.
