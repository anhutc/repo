Teaching a setup

All elements of a setup that need to be measured and saved are displayed.

Tip: If you teach the component shapes before the placement positions, the component shapes can be shown in the camera image when measuring the placement positions.

Elements that can be taught

Component shape

Name of the component shape.
All component shapes that can be taught are shown.

Component name

Name of the component.

Position

Position at which the component is set up.

Nozzles

List of all the nozzles that can be used to pick up a component with this component shape.

Status

Teaching is required.

Teaching not possible: There is a problem with the feeder.

Check component orientation: In this event, press Teach pickup position, check the orientation of the component and confirm that it has been checked by pressing Confirm component orientation. Switch back to the Teach setup view.

Other feeder indicators (setup)

Selected item

Teach pickup position

Creates a camera image of the defined pickup position and opens a dialog box for manually correcting the component pickup position.

If the option "Orientation check by operator" or "Check pickup orientation on component change" has been set for the component in the programming system, the orientation must be confirmed at this point.

Teach component at pickup position

Starts SIPLACE Vision to check and, if necessary, change the pickup position of the selected component on the basis of the surface structure.
This button is only enabled if Measure pickup offset before pick has been enabled for the component.

Teach component shape

Opens the Select pickup parameters dialog box. Here you can check a number of settings (nozzle and cameras used, acceleration values and travel profiles) and then start SIPLACE Vision to check and if necessary change the description of the selected component shape.

Teach tray

Starts the tray editor, which allows you to check and, if necessary, change the description for the selected tray.

This is only possible if the tray has already been activated in the setup editor.

Teach component pocket

Starts SIPLACE Vision to check and if necessary change the description of the selected component pocket.
This button is only enabled if recognition of a custom component pocket has been enabled for the component setup position.

Meaning of the colors in the representation of the setup:

Marked in red: • There is a problem with the feeder which is preventing the component shape from being taught. The status shown in the table can be used to identify the problem.
• No camera has been assigned.
• The required nozzle is not present on the machine.

Orange highlighting:

• The component shape must be taught.
• The component pocket must be taught.

Exception: 2x8 mm feeders are not highlighted in color when only the divisions report a problem, and not the entire feeder.
