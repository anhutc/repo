The measured coordinates for the fiducial are saved in the job, the next fiducial is selected, the camera image of the specified position is displayed and the fiducial position is measured.
Once the final fiducial has been saved, the system switches back to the Teach fiducials view.

Hint:

  • If the fiducial cannot be found in the camera image, its position must first be determined. To do this, press Find fiducial position.
  • If the fiducial description does not match the fiducial that has been found, it can be changed using the functions available in SIPLACE Vision. To do this, press Teach fiducial shape.
