Teaching a board

•   If no CAD data for a board is available to describe it in sufficient detail, individual elements must be measured on the machine and saved (taught). To do this, a sample board is moved into the placement section.

•   It is recommended that you have a second sample board to hand that you can use for taking measurements outside the machine. If this is not available, the sample board must be copied, as you will need a sample to compare with the board in the machine during the teaching process.

•   Positions determined during the teaching process are shown to 3 decimal places. In order to avoid any negative impact on placement accuracy, the values determined during the teaching process should not subsequently be changed by entering values at the keyboard.

•   In order to position a component precisely at the specified position on a panel, the exact position of the panel on the board and the exact dimensions of the board must both have been defined.

Preconditions for teaching a board type:

    -   The machine must have been calibrated and referenced.

    -   A corresponding board type must have been specified for teaching in the job control system before a board can be measured.

    -   Only one board can be measured at a time.

    -   The teaching process terminates any live production operations.

    -   Any boards currently in the machine are completed and moved out of the machine.

Caution: Do not perform any manual operations or calibration steps during the teaching process, as this can lead to inconsistent data.

2. Board handling

•   Place a board in the machine:
When you have described the board on the side of the machine indicated with 1, place the board in the machine as shown in the figure. You must place the board in the other side of the machine rotated by 180°.

•   Press Move board into machine.
The board is moved to the placement section and the system checks whether board position recognition can be performed. This is possible if at least two fiducials have been completely described.

3. Board position recognition

If automatic board position recognition is not possible, you must manually determine any rotational offset of the board in the machine:

To do this, select two points joined by a line parallel to either the longitudinal or lateral edge of the board.
If appropriate fiducials that share the same X or Y coordinate are available, these are displayed.

Hint: Choose two features on the board (e.g. fiducials or long conductor tracks or other lines) that share the same X or Y coordinate, i.e. that are parallel with the X or Y axis and that are as far apart as possible.

4. Teaching the fiducials

If it is not possible to measure all the specified fiducials automatically, you must determine them exactly using the camera image.

It is also necessary to measure all fiducials that have a new fiducial shape and all fiducials for which "Force teach" has been specified.

You can teach either the fiducial position or the fiducial shape.

•   Teaching the fiducial position: The fiducial model is not located precisely over the representation in the camera image.

•   Teaching the fiducial shape: The shape of the model does not correspond exactly to the representation in the camera image.

Tip: If you know neither the exact position of the fiducial nor its shape, you should first teach the shape of the fiducial and then its position.

5. Teaching the panels

If the camera image reveals that all the placement positions of a panel are offset by the same amount, you should determine and save the exact position of the panel. This corrects all the placement positions at the same time.

6. Teaching the placement positions

If only individual placement positions are not located above the representations in the camera image, these should be taught separately.

Also, all placement positions for which "Force teach" has been specified must be determined and saved.

Start by teaching the placement positions of the first panel.

7. Board handling

Move the board out of the machine:

Press Move board to output section or Move board out of machine to transport the board to the next machine.

Make sure that the downstream machines are not in production mode.

 

The job has been prepared for production as soon as no more icons are shown in orange.
