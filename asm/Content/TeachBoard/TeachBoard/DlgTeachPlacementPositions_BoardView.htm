Teaching placement positions

The aim is to check and, if necessary, correct the described placement positions in order to achieve the best possible placement accuracy. You should overlay the model of the placement position (shown in green) over the camera image in such a way that the leads are located above the centers of the soldering points.

The first placement position in the list is selected automatically.

If a placement position has already been exactly described in the job editor, it appears in the image immediately the function is called.

Then use the rotary controls or the rotary button for fine positioning.

X [mm]

Current X coordinate of the selected placement position relative to the parent panel or board.

Y [mm]

Current Y coordinate of the selected placement position relative to the parent panel or board.

Angle [°]

Current rotational offset of the placement position relative to the superordinate panel.

Edit mode

The rotary controls, rotary buttons and the buttons are enabled, allowing you to change the position and rotational offset of the placement position.

Rotary button

To perform fine adjustment of the rotational offset of a placement position, press the mark on the rotary button and rotate it in the required direction with your finger.

Rotary controls

For fine adjustment of the position of the panel.

You can roughly position the panel by pressing and moving it.

Settings: Pseudocolors / high sensitivity / crosshair

+90°

The placement position is rotated by 90° in an anticlockwise direction.

-90°

The placement position is rotated by 90° in a clockwise direction.

The camera image of the next placement position is displayed and you can move and rotate the model of this panel until it matches the camera image.

The previous placement position is selected.

Caution: The previous/next placement position is chosen according to the list in the table on the Details tab. Any sorting and selection that have been performed are taken into account.

Saving changes

Saves any changes made to the settings for the current placement position since the last save operation.
