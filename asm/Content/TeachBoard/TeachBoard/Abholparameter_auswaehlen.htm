Choosing pickup parameters

Before a component assigned to the selected component shape is picked up for teaching, you can check and if necessary change a number of parameters.

Nozzle and camera settings

Nozzle types and camera types are linked. This means that if a nozzle type is changed, the list of possible camera types may also change, and vice versa.

Nozzle type

All the types assigned to the component shape on the programming system and configured on this machine are available for selection.
The currently selected type is to be used for measuring the component shape.

Camera type

All the types assigned to the component shape on the programming system.

Presentation angle

The angle at which the component is presented in front of the camera (possible values: 0째, 90째, 180째, 270째).

Remove selected nozzle

Cancels the assignment of this nozzle type to the component shape in the programming system.
If there is only one nozzle left assigned to the component shape, it cannot be removed.

Acceleration values and travel profiles for picking up components with the selected nozzle

X axis / Y axis / Z axis, up / Z axis, down / star axis / DP axis

Specifies the maximum possible acceleration values for each of the axis movements as a percentage.

The acceleration can be reduced to prevent the component from slipping on the nozzle.

Possible values: 10%, 25%, 50%, 75%, 100%.

Raise

Travel profile for raising the Z axis with the component

Standard

The axis moves at the maximum possible speed.

Slow start

Creep distance on starting (the axis moves 1 mm upwards at a reduced speed)

Creep start

Creep distance on starting (the axis moves 1.5 - 2 mm upwards at a reduced speed)

Set accelerations and travel profiles to reliable pickup values

In order to ensure that a component can be placed reliably, all acceleration values are set to 50% and "Slow start" is entered for the travel profile for upward movement.

Caution: This has an impact on placement performance.

Start

Saves the changes to the component shape in the programming system and starts SIPLACE Vision.

Close

Rejects all changes and closes the dialog box.
