Teaching the position of a fiducial

1.  Switch to the Teach fiducials view.

2.  Select the fiducial whose position you want to teach and press Teach fiducial position.
The Teach fiducial position view opens to allow determination of the precise position of the fiducial.

3.  Move the fiducial description as exactly as possible above the representation of the fiducial in the camera image using the rotary controls.

4.  Press Apply position and continue measuring.
