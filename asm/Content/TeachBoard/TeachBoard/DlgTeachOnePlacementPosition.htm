Teaching a placement position

The aim is to overlay the model of the placement position (shown in green) over the camera image in such a way that the leads are located above the centers of the soldering points.

If a placement position has already been exactly described in the job editor, it appears in the image immediately the function is called.

If the placement position cannot be found in the camera image, switch to the Board overview tab and move the camera to the expected position. Then press once in the new camera image to obtain a rough position.

Then use the rotary controls or the rotary button.

Position X [mm]

X coordinate relative to the panel or board.

Position Y [mm]

Y coordinate relative to the panel or board.

Angle [°]

Angular offset of the placement position relative to the panel or board.

Rotary button

To perform fine adjustment of the rotational offset of a placement position, press the mark on the rotary button and rotate it in the required direction with your finger.

Settings: Pseudocolors / high sensitivity / crosshair

+90° / - 90°

Rotates the placement position by 90° in the relevant direction.

These buttons allow you to roughly align the placement position.

Image selection mode

Draw rectangle

Hint: Use these settings if the component shape is not known.

Select a point in the camera image and drag to produce a rectangle.

The placement position is assumed to be in the center of the rectangle.

Center point (default)

The component shape is known.

Select a point in the camera image, which is then interpreted as the center of the placement position.

Surrounding area

The size of the area around the placement position shown in the camera image. The smaller the value selected here, the more precisely the placement position can be determined.

Small (default)

Wherever possible, this setting should be used in order to localize the placement position as precisely as possible.

Medium

Large

Rough overview to allow the placement position to be located. Move the camera so that the placement position appears in the center of the camera image and then switch back to the more detailed view (Small).

Reset changes

Any settings made are discarded.

The camera image of the placement position is captured and you can move and rotate the model of this position until it matches the camera image.

The previous placement position is selected.

Caution: The previous/next panel is chosen according to the list in the table. Any sorting and selection that have been performed are taken into account.

Saving changes

Saves the changes to the current placement position. The next placement position in the overview is loaded.
