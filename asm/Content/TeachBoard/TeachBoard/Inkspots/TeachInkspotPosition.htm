Teaching the position of an inkspot

1.  Switch to the Teach inkspots view.

2.  Select the inkspot whose position you want to teach and press Teach inkspot position.
The Teach inkspot position view opens to allow determination of the precise position of the inkspot.

3.  Move the inkspot description as exactly as possible above the representation of the inkspot in the camera image using the rotary controls.

4.  Press Apply position and continue measuring.
