Independent placement

If products are specified as being for a specific side of the machine in the programming system, the machine can be operated in "independent placement mode". This means that the two sides of the machine can be controlled independently.

While one lane is being used for production, the other lane can be used to teach component shapes or board fiducials, correcting the pickup position, modifying setups or carrying out a range of manual operations, for instance.

The basic view shows Start and Stop buttons for each side of the machine. These allow you to start and stop production for the corresponding lane.

Prerequisite for production in independent placement mode:

•

Independent jobs have been specified for both conveyor lanes.

•

Each of the gantries on the machine may only be used for one job. This applies both to production and to measurement operations and the positioning of support pins.

Indicator lamps:

The green/white indicator lamps show the status of the corresponding side of the machine. The green indicator lamp also indicates the status of only one side of the machine:

If, for example, the green indicator lamp is lit, production is running on the corresponding side of the machine. If it is not lit, production has been halted on this side of the machine.

All the settings for green/yellow/red indicator lamps refer to the corresponding side of the machine.

For example: In default mode, both green indicator lamps are lit when the machine is in production operation. In independent placement mode, the green lamp on the relevant side of the machine is lit while the machine is in production operation.
