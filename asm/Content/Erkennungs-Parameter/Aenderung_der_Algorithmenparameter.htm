Changing the recognition parameters  

Image evaluation and position recognition in SIPLACE Vision are based on analysis of a grayscale image. An image can contain 256 shades of gray (0=black, 255=white). For each object to be measured (component, fiducial, ink spot, nozzle), the vision system needs to be provided with a description of the object and its orientation (including orientation tolerances).

 

After a job has been specified, the system waits until an object is in front of the camera. Then an image is captured and this image is evaluated by different algorithms which are specific to the object type.

 

By default, the algorithm parameters or recognition parameters are provided with values which need no modification to allow objects to be recognized. These parameters should only be changed in exceptional cases. Before you change the parameters, you should always check whether the measurement results can be improved by adjusting the illumination settings.
