Determination of the coarse position (fiducial recognition)

On the basis of the geometrical description, the number of point pairs used to determine the coarse position of the fiducial is specified automatically:

At each location in the image, the proportion of point pairs is determined where one point is inside the fiducial and one point is outside the fiducial. These point pairs are specified by the distance between points and the minimum contrast.

The coarse position is the position at which most point pairs match, i.e. the contrast between the two points of a pair exceeds the minimum threshold set.

 

If several different coarse positions result in a high quality, this means that the search area contains artifacts and a clear measurement is not possible. Recognition is aborted and an error message is displayed.
