Recognition parameters: Fiducial recognition for geometrically described fiducials (singular features) and component pockets

Component pockets:

The following parameters influence position recognition for component pockets:

Minimum contrast / subsampling / maximum number of point pairs / distance between points

 

Fiducials:

Recognition of geometrically described fiducials is performed in 2 steps:

  1. Determination of the coarse position:
    A template is derived from the geometrical description, and this is used to determine the rough position (coarse position). This template comprises point pairs (pairs of test points).

    The following parameters affect coarse position recognition:
    Minimum contrast / subsampling / maximum number of point pairs / distance between points
  2. Determination of the fine position:
    The edges of the fiducial are determined. The overall position of the fiducial is calculated from the detected edge positions.

To ensure that these steps are carried out as precisely as possible, a number of parameters can be adjusted to match the fiducial properties.
