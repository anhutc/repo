Scanning angle

Rotation of the component in front of the sensor.

Components whose length or width is greater than the field of view of the sensor (50 mm) are automatically aligned longitudinally.

 

  • With some component shapes, parts of the package are below the placement plane (e.g. connectors with centering pins. It is possible for such parts to obscure the laser, which may make it impossible to carry out the coplanarity measurement.
    If this happens, it is possible to set the scanning angle so that the component is oriented in such a way that the shadow does not fall on areas to be measured.
