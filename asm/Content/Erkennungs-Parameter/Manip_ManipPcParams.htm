Recognition parameters: Component recognition for components without leads (leadless, contour position)

Component recognition for components without leads/balls out in 5 steps:
  1. Determination of the coarse position
  2. Determination of the coarse angle
  3. Determination of the fine angle
  4. Determination of the fine position
  5. Dimension inspection (determination of the outer dimensions).

 

The following parameters can be set:

Parameter

Default

Description

General settings

Minimum quality for symmetry method

65 %

The measure of quality which specifies how good and reliable fine position recognition was using the symmetry method. This value must be exceeded for the component to be flagged as "recognized with good quality".

Minimum quality for template method

65 %

The measure of quality which specifies how good and reliable fine position recognition was using the template method. This value must be exceeded for the component to be flagged as "recognized with good quality".

Binning

Depends on camera type and component shape

The larger the value that is set here, the faster an image will be loaded and transferred, and the lower the resolution will be.

1

Each pixel is evaluated. Binning is deactivated.

2

2x2 pixels are merged to form one pixel.

4

4x4 pixels are merged to form one pixel.

Check for tilted components

Brightness threshold for tilted components

75

[75 .. 130]

Grayscale value

If the calculated brightness of the connection surfaces under steep 0° illumination is below this value, a component is identified as being tilted.

Settings for inspection of the connection lengths

Threshold for inner gradients

20

Thresholds for detecting transitions between light and dark, that can be evaluated as the end of the connection.

Connections brighter than component body

Yes

Yes: The connections are bright in comparison with the component body.

Evaluation of inner area of connections

No

Yes: The inner area of the connections is also evaluated. If these contain transitions between light and dark (marked in pink) that are above the specified threshold, the component is rejected.
