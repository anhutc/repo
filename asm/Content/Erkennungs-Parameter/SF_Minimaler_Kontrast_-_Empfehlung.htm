Minimum contrast

This value specifies the minimum contrast necessary for detecting the outer contour of the fiducial.

By default, the minimum contrast is set to 20 grayscale increments. This setting allows fiducials to be detected even if the contrast is poor. If the contrast is very poor, this value should be reduced.

  • Reduction of the value:
    If the contrast is very low and if there are no structures in the fiducial or the background, the minimum contrast can be reduced to 6 - 8 grayscale increments.
  • Increase of the value:
    If the contrast is good and if there are structures in the fiducial or in the background, the minimum contrast can be increased.
    This allows artifacts to be suppressed if these have a lower contrast with the background compared with the fiducial.
