Minimum quality on position measurement

During position recognition, a quality value is calculated for a component which is to be measured. This is derived from the quality of the features found and the correlation between the features found and those searched for, i.e. the model. This quality value must be greater that the specified minimum quality before a component is flagged as recognized.

 

Default

80 %

Recommended

60 - 90 %

 

Only reduce the minimum quality if features are not necessary for position recognition. If you do this, however, inspection no longer makes sense.
