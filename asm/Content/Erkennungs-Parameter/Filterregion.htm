Filter region

The extent to which the tip of the pin is artificially enlarged in the filter region for THT pins.

 

1

Only the filter point itself is evaluated.

2

The filter point and one neighboring point are evaluated.
... ...

 

The larger the value that is chosen here, the larger the region whose brightness is checked will be around each point.

The preset default depends on the description of the tip of the pin.

 

Hint:

  • If a pin is divided into several regions in the filter image, the filter region should be set larger than the default value.

  • If interference (the edges of the housing, reflections) merge with the pin in the filter image, the filter region should be set smaller than the default value.
