Edge detection method: Filter type

248 (default):

Several position candidates are permitted for each edge, additional scan for each edge during which the bad edges are rejected.

For eliminating candidates caused by oscillation.

249: "Outer edge"

The outermost candidate which is above the gradient threshold is taken.

 

For shields with "double edges" (caused by recesses and steep illumination).

250:

Several candidates are possible, no  additional scan is carried out.

Suitable for shields with poor edge quality - the edge contains both light and dark points.

251: Binary image

Detection of the edges with an additional scan.

For shields with oscillation and "double edges".

 

Hint: If this filter is required for a corner, it should also be used for the other corners.
