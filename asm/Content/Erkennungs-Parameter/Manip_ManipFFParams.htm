Recognition parameters: Taught fiducials

 

The following parameters can be set:

Parameters

Default

Description

Threshold for fine search

80 %

The determined quality must be at least this value for a fiducial to be classed as recognized.

If this value is not achieved, the process is iterated until a position with sufficient quality is detected or until the maximum number of iterations is reached.  

Recognition

Robust

Specifies the maximum number of iterations.

Robust

More iterations

Fast
