Detection parameters: Flux inspection

 

The following parameters can be set for flux inspection:

Parameter

Default

Description

Minimum difference from the reference image

4.0

This is the minimum brightness difference for a connection before and after it has been dipped in flux if it is to be identified as being coated.

The smaller this threshold is set, the greater the risk of uncoated connections being incorrectly identified as coated. However, the probability that a coated connection is deemed to be uncoated falls.

Maximum number of uncoated connections

0

The component cannot be placed if the maximum number specified here is exceeded.

Minimum gap between uncoated connections

0 mm

Is only used if a value >0 is set for the "Maximum number of uncoated connections".

A component can only be placed if the gaps between all the uncoated components are above this threshold.

This parameter is used to exclude the possibility of several connections in one region not being sufficiently coated. If this happens, you should check whether the component is creased or whether the flux is distributed evenly (if necessary, start a squeegee operation).

 

If a component has already been checked with an earlier version of flux inspection, the following parameters may be set.

You must reset these parameters to their default values to use the improved flux inspection. To do this, press Default algorithm values.

Parameter

Description

Analysis method

 

3

 

Brightness analysis

The mean brightness per ball on the specified surface is compared with the adjustable threshold.
If the brightness exceeds the threshold, the corresponding ball is flagged as being insufficiently coated.
Characteristic: Mean brightness in the region of interest.

1

Brightness analysis on circle

2

Brightness analysis on annulus

Homogeneity analysis

The amplitudes of the gradients (local differences in brightness) in the region of interest are calculated. A very small gradient indicates a very homogeneous surface, in other words with very little difference in brightness.

The mean value of the gradients on the specified surface forms the characteristic for each ball which is then compared with the adjustable differentiation threshold.
Characteristic: Mean change in value in the region of interest.

Differentiation threshold

 

35

(10-200)

If the determined characteristic (mean brightness or gradient) is above this threshold, a ball is deemed not to be coated. It is indicated with a red circle on the display.

Hint for selecting the analysis method:

Choose the analysis method in which the characteristics of dipped and undipped balls reveal the greatest differences and the characteristics of all undipped balls or all dipped balls revealed the least differences.

 

Hints for specifying the inner and outer circles of an annulus when using the brightness method:

-

Set the parameters on an undipped component.

-

Select the outer circle (1) in such a way that the transition in brightness from the outside to the ring lies outside of this circle.

-

Select the inner circle (2) in such a way that it encloses only the dark center, and not the transition in brightness.

 

Hints for specifying the inner and outer circles of an annulus when using the homogeneity analysis:

-

Set the parameters on an undipped component.

-

Select the outer circle (1) in such a way that the transition in brightness from the outside to the ring lies just inside this circle.

-

Select the inner circle (2) in such a way that it is smaller than the dark center.

 

Hint for setting the differentiation threshold:

First test an undipped component and note the lowest values in the graph, then test a dipped component and note the highest values in the graph. Set the threshold between these values.

If the highest value for the dipped component is greater than the lowest value for the undipped component, check the analysis method selected, the setting for the region of interest and the illumination.
