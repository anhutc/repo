Brightness limit

Corresponds to the maximum brightness in the camera image.

Default: 255.

 

Example:

Several different outlines can be detected in the camera image. To ensure that only the inner area is recognized as a fiducial, the Brightness limit parameter is set to a value < 255.

Brightness limit = 60

The brightness of the camera image is limited to the grayscale value 60. All points that are brighter are shown using this grayscale value.

This allows the inner, darker boundary of the fiducial to be identified clearly.

The detected fiducial matches the specified model.
