Detection parameters: 3D coplanarity sensor

 

The following parameters can be set:

Parameters

Value

range

Description

Common settings

Scanning angle

(automatic, 0°, 90°, 180°, 270°)

Rotation of the component in front of the sensor.
Default: Automatic, in other words the component is scanned at the same orientation at which it is measured with the IC sensor.

Coplanarity tolerance

(Default,

or 1 µm through 999 µm)

Default: The solder past thickness is taken as the upper limit, otherwise the value entered is used.

Caution: In the case of BGAs, this value should be taken from the relevant data sheet.

Repeat scan on error

(yes/no)

Yes

If a measurement with the 3D coplanarity sensor is not completed successfully, it is repeated up to 2 x (default).

No

The failed measurement is not repeated.

Exposure time

(100 µs...250 µs)

Default: 100 µs

Topographic image:

If "black spots" appear in the topographic image, this value should be increased.

 

Caution: Doubling the exposure time also doubles the measuring time.

Resolution in the scanning direction

> 10 µm/line

Reducing this value also reduces the scanning speed. The features are shown enlarged in the topographic image.

Default: The appropriate value is determined on the basis of the size of the features.

Ignore result

(yes/no)

Yes

For test purposes:
The component is placed even if the measurement fails.

No

The component is rejected if the measurement fails (default).
