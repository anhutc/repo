Binning

In image processing, binning is the process whereby neighboring pixels are merged to form one larger pixel. This reduces the quantity of data that makes up the image and results in shorter transfer times and loading times. If too high a value is chosen, however, this can have a negative impact on accuracy, because the image resolution is reduced.

 

1

The option of merging several pixels to form a single pixel is disabled.

2

2x2 pixels are merged to form a single, larger pixel.

4

4x4 pixels are merged to form a single, larger pixel.
This setting is only possible for the type 30 sensor system.

 

A predetermined value is set for binning depending on the camera type and component shape. This value can only be reduced, and not increased.

In the event of extremely high accuracy requirements, you should reduce this value to obtain the desired accuracy.
