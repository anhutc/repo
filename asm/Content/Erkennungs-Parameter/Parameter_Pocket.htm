Recognition parameters: Component pockets

As a rule, you will not have to change these parameters.


Before changing any of these parameters, first check the illumination.

 

You can change the following settings:

Parameters

Value

range

Description

Common settings

Minimum recognition quality

30 .. 100

Threshold for the ratio between edge points found and edge points searched for.

At least 70% (default) must be found.

Recommended setting: 0.5 through 0.9.

Contrast type

-1, 1

-1

Dark component pocket, light tape (default).

1

Light component pocket, dark tape.

Maximum number of point pairs

16 .. 128

Corresponds to the number of edge points searched for. The value is calculated automatically.
If you specify more points, recognition takes longer and accuracy improves.

Minimum contrast

1 .. 80

Defines the requirements on the edge points.

Subsampling

1 .. 20

The image is reduced by this factor. The value is determined automatically on the basis of the component pocket description.

Distance between points

3 .. 30

Specifies the distance between the points of a point pair. One point of each pair should lie on the background and one on the component pocket.
