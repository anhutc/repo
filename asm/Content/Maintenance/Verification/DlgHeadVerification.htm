Placement head verification

Checks the function and state of the placement heads.
Different tests can be performed depending on the type of the head:

    -   State of the vacuum circuit / placement circuit

    -   Freedom of movement of the Z axis bearing

    -   Bearing wear

    -   Component sensor inspection

    -   Z light barrier inspection

    -   Positioning accuracy of the DP and Z axes

    -   Inspection to check that nozzles are seated correctly

Selection

Choose the head for which you wish to perform the inspection.
Only those heads can be selected for which such an inspection is possible.

Prerequisites

Nozzles

Replace the current nozzle magazine with the magazine containing the required calibration nozzles and switch to the Nozzle changer view in order to specify the nozzle configuration.

Calibration tools

Make sure that the appropriate calibration tools are present in the calibration tool pockets provided.

Status

The specified requirement has already been met.

The requirement has not yet been met.

The user is expected to perform an activity.

Verification step

Different individual steps are carried out depending on the placement head selected.

Result

The inspection step has been performed successfully.

The inspection step was aborted after errors. To identify the precise cause, a service engineer should carry out the head inspection.

The inspection step is being performed.

Start

The inspection steps specified for the selected heads are executed in sequence.
This button is only available after all the required nozzles and calibration tools have been configured.

You can also interrupt and resume the inspection.

Stop

The inspection is interrupted and can be resumed later.

Continue

An inspection that has been paused is continued.

Cancel

The inspection is terminated and the state of this dialog box is restored to the state before the inspection was started.

Done

The state of this dialog box is restored to the state before the inspection was started.

Generate report

A report is created documenting the results of all the individual steps carried out. You can enter your name and a comment.

The result is entered for each inspection step that has been carried out. If any step is not completed successfully, the segments on which errors occurred are displayed.

The report is stored under the name: VerificationHead_<MachineType>_<MID>_<Date>_<Time>.pdf.

You can view the reports under LineDiagnostics in the programming system.

You must restore the original nozzle configuration after the inspection has been completed.
