Performing feeder verification

Prerequisite for performing verification: the machine must be calibrated.

1.  Prepare the feeder for which you want to perform verification

  - Insert a reel and set the feeder appropriately for the tape material used (pitch, pickup position on the feeder as necessary).
  - Set the feeder up on a free track.

2.  Switch to the view Maintenance -> Verification and press Feeder verification .

3.   Select the location and track of the feeder.

4.  Press the Start button .
The camera image of the pickup area is updated.

5.  Visually inspect the result that is shown:
The yellow crosshair indicates the position at which components would be picked up, i.e. the pickup position.
The pickup position, the notch for the pickup position that has been set and the center of the component pocket must all coincide for a feeder to be used.

6.  Press Close to close the dialog box.
