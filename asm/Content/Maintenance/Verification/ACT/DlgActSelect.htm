Maintenance > Machine verification

Machine verification is used to determine the placement accuracy of the selected placement head and, where necessary, to correct this.

When machine verification is performed, components are placed on a test board (glass mapping plate) on which there are several fiducials at each placement position. Placement accuracy can be determined by measuring the position of the placed components relative to the surrounding fiducials. To do this, images from the board camera are evaluated using a special measurement procedure.

The offset determined in this way is taken into account during placement. This ensures that the placement head places components with the specified accuracy.
