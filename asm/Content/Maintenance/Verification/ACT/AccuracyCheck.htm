Performing machine verification

  1. Stop the current placement process:

  2. Switch to the view Maintenance -> Verification and press Machine verification

  3. Select one of the available board types.

    • Switch to machine verification mode: Prepare for machine verification.

  4. Press Select machine verification plate.

    The accuracy check is started.

    After the check has been performed successfully, the results view opens automatically.

  5. Save the measurement results as a report:
    Press Save results.

  6. If you want to accept the determined offset values, press Manual correction... and in the dialog box that opens, accept the values by pressing Save all.
