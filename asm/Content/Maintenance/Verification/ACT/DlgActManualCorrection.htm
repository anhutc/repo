Manual correction

The values entered here and saved are added to the specified placement position for all components placed with a particular head on the specified conveyor in order to achieve the highest possible placement accuracy.

Conveyor

Lane

Head position - segment

Gantry on which the head is mounted - segment number (only shown for Twin Head: 1/2).

Camera position - type

Gantry number - SST (camera type)

Stored value for X / Y [µm] or phi [1/1000°]

Currently active manual correction for X / Y or phi stored in the file.

Deviation X/Y [µm] or phi [1/1000°]

The placement offset values determined during machine verification.

These values can be overwritten by the operator.

  

All columns

The entire list is displayed.

Only deviation information

Only those columns showing deviation information are displayed.

Custom selection

Opens the Define custom columns dialog box in which you can specify what information you wish to see in the list.

Reset changed values

The manually changed values are reset to the most recently saved values.

Save all

All the values that have been changed manually are saved. The entries under "Deviation" in the table are deleted and the values under "Stored" are updated.
