Coordinates to view

X

Represents the offset values in X.

Y

Represents the offset values in Y.

Angle

Represents the offset values of the rotational angle.

Segment and SST

The segment selection is only valid for the display (graphic and measurement results). If the data is accepted for the next placement, all segments are included.

Graphic (evaluation for one head)

Mean Ø

Statistical average.

Median

Median: The value that lies precisely in the middle of the distribution of the X, Y or angle values.

0° / 90° / 180° / 270°

The measured values determined for the components placed at the relevant angles.

Statistics

S-StdDev

Sigma standard deviation

Cmk

Critical machine capability value

Save results

Opens a dialog box in which you can define the location to which the current measurement evaluation is to be saved. The measurement evaluation is saved as a PDF document.

Generate report

Generates a report containing the current measurement evaluation.

You can enter your name and edit or add to the existing comment.

The report is saved under:

VerificationMachine_<MachineType>_<MID>_<Date>_<Time>.pdf

You can view the reports under LineDiagnostics in the programming system.

Repeat measurement

Switches back to the Machine verification view and starts the measurement again without assembling the board.

Automatic offset correction...

The values that have been determined are accepted.

Manual correction...

Opens the dialog box for manually correcting the placement offset.

The currently determined values are entered in this window as a deviation.

Close

Switches back to the Machine verification view. You can specify a new board type for machine verification or close machine verification mode.
