CAN bus resistance verification

When this view is opened, the CinX Box is used to measure the resistance at each CAN bus. The values determined are then displayed.

Resistance [Ω]

 

 

Resistance currently measured on the CAN bus.

A value of between 50 Ω and 90 Ω is expected.

If the measured value given in the bar is in the green range, the measured value is OK.

< 50 Ω

Make sure that the cable on the relevant CAN bus is connected correctly.

> 90 Ω

Check the DIP switch setting on the head adapter, the CinX Box and the distribution board.

Actions

Verify resistance of CAN bus

Uses the CinX Box to measure the resistance value for each CAN bus again.
