Leveling assistant - Measured values

The 3D diagram allows you to identify the point at which the surface under investigation is inclined.

Measured values

ID

Number of the measuring point.

X [mm]

X coordinate of the point.

Y [mm]

Y coordinate of the point.

Z abs. [mm]

Measured Z coordinate of the point.

Z rel. (expected) [mm]

Measured Z height with reference to the specified height of the selected surface.

Z delta (last value) [mm]

The difference from the corresponding measured value of the last evaluation.

If no measurement is yet available, a dash (-) is shown.

Each measured point is shown as a cone in the diagram and is color-coded:

The point is above the expected Z height.
The cone points downwards.

The point is at the expected Z height.

The point is below the expected Z height.
The cone points upwards.

Marks the measured Z height of the selected point.

Transport direction.

If a point is selected in the table, it is highlighted in the diagram and its name is shown below the table. The name is the name specified under Teach machine positions.

Expected Z height

The specified Z height of the selected surface.

Diagram

Zoom:

Slider for zooming in the diagram.

Display absolute Z values

Shows the Z heights as machine coordinates.

Display Z values relative to expected height

The values shown are relative to the expected height of the surface.

Display Z values relative to mean height

The values shown are relative to the calculated mean Z height. The surface is on the zero line in the scale.

Start

Starts Z height measurement for the selected surface. This procedure can be repeated at any time.

Cancel

Returns to selection of the surface.

If a discrepancy compared with the expected Z height is identified, adjust the surface using the set screws and restart Z height measurement.
