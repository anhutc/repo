Leveling assistant - Selection of leveling area

Height measurement process that is able to mechanically measure the planarity of a selected surface. A placement head and a standard nozzle are used to move to defined measuring points on the surface, and the Z axis is moved down. The point at which the nozzle makes contact is used to determine the Z height of the measuring point.

Available surfaces

All surfaces that can be measured with the wizard are displayed.

If a vacuum tooling type with built-in adapter plate is selected, the Substrate height is also shown.

Name

Name of the surface.

Selected surface

List of the points at which the Z height is to be measured.

ID

Number of the measuring point.

X [mm]

X coordinate of the measuring point.

Y [mm]

Y coordinate of the measuring point.

Selected

Name of the selected measuring point. This corresponds to the name under Teach machine positions.

Z height

The specified Z height of the selected surface.

Start

Starts Z height measurement of the selected surface and switches to the results page.
