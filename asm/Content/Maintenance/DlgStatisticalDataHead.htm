Statistical data of axes (placement head)

Shows the status of the configuration and statistical data for the axes of the selected placement head.

Moves

Total number of movement commands executed for the selected axis to date.

Moves at last maintenance

Number of movement commands executed for the relevant axis up to the time of the most recent maintenance activity.

Mileage

Distance traveled by axis in [km] or in [revolutions] for the star and DP axes.

Move errors

Positioning operations that were terminated with errors.

Reference runs

Total number of referencing calls.

Reference run errors

Reference runs terminated with errors or canceled.
