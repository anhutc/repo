Preventive maintenance

Regular maintenance has to be carried out in order to restrict the number of errors on a machine to a minimum. The functions listed here will help you in carrying out the recommended maintenance activities.

Tip: Carry out the steps listed from top to bottom (these vary depending on the machine type and head configuration).

Selection

Select one or more placement heads to be checked for correct function.

Comment

Here, you can enter any text you choose, e.g. a description of any malfunctions observed or notes on repair. This text is included in the report.

Prepare for nozzle maintenance

All the nozzles on the selected placement heads are returned to the nozzle changers for maintenance and the placement heads are moved to the service positions.

You can then replace the nozzles and inspect any removed nozzles outside the machine.

Prepare for head exchange and shutdown

The selected placement head is to be exchanged.

If any individual maintenance steps have already been carried out, you can generate a report on the results of the maintenance activities before the machine is shut down.

Heads selected for exchange return all nozzles. The relevant gantries are moved to the setup position and, on certain machines, the component tables are undocked.
A dialog box then opens. Here you must confirm that you want to shut down the entire machine by pressing Shut down.

Functions
(Different functions are available depending on the selected head type.)

Nozzle reference run

Determines the status, vacuum values, altitude values and, depending on the head type, the scan values for all segments again.

Prepare for manual maintenance

Returns all nozzles to the nozzle changer, moves all gantries to the setup position and, on certain machine types, undocks the component tables.

Caution: If glue feeders are mounted on these component tables, you must first disconnect the compressed air hoses and the safety circuit cables.

When this has been done, perform the relevant maintenance steps according to the instructions.

Check tape cutters

Opens a dialog box in which you can select the tape cutter to be checked.

When carrying out the work, follow the instructions in the Maintenance Manual carefully.

Place a sheet of DIN A3 paper (or 2 sheets of DIN A4 paper taped together) in the relevant used tape chute and stick it in place with adhesive tape.

When you then press Start in the dialog box, all selected tape cutters move to the end and back once, cutting the paper that has been inserted.

Then remove the paper. If the cut edge is not smooth, call a service engineer to clean or replace the cutter.

The time required to extend and retract the relevant cylinder is also measured. If the specified maximum values are exceeded, the test is deemed to have failed. You can display the actual and nominal values by pressing ....

Check fans
This button only appears if the machine supports fan monitoring.

Checks that the individual fans are functioning correctly.
If not all fans are working reliably, you can press the entry "Check fans", and the fans that are to be inspected are displayed under the maintenance steps.

Check filter disks

Moves the gantry over the reject position. The Z axis is moved down and a forced air pulse is triggered for each active segment in turn. Any filter disks that are too loose will be lost during this process.

Then the component camera and the component sensor are used to detect that the filter disk is present and seated correctly on each segment, and an error message is output as necessary.

The maximum and minimum component sensor measurements are shown in the table for each segment. If one of these values deviates from the reference value by more than the permitted difference, the filter disk may need to be replaced.

Pick up nozzles

Picks up the nozzles currently required for production for further inspection.
This function can only be performed if a job has already been specified.

Check nozzle seating

Moves the gantry over the reject position and then moves the Z axis down, firstly with the vacuum switched on and then with a forced air pulse. Each time, the position of the nozzle tip is determined using the component sensor. If the difference between the two measurements (Deviation) is above a defined threshold, the nozzle is marked as worn and must be replaced.

Check anti-glare shield

Moves the gantry over the reject position. The analog voltage of the Z axis light barrier is measured using different angles on each of the active segments in turn.

The difference between the maximum and minimum voltages is shown in the table for each segment. If the value determined is greater than the defined threshold, you should check the alignment of the anti-glare shield and it should be replaced if necessary.

Endurance run after maintenance

During the endurance run, the reference runs for all axes and the nozzle scanning operations are performed three times. This means that the head axes move to their zero point and the gantry axes move the placement head over the reject bin and backwards and forwards between the maximum and minimum positions five times.

Maintenance step

All the individual maintenance steps that have already been called are shown for the relevant head.

Result

The maintenance step has been performed successfully.

The maintenance step was aborted after errors.

Generate report

Creates a report documenting the results of all the individual steps carried out to date. You can enter your name and edit or add to the existing comment.

The report is saved under:

Maintenance_<MachineType>_<MID>_<Date>_<Time>.pdf

You can view the reports under LineDiagnostics in the programming system.
