Calibrating the segment offset (method)

 

Note: The "Calibrate head segments" function is used to determine the segment offset for each segment. The head cameras must already have been calibrated before you start this calibration process.

If an FC camera is present, the calibration will be carried out using this camera, otherwise the IC camera will be used.

  1. Make sure that the sleeve of the segment you wish to calibrate has a nozzle of type 517 attached to it.
  2. Put the calibration tool into the calibration tool pocket keeping the printed (matt) side facing downward.
  3. Perform a reference run of all head and gantry axes. To do this, open the "Manual operations > Entire machine" view and press the Overall reference run button.
  4. Open the "Service" view and press the Single calibration button.
  5. In the "Service > Single calibration" view, press the Calibrate head button for the required Twin Head. The dialog box Service > Single calibration > Gantry n > Head opens.
  6. Choose Segments from the drop-down menu.
  7. Under Select segments, you can select both segments or just one segment for which calibration is to be performed.

  8. Press Execute single calibration. Calibration is started for the selected segments.

  9. After calibration has been completed successfully, press Accept to save the values determined during calibration in the machine data.
