Calibrate head cameras (method)

 

Note: The machine zero point must already have been calibrated before you start this calibration process. See Calibrate machine.

  1. Ensure that a type 517 nozzle is on the sleeve of segment 1.
  2. Put the calibration tool into the calibration tool pocket keeping the printed (matt) side facing downward.
  3. Perform a reference run of all head and gantry axes. To do this, open the "Manual operations > Entire machine" view and press the Overall reference run button.
  4. Open the "Service" view and press the Single calibration button.
  5. In the "Service > Single calibration" view, press the Calibrate head button for the required Twin Head. The dialog box Service > Single calibration > Gantry n > Head opens.
  6. Press the Calibrate head cameras button. Calibration of the head cameras will now be performed. This determines the calibration factors and the focus height of the head cameras.
  7. After calibration has been completed successfully, press Accept to save the data determined during calibration in the machine data.
