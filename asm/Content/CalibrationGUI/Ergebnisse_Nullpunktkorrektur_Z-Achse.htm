Calibrate Z zero point - Results tab

After calibration of the zero point correction for the Z axis, a table with the following contents is shown in the Results tab:

Column description:

Parameter

Value

Difference saved in EEPROM of segment

Number of the segment for which the new zero point correction value has been saved in the EEPROM.

Zero correction difference

New zero point correction value

Restart

Calibration of the zero point correction value will be repeated.

Close

Closes the dialog box.
