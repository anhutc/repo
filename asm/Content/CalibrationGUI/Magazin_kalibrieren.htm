Calibrating a magazine (method)

 

Notes:

To be able to carry out the calibration functions, the selected magazine must have been set up.

Before the calibration process is carried out, the calibration data for the board camera, the component camera, the C&P20 head or CPP head, the head offset and the machine zero point must already have been calculated (see Calibrating the head).

To calibrate the height (Z position), a nozzle must be attached to bottom segment.

  1. Perform a reference run of all head and gantry axes. To do this, open the "Manual operations > Entire machine" view and press the Overall reference run button.
  2. Open the "Service" view and press the Single calibration button.
  3. In the "Service > Single calibration" view, press the Calibrate nozzle changer button for the required nozzle changer. This opens the Calibrate nozzle changers dialog box.
  4. Press the button for the required magazine or select the corresponding entry in the table.
  5. Press the Start button. Calibration of the magazine will be performed. The board camera moves over the positioning fiducials of the magazines and measures their X and Y positions.
    The current Z position is then measured with the nozzle in the magazine and thus determines the specified height for picking up and returning the nozzles in the garages for this magazine.
  6. After calibration has been completed successfully, press Save to save the data determined during calibration in the machine data.
