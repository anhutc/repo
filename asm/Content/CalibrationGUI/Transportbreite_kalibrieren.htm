Calibrating the conveyor lane width (method)

 

Note: In order to calibrate the conveyor lane width you will need a board that is 100 mm wide.

  1. Open the "Service" view.
  2. Click the Single calibration button.
  3. In the "Single calibration" view, click Lane 1 or Lane 2 to select the required conveyor lane.
  4. Click the Calibrate conveyor lane width ... button. This opens the Calibrate conveyor lane width dialog box.
  5. Put a 100 mm wide board into the selected conveyor lane and click the buttons Increase (small step) or Decrease (small step) to gradually adjust the conveyor lane width to suit the board.
  6. Click Cancel... to close the dialog box.
  7. Open the Manual operations > Subsystems view .
  8. Click Lane 1 or Lane 2 for the required lane to switch to the Manual operations > Subsystems > Lane x Functions view.
  9. Test to make sure that the board can be transported through the machine without causing problems.
  10. To do this, click the following buttons in sequence: Transport board to Input section, Placement section / Next stop position, Intermediate section, Placement position / Next stop position and Output section.
  11. Click to return to the Service > Single calibration > Lane x view.
  12. Click the Calibrate conveyor lane width ... button. This opens the Calibrate conveyor lane width dialog box.
  13. Press the OK button. The dialog box closes and the conveyor lane width is calibrated to the board width.
  14. After calibration has been completed successfully, press Save machine data for conveyor to save the data determined during calibration in the machine data.
