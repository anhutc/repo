Calibration results

This shows the detailed calibration results for the selected processing area.

Description

Name of the calibration step (e.g. Zero point offset, gantry 1).

Result

Result of the calibration step (e.g. OK , failed ).

Difference

Compared with the last calibration.

Deviation XY

Graphical representation of the segment offset in X and Y.

Expand all

The detailed view for the selected entry is shown.

Collapse all

The detailed view for the selected entry is hidden.

Close

You are returned to the results view for both processing areas.
