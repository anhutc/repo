Service > Automatic calibration - Equipment needed

This view shows the equipment needed for the calibration steps that have been selected.

Number of mapping plates

Setting for the number of mapping plates to be used for mapping. One mapping plate is selected in the example below.

 

Head 3 / Head 4 / Head 2 / Head 1

Specifies the nozzle type required and shows the status (OK or Check nozzles) for the relevant head.

Processing area 2 / Processing area 1

Displays the possible calibration tools for the corresponding processing area:

Large calibration tool (external size: approx. 14 mm)

Small calibration tool (external size: approx. 7 mm) - in this case, the small white or small blue calibration tool can be used.

Small calibration tool (external size: approx. 7 mm) - the small blue calibration tool can be used.

If several calibration tools are shown, any of them can be used.

Left / right (only for type X4I machines)

Position of the calibration tool holder: Left or right side of the machine.

Note on the location of the large calibration tool: the large calibration tool should always be placed in the section of the calibration tool holder that is not directly adjacent to the secured end of the holder.

Start

This button starts calibration of the selected components after all the necessary settings have been made. After calibration has been completed, the results for the relevant calibration steps appear on screen.

Next

Switches to the view for setting further options required to perform mapping.

Previous

Returns to the previous view.

Cancel

Cancels calibration. You are returned to the view for selecting the calibration steps.
