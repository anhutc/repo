Calibrate head segments

Select the segments to calibrate

A button with the corresponding segment number is provided for each segment. You can select the individual segments by pressing the corresponding button or you can select all the segments by pressing the following button:

Select all

All segments are selected for calibration. All the buttons for the individual segments are pressed.

Deselect all

The selection is reset. The segments can be selected again.

 

Start calibration

Calibration is started for the selected segments. The Service > Single calibration > Gantry n > Head dialog box shows the status of calibration using a progress bar.

Cancel

Closes the dialog box.
