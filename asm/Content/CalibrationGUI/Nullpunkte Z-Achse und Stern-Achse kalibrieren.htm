Calibrate Z axis and star axis zero points

Note:

The zero point correction values for the Z axis and the star axis must be stored in the axis data and in the head EEPROM. For safety reasons, the values from the axis data and the EEPROM are compared during a reference run. If the values are the same, calibration is not necessary. An appropriate message is displayed. If the values are not the same, calibration must be carried out as described below.

Preparation and determination of zero point correction for the Z and star axes:

  1. Open the "Manual operations > Subsystems" view.
  2. Press the button for the required head, e.g. CP20 head gantry 1 for head 1.
  3. Press to switch to the "Head x > Functions" view.
  4. Press the Calibrate zero point correction button.
  5. Three dialog boxes now appear in sequence. These show the preconditions that must be met for determining the values and you must confirm each with OK.
  6. Confirm the dialog box with OK if the condition is fulfilled that the head is not at operating temperature.
  7. Confirm the dialog box with OK after you have made sure that the Z axis is in the up position.
  8. Turn segment 1 to the placement position manually.
  9. Check that the star is in the correct position. Do this by pressing the return cylinder down and moving the Z axis up and down. If it moves freely, the star is in the correct position.
  10. Make sure that the Z axis is in the up position again and confirm the currently open dialog box by pressing OK.
  11. After the zero point correction values have been determined, a dialog box is opened showing the old and new zero point correction values.
  12. Press the Accept button to save the determined values for zero point correction to the head EEPROM and to close the dialog box.
