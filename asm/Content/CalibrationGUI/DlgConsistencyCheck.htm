Service > Automatic calibration > Consistency check

This check makes it possible to detect faulty board mapping and head mapping results.

Board mapping

Determines and compensates for any deformation of the gantries and machine frames above the placement section.

Head mapping

Determines and compensates for the position-dependent distance between the tip of the nozzle and the board camera.

Mapping consistency values

Outlier

Shown in yellow: One or more mapping values lie outside the expected tolerance.

In each case, the value of the largest outlier is displayed in µm.

Delinearized range

 Measure of the deformation of the gantry or the machine frame. The maximum amplitude of the model determined is displayed.

Rms

Measure of the measurement noise caused by mapping.

 The RMS (root mean square) value of the deviations from the model is displayed.

Limit

The values tolerated by the machine manufacturer.

Missing mapping data

Mapping data outside tolerance

Mapping should be repeated once, even if the machine capability analysis results are good.

Ensure that the machine is at operating temperature:

  • The axes must have been referenced.
  • The machine must have been switched on for at least 4 hours and must not have placed components in the last 2 hours.

If the values still do not lie within tolerance, a specially trained service engineer should inspect the data. To do this, the following files should be transferred:
- C:\Sirio\Work\Individual\Gantry_*Mapping*.xml
- all files in: C:\Sirio\Work\History
