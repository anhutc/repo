Calibrating the conveyor

Calibrate conveyor rails

Opens a dialog box in which you can start calibration. During calibration, the width is adjusted in individual steps and measured each time. This determines the difference between the set width and the measured width. The results are saved in the conveyor controller as correction values and taken into account later when setting and measuring the conveyor lane width.

  1. Press Start to start the machine function for calibrating the conveyor rails. The calibration start and finish times are shown after successful execution.

  2. Press Close to close the dialog box or Restart if you wish to repeat calibration.

Calibrate conveyor lane width ...

Opens the Calibrate conveyor lane width dialog box. You can start calibrating the conveyor lane width for the selected conveyor lane here.

Calibrate board sensor

Opens a dialog box in which you can start calibration. First make sure that there are no boards on the conveyor lane.
The sensors are adjusted during calibration.

Machine data for conveyor

Machine data for conveyor:

Shows the status of the conveyor machine data (changed / not changed).

Save machine data for conveyor

Opens a dialog box in which you can start the saving the conveyor machine data by pressing Yes. The data is saved to hard disk in the folder 'C:\Sirio\Work\Backup'. It is saved in different files for each conveyor side: 'ParameterLane_R.xml' for the right lane and 'ParameterLane_L.xml'' for the left lane.

Restore machine data for conveyor 

Opens a dialog box in which you can start restoring the conveyor machine data by pressing Yes. The data is read from the file 'ParameterLane_R.xml' or 'ParameterLane_L.xml' and written to the conveyor firmware. If the file is not present in the folder 'C:\Sirio\Work\Backup', an error message to this effect is issued and the machine data is not restored.
