Calibrate zero point of DP axis - Results tab

After calibration of the zero point correction for the DP axis, a table with the following contents is shown in the Results tab:

Column description:

Parameter

Value

Segment

Number of the segment that has been calibrated.

Additional zero point correction

New zero point correction value.

Restart

Calibration of the zero point correction value will be repeated.

Close

Closes the dialog box.
