Automatic calibration of smart pin support

Move pin picker to service position

The pin picker should be in this position to allow a support pin to be attached to it.

All the pin pickers required for calibration are moved to their service positions.

Refresh

The system checks whether there is a support pin on the pin pickers required for calibration.

If calibration of the pin magazines has been selected, the board camera is also used to check if the relevant garage for calibration in the affected pin magazines is empty.

Press this button as soon as you have manually changed the pin configuration.
