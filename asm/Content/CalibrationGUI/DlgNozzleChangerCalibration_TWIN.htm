Service > Calibrate nozzle changers

Single selection

Runs the calibration functions for the selected object (magazine or centering and reject station).

When you center the centering and reject station, the offsets for the X, Y and Z positions for disposing of nozzles in the reject station are determined during calibration.

See Calibrating a magazine (method) or Calibrating the reject position (method)

Select all

Runs the calibration functions for all magazines and the centering and reject station.

See Calibrating the nozzle changer (method)

Start

Starts the calibration operation for the selected objects.

A progress bar is displayed during calibration.

Magazines

Each magazine that is set up in the magazine carrier is shown as a button with the corresponding magazine number. The centering and reject station is selected in the example below.

See also Representation and labeling of magazines

Number

 

Index of the magazine carrier / index of the magazine

RS (Reject Station)
X [mm] Determined deviation from the specified position of the magazine or reject station in X.
Y [mm] Determined deviation from the specified position of the magazine or reject station in Y.
Height [mm]

Determined deviation from the specified height for picking up and returning the nozzles in the relevant magazine or for rejecting the nozzles in the reject station.

Save

Saves the values determined during calibration in the machine data.

Cancel

Rejects the currently determined values. The data most recently saved is used.
