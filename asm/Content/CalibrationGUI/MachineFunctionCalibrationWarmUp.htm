Starting calibration: Endurance run before or after calibration

After starting automatic calibration, you can choose whether an additional endurance run is to be performed before or after calibration.

The following options are available:

Calibration

Only calibration is performed.

Endurance run and calibration

An endurance run is performed immediately before calibration to bring the machine up to operating temperature.

Calibration and endurance run

After calibration has been completed successfully, a reference run is performed in order to keep the machine at operating temperature.

Endurance run and calibration and endurance run

An endurance run is performed before and after calibration.

During the endurance run, all head axes move to their zero point and the gantry axes move the placement head over the reject bin and backwards and forwards between the minimum and maximum positions five times.

Before calibration

Execute endurance run

As soon as calibration is started by pressing Start, an endurance run of the specified duration is first performed.

This ensures that calibration is performed when the machine is at operating temperature.

After successful calibration

Accept result automatically and execute endurance run

When calibration has been completed successfully, an endurance run is started immediately to ensure that the machine is kept at operating temperature.

Duration

Specify the duration of the endurance run. You can specify hours or minutes.

 

Start

Depending on what has been selected, executes calibration, endurance run + calibration, endurance run + calibration + endurance run or calibration + endurance run.

Stop

During an endurance run before calibration:
The current endurance run cycle is stopped. Calibration and any endurance run after calibration are not started. You can resume at any time by pressing Start again.

During an endurance run after calibration:
Executes the current endurance run cycle and terminates calibration.
