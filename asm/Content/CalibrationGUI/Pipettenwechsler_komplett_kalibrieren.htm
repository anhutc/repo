Calibrating the nozzle changer (method)

 

Notes:

Depending on the option set, all magazines must be set up on the selected magazine carrier or the selected magazine must be set up in order to carry out the calibration operation.

Before the calibration process is carried out, the calibration data for the board camera, the component camera, the C&P20 head or CPP head, the head offset and the machine zero point must already have been calculated (see Calibrating the head).

To calibrate the heights (Z positions), a nozzle must be attached to bottom segment. In each magazine, at least one nozzle garage must be empty. If a particular nozzle type has been specified for a magazine, a nozzle of this type must be either attached to the segment or located in the magazine.

  1. Perform a reference run of all head and gantry axes. To do this, open the "Manual operations > Entire machine" view and press the Overall reference run button.
  2. Open the "Service" view and press the Single calibration button.
  3. In the "Service > Single calibration" view, press the Calibrate nozzle changer button for the required nozzle changer. This opens the Calibrate nozzle changers dialog box.
  4. Press the Select all button.
  5. Press the Start button. Calibration of the entire nozzle changer will now be performed. The board camera moves over the positioning fiducials of the magazines and the positioning fiducials of the reject station and measures their X and Y positions. The current Z position is then measured with the nozzle for each the magazine and for the reject station and thus determines the specified height for picking up and returning the nozzles in the garages for each magazine and the specified height for dropping nozzles into the reject station.
  6. After calibration has been completed successfully, press Save to save the data determined during calibration in the machine data.
