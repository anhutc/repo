Calibration settings

 

Conveyor fiducials for PA 2 / Conveyor fiducials for PA 1

Selects the conveyor lane for calibrating the reference corner in placement area 2 or 1. In the example below, the right-hand conveyor lane of a dual conveyor has been selected.

Mapping resolution

Sets the resolution with which the data is determined during the mapping run. The number of measuring points determined during the mapping run varies, depending on the resolution set.

Number of mapping plates

Setting for the number of mapping plates to be used for mapping. One mapping plate is selected in the example below.

Mapping mode (Only shown for certain machine types.)

All mapping plates available for the machine concerned are shown. The size of the mapping plate is shown in parentheses.

Dual mode

Both conveyor lanes are prepared in such a way that an appropriate mapping plate can be inserted in each.

Default

Mapping is carried out with the lane widened and with the standard mapping plate.

Vacuum tooling (only shown if a vacuum tooling type is used where it is possible to calibrate the Z height)

Board height

Specify the height of the board for which vacuum tooling is to be used.

The objective of calibration is to adjust the height of the vacuum tooling in such a way that a board that is clamped during production is at the focal length of the board camera. This adjusts the vacuum tooling precisely for the intended board.

Multistep calibration for machines on which the placement area is larger than the mapping plate:

Mapping plate V3 right

Board mapping for the right-hand side of the lane: Determines and compensates for any deformation of the gantries and machine frames above the right-hand part of the placement section.

Mapping plate V3 left

Board mapping for the left-hand side of the lane.

Head mapping plate

Head mapping: Determines and compensates for the position-dependent distance between the tip of the nozzle and the board camera.

Default

All the selected mapping steps will be performed with only one V3 mapping plate.

Skip calibration of vacuum with closed nozzle (Twin Head)

The calibration step will not be performed. This is recommended if the nozzle changer has not yet been calibrated. Perform this calibration step after all other steps have been concluded: Switch to the view Manual operations > Twin Head > Vacuum > Specialized and press Calibrate vacuum with closed nozzle.

Places the nozzle on the conveyor edge so that the vacuum surface is closed and measures the vacuum value.
Prerequisite: Two type 518 nozzles must be located in the garages (No. 2).
