Calibration steps

Axis zero correction

Determines the values for the zero point offset for the X and Y axis.

Board camera calibration

Determines the following camera parameters: X/Y position, installation angle and scaling factor.

Board mapping

Determines and compensates for any deformation of the gantries and machine frames above the placement section.

Closed vacuum calibration

Twin Head: Determines reference values for the vacuum of the two segments with the nozzles closed.

Component camera calibration

Determines the following camera parameters: X/Y position, installation angle and scaling factor.

In the case of stationary cameras, the distance between the calibration tool and the camera is also stored as a reference value.

Component sensor calibration

Determines the angle of the individual segments to the component sensor to improve presence checking with very small components.

Feed unit calibration

Determines the X and Y positions for the rotation of the feed unit (component table, MTC or WPC).

Head mapping

Determines and compensates for the position-dependent distance between the tip of the nozzle and the board camera.

Height calibration

For Twin Head, P&P head and VHF P&P head:

Determines the installation height of the head and hence the exact height of the segments relative to the conveyor. On the Twin Head, the Z axes of the two segments are also leveled in relation to each other.

Lifting table calibration

Determines the inclination of the lifting table: Several support pins are positioned on the lifting table and the positions of the tips of the pins are measured.

Pin magazine calibration

Determines the position and inclination of the pin magazine: The exact position of the pin magazine is determined on the basis of the fiducials. A support pin is then placed in the calibration garage and the position of the tip of the support pin is measured.

Pin picker offset calibration

Determines the pin picker offset: Support pins are positioned on the lifting table and measured.

Board reference corner calibration

The individual steps for measuring a board reference corner are carried out in sequence for each of the board reference corners.

Segment offset calibration

Determines the X and Y offset in the top and bottom positions.

System identification

Performs location-dependent setting of the control parameters for the X and Y axes to ensure reliable control of axis movement. The control parameters vary depending on the location in which the machine is installed, the head type and the individual machine frames.

This procedure should be repeated if the machine has been subject to mechanical jolts or if conversion work has been carried out.

Travel range calibration

Starts a run to determine the minimum and maximum end position of and the possible travel path of the X and Y axes.

X axis parametrization

Sets the X axis control parameters to match the location in order to optimize the settings to avoid potential natural resonance.

Y axis encoder alignment

Determines the offset of the encoders on both Y axis drives (depending on the gantry type) in order to ensure correct position control of the Y axis and where necessary to allow a reference run to be performed for the main axes.

Y axis mapping

Records the gantry offset along the travel range in order to avoid stresses by deliberately setting the gantry at an oblique angle. This serves to increase the service life of bearings and motors.

Z level calibration

Determines the height of the vacuum tooling.

Vacuum tooling calibration

•

Determines the exact position of the vacuum tooling in the machine.

•

On certain machine types (e.g. TX 2 Micron, CA 2 V3), additionally measures positions and offsets of the fiducial bars (Y direction).

Prerequisites for this additional measurement:

The machine is at operating temperature:

  • The axes must have been referenced.
  • The machine must have been switched on for at least 4 hours and must not have placed components in the last 2 hours.

Machine alignment

(Only shown for certain machine types.)

Measures certain positions of the calibration plate for machine alignment using all the cameras of the gantries to be selected. Calculates the corresponding correction values and saves them in the mapping data for the gantries.

It is urgently recommended that this calibration step should be carried out after the station has been moved and if boards with less than three fiducials are to be produced.

SIPLACE BulkFeeder X

(Only appears when the setup contains a feeder of this type.)

On all tables with at least one BulkFeeder X, first the two table fiducials are measured in order to determine the position of the table. The positions of the feeder and the pickup plate are then determined. Height calibration is carried out by detecting four points on the pickup plate.
The exact position of the feeder camera is determined on the basis of the distance between the feeder camera and the board camera.
