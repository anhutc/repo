Service > Automatic calibration - Mapping

 

Next action

Follow the instructions on preparing for calibration in the green highlighted box.

Confirm manual actions

Actions have been done

First follow the instructions shown and then press this button.

Mapping plate definition

ID of next mapping plate

Enter the unique ID of the plate to be used for mapping.

This is a mapping plate or a machine verification plate.

Load mapping file

Reads the data from the mapping file that was selected by entering its ID.

If the correct ID was not entered or if mapping is being carried out for the first time, a file selection dialog box is opened to select a mapping file that corresponds to the mapping plate.

Mapping plate handling - Move in

Adjust conveyor lane width for mapping

Sets the conveyor lane width to that of the mapping plate or machine verification plate and adjusts the positions of the fixed conveyor rails accordingly.
Conveyor lanes that are not used for mapping are deactivated.

If there is already a mapping plate or machine verification plate in the machine, the button is disabled.

Move plate into machine

Moves the mapping plate or the machine verification plate into the machine.

Confirm that the plate is already in the machine

(Only appears if a plate is already in the machine and the mapping width has been set when this view is called.)

Press this button if a valid mapping plate or machine verification plate is in the machine.

The plate is then moved to the required position and clamped.

Mapping plate handling - Move out

Move plate out of machine

Moves the mapping plate or machine verification plate out of the machine to allow the conveyor to be reset to the standard width.

Restore conveyor configuration

Resets the conveyor lane width and the positions off the fixed conveyor rails to the most recently valid values prior to mapping.

This button is disabled as long as the mapping plate is still in the machine.

Previous

Switches to the previous calibration step.

Next

Switches to the next calibration step.

Start

This button starts calibration of the selected components after all the necessary settings have been made. After calibration has been completed, the Results view is displayed in which status of each calibration step is shown separately for each processing area. Any failed calibration steps are highlighted in red.

Cancel

Stops calibration prematurely. You are returned to the view for selecting the calibration steps.
