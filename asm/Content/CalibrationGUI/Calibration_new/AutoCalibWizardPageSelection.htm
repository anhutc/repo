Service > Automatic calibration - selecting the calibration steps

Select the calibration steps to be performed the next time calibration is started.

Select calibration steps

Combined steps

Main axes calibration: All calibration steps related to the gantry axes.

Calibration of fiducials, cameras and heads: Move to all fiducials, calibrate head, component camera(s) and board camera.

Board and head mapping: Mapping of the board and head.

Calibrate pin picker.

Single steps only

The calibration steps that can be performed individually and that are not included in the combined calibration steps and have to be performed additionally.

All steps

All calibration steps that can be performed individually and are relevant to the machine.

Select only steps that have to be done

For the current selection selects all calibration steps indicated with and which have to be performed. At the same time, a corresponding selection for feeder units, gantries, etc. along with various settings is displayed.

Those modules for which calibration has to be performed are highlighted in red. If radio buttons are shown for selection, calibration must be performed several times (e.g. once for conveyor lane 1 and then for conveyor lane 2).

Display all steps

All calibration steps are shown in the list, irrespective of their status.

Display only steps that have to be done

Only the single calibration steps indicated by are shown.

The status of the parent level depends on the status of the single steps below that level. If at least one calibration step is indicated by , the parent level will also have this status.

Tip:

First perform calibration with the selected combined calibration steps and then with the single calibration steps.

Selection

Select all

Selects all the positions at which at least one of the selected calibration steps will be performed.

Exception: conveyor lanes for mapping. Mapping can only be selected for one lane at a time and must be repeated as necessary.

Select highlighted

Selects all the positions necessary for the modules marked in red.

Clear selection

Deselects the positions already selected.

Next

Switches to the next calibration step.

Cancel

Stops calibration prematurely. You are returned to the view for selecting the calibration steps.
