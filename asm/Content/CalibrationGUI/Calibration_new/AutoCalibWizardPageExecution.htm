Continue

Continues calibration that has been paused, starting with the next calibration step.

Save

Saves the results of the calibration process for the selected components.

Retry failed calibration

All failed calibration steps are repeated.

Previous

Switches to the previous calibration step.

Start

This button starts calibration of the selected components after all the necessary settings have been made. After calibration has been completed, the Results view is displayed in which status of each calibration step is shown separately for each processing area. Any failed calibration steps are highlighted in red.

Firstly, the machine function 'Endurance run to warm-up machine' is opened: You can select whether an endurance run is to be started before calibration or after calibration has been completed successfully.

Cancel

Finishes calibration and switches back to selection of the calibration steps.
