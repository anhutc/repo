Calibration results

The status of the relevant calibration step is shown separately for each processing area in the results table. Any failed calibration steps are highlighted in red.

Description

Name of the calibration step (e.g. Zero point offset, gantry 1).

Result

Result of the calibration step (e.g. OK , failed ).

Difference

Compared with the last calibration.

Deviation XY

Graphical representation of the segment offset in X and Y.

Expand all

The detailed results are shown for all entries.

Collapse all

The detailed results are hidden for all entries.

Close

Closes the dialog box.
