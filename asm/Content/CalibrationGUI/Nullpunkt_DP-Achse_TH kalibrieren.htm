Calibrate DP zero point (method)

Note:

The zero point correction value for the DP axis must be stored in the axis data and in the segment EEPROM. For reasons of safety, the value from the axis data is compared with the value from the EEPROM when the station software is booted. If the values are different, a message to this effect appears. Calibration must always be performed once as described below when the machine is commissioned.

Preparation and determination of zero point correction for the DP axis:

  1. Open the "Manual operations > Subsystems" view.
  2. Press the button for the required head, e.g. Twin Head, gantry 3 for head 3.
  3. Press to switch to the "Twin Head > Functions" view.
  4. Press the Calibrate DP zero point correction button. A dialog box opens showing the prerequisite for determining the zero point correction value.
  5. Press OK if the calibration nozzle for the DP axis is fitted to the segment to be calibrated.
  6. The dialog box for starting calibration opens. If you want to cnage the selected segment, choose the entry for the current segment in the table and press Edit....
  7. In the dialog box that then appears, select the segment you require and confirm your selection with Accept.
  8. Press the Start button.
  9. When the zero point correction value has been successfully determined, the new zero point correction value for is stored automatically and shown in the Results tab.
  10. Press Close to close the dialog box.
