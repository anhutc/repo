Calibrating the head camera (method)

 

Note: The machine zero point must already have been calibrated before you start this calibration process. See Calibrate machine.

  1. Make sure that nozzles of the correct type are fitted to all segments of the head: Type 1235 on the C&P20 head, type 2020 on the CPP head.
  2. Put the calibration tool into the calibration tool pocket keeping the printed (matt) side facing downward.
  3. Perform a reference run of all head and gantry axes. To do this, open the "Manual operations > Entire machine" view and press the Overall reference run button.
  4. Open the "Service" view and press the Single calibration button.
  5. In the "Service > Single calibration" view , press the Calibrate head button for the required head. The dialog box Service > Single calibration > Gantry n > Head opens.
  6. Choose Head cameras from the drop-down menu and press Execute single calibration. Calibration of the head camera will now be performed. Values are determined for the camera coefficient and the camera angle.
  7. After calibration has been completed successfully, press Accept to save the values determined during calibration in the machine data.
