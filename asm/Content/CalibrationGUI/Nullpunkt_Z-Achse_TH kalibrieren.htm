Calibrate Z zero point (method)

Note:

The zero point correction value for the Z axis must be stored in the axis data and in the segment EEPROM. For reasons of safety, the value from the axis data is compared with the value from the EEPROM when the station software is booted. If the values are different, a message to this effect appears. Calibration must always be performed once as described below when the machine is commissioned.

Preparation and determination of zero point correction for the Z axis:

  1. Open the "Manual operations > Subsystems" view.
  2. Press the button for the required head, e.g. Twin Head, cantry 3 for head 3.
  3. Press to switch to the "Twin Head > Functions" view.
  4. Press the Calibrate Z zero point correction button. Calibration starts.
  5. When the zero point correction value has been successfully determined, the new zero point correction value for is stored automatically and shown in the Results tab.
  6. Press Close to close the dialog box.
