38523 Cartridge is not connected correctly, risk of crash

The cartridge is not engaged properly and can therefore cause a crash with the placement head.

Check whether the cartridge has encountered an obstacle and attempt to push the cartridge home.

If problem still exists:

Switch to the Setup view, select the BulkFeeder X and press Settings. Then press Remove cartridge, remove the cartridge and remove any obstacles between the cartridge and the base unit. Insert the cartridge again and make sure it engages properly.
