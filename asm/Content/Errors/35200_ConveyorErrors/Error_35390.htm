35390 Board has not arrived in input section of the next machine

The board has left the machine, but has not arrived in the next machine.

Press the Start button of the next machine.

 

Service engineer:

Check the interface cable using the inputs/outputs.

Switch to the view Manual operations > Conveyor inputs/outputs and press Interface.

 

Check that the interface types (Siemens/SMEMA) of both machines match and that they match the connected cable: Switch to the view Service > Conveyor configuration > Teach fixed conveyor rail and check the entries under Machine configuration.
