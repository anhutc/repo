35314 Width adjustment: Limit switch for maximum width has been reached

Fixed conveyor rail set incorrectly.

Service engineer: Correct the positions of the fixed rails:

Start the Service > Conveyor > Options and configuration view and call the Teach left conveyor rail or Teach right conveyor rail option and save the values: Store reference position of fixed conveyor rail.

Calibration of the conveyor lane width is incorrect.

Recalibrate the conveyor lane width: Start the Service > Conveyor > Calibrate conveyor and functions view and call the Calibrate conveyor function.

 

Calibration of the conveyor rails is incorrect.

Recalibrate the conveyor rails: Start the Service > Conveyor > Calibrate conveyor and functions view and call the Calibrate conveyor rails function.
