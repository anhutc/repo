35283 Stopper for long board is not present

In the programming system an attempt has been made to configure an additional stop unit for placing long boards. The machine, however, does not have an additional stop unit installed, or the stop unit is not connected to the conveyor controller.

Install the additional stop unit on the relevant conveyor and connect it correctly.

There is no stopper present in the specified section.

Switch to the view Service > Conveyor configuration > Lane x > Teach fixed conveyor rail

:

  • Press Change configuration

  • Choose the entry Optional stopper

  • Press Edit.

  • Choose the entry -

  • Press Accept

  • Start configuration by pressing Start.
