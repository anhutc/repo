35309 Board is not clamped correctly

The lifting table could not travel upwards sufficiently to clamp the board correctly.

There are components on the lifting table. The board cannot be clamped uniformly.

Remove the obstructions and restart.

The board high is not specified correctly in SIPLACE Pro.

Compare the settings in SIPLACE Pro with the measured height of the board and changed the placement program if necessary.

The tolerance for the board high is not specified correctly.

Switch to the view Service > conveyor configuration > Lane n > Parameters and set the parameter Tolerance of board height.
