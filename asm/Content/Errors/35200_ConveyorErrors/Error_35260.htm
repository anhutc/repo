35260 Cannot unblock input when option 'Block input section and deactivate related conveyor lane' is active

A problem during the reference run caused the input section of the relevant lane to be locked and the lane to be deactivated.

The input section can only be unlocked after the problem on the lane has been rectified and the option reset:

Switch to the view Machine options - Fault detection and uncheck the box Block input section and deactivate related conveyor lane. Then perform the overall reference run.
