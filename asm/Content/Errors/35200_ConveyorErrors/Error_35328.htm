35328 Function cannot be executed if option is installed

The complete travel range of the conveyor rails is required in order to execute the functions "Calibrate conveyor rails" and "Adjust conveyor lane width for mapping".

If one of the following options is installed, however, the conveyor width adjustment range is restricted:

- Stopper for long board

- Ceramic substrate centering

- Long board support

- Vacuum tooling

Uninstall the relevant option to allow the function to be executed.
