35290 Vacuum tooling: Vacuum value is not achieved

After the lifting table has been moved up, the system checks whether the vacuum is switched on.

There is no board on the vacuum tooling or the board is not positioned completely on the vacuum tooling.

Check the specified stop positions in the programming system.

No vacuum.

Check the vacuum supply.

The vacuum sensor or the vacuum valve is faulty or incorrectly adjusted.

Check the settings for the vacuum sensor and replace the vacuum sensor or vacuum valve as necessary.
