35329 Board cannot be moved into next machine

The downstream machine is not in production mode.

Press the Start button of the downstream machine.

The input section is locked.

Unlock the input section of the downstream machine: Switch to the Production view, press , choose Unblock and press OK.

The input section is occupied.

Repeat the action as soon as the input section is free again.

Teaching is active. Move board out of machine has been pressed on the current machine.

Move the board out of the machine again and immediately press Drive board into machine on the downstream machine.
