35248 Advanced PCB interfaces of consecutive machines do not match

The interfaces of this machine and the specified machine (either the upstream machine or the downstream machine) do not match.

Make sure that the same embedded software version is installed on both machines:

On both machines, switch to the view Service > Embedded software > Update subsystem , choose Conveyor / Conveyor lane / Conveyor I/O and perform an update on the machine on which the older embedded software version is installed.
