35261 Actual position of fixed conveyor rail and configured position do not match

During the reference run, the system identified that the position of the fixed rail is outside the tolerance for the specified position.

Switch to the view Service > Conveyor configuration > Lane x > Conveyor fixed rail options and compare the value entered for the position of the fixed rail with the actual position of the rail.

  • If the conveyor rail is at the position specified by the programming system, change the configuration settings appropriately.

  • If the conveyor rail is not at the position specified by the programming system, adjust the rail accordingly.
