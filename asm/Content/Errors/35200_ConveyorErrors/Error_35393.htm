35393 Error moving lifting table motor

The lifting table does not move freely or is overloaded.

Check the motor current and the mechanical components of the lifting table.

 

The lifting table does not move:

The motor is not connected or the connection cables are faulty.

Check that the motor is connected correctly and that the motor cables are in working order.

Faulty motor.

Service engineer: Replace the motor.

Faulty conveyor controller

Service engineer: Replace the conveyor controller.

 

The lifting table judders on startup:

The motor cable of the measuring system is faulty.

Check the corresponding cable and replace it if necessary. 

 

If the error occurs after mapping has been performed with the lifting table distance rail fitted:

Move the lifting table to the bottom position:

Switch to the view Manual operations > Subsystems > Lane x > Functions and uncheck Actuate clamping.

Then remove the distance rail.

 

 

Note:

The motor functions can be checked using the following conveyor outputs:

View Conveyor - inputs / outputs -> Lane: Motor lifting table, move up and Motor lifting table, move down.
