37854 Login request denied

An attempt was made to log in from a different system. The specified user could not be logged in.

An operator is already logged in on the machine:

Log in again after this operator has been logged out.

It has been specified that only local logins are permitted on this machine:

Switch the user login mode to central authentication.
Switch to the view Settings > User and in the drop-down menu, choose User login mode: Use central authentication.
