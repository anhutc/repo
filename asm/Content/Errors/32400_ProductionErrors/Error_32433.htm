32433 Pin ID (nominal/actual position)

The actual and nominal positions of the specified support pin differ.

The board was clamped in such a way that a support pin could damage components on the underside of the board.

Isolate the affected region of the underside of the board using the ID of the support pin in SIPLACE Pro. Visually inspect the components for damage and cancel the board if necessary.

The tolerance for support pins specified in SIPLACE Pro is smaller than the tolerance of the board stop position.

Check whether the tolerance values specified in SIPLACE Pro are unnecessarily small.

The board reference corner was not measured correctly.

Teach the board reference corner again.
