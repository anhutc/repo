32421 Component accuracy check failed

If component fiducials have been described for a component, the exact position of the component is determined after placement by measuring these fiducials and comparing the results with the expected position.

At least one value at the placement position shown exceeds the maximum permitted deviation from the specified position: exact position in X or Y or placement angle.

Use the values shown in the dialog box for error message 40087 to decide whether the panel or board is to be canceled or placement should continue.
