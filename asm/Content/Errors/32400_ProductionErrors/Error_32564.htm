32564 Machine has stopped: Cannot set configuration of conveyor

An error occurred when changing the conveyor configuration (e.g. flexible or fixed configuration of the conveyor rails).

Check the specifications in SIPLACE Pro: Position of fixed rail, side of fixed rail, conveyor lane width.

The previous attempt to configure the conveyor was not completed successfully.

Switch to the view Service > Conveyor > Options and configuration and set the conveyor by hand.
