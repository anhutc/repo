32411 The operator has postponed placement of component

The "Options for selected positions" dialog box allows placement positions to be posponed for subsequent placement. If placement was halted due to a postponed position, this is displayed accordingly.

To place the postponed position:

  1. Switch to the board view .

  2. Select the relevant board.

  3. Select the positions.

  4. Click the Options button.

  5. Choose the Place option and then click OK.

  6. Then click the Start button to continue placement.
