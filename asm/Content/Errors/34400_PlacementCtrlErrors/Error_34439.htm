34439 Component present before pick up

A component is still on the nozzle before pickup or the nozzle is dirty.

The cleaning pulse is triggered automatically.

The drilled hole in the sleeve is dirty.

Clean the sleeve or insert a new sleeve.

The valve is dirty.

Remove the valve plunger and clean the valve.

If the error recurs:

Perform a nozzle reference run.

If problem still exists:

Replace the nozzle.
