34499 Component sensor covered before pick

Ensure that the gap between the light source and the sensor is not obstructed by tape foil, the tape or a component.

If a beam that has not been covered can be seen, the component sensor is too dirty.

Clean the component sensor.

If it is no longer possible to clean the sensor:

Replace the component sensor.

 

Note:

The voltage value of the component sensor can be checked in the view C&P20 or CPP head > Z axis .

The lower the measured voltage, the greater the degree of soiling.
