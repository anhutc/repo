30401 Encoder counting error

If the error occurs on one segment only:

Sleeve does not move freely or incremental disk soiled along its circumference.


Reference run:

Index segment with error number to the removal position, manually check for free movement and replace, if necessary. Clean the circumference of the incremental disk if it is dirty.

Placement operation:

Reject all components. Switch to the view Manual operations > Head functions and press Reject components. Index segment with error number to the removal position, manually check for free movement and replace, if necessary. Clean the circumference of the incremental disk if it is dirty.

 

If the error occurs on several segments:

Encoder soiled.

Clean the encoder and replace it if necessary.

O-ring on the DP station is defective or dirty.

Clean the O-ring and replace it if necessary.

Scanning unit of turning station faulty.

Line engineer: Replace the scanning unit.
