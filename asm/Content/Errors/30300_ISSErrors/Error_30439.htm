30439 Star axis needs to be realigned on driveshaft

Motor alignment of the star axis could not be performed correctly because the star is not mounted exactly at the magnetic resting position.

Service engineer (SIPLACE): Mount the star on the drive shaft again and determined the zero point correction using the zero point gauge. Switch to the view C&P12 head > Star axis > Zero correction data, enter the current zero point correction value under Head EEPROM and apply this value by pressing the button. Then press Write head and machine data.
