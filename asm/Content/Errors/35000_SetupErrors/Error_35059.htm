35059 Component is not available

Alternative components have been defined at the specified pickup location. The specified component is, however, currently not set up.

Check which deactivated feeder module is set up with the same component and take this module into operation again:

  • Switch to the view Setup > X-Table , Functions > Locations > WPC x or Functions > Locations > MTC x and sort the list by the component name by clicking the Component column header.

  • Rectifying the error on the feeder module on which the specified component is set up.

  • Take the feeder module back into operation again.

If this is not possible:

Set the specified component up at the specified pickup location.
