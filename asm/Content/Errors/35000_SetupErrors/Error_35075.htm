35075 Cracked DIE detected

Cracked dies were detected in the feeder before pickup. These are not allowed to be placed.

The feeder will be deactivated when the maximum number of cracked dies per inspection interval as specified in the programming system is reached.

Check the components that have been fed and put the feeder into operation again if appropriate:

Switch to the view Setup > Deactivated tracks, choose the appropriate entry and press Selected track > Activate.
