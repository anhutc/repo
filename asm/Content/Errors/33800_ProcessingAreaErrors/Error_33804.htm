33804 The measured value of minimal gantry distance is too large

The minimum gantry offset measured during calibration deviates too much from the design data. The value is greater than the permitted range.

Service engineer: Check the buffers between the gantries and replace them if necessary.

Then measure the travel range again: Switch to the view Service -> Automatic calibration , select the calibration step Travel range and start calibration by pressing Next, then select the gantries and press Start.
