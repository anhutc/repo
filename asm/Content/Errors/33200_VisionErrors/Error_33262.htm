33262 GigE interface card not present in PC or driver incorrectly installed (251)

Note: The activities described in the following instructions may only be performed by service engineers:

1

 

Switch to operating system level and go to Control Panel -> Network and Internet -> Network Connections. Check that the necessary GigEVisionGx_192.168.20x.1 connections are displayed and that a connection has been established.

x: The relevant gantry number.

2

Log in as Administrator and check the network parameters for all GigEVision connections:

Double-click a network connection and press "Properties". Ensure that the following boxes are checked: "Baumer Filter Driver" and "Internet Protocol Version 4 (TCP/IPv4)".

Double-click the entry "Internet Protocol Version 4" and check that the IP address and Subnet mask match the values below:

3

Check the network settings for each of the connections:

Double-click a network connection and press "Properties".

Press "Configure..."

Open the "Advanced" tab.

Ensure that a value of at least 9014 bytes has been selected for the entry Settings -> Jumbo Packet.

Ensure that a value of at least 2048 has been specified for the entry Settings -> Performance Options -> Receive Buffers.

 

4

If necessary, re-install the station software.

For more detailed information:

Switch to the view Maintenance -> Verification and press Camera connections .

Then choose Check all camera connections from the Actions drop-down menu to identify the affected connection.
