33316 Installation of the driver for the 3D interface card incomplete (246)

  • Switch to operating system level.

  • Use Remote Desktop Connection to connect to the 3D coplanarity computer vr2host as Administrator.

  • Open the Device Manager with Control Panel->System->Hardware->Device Manager and check whether the driver for the 3D interface adapter has been found (entry "IMAQ PCI-1428" under National Instruments IMAQ Interfaces).

  • If the entry "National Instruments IMAQ Interface" is not in the list, proceed as follows:

  1. Right-click on "VR_2" and choose "Scan for hardware changes"

  2. Check whether the entry "Multimedia Video Controller" can be found under "other devices". If this entry is not present, the 3D interface adapter is faulty.

  3. Select the entry and select "Update Driver..." by right-clicking.

  4. Follow the instructions in the Hardware Update Wizard. Respond to the query "Can Windows connect to Windows Update to search for software?" with "No, not this time".

  5. Respond to the next question with "Install the software automatically".

  6. After the driver "IMAQ PCI-1428" has been installed, exit the Device Manager.

  7. Close the connection to the 3D coplanarity computer.

  8. Switch back to the machine GUI and shut the machine down.
