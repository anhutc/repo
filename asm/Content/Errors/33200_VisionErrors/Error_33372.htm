33372 3D measurement: Not all connections were found (881)

The sensor could not locate all the connections. The component may be damaged.

Check the component description (SIPLACE Vision).

  • In the "Topographic image", check whether the component is in the measuring range of the 3D sensor. Select the pseudo-color image to do this. The connections must appear as green through red. If this is not the case, check the component height in the programming system.

  • In the "Topographic image" again, check whether the contact surfaces of the connections have relatively large dark spots. This is an indication that the surface of the connections is too reflective. Increase the exposure time (algorithm parameter). A component is on the 3D sensor. Check the sensor.
