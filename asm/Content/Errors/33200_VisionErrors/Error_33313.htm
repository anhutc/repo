33313 Camera interface card not detected or driver not installed (241)

The camera interface card was not detected by the SIPLACE Vision PC or the driver for this card is not installed correctly.

Single computer configuration:

Service engineer: Check whether the driver for the camera interface card has been installed correctly in the Device Manager (Control Panel --> System --> Hardware --> Device Manager). An entry with the name "Baumer PCI-A14 or "Baumer PCI-A24" must be present for every card under the entry "Baumer Optronic Interface Cards"

The driver must be reinstalled if none of these entries are present.

Service engineer: Under Properties --> Drivers tab, click "Update Driver". Choose "Select from list" and navigate to the following folder on the installation CD: SIPLACE\driverinstallation\hotlink_pcia14\ boifcxdrv.inf.

If the system calls for the file boifcxdrv.sys, it is located in the same directory.

If you also cannot find the entry "Coprocessor":

Service engineer: Replace the camera interface card.

Configuration with separate SIPLACE Vision computer:

Service engineer: Use Remote Desktop Connection to connect to the Vision computer 192.168.255.252.

Check whether the driver for the camera interface card has been installed correctly in the Device Manager (Control Panel --> System --> Hardware --> Device Manager). An entry with the name "Baumer PCI-A14 or "Baumer PCI-A24" must be present for every card under the entry "Baumer Optronic Interface Cards"

The driver must be reinstalled if none of these entries are present.

Service engineer: Under Properties --> Drivers tab, click "Update Driver". Choose "Select from list" and navigate to the following folder on the installation CD: SIPLACE\driverinstallation\hotlink_pcia14\ boifcxdrv.inf.

If the system calls for the file boifcxdrv.sys, it is located in the same directory.

If you also cannot find the entry "Coprocessor":

Service engineer: Replace the camera interface card.

A single-computer configuration was specified during installation. In fact, however, there is a separate SIPLACE Vision computer. The software is trying to locate the interface card on the wrong computer.

Correct the faulty configuration manually.

Open the file VisionCommunication.xml in the folder C:\Sirio\Type using XML Notepad.

Navigate to the top entry VisionComputer (with the ID 1).

Then continue navigating to the entry NetworkSetting \ VisionIpAdr.

If the IP address 127.0.0.1 is entered there, replace it by 192.168.255.252.

Save this change.

 

Then open the file VisionCommunication.Indiv.xml in the folder C:\Sirio\Work\Individual using XML Notepad.

Navigate to the top entry VisionComputer with the ID 1.

Then continue navigating to the entry NetworkSetting\VisionIpAdr.

If the IP address 127.0.0.1 is entered there, replace it by 192.168.255.252.

Save this change.
