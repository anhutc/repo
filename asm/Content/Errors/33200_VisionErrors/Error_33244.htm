33244 Pattern matching: Measurement takes too long (776)

There are too many places at which the model that has been taught needs to be compared to the image. This can lead to an excessive evaluation time.

Unsuitable settings have been made for the component.
E.g.

  • The presentation tolerances (X, Y or angle) that have been set are too large.
    Even the default angular tolerance of 30° can be too large for certain components.

  • The value for "Pyramid level: Start" is too low.

  • The region selected for teaching the pattern was too small.

Adjust the relevant settings.
