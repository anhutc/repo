33278 No group defined for X orientation (906)

No connection group / feature for determining the position of the component in X has been activated. The position of the component cannot be identified.

In the component description in SIPLACE Vision, check the settings for the individual connection groups / features under Evaluation of position. In order to allow the position of the component to be identified, make sure that at least one connection group or feature has been activated for determining each of the following: the X position, the Y position and the angle.
