30911 Component no longer present after measuring

There is no component on the nozzle after measurement.

Check whether the component is on the MeasuringFeeder X, and remove it.

Faulty membrane or nozzle suction hole.

Replace the nozzle.

The component was thrown from the nozzle as a result of excessive acceleration.

Visual inspection: Remove the component from the machine by hand if necessary and check the specified handling data in SIPLACE Pro (acceleration).
