33587 Cannot use segment for calibration

The specified segment is disabled.

Switch to the view Manual operations > Head > Segments   Status overview (e.g. for the CPP head) of the relevant placement head, select the segment and press Enable.

The segment has not yet been referenced.

Switch to the view Manual operations > Head > Segments   Status overview (e.g. for the CPP head) of the relevant placement head, select the segment and press Nozzle reference run.

A component is present on the nozzle.

Make sure that the nozzle is empty.
