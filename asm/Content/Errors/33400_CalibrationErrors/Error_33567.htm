33567 Glue Feeder cannot be calibrated: No suitable, empty nozzle attached to an activated segment

Measuring the height of the dispenser nozzle:

Only nozzles whose length has already been measured can be used for calibration. Grippers, for example, which cannot be moved to the edge of the conveyor to measure their length, cannot be used.

Make sure that at least one suitable nozzle is available on a segment that has not been deactivated.

Determining the position of the dispenser nozzle:

It is no longer possible to detect the dispenser nozzle because it is too dirty.

Clean the dispenser nozzle.

The position at which the glue feeder has been installed does not correspond to the expected position.

Make sure that the glue feeder is connected correctly.
