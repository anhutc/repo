33542 Calibration plate has already moved, measurement cannot continue

For the current mapping step, it is necessary for all the relevant fiducials to be able to be measured before the position of the mapping plate is changed. The calibration plate must therefore not be transported during measurement.

  • Press Retry to perform as many calibration steps as possible.

  • Press Save to save the results.

  • Press Close.

  • From the view Service > Automatic calibration , select the failed steps again and start calibration by pressing Next.
