38089 Board aborted: Only one panel allowed to be placed, but more panels are detected

Variant production:

Different panels have been defined for the current recipe, but only one panel (one variant) is permitted to be assembled.

In this error scenario, multiple panels that are to be assembled are detected, but only one of these is permitted to be assembled.

•

 

The recipe contains multiple panels (variants) that are to be assembled. Only one of these is permitted to be assembled.

In the programming system, ensure that "Place this Panel" is enabled for only one child panel in the panel that contains variants.

•

Ink spot data is transfered to the machine from an external system via whispering.

Ensure that "Place" is enabled for only one child panel in the panel that contains variants.

•

The ink spots are measured on each machine. Too few panels are omitted during this process.

Check the quality of the ink spots and the assignment of the ink spots to the panels and make sure that only one child panel is active.

•

In the programming system, a panel was incorrectly assigned the "Variant production" property.

Disable the "Variant production" property in the programming system.

•

If the ink spot data is transfered by whispering:

Check the whispering functionality: Go to Settings > Machine options General and make sure that Skip whispering actions is disabled and check that the data transfered by whispering is correct.
