35469 The tray could not be moved to the tower, the level is already occupied.

Check whether the specified level is actually occupied.

If the level is not occupied:

Perform a reference run on the feeder:

  1. Switch to the view Manual operations > Location > X-Table , select the JTF-ML2 feeder and press Details...

  2. Press Reference run.

If there is a tray in the level:

Remove the tray from the specified level by hand from the cassette.
