35457 Tray did not clear gap sensor

The gap sensor checks whether there is a tray between the tower and the conveyor of the Multiple JEDEC Tray Feeder, i.e. whether the gap between the tower and the conveyor is free. If the sensor is covered, the tower cannot be moved. The tray did not clear the gap sensor when returning to or moving out from the Multiple JEDEC Tray Feeder.

Open the protective cover and push the tray fully home in the cassette or push it fully onto the conveyor.

If problem still exists:

Ensure that the gap sensor cable is correctly connected to the driver board.
