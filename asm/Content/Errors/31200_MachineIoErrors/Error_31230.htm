31230 The motor of the frequency converter was stopped unexpectedly

The motor of the frequency converter (frequency-controlled vacuum pump control unit) has been stopped to prevent it from being damaged. It is possible that components have been lost from the placement head and are lying on the PCB.

Service engineer:

  • Check the cause of the error using the error codes on the SIRIO console and rectify the problem.

  • Shut down the machine.

  • Remove any lost components from the machine.

  • Restart the machine.

Possible error codes:

F1

Overcurrent:

  • Motor output does not match the converter output

  • Short circuit in motor cable

  • Line-to-ground fault

Make sure that the following conditions are met:

  • The maximum cable lengths have not been exceeded.

  • The motor connection cable and the motor are not shorted and have no line-to-ground fault.

  • The motor is not jammed or overloaded.

F2

Supply voltage too high.

Check the supply voltage. It must be within the limits given on the rating plate.

F3

No supply voltage

Check the supply voltage.

F4

Converter overheating:

  • Converter overloaded

  • Pulse frequency too high

  • Ambient temperature too high

  • Fan not working

  • Adjust the load or load cycle.

  • Set the pulse frequency to the default value.

  • Try to reduce the ambient temperature.

  • Make sure that the fan is running.

 

F5

  • Converter overloaded

  • Excessive load cycle

  • Motor output exceeds converter output

  • Adjust the load cycle so that it is within the defined limits

  • Make sure that the motor output matches the converter output

F6

Chip temperature exceeds critical value

  • Adjust the load or load cycle

  • Make sure that the motor output matches the converter output

F11

Motor overheating

Adjust the load or load cycle.

F12

No converter temperature signal

Check the sensor on the heat sink.

F20

DC ripple too high (one power input phase has been lost)

Check the power supply cables.

F41

Motor data ID lost (no load)

Make sure that the motor wiring is correct (star, delta).

F51

Parameters: EEPROM error
(Read or write error on accessing EEPROM)

Switch the unit off and on again.

If problem persists: Reset the parameters to factory defaults.

F52

Error reading converter data or data invalid.

Switch the converter off and on again.
