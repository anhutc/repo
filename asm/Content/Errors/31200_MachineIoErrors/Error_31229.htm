31229 A temporary problem with the frequency converter occurred. Press start button to continue.

Production is paused (1-2 minutes) after the components on the placement head have been processed.

Press the Start button after the waiting time has elapsed.

If the warning recurs:

Notify a service engineer:

  • Check the cause of the warning using the warning codes on the SIRIO console and rectify the problem.

  • Then press the Start button again to continue production.

Possible warning codes:

 

A501

Overcurrent:

  • Motor output does not match the converter output

  • Short circuit in motor cable

  • Line-to-ground fault

Make sure that the following conditions are met:

  • The maximum cable lengths have not been exceeded.

  • The motor connection cable and the motor are not shorted and have no line-to-ground fault.

  • The motor is not jammed or overloaded.

A502

Overvoltage threshold reached

Check the input voltage at the converter.

A503

  • Power supply network down

  • Supply voltage below threshold value

Check the supply voltage.

A504

Overheating of the heat sink or the chip junction.

Make sure that the following conditions are met:

  • The ambient temperature is within the specified limits.

  • The fan is running.
A505

  • Converter overloaded

  • Excessive load cycle

  • Motor output exceeds converter output

  • Adjust the load cycle so that it is within the defined limits.

  • Make sure that the motor output matches the converter output.

A506

Increase in temperature at the IGBT junction

Check whether the load steps are within the defined limits.

A507

No converter temperature signal

Check the sensor on the heat sink.

A511

Motor overheating

Adjust the load or load cycle.
