37716 Error in the 15 V power supply

Check the connection cable for damage and replace it if necessary.

If the cable is not faulty:

Check whether the corresponding red Powerfail LED on the base adapter is lit.

If the LED is not lit:

Service engineer: Replace the entire mechanical unit for Smart Pin Support.

If the LED is lit:

Disconnect the mechanical unit for Smart Pin Support.

If the LED is still lit, the problem is with the head interface, base adapter or other components connected to it.

If the LED is no longer lit, the unit for Smart Pin Support is faulty.

Service engineer: Replace the controller board.
