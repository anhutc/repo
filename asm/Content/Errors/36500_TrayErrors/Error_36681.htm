36681 Components are too high for this type of WPC

The level below the current one was occupied by high components in the previous setup. It has not been confirmed that this level has already been converted.

Fill the level below the current one as specified, select this level in the graphical representation in the view Setup > Waffle pack tray detail view and click the button Confirm exchange of tray carrier.

Then load the specified level, choose this level in the graphical representation and again click the button Confirm exchange of tray carrier.
