36657 Zero point or reference sensor of loading axis incorrectly set

The position of the reference sensor does not match the saved position:

• Reference sensor has been moved, is incorrectly adjusted or not calibrated

• Zero point of the load axis is incorrect, work has been carried out on the motor or on the belt of the load axis

• Belt has slipped on the load axis

Check the zero point for the load axis and set it again if necessary.

Switch to the Manual operations > Subsystems > WPC x > Teach axes view, select the load axis, click the Teach zero position button and start the operation by clicking Next.

 

If the zero point of the load axis is correct:

Check and save the position of the load axis reference bero:

Switch to the Manual operations > Subsystems > WPC x > Teach axes view, select the load axis, click the Reference bero button and start the operation by clicking Next.
