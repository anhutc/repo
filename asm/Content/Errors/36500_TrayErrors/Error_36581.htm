36581 Sensor error, reference sensor of feed axis

The position of the reference sensor does not match the saved position:

• Reference sensor has been moved, is incorrectly adjusted or not calibrated

• Zero point of the feed axis is incorrect, work has been carried out on the motor or on the belt of the feed axis

• Belt has slipped on the feed axis

Check the zero point for the feed axis and set it again if necessary.

 

If the zero point of the feed axis is correct:

Check and save the position of the feed axis reference sensor

(Settings: Reference bero button).
