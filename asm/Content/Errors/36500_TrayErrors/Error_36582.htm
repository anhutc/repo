36582 Height checking for normal height components triggered at the feed area

Components are not resting correctly in the waffle pack tray carrier. The components are too high. The crash light barrier for monitoring the height of normal height components has triggered when moving a tray in or out. The system only expects low components. This means that components have jumped from the tray when it moved, are lying on edge or are lying on top of the tray.

Remove these components and check the acceleration setting in the programming system.
