31867 Component lost

There is no component on the nozzle after it has been dipped or after it has been measured with the MeasuringFeeder X.

LDU:

Check whether the component is in the dipping area of the specified dip module, and remove it, with tweezers if possible.

If the component cannot be found:

Clean the linear dipping unit and then take it into operation again.

 

MeasuringFeeder X:

Remove the component from the MeasuringFeeder X, if necessary by blowing it off with compressed air.

Then perform a self-test.
