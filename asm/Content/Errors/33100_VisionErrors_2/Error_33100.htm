33100 Cannot initialize feeder camera

A serious hardware error has occurred on the SIPLACE BulkFeeder X on the specified track.

Remove the feeder and replace it.

If problem still exists:

Switch to the view Service > Embedded software > Update subsystem , find the entry for the relevant feeder and update the embedded software as necessary.

If the component table does not support the SIPLACE BulkFeeder X:

Mount the feeder on a different table.
