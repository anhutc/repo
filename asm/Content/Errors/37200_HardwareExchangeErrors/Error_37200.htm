37200 New heads detected

Switch to the view Service > Fast hardware exchange > Head exchange and follow the instructions.

 

 

If this message is issued after downloading embedded software and if no changes to the hardware have been made:

CPP head: Switch to the view Manual operations > CPP head > Functions and first press the Write calibration data to EEPROM button, followed by Activate previous calibration data for component sensor.

C&P20 head: Switch to the view Manual operations -> C&P20 head > Functions and press Write calibration data to EEPROM.
