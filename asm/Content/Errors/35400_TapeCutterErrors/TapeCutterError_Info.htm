Additional information for tape cutter errors

Bit information

Bit 0 = 0 Left valve on

Bit 0 = 1 Left valve off

Bit 1 = 0 Right valve on

Bit 1 = 1 Right valve off

Bit 2 = 0 Left cylinder is not retracted

Bit 2 = 1 Left cylinder is retracted

Bit 3 = 0 Left cylinder is not extended

Bit 3 = 1 Left cylinder is extended

Bit 4 = 0 Right cylinder is not retracted

Bit 4 = 1 Right cylinder is retracted

Bit 5 = 0 Right cylinder is not extended

Bit 5 = 1 Right cylinder is extended

Bit 6 = 0 24 [V] for the valve is not available

Bit 6 = 1 24 [V] for the valve is available

Bit 7 = 0 Cycle initiated: old interface

Bit 7 = 1 Cycle initiated: not old interface
