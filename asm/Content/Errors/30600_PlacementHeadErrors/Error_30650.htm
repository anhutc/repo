30650 Z axis zero correction values do not match stored data

On initial startup or after a head has been replaced:

  1. Switch to the view "Head > Star axis" , select the required zero point correction entry and click Edit...

  2. The values on the label attached to the head must match the values entered in the machine data and the head EPROM.

  3. If this is not the case, transfer the values for the star axis on the label attached to the head.

  4. Click Write head and machine data to save the data.

  5. Switch to the view "Head > Z axis" , select the required zero point correction entry and click Edit...

  6. The values on the label attached to the head must match the values entered in the machine data and the head EPROM.

  7. If this is not the case: Transfer the values for the Z axis on the label attached to the head.

  8. Click Write head and machine data to save the data.

Zero point values calibrated incorrectly:

Service engineer: Calibrate the zero point values.
