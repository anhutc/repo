30791 Valve plunger in wrong position: Component not present before dipping

Valve plunger moves too freely.

Check how freely the valve plunger moves and replace if necessary.

Valve actuator not adjusted correctly.

Check the setting with an adjustment valve plunger and correct if necessary.

If the error occurs after the valve plunger has been replaced or after the valve actuator has been adjusted:

Replace the valve housing.

No component is present on the nozzle for dipping:

Inspect the table area visually and remove the component from the machine by hand if necessary.
