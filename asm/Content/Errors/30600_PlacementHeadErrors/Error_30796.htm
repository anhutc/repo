30796 Component was not placed properly because Z axis did not adhere to given creep distance

A creep distance for Z axis movement has been set for the component in SIPLACE Pro (for instance if centering pins or leads need to be passed through holes in the board.). This movement has been terminated too early because, for instance, the holes have not been hit or an obstacle is preventing further lowering of the Z axis.

Change to the view Board > Placement position list for one lane, select the corresponding placement position and press View placement position.

Use the Vision image to decide how to proceed:
- Cancel board
- Flag board for inspection
- Continue placement.
