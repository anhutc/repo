30797 Component was not placed properly (snap in detection failed); measured difference of Z height greater than defined threshold

The difference determined between the measured Z height when the component is actually placed and the Z reference height is greater than the threshold specified in the programming system (Snap-in, Z threshold).

The component appears not to have snapped into place. The board stops as specified under Settings > Machine options - Inspection , "If snap-in/pin-in-paste confirmation is needed".

Remove the board and visually check whether the component has snapped in. If this is the case, you should adjust the threshold in the programming system accordingly in order to avoid unnecessary errors.

If not, you can try to snap the component into place by hand and then set the placement status to OK.
