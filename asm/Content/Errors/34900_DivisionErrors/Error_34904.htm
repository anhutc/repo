34904 Pickup location deactivated because of frequent vision errors

Measurement of the picked-up components using the component camera has failed frequently.

At the specified pickup position, the proportion of pickup operations that return a Vision error is greater than the specified threshold value.

Note: Storage space constraints mean that it may not have been possible to include all the Vision dumps (measurement contexts) for the Vision errors used for calculation.

Error distribution

Distribution of individual error messages from SIPLACE Vision.

Segment distribution

 Distribution of all SIPLACE Vision error messages for each segment.
If a disproportionate number of errors occur for a given segment, you should check the nozzle.

Vision dumps

 List of saved Vision dumps (measurement contexts). The Vision dump for the selected Vision error is displayed.

 

First check whether specific error messages occur frequently, distributed across all segments reasonably evenly.

If the same error message occurs repeatedly,

check the Vision dumps for this error message and correct the settings for the component accordingly.

If the error messages occur frequently for one segment:

The nozzle is contaminated with solder paste:

Clean the nozzle.

The nozzle is worn:

Replace it.

 

 

Then press the Start button .

The error counter is reset.
