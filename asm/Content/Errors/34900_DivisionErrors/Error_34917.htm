34917 Pickup location deactivated after a set number of vacuum or component sensor errors

A specification is made in the programming system defining how many vacuum or component sensor errors are permitted before a division is deactivated. The defined number has been exceeded.

Check the component feed facility: Foil removal device, pickup position, ... 

 

Component sensor error:

It was possible to pick up and measure the component, but it was no longer detected by the vision system or the component sensor before placement. The feeder from which the component was picked up is deactivated after a number of errors of this type set in the programming system to prevent components from being picked up that can then not be placed.
