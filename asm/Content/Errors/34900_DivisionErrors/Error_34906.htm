34906 Pickup location deactivated after feeder recognition error

Incorrect feeder type has been set up.

Set up the feeder as specified in the setup list.

A component pocket was not recognized. The dimensions of the component pocket are incorrect or a bad choice of illumination was made.

Create a corresponding component pocket by hand on the programming system or teach the component pocket.

Travel ranges of the machine have not been set up correctly.

Notify service engineer: Check the travel ranges of the axis by calibrating the gantry axes.
