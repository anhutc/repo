34922 Feeder: Component was not transported into the specified position in the defined time

Tape is jammed, tape is twisted.

Check the tape feeder.

Tape drive faulty.

Replace the tape drive or the feeder.

Toothed wheel jammed by components.

Check the toothed wheel and remove any components.

 

Then reset the feeder status by pressing the following feeder buttons simultaneously: SET button and Forward button .

Then set the feeder pitch correctly:

Set the feeder pitch directly at the feeder

or switch to the view Setup > Table , select the relevant feeder, press Settings... and then press Synchronize with expected settings.
