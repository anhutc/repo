34943 Pickup location deactivated for component orientation check

"Orientation check by operator" has been enabled for the specified component in SIPLACE Pro.

The machine can stop if
- a new feeder with this component has been used
- a new batch has been started
- a splice has been detected
- a new filling level for trays has been entered
- a new setup has been specified

 

Switch to the view Functions > Pickup offset correction . Use the camera image to check whether the component is at the expected position in the tape or on the waffle pack tray. If necessary, correct the specified orientation by pressing the +90° / -90° buttons. Then press Confirm component orientation.
