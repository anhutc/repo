34947 Pickup location deactivated after component barcode reading errors

The permitted number of failures when reading the component barcode has been exceeded. No more components can be picked up from the specified pickup location.

Correct the position at which the barcode is to be read:

To do this, switch to the view Setup > Fiducials for feeder modules.Choose the required entry (barcode type) and press Test.

 

If the error was caused by a mechanical issue that has already been rectified:

Take the feeder into operation again:

Switch to the view Setup > Table , select the specified feeder and press Activate.
