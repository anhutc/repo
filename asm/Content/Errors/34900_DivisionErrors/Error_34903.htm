34903 Pickup location deactivated after specified number of component pickup errors

A specification is made in the programming system defining how many pickup errors are permitted before a pickup location is deactivated. The defined number has been exceeded.

Check the component feed facility: Foil removal device, pickup position, ... 

 

Pick-up error:

No component in pickup position:

Replenish components, fill up tracks.

Component faulty:

Refill components in perfect condition.

Faulty membrane or nozzle suction hole or nozzle too small:

Replace nozzle.
