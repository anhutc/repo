34860 Cannot assign selected component shape

Multiple component shapes have been described for the specified component in the programming system.

•

 

The currently selected component shape does not match the feeder that has been set up.

Switch to the view Setup > Teach component shape , select the appropriate entry from the list and choose Change component shape from the Actions drop-down list.

Choose the component shape that is being used and press Apply.

•

The system attempts to assign a suitable component shape automatically.
No component shape can be selected manually at the same time.
