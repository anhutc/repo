Absolute position of the axis after movement.

Travel profile

The speed of the axis and the force can be set depending on the movement (down for placement, down for pickup, up for placement, up for pickup or simulation).

  1. Use the up and down buttons in the Configuration view to select the parameter you wish to change.

  2. Click Edit, change the setting and then click Accept.

  3. When you have changed all the parameters you wish to, click Start. The gantry then moves to the specified position.

Start

The gantry axes are positioned according to your specifications.

Close

The dialog box is closed.
