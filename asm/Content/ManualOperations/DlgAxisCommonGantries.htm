Gantry > X and Y axis

Common functionalities

Position

Shows the current position of the X or Y axis.

Axis functions

Servo on

Switches the servo of the axis on.

Servo off

Switches the servo of the axis off.

Axis reference run

A reference run is performed with the axis.

Move Axis...

Opens the Move axis dialog box in which you can specify the target position, the acceleration, the travel profile and the force.

The axis is moved to the specified target position.
