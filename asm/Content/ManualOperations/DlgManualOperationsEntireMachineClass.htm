Move gantries to preset position

Park position

When the gantry is in the this position, it is not required for the placement process.

Service position (Not available on machine types where it is not possible to move the gantries to this position simultaneously.)

When the gantry is in the service position, it is possible to perform work on the placement head.

Setup position

The gantry is moved away from the feeder area to allow free access to the feeders.

Conveyor edge

The gantry moves to the nozzle reference position for test purposes:
Tests the sensors or the Z axis travel profiles (free movement of the Z axis with the 1N travel profile).

  1. Select the position to which you want to move the gantries.

  2. Press Start button to move the gantries.

  3. Click the Close button to close the dialog box.
