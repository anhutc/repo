Location > X-Table > JTF-ML 2 > Functions

 

Move tower into transport position

If a waffle pack tray from the tower is still available for picking up components, it is moved back in and the Multiple JEDEC Tray Feeder is prepared to allow it to be removed from the table.

Tip: Remove both cassettes if you wish to remove the tower.

Reference run

Moves all the axes of the Multiple JEDEC Tray Feeder to a defined position.

 

Transfer position

Position of the lifting axis at which the waffle pack tray can be moved or pushed onto the rail of the feed axis with no height difference.

Teach position

Starts the procedure for setting the lifting axis:
The lifting axis must be set in such a way that the bottommost waffle pack tray in the topmost cassette can be moved onto the feed axis with no difference in height.

To allow the setting to be made, both cassettes must be in the JTF-ML2 and one tray must be in the lowest level of the top cassette.

Reset

Discards the most recently determined transfer position and restores the factory setting.

Refill position

The position of the lifting axis at which the required cassette can be removed or inserted smoothly.

Teach position

Starts the procedure for setting the lifting axis for replacing the cassettes:

The lifting axis must be set in such a way that the top cassette can be removed from the tower smoothly.

Both cassettes must be in the JTF-ML2 to allow this setting to be made.

Reset

Discards the most recently determined refill position and restores the factory setting.
