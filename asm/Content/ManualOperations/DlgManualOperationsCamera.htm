Save to file...

Opens a dialog box in which you can specify the directory and file name under which you want to store the current information from the SIPLACE Vision measurement context.

Test illumination

Checks correct operation of the individual illumination planes:

Set an illumination intensity for each of the illumination planes in sequence and check the displayed image and whether the camera triggered.

(The values set here are not saved and are only used for test purposes.)

Actions

Shows all the illumination planes available for the selected camera system and the corresponding illumination values in percent that have been set.

Illumination plane

All the illumination planes available for the selected camera system:

The components are illuminated with LEDs arranged in a ring or as a flat surface at different angles to the component surface. This means that the components are illuminated from different angles.

Those LEDs arranged at a particular angle to the component surface are grouped as an illumination plane.

Illumination intensity

Illumination intensity for an illumination plane.

Edit...

Opens a dialog box in which you can enter the required value.
