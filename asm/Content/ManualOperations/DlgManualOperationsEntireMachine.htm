Entire machine functions

Overall reference run

Before the stations can start assembly, the gantries and placement heads have to perform a reference run. During the reference run all axes move to their zero points. The head height and the nozzle offsets are measured and a vacuum test is performed.

Move all boards to the output section of the machine

All boards in the machine are transported to the output section and must be removed by hand.

Move gantries to preset position...

Opens a dialog box in which you can specify the position to which the gantry is to move:

Park position, service position, setup position or conveyor edge.

Gantry functions
(Is only shown for machines on which the gantries cannot be moved to the service position simultaneously.)

Move gantry to service position

When a gantry is in the service position, it is possible to perform work on the placement head.

Operating hours

Shows the number of hours the station software has been booted since initial commissioning.

Component count (Only appears if no component counter (hardware) is fitted to the machine.)

Shows the number of components placed on this machine to date (the value is regularly saved in the EEPROM of the I/O module.)

Caution: If I/O modules are exchanged between machines, the counter values are also exchanged. This is indicated by a warning issued at the GUI.

Vacuum pump functions
(Only shown for machines on which a frequency-controlled vacuum pump is present.)

Shows the current status:

On

The vacuum pump is switched on.

Off

The vacuum pump is switched off and no vacuum is being generated (power-saving mode).

Press the appropriate radio button to change the status.

Caution: The vacuum pump may only be switched off if there are no more components on the placement heads.

Hardware start button behavior

Start production automatically

Recommended setting during production: Automatic mode.

Pressing the Start button starts production immediately.

Do not change production status

Recommended setting during service activities:

Pressing the start button does not start production, but only closes the electrical circuit.

Hardware stop button behavior

Stop immediately

Default:

Pressing the Stop button stops production immediately. The protective covers can be opened immediately. Components that have already been picked up remain on the placement heads.

Stop after placement cycle (can only be selected on machines with a protective cover locking mechanism)

Recommended setting if extremely high components are to be placed in the job:

Pressing the Stop button only stops production after all components that have been picked up have been placed. The protective covers can only be opened after the current placement cycle is complete.

Run options

The following settings are only designed for use during testing and would lead to considerable delays during productive placement operation.

Cycle mode (for test purposes only)

On: The Start button must be pressed each time the machine moves a board.

Off: Normal mode.

Safe mode (reduced speed)

The gantry axes and star axes move slowly (at approximately 10% of the maximum speed).

All axes move at maximum possible speed.

Pass-through mode

The boards are not assembled, but are simply passed through the machine.

Default: The boards are assembled.
