Location > X-Table > X-Feeder

 

Get version

The versions of the currently installed firmware are shown in a dialog box for the selected X-Feeder.

Transport component

The components in the tape are transported forwards by one component pocket each time this button is pressed.

Pickup offset correction

X [mm]:

Shows the pickup offset correction value in X that has been automatically determined.

Y [mm]:

Shows the pickup offset correction value in Y that has been automatically determined.

Z [mm]:

Shows the pickup height correction value in Z that has been automatically determined.

Reset

Resets the automatically determined pickup offset correction values to the default values.

Feeder position recognition already performed?

Component pocket position recognition: No/Yes

Indicates whether position recognition has already been carried out.

Start

Starts the machine function for component pocket position recognition.

Measurement of pickup position height: No/Yes

Indicates whether measurement of the component pickup height has already been carried out.

Start

Starts the machine function for measurement of the component pickup height.
