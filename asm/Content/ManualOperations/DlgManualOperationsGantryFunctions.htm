Manual operations > Gantry functions

Current position

The X and Y coordinates of the gantry are shown.
The coordinates are updated each time a gantry function is performed.

Position:

Axis coordinates.

Position (global coordinates):

Coordinates with reference to the machine zero point.

Update position

The current coordinates of the gantry are determined and displayed.

Functions - Reference run

Head and gantry reference run

Moves all axes of the placement head to a defined position and determines the position of the axes in the machine.

Gantry reference run

Determines the position of the axes in the machine.

Functions - Move axis

Move to preset position...

Opens a dialog box in which you can move the selected gantry to a defined position:

Park position, service position or setup position.

Move to X/Y position...

Opens a dialog box in which you can specify the coordinates to which the gantry is to move and in which you can select a travel profile for each axis

Move to zero pulse

The zero pulse denotes the zero point of the gantry to which the counting operations of the incremental encoder refer. The gantry moves to this zero point during the reference run.
This function can be used to check the zero pulse of the gantry.

Functions - Data

Gantry data...

Opens a dialog box in which you can read the following information for the X and Y axes: Number of moves, number of hardware errors, number of processing errors.

Travel range...

Specifies the minimum and maximum limit position of the selected axis and the possible travel range.

Restore Y axis mapping data (only displayed for the Service activity level)

Stores the gantry offset values determined during mapping for the current gantry along the travel range in the gantry EEPROM.

Confirm exchange of sensor module for Y axis (only displayed for the Service activity level)

Stores the gantry data in the gantry EEPROM.
This function must not, however, be executed if the entire gantry has been replaced and not just the sensor module.
