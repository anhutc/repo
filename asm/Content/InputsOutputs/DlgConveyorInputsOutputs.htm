Conveyor inputs/outputs

Lane 1/2

Each of the following buttons represents an input/output module. When you press the button for the required module, a dialog box opens showing all the outputs that belong to the selected module as buttons. Press the button for a particular output in order to activate or deactivate the corresponding output in the dialog box that then opens.

Input/output modules for the right or left conveyor lane:

(Different modules are shown here depending on the machine type.)

Belt motors

Input/output module for the belt motors and adjustment units of the individual sections of the conveyor.

Lifting table

Input/output module for lifting table up/down and the associated laser light barriers.

Interface

Input/output modules for the interfaces to the upstream and downstream stations and the fault circuit.

Options

Input/output module for options such as the barcode reader, vacuum tooling, ceramic substrate centering, stopper for long board.

Stop unit

Input/output module for the standard stoppers and optional stoppers (combined PCB) in the individual sections of the conveyor.

Width adjustment

Width adjustment

Input/output module for width adjustment (width adjustment spindle, right/left limit switch, right/left limit switch for side panel).
