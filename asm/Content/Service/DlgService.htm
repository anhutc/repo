Calibration and teaching

Automatic calibration

Opens the Automatic calibration view. All calibration steps which are needed to calibrate the entire machine are offered for selection here.

Single calibration

Opens the Single calibration view. All machine components for which single calibration is possible are offered for selection here.

Teaching machine positions

Opens the Teach machine positions view. Here you can select the gantry you want to use to check a particular point in the machine (e.g. the machine zero point or a board reference corner).

Manual correction (Only displayed on the Service activity level.)

Opens the Manual correction view in which you can change the placement offset values determined by the accuracy check.

MAC test (machine accuracy capability test) (Only shown for certain machine types.)

Opens the MAC test view. You can select the head for which the Machine accuracy capability test (MAC test) is to be carried out.

Embedded software

Archive and restore

Archive machine data...

Compresses the machine data to the ZIP file _MachineData_<MachineFrameSerialNumber>_MID<UniqueMachineId>.zip and copies the file to the selected directory.

This also reads all conveyor machine data for the existing conveyor lanes from the conveyor controller and saves it.

Restore machine data...

Extracts the machine data contained in the ZIP file _MachineData_<MachineFrameSerialNumber>_MID<UniqueMachineId>.zip from the selected directory or copies the files stored in the specified directory to corresponding directories on the station computer.

The machine must be restarted in order to activate the restored machine data.

Archive SIPLACE system... (not shown for some operating modes)

Compresses the machine data, the data from the programming system and the data from the monitoring system to a file (*.sbu) and copies this file to the selected folder.

Restore SIPLACE system... (not shown for some operating modes)

Extracts the data stored in an *.sbu file to the relevant folders.
The machine must be restarted in order to activate the restored data.
