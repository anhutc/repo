Change selected position

Each press of these buttons moves the gantry one step of the selected step size in the corresponding direction.

Specify the step size and the coordinate system.

First press this button and then press a position in the camera image. The camera is moved to the corresponding position.

Move to original position

(only shown for certain machine positions)

Moves the camera to its original position and resets the previously saved offsets.

If you then press the Save button, all offsets taught previously are rejected.

The changed values are shown immediately.

False colors

Represents grayscale values using a color palette.

Previous

Switches back to the Teach machine positions view.

Next

(Disabled in this view.)

Save

The current settings are saved.

Cancel

Any changes made are discarded.

If at least one value has been changed, it is only possible to close the view by clicking Save or Cancel.
