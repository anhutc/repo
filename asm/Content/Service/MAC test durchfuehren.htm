Carrying out a machine accuracy capability test

Prerequisite for starting the machine accuracy capability test:

The machine must have been fully calibrated.

  1. Switch to the Service view and click the MAC test button.

  2. Choose the head for which you wish to perform the machine accuracy capability test.

  3. If the test is to be carried out for all segments of the selected head

    Choose Perform MAC on all segments from the Actions pull-down menu.

    If the test is only to be performed on individual segments

    Choose Perform MAC on specific segments from the Actions pull-down menu, select the relevant segments and start the test by clicking Start capability test.

  4. After measurement is complete, click Save to save the results in a .pdf file for subsequent evaluation.
