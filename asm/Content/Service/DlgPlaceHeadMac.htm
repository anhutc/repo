The machine accuracy capability is understood to be the stability and reproducibility of a production step on a machine in live production.

Segment

Measured segment

Deviation

Standard deviation at 3 sigma

Mean Offset

Statistical average

CM

Machine capability:

The higher this value, the greater the accuracy with which the machine places components. The tolerance bands are not taken into account for this calculation.

CMK

Machine capability index: Variance of the parameter under investigation.

Position of the average of the random sample with reference to the specified tolerance. Ideally, CMK will be equal to CM and both values will be high.

Spec.

Value according to the specification of the camera used to carry out the test.

Actions

Perform MAC on specific segments

Opens the Select the segments for machine accuracy capability test dialog box. The machine accuracy capability test is started when you confirm your selection by pressing Start capability test.

Perform MAC test on all segments

Starts the machine accuracy capability test.

Set options

Opens the Select options dialog box, in which you can enter the number of repetitions for the individual measurements.

Save

The results of the current test can be stored as a .pdf file. You can choose the file name and folder. When you confirm that the file is to be saved, the system switches back to the Service > MAC test view.

Cancel

Cancels the current measurement and switches back to the Service > MAC test view. No results are saved.
