Teaching machine position

Machine position

Select the position that the gantry is to move to.

Move to position

Moves the gantry to the selected machine position.

Move to next

Moves the gantry to the position below the current position in the list of machine positions.

The current gantry position is shown in the output field to the right of the list of machine positions. You can adjust the coordinates of this position if you switch to the "Change selected position" view.

 

Previous

(Disabled in this view.)

Next

Switches to the "Change selected position" view.

Save

The current settings are saved.

Cancel

Any changes made are discarded.

If at least one value has been changed, it is only possible to close the view by clicking Save or Cancel.
