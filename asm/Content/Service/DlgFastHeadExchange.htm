Service > Fast hardware exchange > Head exchange

Exchanged heads

Each exchanged head is selected here for the additional steps required (see the following sample configuration).

On certain machines, it is possible to replace not only the head, but also the gantry.

The head or gantry has not been replaced. (Default setting)

The head or gantry has been replaced.

If the system automatically detects that placement heads have been replaced, these are selected for calibration and shown in orange in the diagram. These heads must be calibrated.

As soon as a head or gantry is selected for which different calibration options are available, the following selection options are displayed:

Calibration type

Standard calibration

Completely calibrates the head. The calibration data stored in the head EEPROM are not used.

Brief calibration

Calibrates the component camera(s) and segment 1. The calibration data stored in the head EEPROM are used for this.

It is not possible to select the calibration type for the Twin Head, as it is always necessary to perform a complete calibration on the Twin Head.

Next

Switches to the relevant view for the next work step for the exchanged head (see description below). 

Adjust nozzle changer data (optional)

This view is only shown if not all nozzles were returned before the head is exchanged.

The nozzles on the head must be returned to the correct garages of the nozzle changer manually and you must then confirm that the garages are full.

You can do this for each individual garage or for all garages at once by checking the All garages have been refilled box.

Next

Switches to the relevant view for the next work step for the exchanged head (see description below). 

Prerequisites for reference run and calibration

Make sure that the required calibration nozzles are present on all segments and that the calibration tools are in the appropriate calibration pockets.

Not all calibration tools can be provided at the same time.

If insufficient calibration tools are available, it is also possible to perform calibration step by step.
To do this, specify the position in the machine at which a calibration tool is available.

Start

Carries out a reference run and then calibrates the placement head.

After calibration, the calibration nozzles are returned to the nozzle changer. If this is not possible, they are rejected and must be removed from the machine along with the calibration tools.
