Nozzle detection

Depending on the selection you make here, the board camera is positioned over a single garage, over all the garages of a specific magazine in sequence or over all the garages of the nozzle changer and nozzle detection is started.

The system checks whether there is a nozzle in the garage and whether this nozzle has a nozzle ID. If a nozzle with a nozzle ID is detected, the nozzle type can be determined and entered under "Current" in the table.

 

The following results are possible:

The garage is occupied, and the nozzle ID is recognized.

The nozzle type detected is entered under "Current".

The garage is occupied, but there is no nozzle ID.

The previous occupancy remains displayed and an error message is issued.

The garage is empty.

An empty garage is shown in the detail view and "-" is shown in the table.

The garage is occupied, and the nozzle ID is not recognized.

The previous occupancy remains displayed and an error message is issued.
