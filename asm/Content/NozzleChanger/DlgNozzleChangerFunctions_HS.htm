Reference nozzles 

Determines the nozzle heights for all segments again. To do this, the gantry moves to a defined position above the board conveyor. The Z axis moves down with all nozzles in turn until the nozzles make contact with the conveyor.

Nozzle functions

Set another nozzle... (only shown on the Service activity level)

Opens a dialog box in which you can specify an actual nozzle type. The nozzle type is entered in the list when you confirm your selection.

Confirm change

Accepts the nozzle change when no nozzle changer is used. The actual nozzle and the specified nozzle match in the table.

Confirm nozzle is not present

First, visually check whether the nozzle is present on the selected segment.

Change

Replaces the actual nozzle in its garage and then takes the specified nozzle from its garage.

Change with... (only shown on the Service activity level)

Opens a dialog box in which you can select a nozzle to be replaced by the current one.

Return to nozzle changer

Places the nozzle in the selected garage of the selected magazine.

Pick up from nozzle changer (can only be enabled if the selected segment is empty and there is a default value)

Picks up the nozzle from the predefined garage.

Return dirty nozzle and pick up

Replaces the dirty nozzle in the nozzle changer, marks it as dirty and then picks up a nozzle of the same type from the nozzle changer.

 Reject

Positions the nozzle over the reject bin and then pulls it off. This allows a faulty nozzle, for instance, to be removed from the segment.
