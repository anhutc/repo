Configure magazines (Twin Head)

1. Select magazine

Select the magazine you wish to change.

indicates the magazine carrier you see first when you are standing on the relevant side of the machine.

2. Select magazine type

Select the appropriate type:

Magazine with 1 garage (suitable for special nozzles).

Magazine with 2 garages.

Magazine not present (corresponds to an empty magazine location).

The magazine cannot be reached by the head.

Accept

The changes are saved.

Close

The dialog box is closed and any changes which have not been saved are discarded.
