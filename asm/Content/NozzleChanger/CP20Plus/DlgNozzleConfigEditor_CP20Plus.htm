Configure nozzles

 

1. Select magazine and garages

When this dialog box is called, the magazine selected in the higher-level view is automatically selected.

First select the magazine and then the garages for which you want to configure a nozzle type.

Select all

Selects all garages.

Deselect all

Cancels the current selection. All garages are deselected.

2. Select nozzle type

Calibration nozzle

Automatically selects the nozzle for calibration purposes.

List of possible nozzle types for the selected magazine

Select the nozzle type contained in the garage.

 

Select all

All the garages of the magazine that is displayed are selected.

Deselect all

None of the garages of the magazine that is displayed are selected.

Accept

The selected garages are loaded with the selected nozzle type.

Close

Your entries are canceled and the dialog box is closed.
