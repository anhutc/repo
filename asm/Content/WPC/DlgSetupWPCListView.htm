Setup > Locations > WPC x

Position

Level - Number of the waffle pack tray

Random Setup: In the programming system it was specified that the component must not necessarily be set up at the position specified in the required setup.
It is also possible to set up additional alternative tracks.

Prerequisite: Component verification.
 

The component must be set up at the position specified in the required setup.

Status

Status and filling level of the waffle pack tray.

Filling level

The number of components still in the waffle pack tray.
(Current quantity / Max. comp. level)

Component

Name of the component which is set up in the waffle pack tray.

At least one alternative component has been specified for this pickup location.
First select the corresponding row in the table and press the field to display all possible alternatives.

The component that is currently being used is shown in the first row of the field.
If, however, no component supply has yet been scanned in, the top row is empty.

Currently, all alternative components are displayed.
Press the field to hide the alternatives.

Component shape

Name of the component shape.

Activated if the component position has been corrected manually and differs from the specified component position.

Selected waffle pack trays

Selection

Select empty trays

Selects all waffle pack trays which do not contain components.

Select trays deactivated automatically

Selects all waffle pack trays that have been deactivated by the system.

Select trays deactivated by operator

Selects all waffle pack trays that have been deactivated by the operator.

Select trays not used by setup

Selects all waffle pack trays that are not used by the currently valid setup.

Select all

All the entries in the list are selected.

Deselect all

All the entries in the list are deselected.

Actions

Set full

The maximum number of components is entered as the current component level and the magazine is taken back into operation.

Set empty

Confirms that the selected waffle pack trays have been emptied.

Enable

Takes the selected waffle pack trays into operation.

Disable

Takes the selected waffle pack trays out of operation.

Clear entry

Confirms manual removal of the selected waffle pack trays and removes the corresponding row from the list.

Single tray

Refill level... (displayed depending on the WPC type)

Moves the waffle pack tray carrier containing the selected waffle pack tray into the refilling unit and switches to the Refill level view.

Provide level

The lifting axis moves the selected level to the position of the feed axis and the waffle pack tray carrier is then pushed into the staging position .

Correct position... (Can only be enabled as of the "Service" activity level.)

Opens a dialog box for manually correcting the component pickup position.

Change acceleration...

Opens a dialog box that allows you to specify the maximum permitted acceleration when moving a waffle pack tray carrier with the specified component.

Refill position

Move to refill position

Moves to the position at which the waffle pack trays of all levels can be exchanged.

While this process is taking place, it is not possible to pick up components from the WPC.

Quit refilling mode

Moves the entire WPC back into the processing position.
It is now possible to pick up components again.

Check WPTC occupancy

The waffle pack tray carriers that are present are compared with those specified in the setup. The list is updated.

FIFO strategy

As soon as components have been picked up from a waffle pack tray on an alternative track, the tray is completely emptied before the machine switched back to the original track.

The components are picked up with optimized travel.
