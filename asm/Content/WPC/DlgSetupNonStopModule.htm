Setup > WPC refilling unit (non-stop module)

 

Selected tray

Set full

Confirms that the waffle pack tray has been completely refilled.

Set empty

Confirms that the waffle pack tray has been completely emptied.

Change filling level...

Opens a dialog box in which you can enter the number of components to be used to refill the waffle pack tray.

 

How to refill a waffle pack tray:

  1. Open the loading flap.

  2. Remove the waffle pack tray carrier and insert a new waffle pack tray carrier.

  3. Enter the appropriate component level.

  4. Close the loading flap.

    The waffle pack tray carrier is moved back to its initial position.

 

Tip:

If an incorrect waffle pack tray carrier was moved into the refilling unit, you can stop the process by pressing the operating button on the refilling unit without impacting on setup verification.
