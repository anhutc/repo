Manual operations > Subsystems > WPC x

Positions - WPC

Move lifting axis to service position

Moves the lifting axis to the position at which service work can be performed on the WPC.

Move feed axis to service position

Moves the feed axis to the position at which service work can be performed on the WPC.

Move load axis to service position

Moves the loading axis to the position at which service work can be performed on the WPC.

Move into transport position

Moves the WPC into a defined transport position so that it can be transported safely.

Functions

Reference run

Moves the lifting axis and the feed axis to a defined position.

Refill position

After selection from the drop-down menu, starts the machine functions for moving to the refill position for the whole WPC. Moves to the position at which the waffle pack trays of all levels can be exchanged.

Occupancy check

Starts the machine functions for checking the waffle pack tray carriers. The position of the waffle pack tray carriers (level) is then displayed. If waffle pack tray carriers specified by the setup are missing, this is displayed.

Functions - Level

Provide level...

Opens a dialog box to select the level in which the waffle pack tray to be changed is located. The machine function for providing the level is started after you confirm your selection with Accept. The lifting axis moves the selected level to the position of the feed axis and the waffle pack tray carrier to be changed is then pushed into the staging position.

 

Advanced functions... (only shown on the Service activity level)

Switches to the most recently displayed detailed view for the WPC: Teach axes, Axis properties, Input/Output or Machine data.
