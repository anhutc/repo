WPC: Teaching the lifting axis

The individual steps must be performed in sequence from top to bottom during initial setup:

Teach zero position

Performs a reference run and determines the zero point correction value. The buttons for teaching the zero point are then displayed.  

See

Teach lifting axis zero position (method).

Reference bero

Moves the lifting axis to the reference bero in order to re-determine the position of the bero. The determined position is saved.

 

Calibrate limit switch -

Moves the lifting axis to the bottom limit switch in order to re-determine the minimum position which can be reached. A reference run is then performed with the axis.

Calibrate limit switch +

Moves the lifting axis to the top limit switch in order to re-determine the maximum position which can be reached. A reference run is then performed with the axis.

 

Teach offset load axis

Setting the correct height of the load axis.

 

Additional action

Move lifting axis

The view for stepping the lifting axis up and down is opened.
