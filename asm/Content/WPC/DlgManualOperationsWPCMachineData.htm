Manual operations > Subsystems > WPC x > WPC machine data

Set serial number

Serial number:

Currently entered serial number.

Click in the text box to change the serial number.

(The serial number can be found on the WPC frame.)

Submit

Transfers the currently entered serial number to the internal machine data of the WPC.

Machine data

Restore machine data...

Opens a dialog box in which you can navigate to the location in which the machine data is stored and select the file you require with Open.

Save machine data...

Opens a dialog box in which you can specify the storage location and file name for the changed machine data and save it with Save.
