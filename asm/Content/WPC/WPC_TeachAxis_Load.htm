WPC: Teaching the load axis

Prerequisite for teaching the load axis: the lifting axis and feed axis must already have been taught.

Teach zero position

Performs a reference run and determines the zero point correction value.

See

Teach zero position of the WPC load axis (method).

Reference bero

Moves the load axis to the reference bero in order to re-determine the position of the bero. The determined position is saved.

 

Calibrate limit switch -

Moves the load axis to the limit switch on the tower side in order to re-determine the minimum position which can be reached. A reference run is then performed with the axis.

Calibrate limit switch +

Moves the load axis to the limit switch on the side of the non-stop module in order to re-determine the maximum position which can be reached. A reference run is then performed with the axis.
