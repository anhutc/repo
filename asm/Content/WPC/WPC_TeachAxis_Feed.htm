WPC: Teaching the feed axis

 

Teach zero position

Performs a reference run and determines the zero point correction value.

See

Teach zero position of the WPC feed axis (method).

Reference bero

Moves the lifting axis to the reference bero in order to re-determine the position of the bero. The determined position is saved.

 

Calibrate limit switch -

Moves the feed axis to the limit switch on the transfer side in order to re-determine the minimum position which can be reached. A reference run is then performed with the axis.

Calibrate limit switch +

Moves the feed axis to the limit switch on the tower side in order to re-determine the maximum position which can be reached. A reference run is then performed with the axis.

 

Teach WPTC alignment position

Notes on the procedure are then displayed. You must confirm each of these with Next. After all the steps have been performed and the machine functions have been executed, click Finish to accept the current position as the WPTC alignment position.

Teach WPTC alignment position (method).
