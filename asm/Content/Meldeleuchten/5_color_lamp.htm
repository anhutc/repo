Indicator lamp (red-yellow-blue-green-white)

Default settings:

Lamps

 

Machine status

Red indicator on

Machine fault,

safety circuit interrupted

Yellow indicator on

Setup / manual operations:

- Placement halted

- Low filling level

- Last available alternative pickup location is active

- Some pickup locations are deactivated

- Barcode must be entered by hand

- Measurement error

- Board conveyor error

- Teaching of a fiducial is required

- Machine has been stopped following an error message

 

 

Blue indicator on

- Setup change

- Setup incomplete

- Production preparation

Green indicator on

The machine is in production mode,

automatic mode

White indicator on

The machine is switched on, but not in production mode:

The machine is waiting for a board or a recipe.

All indicators off

The machine is switched off.
