Show board details in list

A list with the following contents is displayed:

Board

Unique identification of a board using the board conveyor and ID.
This ID is also shown if you press a board in the main view.

Components/hour

Number of components placed per hour.

This value is calculated on the basis of the number of components placed in the specified period: 3600 s / period * number of components.

Components

Number of components placed for the specified board in the processing area (or a different stopper).

Time span [s]

Period between the start time and end time.

Start time

Start time for recording data for the current board.

End time

End time for recording data.

If "not available" is shown in a column, this indicates that no boards are being transported, for instance due to a "dry run" for test purposes, and that the start time is therefore not set and that no components are placed.

Performance data parameters

Here you can specify the parameters for the individual traces.

Start time trigger

Here you can specify the time that the evaluation should start.

• As soon as the first fiducial is measured (default).

• As soon as the board has started moving into the processing section.

• After the board has been clamped.

• When placement starts.

• The first component has been placed.

End time trigger

Here you can specify the time that the evaluation should end.

• As soon as the last component has been placed (default).

• As soon as the board is unclamped.

• The board has reached the next section
(intermediate section or output section).

Maximum number of boards

Number of boards captured in the statistics. If more boards than the specified number have been assembled since the evaluation started, the oldest are removed from the evaluation.

Note:

Because the Operator Information System (OIS) takes the measurement of the first fiducial as the start time and placement of the last component as the end time for recording performance data, you should also use these settings. If you do not do so, the performance data will differ.

Select triggers for "Build time"

Sets measurement of the first fiducial as the start time and placement of the last component as the end time.
These settings correspond to the specifications in the Operator Information System (OIS) for recording the performance data.

Bar chart

The calculated placement performance (number of components placed per hour) is shown for each board.

The horizontal gray line indicates the average placement performance across all boards in this evaluation (the boards captured to date up to the maximum number specified).

The board counter shows how many boards have been included in the evaluation to date.

Export

Select the data you wish to save by pressing the button.

Export all data to file...

The performance data for all the boards in the list is saved to a file.

Export selected data to file...

The performance data for all the boards selected in the list is saved to a file.
