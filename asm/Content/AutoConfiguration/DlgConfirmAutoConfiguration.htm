Service > Machine configuration

Shows the automatically detected machine configuration.

You can edit all the parameters.

If "?" is shown in the "Value" column, it was not possible to detect the parameter, and it must be entered manually.

You cannot proceed as long as the message "Manual selection required" is shown.

Restart auto-configuration

Calls automatic detection of the machine configuration.

Save new configuration and continue startup

The system continues to boot using the current data.

Continue startup with previous configuration

The system continues to boot using the most recently detected machine configuration.
