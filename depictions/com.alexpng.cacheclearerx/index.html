Little12
Tweaks
Bring iPhone X features and more to devices running iOS 14

Features

  • iPhone X home and multitasking gestures.
  • iPhone X and iPad Status Bar styles
  • Bottom Home Bar
  • Battery Percentage in Battery Icon
  • Hide the Status Bar in the Control Center
  • iPhone X Keyboard
  • Lockscreen Shortcuts
  • iPhone X Device Spoofing
  • Rounded Corners, Dock and Switcher
  • Face ID Padlock
  • Reduce Home Screen Rows
  • Bottom Inset
  • Control Center Grabber
  • Gestures option for Keyboard
  • iPad Dock and Multitasking
  • iPhone 11 Camera UI

Little12 is partially open source and can be found here: https://github.com/ryannair05/Little12
